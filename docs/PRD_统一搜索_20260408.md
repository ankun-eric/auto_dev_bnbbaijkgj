# 比尼健康 — 统一搜索功能 产品需求文档（PRD）

## 1. 需求概述

### 1.1 背景与目的

比尼健康（bini-health）H5 首页目前有一个搜索栏（placeholder 文字为"搜索症状、疾病、药品"），用户输入内容后点击搜索会跳转到 `/drug?q=搜索词`（拍照识药页面）。这种设计存在以下问题：

1. 搜索结果类型单一，仅指向拍照识药，无法满足用户搜索文章、视频、服务项目等多类型内容的需求
2. 搜索体验缺乏联想提示、搜索历史等辅助功能
3. 不支持语音输入，移动端输入体验不够便捷

本次需求将首页搜索**完全改造**为统一搜索体系，用户可在一个搜索入口中查找平台上的多类型内容，提升信息获取效率和用户体验。

### 1.2 目标用户

- 比尼健康 H5 端所有注册及未注册用户（本期仅实现 H5 端）

### 1.3 核心价值

- **一站式搜索**：用户通过一个入口即可搜索文章、视频、服务项目、积分商品 4 类内容
- **智能搜索体验**：联想词提示、热门搜索、搜索历史、语音输入，全方位提升搜索效率
- **运营可控**：管理后台可管理推荐搜索词、屏蔽词、查看搜索统计、配置语音识别参数

### 2.2 功能详细描述

#### F01 — 独立搜索页

**入口**：用户在 H5 首页点击搜索栏后，跳转到独立的搜索页面（路由 `/search`）。

**页面元素**：
- 顶部：返回按钮 + 搜索输入框 + 语音按钮
- 搜索框 placeholder："搜索文章、视频、服务、商品"
- 下方区域（未输入时）：
  - 热门搜索区域（F02）
  - 搜索历史区域（F03）
- 下方区域（正在输入时）：
  - 联想词下拉列表（F04）

**首页改造**：原首页搜索框输入后不再跳转拍照识药（`/drug?q=`），完全改为跳转到统一搜索页 `/search`。

---

#### F02 — 热门搜索展示

**展示规则**：
1. **推荐搜索词（管理员配置）**：置顶展示，按管理员设置的排序显示
2. **自动统计热门词**：系统自动统计用户搜索频率最高的词，取 Top N 展示在推荐词之后
3. 推荐词与热门词合并去重后展示，**总数最多 10 个**（管理员推荐词优先展示，剩余名额由自动统计热门词补充）

**展示样式**：
- 区域标题："热门搜索"
- 标签式布局（Chip/Tag），支持换行
- 点击标签直接触发搜索，跳转到搜索结果页

---

#### F03 — 搜索历史

**存储方式**：保存在服务器端（与用户账号关联），登录后多端同步。

**展示规则**：
- 区域标题："搜索历史" + 右侧"清空"按钮
- **默认显示最近 6 条**，底部有展开下拉箭头（类似微信），点击展开查看全部历史
- 展开后最多展示 **20 条**，超出 20 条的旧记录自动丢弃不再保存
- 每条记录右侧有**垃圾桶图标**，点击直接删除该条（不做二次确认，Toast 提示"已删除"）；同时支持**左滑删除**
- 点击某条历史记录直接触发搜索
- 支持一键"**全部清空**"，点击后弹出**二次确认弹窗**（"确定清空所有搜索历史？"），确认后清空

**数据记录**：用户每次发起搜索时，将搜索关键词记录到服务器（去重：同一用户搜索相同关键词更新时间戳）。

---

#### F04 — 联想词/建议词

**触发条件**：用户在搜索输入框中每输入一个字符，停止输入 300ms 后发起联想请求。

**匹配范围**：仅匹配热门搜索词（推荐词 + 自动统计热门词），不做全文搜索。

**展示样式**：
- 搜索框下方弹出下拉列表
- 每条建议词格式：`搜索图标 + 建议词文本 + [类别标签]`
  - 类别标签示例：`[文章]`、`[视频]`、`[服务]`、`[积分商品]`、`[功能]`
- 当用户输入"药""拍照"等关键字时，联想词中出现 `拍照识药 [功能]` 建议项
- 点击联想词：
  - 普通搜索词 → 跳转到搜索结果页
  - `拍照识药 [功能]` → 直接跳转到拍照识药页面 `/drug`

---

#### F05 — 语音搜索

**入口**：搜索输入框最右侧的声音图标按钮。

**交互流程**：
1. 用户点击声音按钮
2. 弹出半屏弹窗：
   - 中间大麦克风图标 + 波形/脉冲动画
   - 文案提示："请说出您想搜索的内容"
   - 支持普通话和粤语识别
3. 用户说话时展示实时波形动画
4. 用户说完后（静音检测 / 超时），自动转为文字填入搜索框
5. 弹窗关闭，用户可查看/修改转写文字，确认后点击搜索按钮触发搜索

**技术方案**：接入腾讯云一句话识别（ASR），支持普通话 + 粤语。

**管理后台配置项**：
- AppID
- SecretId
- SecretKey
- 启用/禁用开关
- 方言支持：普通话、粤语

---

#### F06 — 搜索结果页（分Tab展示）

**路由**：`/search/result?q=关键词`

**Tab 列表**：`全部` | `文章` | `视频` | `服务` | `积分商品`

**默认选中**：`全部` Tab

**搜索匹配范围**（所有类别统一）：
- 标题匹配
- 内容/描述匹配
- 标签/关键词匹配

---

#### F07 — "全部"Tab 行为

**排序规则**：按类别优先顺序展示 — 文章 → 视频 → 服务项目 → 积分商品。

**展示方式**：
- 每个类别区块标题：`文章相关（N条）`
- 每类最多展示 3~5 条（建议 3 条）
- 每个区块底部显示"查看更多 >"链接，点击切换到对应类别 Tab
- 如某类无匹配结果，则不展示该区块

---

#### F08 — 单类别 Tab

**结果卡片样式（精简模式）**：
- 缩略图（左侧或右侧，如有）
- 标题（1行，超长截断）
- 类别标签（如 `文章`、`视频` 等彩色小标签）
- 点击卡片进入对应详情页

**分页方式**：
- 底部"加载更多"按钮
- 每页数量建议 10 条
- 加载完所有结果后按钮变为"没有更多了"

**排序规则**：类内按相关度排序（标题命中 > 标签命中 > 内容命中）。

---

#### F09 — 拍照识药入口

**触发方式**（双重触发）：

1. **联想词触发**（F04 已描述）：输入"药""拍照""识药"等关键字时，联想词下拉中出现 `拍照识药 [功能]`
2. **搜索结果页触发**：当用户搜索"拍照识药""识药""药品识别"等特定关键词时，搜索结果页顶部（Tab 下方）展示一个**拍照识药快捷入口卡片**：
   - 卡片样式：图标 + "拍照识药" + 简要描述 + 箭头
   - 点击跳转到拍照识药页面 `/drug`

**关键词列表**（建议后端可配置）：`拍照识药`、`识药`、`药品识别`、`拍药`、`认药` 等。

---

#### F10 — 空结果处理

当用户搜索的关键词在所有 4 个类别中都没有匹配结果时：

1. **空状态提示**：显示插图 + "暂无搜索结果"文案
2. **推荐热门内容**：下方展示热门文章/热门服务等推荐卡片（从热门搜索词关联的内容中获取）
3. **AI 健康咨询入口**：底部按钮"试试 AI 健康咨询"，点击后将搜索词作为 AI 对话的第一句话，跳转到 AI 聊天页

---

#### F11 — 管理后台：推荐搜索词管理

**菜单位置**：顶级菜单"搜索管理" → "推荐搜索词"

**功能**：
- 列表展示所有推荐搜索词（词条、排序值、关联类别、创建时间）
- 新增推荐词：输入词条 + 设置排序值 + 可选关联类别
- 编辑推荐词
- 删除推荐词
- 拖拽或输入排序值调整顺序
- 推荐词在 H5 搜索页"热门搜索"区域置顶展示

---

#### F12 — 管理后台：搜索数据统计

**菜单位置**：顶级菜单"搜索管理" → "搜索统计"

**统计内容**：
- **热门搜索词排行**：按搜索次数降序排列，支持**自定义日期范围**筛选（管理员自行选择起止日期）
- **搜索次数趋势**：折线图展示每日搜索总次数趋势
- **无结果搜索词 Top N**：用户搜索但无结果的关键词排行，帮助运营补充内容
- **各类别点击分布**：饼图/柱状图展示用户点击搜索结果的类别分布

---

#### F13 — 管理后台：搜索屏蔽词管理

**菜单位置**：顶级菜单"搜索管理" → "屏蔽词管理"

**功能**：
- 列表展示所有屏蔽词（词条、屏蔽模式、创建时间）
- 新增屏蔽词：输入词条 + 选择屏蔽模式
  - **完全屏蔽**：搜索该词时不返回任何结果，显示"暂无搜索结果"
  - **替换提示**：搜索该词时显示自定义提示文案（如"该内容暂不可搜索"）
- 编辑屏蔽词
- 删除屏蔽词
- 支持批量导入（文本框粘贴，一行一个）

---

#### F14 — 管理后台：语音配置（ASR）

**菜单位置**：顶级菜单"搜索管理" → "语音配置"

**配置项**：
- **腾讯云 AppID**：文本输入
- **腾讯云 SecretId**：文本输入
- **腾讯云 SecretKey**：密码输入（已保存时显示"已设置"，支持重新输入覆盖）
- **启用/禁用开关**：Switch 控件
- **方言支持说明**：页面提示文字说明当前支持普通话和粤语

**测试识别功能**：
- 管理员填写完 AppID / SecretId / SecretKey 后，可点击"**测试识别**"按钮
- 系统调用腾讯云 ASR 接口进行验证，返回"配置验证通过"或"配置错误：xxx"的提示
- 可避免管理员配置错误后影响 H5 端用户体验

## 3. 页面/界面设计

### 3.1 页面结构与导航

```
H5 首页
  └─ 点击搜索栏 → /search（独立搜索页）
       ├─ 未输入时：热门搜索 + 搜索历史
       ├─ 输入中：联想词下拉
       ├─ 点击搜索/回车 → /search/result?q=xxx（搜索结果页）
       │    ├─ 全部 Tab
       │    ├─ 文章 Tab
       │    ├─ 视频 Tab
       │    ├─ 服务 Tab
       │    └─ 积分商品 Tab
       └─ 点击语音按钮 → 半屏弹窗（语音识别）

管理后台
  └─ 搜索管理（顶级菜单）
       ├─ 推荐搜索词
       ├─ 搜索统计
       ├─ 屏蔽词管理
       └─ 语音配置
```

### 3.2 各页面功能说明

#### 3.2.1 搜索页（/search）

| 区域 | 元素 | 交互说明 |
|------|------|----------|
| 顶部导航栏 | 返回按钮 + 搜索输入框 + 语音按钮 | 输入框自动获取焦点 |
| 热门搜索区 | 标题"热门搜索" + Tag 列表 | 点击 Tag 直接搜索 |
| 搜索历史区 | 标题"搜索历史" + "清空"按钮 + 历史列表（默认6条）+ 展开箭头 | 点击历史项搜索；垃圾桶图标/左滑单条删除；展开查看全部（最多20条）；清空需二次确认弹窗 |
| 联想词下拉 | 建议词列表（输入时浮层） | 点击直接搜索或跳转功能页 |

#### 3.2.2 搜索结果页（/search/result）

| 区域 | 元素 | 交互说明 |
|------|------|----------|
| 顶部 | 返回按钮 + 搜索框（带当前关键词） | 可修改关键词重新搜索；点击返回**直接返回首页**（不回搜索页） |
| 拍照识药卡片 | 仅特定关键词触发时展示 | 点击跳转 /drug |
| Tab 栏 | 全部 / 文章 / 视频 / 服务 / 积分商品 | 点击切换 |
| 结果列表 | 精简卡片列表 | 点击卡片进入详情 |
| 底部 | "加载更多"按钮（单类别Tab） | 点击追加下一页 |
| 空结果 | 插图 + 提示 + 推荐内容 + AI咨询按钮 | 推荐区可滑动 |

#### 3.2.3 语音搜索弹窗

| 区域 | 元素 | 交互说明 |
|------|------|----------|
| 半屏弹窗 | 麦克风图标 + 波形动画 + 提示文案 | 触发录音 |
| 录音中 | 波形实时动画 + "正在聆听…" | 用户说话 |
| 识别完成 | 转写文字回填搜索框 | 弹窗自动关闭，用户确认后搜索 |
| 识别失败 | "未识别到语音，请重试" | 可重试或关闭 |

## 4. 数据模型设计

### 4.1 数据实体与字段

#### 4.1.1 SearchHistory（搜索历史）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK, 自增 | 主键 |
| user_id | Integer | FK → users.id, NOT NULL | 用户ID |
| keyword | String(200) | NOT NULL | 搜索关键词 |
| search_count | Integer | DEFAULT 1 | 该用户搜索该词的累计次数 |
| created_at | DateTime | NOT NULL | 首次搜索时间 |
| updated_at | DateTime | NOT NULL | 最近搜索时间 |

唯一约束：`(user_id, keyword)`

#### 4.1.2 SearchHotWord（搜索热词统计）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK, 自增 | 主键 |
| keyword | String(200) | UNIQUE, NOT NULL | 搜索关键词 |
| search_count | Integer | DEFAULT 0 | 全站搜索次数 |
| result_count | Integer | DEFAULT 0 | 最近一次搜索的结果数量 |
| category_hint | String(50) | NULL | 关联类别提示（article/video/service/points_mall） |
| created_at | DateTime | NOT NULL | 首次出现时间 |
| updated_at | DateTime | NOT NULL | 最近更新时间 |

#### 4.1.3 SearchRecommendWord（推荐搜索词）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK, 自增 | 主键 |
| keyword | String(200) | NOT NULL | 推荐词 |
| sort_order | Integer | DEFAULT 0 | 排序值（越小越前） |
| category_hint | String(50) | NULL | 关联类别提示 |
| is_active | Boolean | DEFAULT TRUE | 是否启用 |
| created_at | DateTime | NOT NULL | 创建时间 |
| updated_at | DateTime | NOT NULL | 更新时间 |

#### 4.1.4 SearchBlockWord（搜索屏蔽词）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK, 自增 | 主键 |
| keyword | String(200) | UNIQUE, NOT NULL | 屏蔽词 |

## 6. 非功能性需求

### 6.1 性能要求

| 指标 | 要求 |
|------|------|
| 搜索响应时间 | ≤ 500ms（95th percentile） |
| 联想词响应时间 | ≤ 200ms |
| 语音识别延迟 | ≤ 3s（从停止说话到文字回填） |
| 搜索接口 QPS | 支持 ≥ 100 QPS |

### 6.2 安全要求

- 搜索屏蔽词过滤在服务端执行，客户端不可绕过
- ASR SecretKey 加密存储，接口返回时脱敏
- 搜索日志中用户手机号脱敏显示（管理后台）
- 防止恶意高频搜索请求（IP/用户级限流）

### 6.3 兼容性要求

- H5 端：iOS Safari 12+、Android WebView / Chrome 70+、微信内置浏览器
- 语音搜索：依赖腾讯云 ASR，需 HTTPS 环境 + 麦克风权限
- 管理后台：Chrome 90+ / Edge 90+

## 7. 业务规则与约束

| 编号 | 规则 | 说明 |
|------|------|------|
| R01 | 搜索防抖 | 联想词请求防抖 300ms；搜索按钮/回车触发完整搜索 |
| R02 | 搜索历史去重 | 同一用户搜索相同关键词只更新时间戳和累计次数 |
| R03 | 屏蔽词优先级 | 搜索时先检查屏蔽词，命中则不执行实际搜索 |
| R04 | 热门词统计 | 基于 SearchLog 表定期（每小时或实时）聚合更新 SearchHotWord |
| R05 | 搜索排序 | 全部Tab：类别优先（文章→视频→服务→积分商品）；单类别Tab：相关度优先（标题>标签>内容） |
| R06 | 搜索日志记录 | 每次搜索记录日志（含来源text/voice），不记录联想词请求 |
| R07 | 拍照识药触发 | 双重触发：联想词匹配 + 搜索结果页顶部卡片 |
| R08 | 分页规则 | 单类别Tab 每页10条，"加载更多"按钮追加；全部Tab每类3条+查看更多 |
| R09 | 语音识别限制 | ASR 未启用时隐藏语音按钮；识别超时10s自动结束 |
| R10 | 搜索历史展示 | 默认显示6条 + 展开箭头；展开后最多20条；支持垃圾桶图标删除（单条）+ 左滑删除 + 全部清空（二次确认） |
| R11 | 搜索结果页返回 | 点击返回按钮**直接返回首页**，不经过搜索页 |
| R12 | 热门搜索词上限 | 最多展示10个（管理员推荐词优先 + 自动统计词补充） |

## 9. 异常处理与边界情况

| 场景 | 处理方式 |
|------|----------|
| 搜索关键词为空 | 不发起搜索请求，停留在搜索页 |
| 搜索关键词过长 | 前端限制最大 100 字符 |
| 搜索关键词含特殊字符 | 后端转义处理，防止 SQL 注入 |
| 命中屏蔽词（完全屏蔽） | 返回空结果 + "暂无搜索结果" |
| 命中屏蔽词（替换提示） | 返回空结果 + 自定义提示文案 |
| 搜索接口异常/超时 | 前端展示"搜索失败，请稍后重试" + 重试按钮 |
| 语音识别未开启 | 隐藏语音按钮，不影响文本搜索 |
| 语音识别无权限（麦克风） | 弹出提示"请允许使用麦克风权限" |
| 语音识别失败 | 弹窗显示"未识别到语音，请重试" + 重试/关闭按钮 |
| 语音识别超时（10s） | 自动结束录音，尝试识别已有音频 |
| 用户未登录访问搜索历史 | 搜索历史区域显示"登录后可查看搜索历史" |
| 推荐词/热门词均无数据 | 热门搜索区域不展示 |
| 搜索结果仅部分类别有数据 | 全部Tab只展示有数据的类别区块，无数据类别Tab显示空状态 |

## 10. 补充说明

### 10.1 现有功能影响

- 文章/资讯 → `Article` 表（匹配 title、summary、tags）

- 服务项目 → `ServiceItem` 表（匹配 name、description、tags）
- 积分商品 → `PointsMallItem` 表（匹配 name、description）

  └─ 语音配置          ← /admin/search/asr-config
