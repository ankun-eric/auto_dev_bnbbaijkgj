## 三、铁律（CI lint + 人工评审共同遵守）

1. 全域统一天蓝色阶（brand-50 ~ brand-900），不得引入新色板
2. 严禁旧版本绿色主色 `#52c41a`（含 hex/rgb/hsl/语义命名）
3. 阴影必须使用天蓝 `rgba(56,189,248,*)`，禁止 `rgba(0,0,0,*)` 纯黑
4. 病历卡：白底 + 左 3px brand-400 竖线 + shadow-2
5. 可点击区域 ≥ 44pt，AA 对比度 ≥ 4.5:1
6. 字号四档（标准/大/超大/特大）必须支持系统跟随 + App 内独立开关
7. 图标统一双色 SVG（主色 brand-600 / 辅色 brand-200），viewBox 0 0 24 24
8. 商品现价用 brand-700 深医蓝，不用红色
