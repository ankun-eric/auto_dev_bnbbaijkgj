# H5 订单预约信息 Bug 方案确认

## 对话时间
2026-04-28 23:07

## 用户原始需求
用户反馈 H5 客户端订单详情页看不到预约信息，经排查确认为 H5 Web 前端缺失预约信息展示模块。

## 关键决策记录

### 多轮对话确认的需求细节
1. **问题端**：H5 网页版（非 Flutter APP、非微信小程序）
2. **展示内容**：预约时间 + 预约门店名称
3. **展示位置**：商品信息卡片下方、核销码区域上方（参考主流平台做法：美团、抖音团购等）
4. **展示样式**：带图标的信息行卡片（📅 日历图标 + 预约时间蓝色高亮，📍 定位图标 + 门店名称正常深色）
5. **无预约数据时**：直接隐藏，不显示占位提示

### 技术排查结论
- 后端接口已返回预约数据
- H5 前端 OrderDetail 接口类型未定义预约字段
- H5 前端页面未渲染预约信息区域
- Flutter APP 端预约信息展示正常
