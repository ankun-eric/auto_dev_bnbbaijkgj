## 沟通时间

## 背景

延续AI对话页面设计沟通，用户对v2效果图提出重大调整，核心是取消顶部Tab、以AI对话为唯一入口、功能按钮分发。

## 本轮用户回答

### 调整1 — 取消顶部Tab
- 原来的四个Tab（咨询/档案/计划/服务）全部取消
- 默认直接进入AI咨询对话页面
- 不再有Tab切换导航

### 调整2 — Admin后台控制页面风格
- 页面风格切换在admin管理后台配置，不是用户端切换
- 菜单模式是过渡期保留，未来完全抛弃
- 代码上需要新旧分离，方便后续一键删除旧代码

### 调整3 — 浮动打卡图标
- 「计划」从顶部Tab移除，改为咨询页面右侧浮动按钮
- 点击跳转到计划/打卡页面
- 当天有未完成打卡任务时显示红点

### 调整4 — 6个功能按钮行为确定
在admin后台【AI咨询配置 - 功能按钮配置】中管理：

| 按钮 | 点击行为 |
|------|---------|
| 看报告 | AI对话框内弹出功能引导卡片（引导上传） |
| 查用药 | AI对话框内弹出功能引导卡片（引导查询） |
| 看档案 | 页面跳转 → 健康档案设置页面（原档案Tab） |
| 查订单 | 页面跳转 → 订单列表页面 |
| 找专家 | 页面跳转 → 服务的「专家服务」Tab |
| 找服务 | 页面跳转 → 服务页面（原服务Tab） |

### 功能按钮出现位置
- 首页3个功能卡片区（后台配置选哪3个展示）
- 底部快捷标签栏（后台配置展示哪些）
- 轮播图点击（后台配置对应哪个功能）
