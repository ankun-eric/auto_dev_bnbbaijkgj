# 验证码字号升级 v1.6（192px + 4× 高清画布）

## 用户需求
用户反馈验证码字体太小，要求将字号改为 192px，画布不变（前端 CSS 显示尺寸不变）。

## 执行过程
1. 后端 `captcha_service.py` 字号从 48px → 192px
2. 画布物理像素从 160×60 → 640×240（4× 等比放大以容纳 192px 字号）
3. 描边从手动偏移 1px → Pillow 原生 `stroke_width=4`
4. 干扰曲线振幅/噪点数量按 4× 面积等比放大
5. 前端两个 `CaptchaImage` 组件（admin-web / h5-web）CSS 显示尺寸不变

## 涉及文件
- `backend/app/services/captcha_service.py`（核心修改）
- `admin-web/src/components/CaptchaImage.tsx`（未改动，确认兼容）
- `h5-web/src/components/CaptchaImage.tsx`（未改动，确认兼容）

## 关键结论
- 4× 高清渲染方案：后端生成 640×240 高分辨率图，前端 CSS 仍 160×60，浏览器自动缩放
- 前端零改动，部署后端即可生效
- 版本号从 v1.5 升级到 v1.6
