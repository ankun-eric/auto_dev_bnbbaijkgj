# 聊BUG：多规格商品用户端列表页价格显示为 0

**时间**：2026-04-28 00:02

**用户诉求**：用户发现管理后台配置的多规格商品，在管理后台列表显示「¥1300起」正确，但在用户端（H5 + APP）服务列表页价格显示为 ¥0.00

## 关键确认信息

- 现象：多规格商品在用户端列表页价格显示为 ¥0.00（带小数点）
- 影响端：H5 网页端 + Flutter APP 端均有问题
- 影响范围：仅列表页，商品详情页价格正确
- 触发条件：仅多规格商品，单规格商品正常
- 期望效果：显示「¥最低SKU价起」格式
- 样式要求："起"字用小一号字体显示，参考前端统一样式

## 根因分析

用户端列表页直接读取商品主表 sale_price 字段，多规格商品主表 sale_price 为 0（实际价格存在 SKU 表），未做最低 SKU 价格计算。管理后台已有正确逻辑可复用。
