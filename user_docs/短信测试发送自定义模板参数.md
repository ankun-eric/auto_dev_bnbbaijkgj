## 功能简介

本次更新为管理后台的**短信管理模块**新增了「自定义模板参数」功能，主要包含以下改进：

1. **模板变量配置**：在新增/编辑短信模板时，可以为模板定义变量信息（变量名称、说明、默认值），变量顺序对应云厂商模板中的 `{1}`、`{2}` 等占位符
2. **测试发送改造**：测试发送短信时，可以选择模板并自定义模板参数值，而不是系统固定生成验证码
3. **发送前预览确认**：发送前弹出预览框，显示手机号、模板信息、参数列表和短信预览内容，确认无误后再发送
4. **测试结果增强展示**：发送结果中展示实际使用的参数列表和完整的短信预览内容

## 使用说明

### 一、配置模板变量

1. 登录管理后台，进入「短信管理」页面
2. 点击「短信模板」标签页
3. 点击「新增模板」按钮，或点击已有模板的「编辑」按钮
4. 在弹窗中填写模板基本信息后，找到下方的「模板变量」区域
5. 点击「+ 添加变量」按钮，添加一行变量配置：
   - **序号**：自动生成，如 `{1}`、`{2}`，对应云厂商模板的占位符
   - **变量名称**（必填）：如"验证码"、"有效时间"
   - **变量说明**（选填）：补充描述，如"6位数字验证码"
   - **默认值**（选填）：测试发送时自动填充的值，如"123456"
6. 可添加多个变量，按需点击右侧「删除」按钮移除
7. 点击「保存」完成模板配置

> 提示：模板列表中，未配置变量信息的模板会显示「未配置」标识，提醒您去补全变量信息。

### 二、测试发送短信

1. 点击「测试发送」标签页
2. 选择**服务商**（腾讯云/阿里云）
3. 输入**收信手机号**（11位手机号）
4. 在**短信模板**下拉框中选择要测试的模板（必选）
5. 选择模板后，系统根据模板变量配置自动生成参数输入区域：

#### 自动模式（默认）
- 如果所选模板已配置变量信息，系统会自动生成对应数量的输入框
- 每个输入框标注变量名称，变量说明显示为辅助提示
- 配置的默认值会自动填充到输入框中
- 您可以直接使用默认值，也可以手动修改

#### 高级输入模式
- 点击「切换到高级输入」链接，切换为文本框输入方式
- 在文本框中按英文逗号分隔输入所有参数值（如：`123456,5`）
- 适用于模板未预配置变量信息的场景

6. 点击「发送测试」按钮

### 三、预览确认

点击发送后，系统会弹出**预览确认框**，显示以下信息：

- **收信手机号**
- **模板名称和模板ID**
- **各变量参数及其值**（如：验证码 = 123456，有效时间 = 5）
- **短信预览内容**：将参数值填入模板后生成的最终短信文本

确认无误后，点击「确认发送」按钮，系统才会真正调用短信服务商 API 发送短信。

### 四、查看测试结果

发送完成后（无论成功或失败），在表单下方会展示测试结果：

- **发送状态**：成功（绿色）或失败（红色）
- **错误信息**：失败时显示服务商返回的错误信息
- **使用的参数**：本次测试实际使用的各变量参数值
- **短信预览内容**：将参数填入模板后的完整短信文本

## 注意事项

1. **权限要求**：短信管理模块仅限**系统管理员（admin 角色）**操作
2. **模板变量顺序**：变量的排列顺序对应云厂商模板中 `{1}`、`{2}` 等占位符的顺序，请确保顺序正确
3. **旧模板兼容**：已有的旧模板未配置变量信息时系统不会报错，您可以：
   - 进入模板编辑页面手动补录变量信息
   - 或在测试发送时切换到「高级输入模式」手动输入参数
4. **测试发送限制**：测试发送会实际调用短信服务商 API，请勿频繁测试，避免产生不必要的费用
5. **短信服务配置**：测试发送前请确保在「短信配置」标签页中正确配置了服务商的 API 密钥和相关信息
6. **参数校验**：
   - 自动模式下，系统会检查所有变量输入框是否已填写
   - 高级模式下，如果模板已配置变量信息，系统会校验参数个数是否匹配
