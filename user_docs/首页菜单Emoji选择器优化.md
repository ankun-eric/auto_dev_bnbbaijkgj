## 功能简介

本次更新对管理后台「首页配置 → 首页菜单管理」中的 Emoji 图标选择功能进行了全面优化：

1. **关键词映射扩充**：Emoji 智能推荐从 11 组扩充至 48 组，覆盖中医、药品、疫苗、保险、慢病管理、急救、康复、口腔、眼科、皮肤、妇科、儿科、养生、减肥等 40+ 个健康管理常见场景
2. **Emoji 选择面板**：新增独立的 Modal 弹窗选择器，支持 7 大分类浏览（医疗健康/运动健身/食物饮料/自然植物/活动物品/符号标志/动物），每个分类包含数十个精选 Emoji
3. **中文搜索**：支持输入中文关键词搜索 Emoji（如输入"医院"即可找到 🏥），覆盖 260+ 个 Emoji 的中文标签
4. **智能推荐整合**：选择面板顶部根据菜单名称实时推荐匹配的 Emoji（最多 8 个）
5. **去除手动输入框**：原有的 Emoji 文本输入框替换为可视化预览区域 + 选择按钮，避免手动输入出错

## 使用说明

### 步骤 1：进入首页菜单管理

1. 登录管理后台：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单中找到「首页配置」→「首页菜单管理」
3. 点击「新增菜单」按钮，或点击已有菜单的「编辑」按钮

### 步骤 2：填写菜单名称

1. 在弹窗中输入菜单名称（如"体检预约"、"中医养生"、"口腔护理"等）
2. 系统会根据名称自动匹配推荐的 Emoji 图标

### 步骤 3：选择 Emoji 图标

1. 确保「图标类型」选择为「Emoji」
2. 点击「选择图标」按钮，打开 Emoji 选择面板

### 步骤 4：在选择面板中挑选 Emoji

面板提供三种选择方式：

#### 方式一：使用智能推荐（推荐）
- 面板顶部会根据已输入的菜单名称显示推荐的 Emoji
- 直接点击推荐的 Emoji 即可选中

#### 方式二：分类浏览
- 面板中部有分类标签栏：医疗健康、运动健身、食物饮料、自然植物、活动物品、符号标志、动物
- 点击不同分类切换浏览，在网格中找到合适的 Emoji 点击选中

#### 方式三：中文搜索
- 在面板顶部搜索框中输入中文关键词（如"医院"、"跑步"、"苹果"）
- 系统会实时搜索匹配的 Emoji 并展示
- 清空搜索框可返回分类浏览模式

### 步骤 5：确认选择

1. 选中的 Emoji 会以蓝色边框高亮，底部显示"已选择：[Emoji]"
2. 点击「确定」按钮，Emoji 会填入表单
3. 填入后可在预览区域看到大号的 Emoji 效果
4. 如需更换，再次点击「选择图标」重新选择
5. 如需清除，点击「清除」链接

### 步骤 6：保存菜单

填写完其他信息（跳转类型、跳转地址等）后，点击「确定」保存菜单。

## 注意事项

1. **图标类型选择**：选择"Emoji"类型时，必须通过选择面板选择一个 Emoji 图标，不再支持手动输入
2. **菜单名称影响推荐**：建议先填写菜单名称，再打开选择面板，这样能获得更精准的推荐
3. **搜索技巧**：搜索支持模糊匹配，输入部分关键词即可（如输入"医"可匹配"医院"、"医生"等）
4. **复合 Emoji 支持**：系统支持所有 Unicode Emoji，包括复合 Emoji（如 👨‍⚕️、👨‍👩‍👧 等）
5. **权限要求**：仅管理员角色可操作首页菜单管理
6. **兼容性**：Emoji 在不同操作系统和浏览器中的显示风格可能略有不同，属于正常现象
