## 功能简介

本次新增「用药提醒 - 历史打卡记录」功能，为老年用户提供了便捷的历史用药打卡查看和补打卡能力。

### 新增功能

1. **历史记录入口**：在用药提醒主页新增「历史记录」按钮，一键进入历史打卡记录页面
2. **日历月份切换**：可通过左右箭头切换月份，查看任意月份的用药打卡情况
3. **日历状态标记**：每个日期格子右下角用彩色圆点标记当天打卡状态（绿=全部打卡 / 橙=部分打卡 / 红=全部漏打）
4. **打卡详情查看**：点击日历上任意日期，下方展示该天的用药打卡记录详情
5. **补打卡功能**：支持对最近 3 天内的漏打卡记录进行补打卡操作
6. **与主页布局一致**：历史记录页的打卡卡片布局与主页完全一致，老人无需重新学习

## 使用说明

### 1. 进入历史打卡记录

从用药提醒主页，点击「📋 历史记录」按钮即可进入历史打卡记录页面。

### 2. 日历操作

- **月份切换**：点击日历顶部的 `<` 或 `>` 按钮切换月份
- **选择日期**：点击日历上任意日期格子，下方将展示该天的用药打卡记录
- **默认选中**：进入页面时自动选中昨天（因为用户进历史记录主要为了补昨天的漏打卡）
- **状态识别**：日历格子右下角圆点颜色标示当天打卡状态

| 圆点颜色 | 含义 |
|----------|------|
| 🟢 绿色 | 当天所有用药全部打卡完成 |
| 🟠 橙色 | 当天部分打卡（有多个药，只打了部分） |
| 🔴 红色 | 当天全部漏打 |
| 无圆点 | 当天无用药计划 |

### 3. 查看打卡记录

点击日期后，下方展示该天的用药打卡记录。每条记录采用与主页一致的卡片布局，方便老人识别。

### 4. 打卡记录卡片状态

记录卡片有四种视觉状态，清晰区分：

| 状态 | 左侧竖条 | 背景色 | 状态标签 | 操作 |
|------|---------|--------|---------|------|
| 已打卡 | 绿色 | 白色 | 浅绿底+绿色字「已打卡」| 显示打卡时间 +「OK 完成」 |
| 已补 | 橙色 | 浅黄色 | 浅橙底+橙色字「已补」| 显示补打卡日期时间 +「OK 完成」|
| 未打卡（可补）| 红色 | 白色 | 浅粉底+红色字「未打卡」| 橙色「补打卡」按钮 |
| 未打卡（不可补）| 灰色 | 白色 | 浅粉底+红色字「未打卡」| 灰色「不可补」按钮 |

### 5. 补打卡操作

补打卡仅支持**最近 3 天内**的漏打卡记录：

1. 在历史记录页面中，找到需要补打卡的日期（默认为昨天）
2. 点击未打卡记录右侧的「补打卡」按钮
3. 在弹出的确认窗口中确认药品名称和计划时间
4. 点击「确认补打卡」完成操作
5. 补打卡成功后，卡片状态立即变为「已补」（橙色竖条+浅黄背景），日历圆点同步更新

**重要规则**：
- 今天的用药请在用药提醒主页直接打卡，历史记录页不支持今天的补打卡
- 超过 3 天的漏打卡记录无法补打，显示灰色「不可补」按钮
- 补打卡后如果误操作，可在主页通过「OK 完成」按钮撤销

### 6. 返回主页

点击页面顶部导航栏的「← 返回」按钮，回到用药提醒主页。

## 设计特点

本功能延续了宾尼健康一贯的老年用户友好设计理念：

- **字体够大**：时间 16px 粗体，药品名 14px 粗体，日历数字 16px 以上
- **触控友好**：日历格子触控区域 44x44px 以上，按钮高度 44px 以上
- **颜色醒目**：深色文字+白色背景，状态标签用绿/橙/红醒目颜色
- **信息清晰**：按日期→时间→药品→状态的自然阅读顺序排列
- **操作反馈**：补打卡后卡片和日历圆点同步更新，视觉反馈明确
- **布局统一**：与用药提醒主页卡片布局完全一致，老人无需重新适应
