## 功能简介

本次更新包含三大模块的系统性优化升级：

### 1. AI 对话页面切换咨询人体验优化
- 去掉了切换咨询人时的顶部蓝色横条提示（含"返回上一会话"撤销功能）
- 保留中央 Toast 胶囊提示（"已切换为XX咨询"，2秒自动消失）和聊天区系统分割消息

### 2. 家人健康看板（新增功能）
- 子女/家人可远程查看被守护人的健康状态总览
- 包含仪表盘（血压/血糖/心率最新值+异常高亮）、健康评分、趋势曲线（7/30/90天）、今日动态时间线、用药打卡清单、体检报告摘要

### 3. 健康提醒增强（新增功能）
- 复诊提醒：手动设置复诊日期，系统自动提醒
- 定期体检提醒：系统根据年龄智能推荐体检频率
- 健康指标复查提醒：血压/血糖异常后系统自动创建3天后复查提醒
- 异常预警：超出医学标准阈值时自动通知守护者

## 使用说明

### 一、AI 对话切换咨询人（优化后）

**操作步骤：**

1. 打开 [AI 健康咨询页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)
2. 点击页面顶部的咨询人选择区域
3. 在弹出的家人列表中，选择要咨询的对象
4. 系统会自动完成切换：
   - 页面中央显示半透明黑底胶囊提示"已切换为XX咨询"（2秒后自动消失）
   - 聊天区域清空并显示系统消息"—— 现在开始为XX提供健康咨询 ——"
   - 之前的对话记录已后台保存，不会丢失

**变化说明：**
- 与之前相比，切换时不再出现顶部蓝色横条和"返回上一会话"撤销按钮
- 操作更简洁清爽，减少了中老年用户的操作干扰

### 二、家人健康看板

**入口路径：**

1. 打开 [健康档案首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)
2. 在页面顶部的成员切换栏中，点击切换到某位家人（如"妈妈"、"爸爸"）
3. 切换后，成员栏下方会出现一个醒目的蓝色渐变按钮 **"查看 XX 的健康看板 →"**
4. 点击该按钮，进入该家人的健康看板详情页

**看板页面内容说明：**

#### 顶部导航栏
- 左侧返回按钮
- 中间显示家人姓名
- 右侧显示健康评分圆形徽章

#### 仪表盘区（核心指标一目了然）
- 三张小卡片横排展示：**血压**、**血糖**、**心率** 的最新数值
- **异常值**会以红色高亮显示，提醒您关注
- **正常值**以绿色显示

#### 健康评分（综合健康状态）
- 系统根据多个维度综合打分（满分100分）
- 圆形进度环 + 大字号分数
- 评分颜色：≥80分绿色（健康）、60-79分黄色（需关注）、<60分红色（需警惕）
- 评分维度：血压达标(25%) + 血糖达标(25%) + 心率达标(15%) + 用药按时完成度(20%) + 健康数据录入规律性(15%)

#### 趋势曲线区（健康趋势一览）
- 默认展示最近 **7天** 的趋势
- 顶部 Tab 可切换 **7天 / 30天 / 90天**
- 显示血压（收缩压+舒张压双线）、血糖、心率三条趋势曲线
- 曲线中以浅色背景区域标注正常范围，超出的数据点会红色标记

#### 今日健康动态时间线
- 按时间倒序展示当日所有健康事件
- 包含：体征录入事件（如"10:30 血压 130/85"）、用药打卡事件（如"08:00 服用降压药 ✓"）、就医事件等
- 异常事件红色标注（如"血压 160/100 ⚠️ 偏高"），正常事件绿色标注
- 当日无记录时显示"今日暂无健康动态"

#### 用药打卡清单
- 按 **早 / 中 / 晚 / 睡前** 四个时段分组展示
- 每个时段列出应服用的药品名称
- 标注服用状态：✓ 已服（绿色）/ ✗ 未服（橙色）
- 顶部显示"今日用药完成率 XX%"

#### 体检报告摘要
- 展示最近一次体检报告的关键信息
- 异常项以红色标签高亮
- 显示距离下次复诊/体检还有多少天（如"距离下次体检还有 23 天"）

### 三、健康提醒管理

**入口路径：**

1. 在 [健康档案首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile) 的预警区域下方，点击 **"提醒管理"** 卡片
2. 或直接访问 [健康提醒页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-reminders)
3. 也可以从健康看板页面底部的"查看提醒"按钮进入

**页面包含三个标签页：**

#### 复诊提醒（M11）

**添加复诊提醒：**
1. 点击页面右下角的 **"+"** 浮动按钮
2. 填写提醒信息：
   - **标题**（必填）：如"心内科复诊"
   - **复诊日期**（必填）：选择预约的复诊日期
   - **就诊科室**：如"心内科"
   - **就诊医院**：如"北京协和医院"
   - **周期性设置**：可选"不重复"、"每3个月"、"每6个月"、"每12个月"
   - **备注**：如"带上近期血压记录"
3. 点击"保存"完成创建

**提醒时机：** 系统会在复诊日 **提前1天** 和 **当天** 各提醒1次。

**标记完成：** 复诊完成后，点击提醒卡片上的"完成"按钮进行标记。

#### 体检提醒（M12）

**系统智能推荐：**
- 页面顶部会显示系统根据年龄推荐的体检频率
  - 40-49岁：建议每年体检 1 次
  - 50-59岁：建议每年体检 1 次，重点项目每半年 1 次
  - 60岁以上：建议每半年体检 1 次

**设置体检提醒：**
1. 查看系统推荐，确认后点击"+"按钮
2. 设置首次体检日期
3. 系统自动按推荐频率创建后续提醒
4. 也可以不采纳推荐，自行手动添加

#### 复查提醒（M13）

- 此页面为 **只读列表**，展示系统自动创建的指标复查提醒
- 当血压或血糖数据异常时，系统会自动在 **3天后** 创建复查提醒
- 提醒文案示例："您 3 天前的血压偏高（160/100），建议今天再次测量确认"
- 如果 3 天内您已主动录入了该指标的新数据，复查提醒会自动取消

### 四、异常预警机制

系统采用医学标准默认值进行异常检测，无需手动设置：

| 指标 | 正常范围 | 异常判定（触发预警） |
|------|---------|-------------------|
| 血压（收缩压/舒张压） | 90-139 / 60-89 mmHg | ≥140/90 或 <90/60 mmHg |
| 空腹血糖 | 3.9-6.1 mmol/L | ≥7.0 或 <3.9 mmol/L |
| 餐后2h血糖 | <7.8 mmol/L | ≥11.1 mmol/L |
| 心率 | 60-100 次/分 | >100 或 <50 次/分 |

**异常时系统会自动：**
1. 在健康看板中以红色高亮显示异常数值
2. 通知所有绑定的守护者（子女/家人）
3. 创建 3 天后的复查提醒
4. 同一异常指标同一天内最多通知 3 次，避免频繁打扰

## 注意事项

1. **健康看板仅对守护者可见**：只有通过绑定守护关系的子女/家人才能查看被守护人的健康看板。解除守护关系后，数据将立即不可见。

2. **看板数据为只读**：子女/家人在健康看板中只能查看数据，无法修改。录入体征数据、用药打卡等操作需要由老人（主体用户）本人在自己的账号中完成。

3. **提醒可由守护者创建**：子女可以帮父母创建复诊提醒和体检提醒，方便远程管理。

4. **核心指标聚焦三项**：本版本仅支持血压、血糖、心率三项核心指标，不含体温和体重。

5. **通知渠道**：异常预警和提醒通知会通过 APP 内消息推送，微信服务号/小程序消息通知也会在后续版本中接入。

6. **数据录入越规律，评分越高**：健康评分中有 15% 的权重来自"录入规律性"，建议每天定时测量并录入数据。
