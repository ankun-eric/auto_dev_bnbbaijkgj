## 功能简介

本次全面升级了体检报告智能解读模块，覆盖小程序、H5、App（Flutter）三端及管理后台，新增/优化以下功能：

- **百度智能云OCR接入**：替换模拟数据，接入百度OCR标准版/高精度版进行真实文字识别
- **后台OCR配置管理**：管理员可在后台配置OCR API Key/Secret Key、选择识别类型、一键开关
- **多格式上传**：支持图片（相册选择）、拍照、PDF三种上传方式
- **图片质量检测**：上传图片后自动检测清晰度，模糊图片提示重新上传
- **AI智能解读双视图**：解读结果支持「按分类展示」和「按异常优先展示」两种视图自由切换
- **历史报告管理**：修复三端假数据问题，全部对接后端真实接口，新增报告详情页
- **指标趋势对比**：折线图展示同一指标的历史趋势，标注正常范围，AI生成趋势分析
- **趋势恶化预警**：上传新报告后自动检测异常趋势，应用内横幅提醒
- **解读结果分享**：支持生成分享链接（7天有效期），接收者无需登录即可查看
- **免责声明**：所有解读结果页底部展示AI免责声明

## 使用说明

### 一、管理后台 — OCR识别配置

1. 登录管理后台：[管理后台](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单找到 **「AI管理」→「OCR识别配置」**
3. 配置以下项目：
   - **OCR功能开关**：启用后用户端可正常上传和解读报告，关闭后用户端会提示"解读功能暂时维护中"
   - **API Key**：填入百度智能云应用的API Key
   - **Secret Key**：填入百度智能云应用的Secret Key（页面上以脱敏方式显示）
   - **识别类型**：选择「通用文字识别（标准版）」或「通用文字识别（高精度版）」
4. 点击 **「连接测试」** 验证配置是否有效
5. 点击 **「保存」** 保存配置

### 二、用户端 — 上传体检报告

**（以下操作在H5/小程序/App三端一致）**

1. 进入体检报告页面（首页点击「体检报告」入口）
2. 选择上传方式：
   - **📷 拍照上传**：点击拍照按钮，调用摄像头拍摄体检报告
   - **🖼️ 相册选择**：从手机相册选择已有的体检报告图片
   - **📄 PDF上传**：选择本地PDF格式的电子体检报告
3. 上传后系统自动进行：
   - 图片质量检测（PDF免检）→ OCR文字识别 → AI智能解读
4. 解读完成后自动跳转到结果页

### 三、AI解读结果查看

解读结果页提供两种视图：

**视图一：按体检分类展示**
- 按血常规、肝功能、肾功能等分类展示指标
- 每项指标显示：指标名称、检测值、参考范围、状态标签
- 异常指标用红色标记

**视图二：按异常优先展示**
- 异常指标排在最前面，用红色/橙色醒目标记
- 每个异常指标附带简要说明
- 正常指标折叠显示

**综合解读区域**（两种视图共有）：
- AI生成的整体健康评估
- 针对异常指标的综合分析
- 饮食、运动、生活方式改善建议

### 四、历史报告管理

1. 在体检报告页面下方查看 **「历史报告」** 列表
2. 每条记录显示：上传日期、报告缩略图、异常指标数量
3. 点击进入 **报告详情页**：
   - 查看报告原图/PDF
   - 查看完整AI解读结果
   - 两种视图自由切换

### 五、指标趋势对比

1. 在报告详情页，每个指标项旁有 **「查看趋势」** 按钮
2. 点击进入趋势图页面：
   - 折线图展示该指标的历史变化
   - 浅绿色区域标注正常参考范围
   - 超出正常范围的数据点用红色标记
3. 图表下方有AI生成的趋势分析文字

### 六、趋势恶化预警

- 上传新报告后，系统自动检测以下情况并发出预警：
  - 某指标连续2次及以上偏高或偏低
  - 某指标数值呈持续恶化趋势
  - 之前正常的指标本次变为异常
- 预警以橙色横幅形式在报告页顶部展示
- 点击可查看详情，确认后不再重复提示

### 七、解读结果分享

1. 在报告详情页右上角点击 **「分享」** 按钮
2. 选择分享方式：
   - **复制链接**：生成一个7天有效期的H5链接，复制到剪贴板，可发送给家人或医生
   - **生成图片**：将解读结果生成长图，保存到相册后分享
3. 分享链接特点：
   - 接收者无需登录即可查看
   - 仅包含解读结果，不包含您的个人信息
   - 7天后链接自动失效

## 注意事项

1. **OCR配置前置**：管理员需先在后台配置百度OCR的API Key和Secret Key，用户端才能正常使用上传和解读功能
2. **图片清晰度**：上传的体检报告图片应清晰可辨，过于模糊的图片会被系统拦截并提示重新上传
3. **AI解读免责**：所有AI解读结果仅供参考，不构成医疗诊断或治疗建议
4. **分享安全**：分享链接7天后自动失效，分享内容不包含个人敏感信息
5. **趋势分析**：至少需要2次及以上的体检报告数据，趋势分析才能发挥作用
