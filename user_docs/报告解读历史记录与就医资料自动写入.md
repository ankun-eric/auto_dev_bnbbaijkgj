## 功能简介

本次修复解决了 **AI 报告解读完成后，历史报告和就医资料未自动写入记录** 的 Bug。

修复前的问题：
- 用户通过"报告解读"功能上传体检报告并完成 AI 解读后，进入"历史报告"页面看不到任何记录
- 进入"档案管理 → 就医资料 → 体检报告"也看不到对应记录

修复后的效果：
- AI 报告解读成功完成后，系统**自动**将解读记录写入历史报告列表
- 同时**自动**在就医资料-体检报告中生成对应记录
- 无需用户进行任何额外操作，全自动完成

## 使用说明

### 一、报告解读（触发自动写入）

1. 打开 H5 首页，点击"报告解读"功能卡片
2. 选择要解读的咨询人（家庭成员）
3. 拍照或从相册上传体检报告图片
4. 等待 AI 完成解读（通常 10~30 秒）
5. 解读结果正常显示后，历史报告和就医资料**已自动保存**

### 二、查看历史报告

1. 进入报告解读页面
2. 点击顶部的"历史报告"按钮
3. 列表中即可看到所有已完成解读的报告记录
4. 点击任一记录可查看解读详情
5. 支持报告对比分析功能（选择两份报告进行对比）

### 三、查看就医资料中的体检报告

1. 从首页进入"档案管理"
2. 点击"就医资料"
3. 选择"体检报告"分类
4. 列表中即可看到 AI 解读自动归档的体检报告
5. 点击可查看报告详情和原始图片

### 四、手动保存到就医资料

如果历史报告中有未自动归档到就医资料的记录，也可手动操作：

1. 进入"历史报告"列表
2. 点击目标报告进入详情
3. 点击"保存到就医资料"按钮
4. 系统会在就医资料-体检报告中创建对应记录

## 注意事项

1. **自动写入时机**：系统在 AI 解读**成功完成**后自动写入，若解读失败或中断则不会写入
2. **去重机制**：同一份报告的同一次解读不会重复写入，即使多次进入历史报告页面也只有一条记录
3. **数据完整性**：自动写入的记录包含报告名称、原始图片、AI 解读结果全文、提取的指标数据等完整信息
4. **两端同步**：H5 网页端和微信小程序端共用同一后端接口，修复对两端同时生效
5. **历史数据**：此前已完成但未自动保存的解读记录，需要重新发起解读才能写入历史报告
