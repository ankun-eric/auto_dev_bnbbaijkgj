## 功能简介

"数字安全绳"是关怀模式下的核心子功能，为独居老人或需要日常安全守护的用户提供每日签到 + 超时自动预警服务。

**核心功能**：
- **每日签到**：点击"我今天平安"按钮，系统记录签到时间和位置（需授权 GPS）
- **超时预警**：如果超过设定的 24 或 48 小时未签到，系统自动通知紧急联系人
- **紧急联系人管理**：可添加最多 3 位紧急联系人，支持 7 种关系类型（子女/配偶/父母/邻居/朋友/护工/其他）
- **暂停守护**：支持临时暂停 3/7/14 天或无限期，适用于外出旅行等场景
- **历史预警记录**：可查看所有历史预警事件和处理状态

**本次部署修复说明**：上一轮数字安全绳 v1 代码已写好但因容器未重建导致未在服务器生效。本次通过强制无缓存重建后端和 H5 容器，13 项端到端冒烟全部通过，功能正式上线。

## 使用说明

### 1. 进入数字安全绳

有两种方式进入数字安全绳功能：

**方式一：关怀模式首页入口卡片**
1. 打开关怀模式首页：[https://域名/care-home](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/care-home)
2. 在快捷胶囊下方，找到绿色渐变的"数字安全绳"横卡
3. 点击横卡，进入数字安全绳功能页

**方式二：右下角绿色悬浮球**
1. 在关怀模式首页，右下角有 SOS 红色悬浮球
2. 紧贴 SOS 左侧，有一个绿色圆形悬浮球，标有"🪢安全绳"字样
3. 点击绿色悬浮球，进入数字安全绳功能页

### 2. 设置超时阈值

在功能页的"守护设置"区域：
- 点击 **24 小时** 或 **48 小时** 按钮，设置超时时长
- 选中项会变成绿色填充，右上角有白底绿勾 ✓
- 设置后即时生效

### 3. 每日签到

在功能页的签到区域：
1. 点击大圆形绿色按钮"我今天平安 ✋"
2. 系统会请求 GPS 位置权限（建议允许，以便预警时提供位置信息）
3. 签到成功后，顶部横幅显示：
   - ✅ 上次签到：MM-DD HH:mm
   - ⏰ 下次签到截止：MM-DD HH:mm
4. 页面底部会弹出绿色提示"收到你的平安信号 ❤"

### 4. 管理紧急联系人

**添加联系人**：
1. 在"紧急联系人"区域，点击"+ 添加"按钮
2. 底部弹出添加表单，填写：
   - **姓名**：必填，如"张儿子"
   - **手机号**：必填，输入 11 位手机号。输完后系统自动校验该号码是否为 bini-health 已注册用户；绿色对勾表示校验通过，此时"保存"按钮才可点击
   - **关系**：选填，从 7 芯片中选择（子女/配偶/父母/邻居/朋友/护工/其他），支持取消选择
3. 填写完成后点击"保存"
4. 保存成功后，联系人立即出现在列表中
5. 被设为联系人的 TA 会收到一条站内信，告知其已被设为紧急联系人

**编辑联系人**：点击联系人右侧的"编辑"按钮，修改信息后保存。

**删除联系人**：点击联系人右侧的"删除"按钮，确认后删除。

> ⚠️ 最多添加 3 位紧急联系人。手机号必须是 bini-health 已注册用户。

### 5. 暂停守护

如需临时外出或旅行：
1. 点击"暂停守护"按钮
2. 选择暂停时长：3 天 / 7 天 / 14 天 / 无限期
3. 暂停期间不会触发超时预警
4. 随时可点击"恢复守护"重新开启

### 6. 查看历史预警

在页面底部的"历史预警"区域：
- 显示预警记录数量
- 点击"展开"查看每条预警的详细信息：触发时间、最后签到位置、通知了哪些联系人、是否已解除

## 注意事项

1. **首次访问需登录态**：所有 `/api/safety-rope/*` 接口都需要带 token，未登录访问会返回 401（这是正常现象）
2. **手机号校验依赖用户表**：紧急联系人必须是 bini-health 已注册用户。如果手机号未注册，请先邀请 TA 注册
3. **关系字段白名单**：后端强制校验关系必须是以下 7 芯片之一：子女 / 配偶 / 父母 / 邻居 / 朋友 / 护工 / 其他
4. **预警仅站内信，不发邮件**：v2 锁死版已彻底移除 SMTP 邮件，所有通知走站内消息
5. **GPS 位置**：签到时的位置信息有助于紧急情况下快速定位，建议授权 GPS
