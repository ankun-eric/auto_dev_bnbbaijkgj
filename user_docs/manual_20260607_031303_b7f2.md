## 功能简介

本次更新对益智乐园（脑力游戏）进行了以下 3 项优化：

### 1. 首页功能卡片紧凑化

三张功能卡片（数学游戏、排行榜、组队挑战）的布局从纵向改为横向并排，配合更紧凑的间距和更小的字号，确保三张卡片在一屏内全部可见，无需上下滚动。

- 卡片由「图标在上、标题在下」改为「图标在左、文字在右」的横向排列
- 卡片内边距、间距、圆角均缩小
- 图标、标题、副标题、描述的字号均减小
- 页面标题和用户栏也做了同步缩小适配

### 2. 分享图标升级为微信图标

原来使用的 🔄 符号替换为微信官方风格的绿色气泡图标，客户一看就知道可以转发到微信。

涉及位置（共 4 处）：
- 组队挑战页「邀请好友来挑战」按钮
- 组队结果页「分享成绩」按钮
- 右下角绿色圆形浮动分享按钮
- 分享弹窗「微信好友」选项

### 3. 地区数据去重优化

修复了排行榜四级联动选择器（省→市→区→街道）中出现重复数据的问题。现在 sync-seed 接口在同步种子数据时会自动检查数据是否已存在，避免重复插入，确保无论调用多少次都不会出现重复数据。

## 使用说明

### 进入益智乐园

1. 打开 H5 用户端首页
2. 在长辈版模式下，点击「乐龄游戏」卡片
3. 或直接访问 [益智乐园页面](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/brain-game.html)

### 功能卡片使用

进入益智乐园后，您将看到三张功能卡片：

| 卡片 | 图标 | 功能 |
|------|------|------|
| 数学游戏 | 🧮 | 基础、进阶、挑战三个难度级别的数学题目 |
| 排行榜 | 🏆 | 查看全国/本地区的游戏成绩排名 |
| 组队挑战 | 🤝 | 邀请好友一起组队挑战题目 |

点击任意卡片即可进入对应功能。

### 排行榜地区筛选

1. 进入排行榜页面
2. 使用四级联动选择器，依次选择「省 → 市 → 区 → 街道」
3. 系统将展示所选地区的排行数据

### 组队挑战分享

1. 进入组队挑战页面
2. 点击「邀请好友来挑战」按钮（按钮前显示微信图标）
3. 在分享弹窗中选择「微信好友」进行分享
4. 也可以点击右下角的绿色浮动分享按钮快速分享

### 分享成绩

1. 完成组队挑战后
2. 在结果页点击「分享成绩」按钮（按钮前显示微信图标）
3. 选择分享方式即可将成绩分享给好友

1. **浏览器兼容性**：建议使用 Chrome、Safari、Edge 等现代浏览器访问，微信内置浏览器也已适配
