## 「本月配额」三卡说明

「本月配额」区域为横向并排的三张独立卡片，宽度均分，顺序固定：

| 顺序 | 图标 | 标题 | 单位 | 说明 |
|------|------|------|------|------|
| 第 1 张 | 📞 | AI 外呼 | 次 | 展示本月已用 / 总额，附进度条 |
| 第 2 张 | 🚨 | 紧急呼叫 | 次 | 展示本月已用 / 总额，附进度条 |
| 第 3 张 | 👥 | 家庭成员 | 人 | 展示已添加 / 上限，附进度条 |

每张卡片从上到下结构一致：**图标 → 标题 → 大数字 X/Y（已用/总额）→ 进度条 → 说明文案**。

**特殊状态**：

- **不限量**：当某项额度为不限时，中部直接显示「不限」，且不展示进度条。
- **已用完**：当已用达到或超过总额时，进度条变为**红色**，并在数字旁出现红色「已用完」小标签。

## 使用说明

1. **进入会员中心**：在 AI 首页右上角「+」菜单中点击「👑 会员中心」，或从左侧抽屉的「会员中心」入口进入；也可直接打开上方「会员中心」链接。
2. **查看顶部会员状态**：付费且未过期的会员将看到蓝紫渐变 Banner、金/紫等级徽章；即将到期或已过期会在 Banner 上方出现对应状态条。
3. **查看「本月配额」**：在用户信息卡下方，可看到横向并排的三张配额卡（AI 外呼 / 紧急呼叫 / 家庭成员），每张展示已用量、总额、进度条与说明。
4. **查看「权益对比」**：继续向下，可看到新版权益对比表，列出免费会员与各付费套餐在「家庭成员 / AI 外呼 / 紧急呼叫」三项上的差异，当前所属套餐列高亮。
5. **邀请家庭成员 / 升级续费**：点击「邀请家庭成员」卡片可发起邀请；页面底部的吸底紫色大按钮会根据会员状态智能切换文案（立即开通 / 升级 / 续费等），点击进行相应操作。
