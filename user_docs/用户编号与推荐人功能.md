## 功能简介

本次更新为系统新增了 **用户编号** 与 **推荐人** 两大功能模块：

1. **用户编号**：每位用户拥有一个独一无二的 8 位数字编号，用于身份标识和分享传播
2. **推荐人绑定**：用户可通过分享链接/二维码邀请新用户注册，新用户注册后自动绑定推荐关系
3. **推广落地页**：新用户打开分享链接后先看到产品推广介绍页，了解产品亮点后再注册
4. **推荐管理（后台）**：管理员可查看推荐统计、排行榜，超级管理员可修改用户推荐人

## 使用说明

### 一、查看用户编号

**H5 端 / 小程序 / Flutter App：**
1. 登录后进入「个人中心」页面
2. 在头像和昵称下方可以看到您的 8 位数字用户编号
3. 点击编号旁边的「复制」按钮可将编号复制到剪贴板

**管理后台：**
1. 登录管理后台，进入「用户管理」页面
2. 用户列表中新增了「用户编号」列，直接显示每位用户的编号
3. 支持通过搜索框按用户编号搜索用户

### 二、邀请好友（生成分享链接/二维码）

**H5 端：**
1. 进入「个人中心」→ 点击「邀请好友」菜单
2. 或点击首页右下角的悬浮邀请按钮
3. 进入邀请页面后，可以看到：
   - 您的专属二维码（好友扫码即可注册）
   - 「复制分享链接」按钮（复制后发送给好友）

**小程序：**
1. 进入「我的」→ 点击「邀请好友」菜单
2. 或点击首页右下角的邀请按钮
3. 进入邀请页面后，可以：
   - 查看专属二维码和邀请码
   - 点击「复制链接」复制分享链接
   - 点击「分享给好友」通过微信直接分享给好友

**Flutter App：**
1. 进入「我的」→ 点击「邀请好友」
2. 或点击首页的邀请卡片
3. 进入邀请页面后，可以：
   - 查看专属二维码
   - 复制分享链接
   - 使用系统分享功能分享给好友

### 三、新用户通过分享链接注册

1. 新用户收到分享链接后点击打开
2. 进入**推广落地页**，了解产品五大核心亮点：
   - AI 健康问答
   - 智能症状自查
   - 体检报告 AI 解读
   - 中医体质辨识
   - 个性化健康计划
3. 点击页面底部的「立即注册」按钮
4. 进入登录/注册页面，完成注册后将自动绑定推荐关系

### 四、管理后台 — 推荐管理

1. 登录管理后台，在左侧菜单找到「推荐管理」
2. 页面展示：
   - **统计概览**：总推荐用户数、今日新增推荐数、本月新增推荐数
   - **推荐排行榜**：按推荐人数降序排列的用户榜单

### 五、管理后台 — 修改推荐人（仅超级管理员）

1. 进入「用户管理」页面
2. 在用户列表中找到目标用户，点击操作列的「推荐人」按钮
3. 在弹窗中输入新的推荐人编号（8位数字）
4. 确认修改后完成

> ⚠️ 此功能仅超级管理员可见和操作，普通管理员看不到「推荐人」按钮。

## 注意事项

1. **用户编号不可更改**：每位用户的 8 位编号在注册时自动生成，之后不可修改
2. **推荐关系仅可绑定一次**：新用户注册时绑定推荐人后，用户自己无法更改推荐人
3. **不允许自推荐**：推荐人编号不能是自己的编号
4. **推荐人编号无效时不影响注册**：如果分享链接中的推荐人编号不存在，注册流程不受影响，只是不会建立推荐关系
5. **修改推荐人权限**：仅超级管理员（is_superuser）可以在管理后台修改用户的推荐人
6. **存量用户**：系统上线后会自动为所有已注册用户补全用户编号
