## 功能简介

本次只换界面与交互，不改变业务含义。家庭成员这边的邀请，仍然是「邀请这个家庭成员加入 / 绑定」的原有业务逻辑，二维码携带的仍是按成员（member_id）绑定的原有邀请，只是承载界面统一为漂亮的二维码页。

具体变化：

1. **健康档案页「邀请家庭成员」**：填资料保存成功后，新增一步「成员已添加成功🎉 / 去邀请 TA / 暂不邀请」提示框，点「去邀请 TA」即跳转到带二维码的邀请页。
2. **家庭成员列表里每条记录的「去邀请 / 查看邀请码 / 重新邀请」**：点击后全部跳转到漂亮二维码页，不再弹出旧的纯文字邀请码弹窗。
3. **二维码页顶部标题**：从家庭成员这边进入时，沿用「邀请 TA 加入我的健康守护」，不做改动。

## 使用说明

### 一、从「邀请家庭成员」新建并邀请（入口 1）

1. 打开 [健康档案页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)。
2. 点击「邀请家庭成员」，在弹出的填写页中填写成员关系、姓名、性别、出生日期等（可展开「其他（选填）」补充身高体重、健康信息）。
3. 点击「保存」。保存成功后，会弹出「**成员已添加成功🎉**」提示框，并提供两个按钮：
   - **去邀请 TA**：跳转到漂亮的二维码邀请页（`/family-invite?member_id=<新成员id>`），可保存二维码、复制链接或转发微信好友。
   - **暂不邀请**：关闭提示框，回到列表（成员已添加成功，可稍后再邀请）。

### 二、从家庭成员列表里发起邀请（入口 2）

1. 打开 [家庭成员列表](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/archive-list)。
2. 在任意一条家庭成员记录右下角，根据该成员当前状态，会出现「去邀请 / 查看邀请码 / 重新邀请」中的一个按钮。
3. 点击该按钮，即**直接跳转到漂亮的二维码邀请页**（按该成员的 member_id 绑定），不再出现纯文字邀请码弹窗。

### 三、在二维码邀请页完成邀请

进入二维码邀请页后，页面会自动生成该成员专属的邀请二维码，您可以：

- **保存到本地**：把二维码图片保存到手机相册，方便发送。
- **复制链接**：复制邀请链接，粘贴到任意聊天窗口。
- **转发微信好友**：点击后按提示通过右上角「…」转发给微信好友。

对方扫码 / 打开链接并确认后，即会绑定到对应的那个家庭成员。

## 注意事项

- **邀请有效期**：二维码邀请页提示「邀请 24 小时内有效」，请尽快发送给对方。
- **绑定对象正确性**：每个入口生成的二维码都严格按 `member_id` 绑定，扫码后绑定的仍是您选择 / 新建的那个家庭成员，不会绑定到错误的人。
- **缺少成员信息时**：若在未携带成员信息的情况下直接打开二维码页，页面会提示「⚠️ 缺少成员信息，无法生成邀请，请从成员档案进入」，请从上述入口进入。
- **本期范围**：本次改动仅涉及 **H5 端**。小程序端如存在同样入口，是否同步对齐待后续确认，本期不在范围内。
- 本功能为纯前端界面/交互统一，无需重新安装任何客户端，刷新页面即可体验最新效果。
