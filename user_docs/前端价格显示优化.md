## 功能简介

本次更新优化了全端（H5 网页端、Admin 管理后台、Flutter APP、微信小程序）的**价格显示格式**：

- **去掉价格末尾多余的零**，让价格展示更简洁清爽
- 优化前：`¥19.00`、`¥19.10`、`¥0.00`
- 优化后：`¥19`、`¥19.1`、`¥0`

所有涉及金额展示的场景均已覆盖，包括商品价格、订单金额、优惠券金额、服务价格等。

> 说明：支付、退款、结算等资金流转的精确金额不受影响，数据库存储精度也保持不变。

## 注意事项

1. **价格精度不受影响**：本次优化仅改变价格的显示格式，不影响实际金额计算。数据库中的价格数据精度保持不变。
2. **支付金额不受影响**：支付、退款、结算等涉及真实资金流转的接口，金额仍精确到分（两位小数），不参与格式化。
3. **小数价格正常显示**：如果某个商品价格本身就有非零小数（如 ¥19.12），则完整显示，不会被截断。
4. **全端统一**：H5 网页端、Admin 管理后台、Flutter APP、微信小程序的价格显示规则完全一致。
