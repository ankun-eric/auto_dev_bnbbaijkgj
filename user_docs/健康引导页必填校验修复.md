## 功能简介

本次修复了 **健康引导页** 第一步"基本信息"中必填校验缺失的问题。

### 修复前的问题

在健康引导页的第一步"基本信息"中，姓名、性别、出生日期虽然标注了红色星号 `*` 表示必填，但用户不填写任何信息也能直接点击"下一步"跳过，导致健康档案中基本信息可能为空。

### 修复后的效果

- 在第一步点击"下一步"时，系统会自动校验姓名、性别、出生日期三个必填字段
- 未填写的字段下方会显示红色提示文字（如"请输入姓名"）
- 同时弹出 Toast 汇总提示，列出所有未填写的必填字段（如"请填写：姓名、性别"）
- 身高和体重仍为选填字段，不做必填校验
- 第二步至第四步（慢性病史、过敏史、遗传病史）均为选填，不受影响

## 使用说明

### 进入健康引导页

1. 打开 H5 移动端首页：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 使用手机号注册或登录
3. 登录成功后，系统会自动弹出"完善健康档案"引导页

### 第一步：填写基本信息

1. **姓名（必填）**：在"姓名"输入框中输入您的姓名
2. **性别（必填）**：点击"男"或"女"按钮选择性别
3. **出生日期（必填）**：点击"请选择"，在弹出的日期选择器中选择出生日期
4. **身高（选填）**：如需填写，输入身高数值（单位：cm）
5. **体重（选填）**：如需填写，输入体重数值（单位：kg）
6. 点击"下一步"按钮

### 校验提示说明

- 如果有必填字段未填写，点击"下一步"后：
  - 未填写字段下方会显示**红色提示文字**，明确告知您需要填写什么
  - 页面顶部会弹出 **Toast 提示**，汇总所有未填字段，格式如："请填写：姓名、性别、出生日期"
- 补全所有必填字段后，再次点击"下一步"即可正常进入第二步

### 第二步至第四步

- **第二步 - 慢性病史**：选择您已有的慢性疾病（选填，可不选直接"下一步"）
- **第三步 - 过敏史**：填写药物过敏、食物过敏、其他过敏情况（选填，可不填直接"下一步"）
- **第四步 - 遗传病史**：选择家族遗传疾病（选填，可不选直接点击"完成"）

### 完成引导

- 在第四步点击"完成"后，档案保存成功，自动跳转到首页
- 如果暂时不想填写，可以随时点击右上角"跳过"按钮

## 注意事项

1. **必填字段仅限第一步**：只有第一步的姓名、性别、出生日期为必填，其余所有步骤的字段均为选填
2. **校验时机**：仅在点击"下一步"按钮时触发校验，填写过程中不会实时校验
3. **数据保存**：每一步点击"下一步"或"完成"时，当前步骤的数据都会自动保存到服务器
4. **跳过功能**：右上角"跳过"按钮在任何步骤都可使用，跳过后可以在个人健康档案页面补充完善
