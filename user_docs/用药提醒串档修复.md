## 功能简介

在「健康档案」中，可以切换查看「本人」或「家人（非本人成员）」的健康卡看板。看板里的「用药提醒」卡片用于展示该对象当前需要服用的药物、用药评分和今日用药时间线。

**本次修复内容**：此前在查看「某位家人」的健康卡看板时，用药提醒里会错误地混入「账号本人」的用药记录，造成数据串档、归属混乱。修复后，看板严格遵循「看谁的看板，就只显示谁的用药记录」原则：

- 查看**本人**看板：只显示本人自己的全部用药记录，一条不少。
- 查看**某位家人**看板：只显示该家人自己的用药记录，绝不混入本人或其他家人的记录。

该规则在「用药评分」「今日用药时间线」「用药汇总」三处口径完全一致。

## 使用说明

1. 打开 [健康档案页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)。
2. 在顶部成员 Tab 中，切换到某位「非本人」家庭成员。
3. 点击「查看 XX 的健康卡看板」，进入该成员的看板页面。
4. 查看「用药提醒」卡片：此时显示的用药记录、用药评分、今日时间线，**均只属于该家人本人**，不会再出现你自己（账号本人）的用药记录。
5. 切回「本人」Tab 查看本人看板时，你自己的全部用药记录会完整、正常显示。

## 注意事项

- 本次修复**不改变任何数据**，只修正了看板的展示口径，历史用药记录不受影响。
- 本人的用药记录在系统中一律为「未绑定具体家庭成员」状态，因此始终归属本人看板，不会出现在家人看板中。
- 「用药提醒-全部页」等其它用药相关功能不受本次修复影响，使用方式保持不变。
