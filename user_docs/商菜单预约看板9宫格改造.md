## 功能简介

本次「商菜单（预约看板）」9 宫格改造完成了商家端 PC 后台预约管理页面的整体重构，严格按照 PRD v1.0 §2.2 的视觉与交互规范实现。改造后：

- 商家进入「商菜单」**默认即看到 3×3 = 9 宫格日视图**，一眼看清当天 9 个时段的预约分布；
- 顶部增加 **4 个并列 Tab**：预约看板（默认） / 服务分组日视图 / 资源视图 / 列表视图；
- **周视图改为 3×7 = 21 格**（上午 / 下午 / 晚上 × 周一 ~ 周日），每格显示「N 个预约」徽标 + 状态色；
- **月视图弹窗改为左右双栏联动**：左栏=当天时间轴预约列表，右栏=选中预约详情卡，点击左栏自动切换右栏详情；
- 抽屉宽度 480 px，**4 个操作按钮一目了然**：核销 / 修改预约时间 / 取消预约 / 联系客户；
- 4 色状态严格采用 Ant Design 5 默认色板（蓝/橙/绿/灰）；
- 「核销」与「修改预约时间」在 PC 端**置灰并悬浮提示「请到手机端核销」/「请到手机端修改时段」**，避免 PC 误改；
- 「取消预约」与「联系客户」PC 端可用（取消带二次确认弹窗）；
- 旧的「按服务项目分组的 9:00–20:00 时间轴大表格」已**直接废弃并从代码中移除**。

## 使用说明

### 1. 进入「商菜单」预约看板

1. 打开 PC 浏览器，访问 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 使用商家管理账号登录
3. 在左侧菜单选择「订单管理」→「商菜单」（或直接访问预约看板链接）
4. 默认进入「**预约看板**」Tab → **日视图 3×3 九宫格**

### 2. 顶部 4 个 Tab 切换

| Tab 名称 | 用途 |
|----------|------|
| 预约看板（默认） | 9 宫格 / 周视图 / 月视图三种核心看板 |
| 服务分组日视图 | 当日预约**按服务项目**横向分组展示（无 9:00-20:00 时间轴大表格） |
| 资源视图 | 按资源（美容师/房间）维度查看预约分布 |
| 列表视图 | 跳转到平铺订单列表页，支持完整搜索筛选 |

### 3. 顶部日期切换控件

在「预约看板」Tab 内提供以下控件，方便快速切换日期：

```
[← 前一天]  [今天]  2026-05-06 周三  [后一天 →]  [📅 日期选择器]  [🔄 刷新]
```

- 「今天」按钮：当前日期为今天时高亮显示（蓝色主按钮）
- 「日期选择器」：可任意跳转日期 / 周 / 月

### 4. 日视图（默认子视图）

- 3 行 × 3 列共 9 个时段格，覆盖 06:00 ~ 24:00 每 2 小时一段
- 每格默认显示：
  - 客户姓名（首条预约）
  - 服务名称（首条预约）
  - 具体时段（如 09:00）+ 时段标签（06:00-08:00）
  - 4 色状态色块（左侧竖条 + 右上角文字标签）
  - 该时段 N 条预约统计 + 已核销数
- 点击任一时段格 → **滑出右侧抽屉**，列出该时段所有订单完整详情

### 5. 周视图（3 行 × 7 列 = 21 格）

- 3 行：上午（06:00-12:00） / 下午（12:00-18:00） / 晚上（18:00-24:00）
- 7 列：周一 ~ 周日
- 每格仅显示：「N 个预约」徽标 + 状态色（不显示客户姓名，保持简洁）
- 点击格子 → 抽屉滑出，列出该时段全部预约

### 6. 月视图（月历 + 双栏弹窗）

- 标准月历布局，每个日期格仅显示徽标「N 个预约」+ 状态色
- 点击某天 → 弹出**左右双栏弹窗**：
  - **左栏**：当天 06:00-24:00 时间轴预约列表，每条一行（时间 + 客户 + 服务 + 状态色）
  - **右栏**：选中那条预约的**详情卡片**（客户/服务/时段/资源/状态/操作按钮）
  - **联动逻辑**：点击左栏任一条 → 该条高亮 → 右栏实时切换为该条详情

### 7. 抽屉详情面板（核心交互）

抽屉为右侧滑出，固定宽度 480 px，每条预约展示：

- **预约头部**：客户姓名、客户手机号
- **预约信息**：服务名称、时段、金额
- **预约状态**：4 色状态色块（待到店蓝 / 已到店橙 / 已核销绿 / 已取消灰）
- **操作区**（4 个按钮）：

| 按钮 | PC 端表现 | 行为 |
|------|----------|------|
| 核销 | **置灰** | 鼠标悬浮提示：「请到手机端核销」 |
| 修改预约时间 | **置灰** | 鼠标悬浮提示：「请到手机端修改时段」 |
| 取消预约 | **可用（红色）** | 弹出二次确认对话框，确认后调用后端取消接口 |
| 联系客户 | **可用** | 调用 `tel:` 协议唤起本机拨号软件 + 复制手机号到剪贴板 |

### 8. 4 色状态识别（Ant Design 5 默认色板）

| 状态 | 名称 | 色值 | 业务含义 |
|------|------|------|----------|
| 🔵 待到店 | colorPrimary | `#1677FF`（蓝） | 已预约、客户尚未到店 |
| 🟠 已到店 | colorWarning | `#FA8C16`（橙） | 客户已到店、部分核销中 |
| 🟢 已核销 | colorSuccess | `#52C41A`（绿） | 全部核销完成 / 已完成 |
| ⚪ 已取消 | colorTextDisabled | `#BFBFBF`（灰） | 已取消 / 已退款 / 已过期 |

每个时段格 / 预约卡左侧的**竖条颜色**即为该状态的快速识别色，配合右上角文字标签使用。

## 注意事项

1. **核销与修改预约时间必须使用手机端**：PC 端这两个按钮始终置灰，悬浮显示提示，请商家在抽屉提示出现时切换到手机端商家端 / 小程序 / App 完成。
2. **取消预约不可恢复**：取消按钮虽 PC 端可用，但会弹出二次确认对话框，请仔细核对客户姓名、服务名称、时段后再确认。
3. **联系客户**：点击后浏览器会同时尝试 `tel:` 协议（部分电脑可能没有拨号软件）+ 自动将完整手机号复制到剪贴板，复制成功会有 toast 提示，可直接粘贴拨打。
4. **时段切片固定为 9 段**：06:00-08:00 / 08:00-10:00 / 10:00-12:00 / 12:00-14:00 / 14:00-16:00 / 16:00-18:00 / 18:00-20:00 / 20:00-22:00 / 22:00-24:00。**凌晨 00:00-06:00 不开放预约**，凌晨脏数据会在统计区域以橙色警示提醒，需到「列表视图」查看处理。
5. **日 / 周 / 月视图记忆**：浏览器会记住您上次选择的子视图，下次进入「预约看板」自动恢复。
6. **响应式范围**：本次改造仅针对 PC 端 1280 px+ 主流分辨率优化；手机端商家请使用小程序 / H5 / App。
7. **旧版「按服务项目分组的 9:00–20:00 时间轴大表格」已彻底废弃**，如有同事仍按旧路径访问，请告知改用新「商菜单」入口。
