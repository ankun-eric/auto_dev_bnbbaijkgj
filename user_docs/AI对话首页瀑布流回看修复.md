## 功能简介

本次修复解决了 H5 端 AI 对话首页"小康"在用户发起对话后，**顶部"欢迎语 + 三大功能卡片（AI 诊室 / 看报告 / 健康档案）+ 推荐快捷胶囊"整片消失、无法回看**的问题。

修复后，原本会消失的顶部内容改为**收缩为右上角圆形小康头像悬浮按钮**，用户可随时点击展开查看完整面板，并直接点击三大卡片或推荐胶囊触发对应动作。整个交互**不依赖任何后台开关**，所有 H5 用户开箱即用。

## 操作步骤

### 1. 进入 AI 对话首页

在手机浏览器（或企业微信内置浏览器）中打开：

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)

登录后进入"小康"页面。

### 2. 首次进入（无对话历史）

如果当前会话还没有任何消息，您会看到完整的顶部欢迎面板：

- 顶部：欢迎语 + AI 头像 + 引导文案
- 中部：三大功能卡片 — **AI 诊室** / **看报告** / **健康档案**
- 下部：推荐快捷胶囊 — 体检解读 / 用药咨询 / 健康档案 / 症状自查 等
- 此状态下**右上角无悬浮按钮**（因为面板已完整展开）

### 3. 发起对话后（核心修复点）

#### 3.1 顶部面板自动收起

只要您发送任意一条消息（手动输入或点击推荐胶囊），原本的顶部欢迎面板会**自动收起**为：

- 屏幕**右上角**保留一个**圆形小康头像悬浮按钮**（44×44 像素）
- 消息列表占据主屏，自动滚动到底部最新消息
- 您可以正常上下滚动查看历史消息

#### 3.2 随时点击右上角小康头像 → 展开欢迎面板

- 点击右上角的小康头像悬浮按钮
- 顶部面板**从屏幕顶部下拉展开**（约 250ms 动画），覆盖在消息列表之上
- 展开态下：欢迎语、三大功能卡片、推荐胶囊**全部完整显示**，与未对话时一致
- 展开态右上角会显示一个 "⌃ 收起" 按钮

#### 3.3 三大功能卡片点击行为（展开态下）

| 卡片 | 点击后效果 |
|------|------------|
| **AI 诊室** | 跳转到 AI 智能问诊页面 |
| **看报告** | 跳转到体检报告解读页面 |
| **健康档案** | 跳转到个人健康档案页面 |

#### 3.4 推荐快捷胶囊点击行为（展开态下）

| 胶囊 | 点击后效果 |
|------|------------|
| 体检解读 | 自动以"体检解读相关问题"作为用户消息发送给 AI，面板自动收起，AI 开始回复 |
| 用药咨询 | 同理，自动发送对应问题 |
| 饮食建议 | 同理 |
| 失眠 | 同理 |

> 💡 推荐胶囊点击后**会自动收起面板**，让您聚焦于 AI 的回复，无需手动关闭。

### 4. 顶部面板"重新收起"的 4 种方式

无论您处于展开态，都可以通过以下任一方式将面板收起为右上角悬浮按钮：

| 方式 | 操作 |
|------|------|
| ✅ 手动收起 | 点击展开态右上角的 "⌃ 收起" 按钮 |
| ✅ 上滑消息列表 | 在面板下方的消息列表区域向上滑动 → 自动收起 |
| ✅ 点击输入框 | 把焦点放到底部输入框 / 开始打字 → 自动收起 |
| ✅ 点击外部空白 | 点击面板外的消息列表空白区域 → 自动收起 |

## 注意事项

1. **常驻生效，不依赖配置**：本次修复**不读取**任何后台 `ai_home_config` 开关，所有 H5 用户进入对话页面后均生效，无需运营配置。
2. **下载/安装无需变更**：本次修复仅涉及 H5 移动网页端代码，**未改动**安卓 APK / iOS / 微信小程序 / 桌面客户端，已安装的客户端无需重新下载。
3. **缓存刷新提示**：若您之前打开过该页面、出现界面与本手册描述不一致的情况，请**下拉刷新**或**关闭页面后重新进入**，以加载最新前端资源。
4. **支持多设备宽度**：iPhone SE / 中屏 / 大屏 Pro Max 等不同设备宽度均已自适应，悬浮按钮位置和点击热区符合系统规范（44~48px）。
5. **历史会话自动滚到底部**：进入页面时若已有对话历史，页面会自动滚动到最底部最新消息，顶部面板默认收起为悬浮按钮。
