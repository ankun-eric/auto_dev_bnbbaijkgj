## 功能简介

本次健康档案 v2 优化涉及以下核心功能升级：

### 1. 用药计划全面升级（M4）
- 新增**每日服药次数**选择（1-6次），支持**自定义服药时间点**（可增删，最多6个）
- 新增**开始日期**和**结束日期**设置，支持**长期服用**标记
- 新增**用药提醒开关**，可随时开启/关闭提醒
- 新增**关联疾病标签**多选（高血压、糖尿病等），支持自定义输入
- 新增**拍照识药**功能，拍药盒即可自动识别药品名称
- 用药计划Tab新增**"今日用药 / 全部用药计划"**双分段切换

### 2. 健康信息模块增强（M6）
- 新增**家族病史**编辑功能：可添加/编辑/删除家族病史记录（关系+疾病名称+备注）
- 新增**手术史**编辑功能：可添加/编辑/删除手术史记录（手术名称+时间+备注）

### 3. 健康档案主页优化（M2）
- **Hero卡片升级**：由原来的5行基本信息改为**四格健康摘要指标**（既往病史/过敏史/家族遗传/长期用药）
- 新增**编辑基本信息**按钮，可直接修改姓名、性别、出生日期、身高、体重、血型

### 4. 病历卡上传功能（M8）
- 新增**上传病历**入口，支持拍照/相册选择上传病历卡图片
- OCR自动解析病历卡内容
- 时间轴中病历卡事件使用**蓝色边框+📋图标**特殊展示

### 5. 共管与提醒优化（M7）
- **共管家人列表**直接在Tab内展示，无需跳转其他页面
- 新增**权限管理**功能：可为每个共管人单独设置"允许编辑"/"允许查看"权限

### 6. 自建药品库扩充（M10）
- 药品库数据量从约200条大幅扩充至**300+条**
- 新增覆盖高血压、糖尿病、高血脂、冠心病、呼吸系统、消化系统、痛风、甲状腺、抗生素等慢病常用药

## 使用说明

### 一、进入健康档案

1. 在应用首页点击底部导航栏的「我的」或「健康档案」入口
2. 进入健康档案主页，可看到家庭成员条、Hero卡片、设备列表、健康数据等

### 二、用药计划 —— 添加用药提醒

1. 在健康档案主页向下滚动，找到「用药计划」Tab区域
2. 点击右上角「+ 添加用药」按钮，进入添加用药页面
3. 填写药品信息：
   - **药品名称**：输入药品名，系统会自动从药品库联想搜索（300+条数据）
   - 点击📷 **拍照识药**按钮可拍摄药盒自动识别药品名称
   - **用药剂量**：如"1片"、"2粒"、"5ml"
   - **备注**：如"饭后服用"、"需空腹"（最多200字）

4. 设置服药时间：
   - **每日次数**：点击1-6数字按钮选择每日服药次数
   - **服药时间点**：点击时间按钮选择具体服药时间（HH:MM格式）
   - 可点击「+ 添加服药时间」增加时间点（最多6个）
   - 点击 × 按钮可删除多余时间点（至少保留1个）

5. 设置日期：
   - **开始日期**：选择开始服药的日期（默认今天）
   - **长期服用**：开关打开表示长期服用，关闭后可设置**结束日期**

6. 其他设置：
   - **开启用药提醒**：开关控制是否接收服药提醒（默认开启）
   - **关联疾病**：点击预设疾病标签可多选（高血压、糖尿病、高血脂等），也可自行输入

7. 点击底部「添加用药提醒」按钮完成添加

### 三、用药计划 —— 查看与切换

1. 在健康档案主页的「用药计划」区域
2. 点击双分段按钮切换视图：
   - **今日用药**：仅显示今天需要服用的药品
   - **全部用药计划**：显示所有用药计划
3. 每张用药卡片显示药品名称、剂量、时间点打卡状态、本周完成率

### 四、健康信息 —— 编辑家族病史

1. 在健康档案主页滚动到「健康信息」区域，点击右上角「编辑」
2. 在弹出的编辑页中滚动到底部找到「家族病史」区域
3. 点击「+ 添加家族病史」按钮
4. 在弹出的表单中：
   - 选择**关系**（爸爸/妈妈/爷爷/奶奶/外公/外婆/兄弟/姐妹/其他）
   - 输入**疾病名称**（如"高血压"）
   - 可输入**备注**（如"确诊于2020年"）
5. 点击「保存」完成添加
6. 已添加的条目可点击「编辑」修改或「删除」移除
7. 最后点击底部「保存」按钮保存所有健康信息

### 五、健康信息 —— 编辑手术史

1. 在健康信息编辑页中找到「手术史」区域
2. 点击「+ 添加手术史」按钮
3. 填写：
   - **手术名称**（如"阑尾切除术"）
   - **手术时间**（可选择日期）
   - **备注**（可选，如"恢复良好"）
4. 点击「保存」完成添加
5. 最后点击底部「保存」按钮保存所有健康信息

### 六、Hero卡 —— 查看与编辑基本信息

1. 在健康档案主页顶部的绿色Hero卡片中查看**四格健康摘要**：
   - 既往病史（项）、过敏史（项）、家族遗传（项）、长期用药（种）
2. 点击Hero卡片右上角的「✏️ 编辑」按钮
3. 在弹出表单中修改：
   - 姓名、性别、出生日期、身高(cm)、体重(kg)、血型
4. 点击「保存」更新基本信息

### 七、上传病历卡

1. 在健康档案主页向下找到「健康事件」Tab区域
2. 点击右上角「📋 上传病历」按钮
3. 点击图片区域拍照或从相册选择病历卡图片
4. 可填写标题和备注（可选）
5. 点击「上传病历」按钮完成上传
6. 上传的病历卡会在事件时间轴中以蓝色边框+📋图标展示
7. 可通过筛选器「病历卡」标签快速筛选

### 八、共管与权限管理

1. 在健康档案主页找到「共管与提醒」Tab区域
2. **查看共管家人**：共管家人列表直接展示在区域内
3. **邀请共管**：点击「+ 邀请共管」添加新的共管人
4. **管理权限**：点击某位共管家人，弹出权限管理窗口：
   - **允许编辑健康信息**：控制共管人是否可以修改您的健康数据
   - **允许查看健康档案**：控制共管人是否可以查看您的健康档案
5. 点击「保存」更新权限设置
6. 下方「漏打卡提醒」区域可设置：
   - 漏打卡阈值（1/2/3/5/7天）
   - 推送通道（站内消息/微信小程序/通知共管家人）
   - 静默时段（设置不推送提醒的时间段）

## 注意事项

1. **拍照识药**需要在手机浏览器中打开才能使用相机功能，电脑端可能无法使用
2. **用药提醒**的推送消息依赖于您在「共管与提醒」中设置的推送通道
3. **家族病史**和**手术史**数据的保存依赖于健康档案（HealthProfile）已创建
4. **病历卡上传**的OCR自动解析依赖于后端OCR服务配置，如未配置则仅保存图片
5. **共管权限**修改后立即生效，请谨慎设置"允许编辑"权限
6. **用药计划**的双分段切换中，"今日用药"会根据当前日期动态筛选
7. 所有健康信息修改需点击底部「保存」按钮才会生效
