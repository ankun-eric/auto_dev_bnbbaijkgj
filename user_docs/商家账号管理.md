## 一、功能介绍

「商家管理 → 商家账号」是 admin 后台对商家**老板主账号**进行集中管理的页面。本次发布完成了以下改造：

1. **角色列按真实数据展示**：每行账号的「角色」列不再硬编码为"老板"，会按账号实际角色（老板 / 店长 / 核销员 / 财务 / 店员）渲染对应中文 Tag 与颜色，避免显示错乱。
2. **列表收紧为只展示老板账号**：默认情况下列表仅返回 `role_code = boss` 的商家主账号，6366（核销员）等员工账号不再出现在主列表里，列表语义完全回归"商家老板管理"。
3. **新增"员工列表"按钮 + 抽屉**：每个老板账号操作列新增 `员工列表 (N)` 按钮，N 为该商家下非老板员工数量，点击后右侧抽屉一次性展示员工的账号 / 姓名 / 手机号 / 角色 / 状态 / 创建时间 / 最后登录时间 / 绑定门店等 8 个字段。
4. **数据修正**：账号 6399（13719036399 / 越秀区果菜西街店）的真实身份已由「店长」修正为「老板」，登录商家后台菜单将自动包含财务、商家设置、员工管理等老板专属模块。

## 三、操作步骤

### 1. 查看老板账号列表

进入页面后，列表默认仅展示老板账号。每行包括：账号 / 商家昵称 / 用户昵称 / 手机号 / **角色** / 状态 / 关联门店 / 创建时间 / 操作。

- 「角色」列对老板账号显示金色 Tag「老板」。
- 若日后开放展示其他角色，会按真实角色渲染：
  - 老板（boss） — 金色
  - 店长（store_manager） — 蓝色
  - 财务（finance） — 紫色
  - 核销员（verifier） — 绿色
  - 店员（staff） — 灰色

### 2. 通过手机号 / 商家名搜索

页面顶部搜索框输入手机号或商家名片段，回车即可过滤；过滤条件不会影响"只展示老板"的固定规则。

### 3. 查看某老板账号的员工

1. 在列表行的"操作"列找到 `员工列表 (N)` 按钮，N 即该老板名下"非老板员工"数量。
2. 点击按钮，右侧抽屉打开，标题为「员工列表 - 商家名称」。
3. 抽屉表格展示该商家下店长 / 核销员 / 财务 / 店员的全部账号（不包含老板自己），列包括：
   - 账号、姓名、手机号
   - **角色**（带颜色 Tag）
   - 状态（正常 / 禁用）
   - 创建时间、最后登录时间
   - 绑定门店（多门店逗号分隔，过长时省略号 + 鼠标悬停查看完整列表）
4. 关闭抽屉，主列表的搜索条件、分页位置保持不变。
5. 若该老板暂无任何员工，抽屉内会显示空状态："该商家暂无员工"。

### 4. 新建商家账号

页面右上角 `新建商家账号` 仍只允许新增老板账号，新建后会立即出现在主列表。
