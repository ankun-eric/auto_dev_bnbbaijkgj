## 功能简介

本次更新修复了**「健康档案 → 家庭成员」里点一下「删除」就被弹出「操作频繁，请明日再试」且删不掉**的问题。

修复前：系统的「防止恶意狂删」机制写错了——只要你点一下删除（哪怕没删成、点了取消、被其他规则挡住），都会被算作一次额度。来回试几次额度就被悄悄扣光，之后删谁都会被拦。

修复后：

1. **只有真正删除成功**，才会算一次额度；点了取消、被其他规则拦住没删成，都**不计入**额度。
2. 每天的额度上限从 **5 次大幅放宽到 50 次**，正常使用完全碰不到上限。
3. 取消了删除时的「双重拦截」，让删除逻辑统一、不再重复扣额度。
4. 同口径的「创建邀请」「解除守护」也一起改成「50 次/天、只算成功的那一次」，规则统一。
5. 登录锁定、短信验证码间隔、图形验证码等**安全防护保持原样，安全性不降低**。

## 使用说明

> 本次修复为后端逻辑修复，无需更新任何 App 或重新导入小程序，**直接在手机浏览器打开页面即可生效**。

### 删除家庭成员的正常操作步骤

1. 用手机浏览器打开 [项目主页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/) 并登录。
2. 进入「**健康档案**」页面。
3. 在「**家庭成员**」区域，找到要删除的成员卡片。
4. 点击该成员的「**删除**」操作。
5. 在弹出的确认框中确认删除，即可正常删除成功，**不会再被「操作频繁」误拦**。

### 体验「修复后」的几种场景

- **正常删除**：点一下删除并确认，能正常删除成功。
- **点了取消 / 反复尝试**：即使来回点、点了取消，都不会消耗删除额度。
- **被其他规则拦住**（如成员仍处于「守护中 / 邀请中」、名下有绑定设备或在途服药计划）：会提示对应的处理方式（如「请先解除守护关系」），且这次失败的尝试**不计入**当天的删除额度。
- **超大量删除**：每天 50 次以内随便删，正常用户完全碰不到上限；只有真的删够 50 次成功删除后，第 51 次才会提示「今日删除次数已达上限，请明日再试」。

## 注意事项

1. **本次为后端修复，无客户端（App/小程序）安装包变更**，无需下载或重新安装任何客户端，刷新页面即可使用最新逻辑。
2. 删除额度按**自然日**计算，每天 0 点自动清零。
3. 「删除 / 创建邀请 / 解除守护」三个动作各自独立计算额度，均为 **50 次/天、仅成功才计数**。
4. 如果遇到「请先解除守护关系 / 请先取消邀请 / 请先解绑设备 / 请先终止服药计划」等提示，属于正常的安全闸门，按提示先处理对应事项后即可删除——这些被拦的尝试不会扣你的删除额度。
5. 安全相关的限制（登录密码错误锁定、短信验证码 60 秒间隔、图形验证码防刷等）保持不变。
