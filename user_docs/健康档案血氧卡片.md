## 功能简介

健康档案页的「今日健康数据」原本只有 4 张卡片（血压 / 血糖 / 心率 / 睡眠），**缺少血氧**。本次更新把血氧卡片补齐，并做成和血压一样的精致样式：

- **位置**：插在「睡眠」前面，2×2 网格里顺序与关怀模式完全对齐——血压 / 血糖 / 心率 / **血氧** / 睡眠。
- **样式**：左上「🫁 血氧」标签 + 右上彩色**档位胶囊** + 中部大号数值（加粗）+ 单位 `%` + 底部「测量时间 · 来源」一行。
- **可点击**：点击整张卡片可进入血氧详情页查看历史趋势。
- **异常提示**：血氧偏低时卡片左侧出现黄色竖条提示。

### 血氧档位判定标准

| 血氧值 | 档位文案 | 胶囊颜色 |
|--------|----------|----------|
| ≥ 95% | 正常 | 蓝色（正常） |
| 90 ~ 94% | 偏低 | 橙色（预警） |
| < 90% | 偏低明显 | 橙红（严重预警） |

> 当暂无血氧记录时，卡片数值显示「—」，不显示档位胶囊。

## 使用说明

1. 打开**健康档案页**（H5 / 小程序 / App 均可），进入「本人」Tab。
2. 找到「今日健康数据」区块，可以看到 2×2 网格的健康指标卡片。
3. **血氧卡片**位于「心率」右侧、「睡眠」之前：
   - 左上角显示 `🫁 血氧`；
   - 右上角是彩色档位胶囊（正常=蓝色，偏低=橙色，偏低明显=橙红）；
   - 中间是大号血氧数值，后面跟单位 `%`；
   - 底部一行显示最近一次的「测量时间 · 数据来源」。
4. **点击血氧卡片**，可跳转到血氧详情页查看历史记录与趋势曲线。
5. 若卡片左侧出现**黄色竖条**，表示本次血氧数值异常（低于 95%），建议关注。

## 注意事项

- **血氧数据来源**：卡片显示的是该档案最近一次血氧记录，可来自手工录入或绑定的设备（如智能手表）自动同步；无记录时显示「—」。
- **档位仅供参考**：档位判定（≥95 正常 / 90~94 偏低 / <90 偏低明显）为通用健康参考标准，不构成医疗诊断。若血氧持续偏低或有不适，请及时就医。
- **H5 网页端无需下载**：网页端已在线更新，直接刷新健康档案页即可看到血氧卡片；小程序与 App 需下载上方对应的新版本。
- **多端样式一致**：H5、小程序、App 三端血氧卡片样式与档位标准保持一致。
