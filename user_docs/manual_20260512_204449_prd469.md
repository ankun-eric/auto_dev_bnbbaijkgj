## 功能简介

本次 PRD-469 完成了健康档案 v2 的视觉与功能升级，对齐 v5 设计稿，并彻底下线了旧版 `/health-profile` 路由。本次更新一次性带来 10 项核心改动：

| 编号 | 模块 | 本次新增/优化 |
|---|---|---|
| M1 | 旧路由下线 | 旧 `/health-profile` 路由统一返回 404，全平台跳转改到 `/health-profile-v2` |
| M2 | 主页 5 Tab + 6 宫格 | 全新健康绿主色 + 圆角病历卡 + 粘性 Tab + 头像化成员条 |
| M3 | 新建家庭成员弹层 | 16 种亲属关系预置（含「其他」自定义）+ 关系自动绑定 emoji 头像 |
| M4 | 用药添加页升级 | 药品库联想搜索（300ms 防抖）+ 处方类型自动带出 + 关联疾病标签 |
| M5 | 用药保存 Bug 修复 | 添加用药保存成功后列表自动刷新（sessionStorage + visibilitychange 双保险） |
| M6 | 健康信息模块 | 既往病史 / 过敏史 / 家族病史 / 个人习惯 四子模块 + 主页摘要胶囊 |
| M7 | 共管与提醒 | 漏打卡阈值（1/2/3/5/7 天）+ 站内消息 + 微信订阅消息 + 静默时段配置 |
| M8 | 健康事件 | 时间轴展示 + 手动写「健康日记」（含 4 种事件标签） |
| M9 | 设备绑定列表 | 10 项设备清单（华为手环已接通，9 项敬请期待 + 上线后通知我） |
| M10 | 自建药品库 | 上线时入库 **200+ 条**核心慢病常用药（覆盖 10 大慢病）+ OCR 拍照识药接口 |

## 使用说明

### 1. 进入新版健康档案

在浏览器中打开：
[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile-v2](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile-v2)

进入页面后，从上至下依次为：

1. **绿色顶部导航条**：「我的健康档案」标题；
2. **家庭成员条**（横向滚动）：本人 + 已添加的家庭成员，每个成员都有圆形 emoji 头像（与 AI 对话页一致）。最末尾固定一个虚线圆框「+ 添加」按钮——点击它会弹出新建家庭成员弹层；
3. **Hero 卡片**：当前选中成员的姓名、关系标签 + 性别 / 年龄 / 身高 / 体重 / 血型；
4. **我的设备区**：默认显示 4 项设备（华为手环已接通，其他敬请期待），点击「查看全部」展开全部 10 项；
5. **粘性 5 Tab**（向下滚动时吸顶）：
   - **今日数据**：6 宫格（血压 / 血糖 / 心率 / 睡眠 / 血氧 / 用药打卡 X/Y）
   - **健康信息**：摘要胶囊 + 4 个统计计数（既往病史 / 过敏史 / 家族病史 / 个人习惯）
   - **用药计划**：今日用药卡片 + 「+ 添加用药」按钮
   - **共管与提醒**：共管列表入口 + 漏打卡阈值 + 推送通道 + 静默时段
   - **健康事件**：时间轴 + 「+ 写日记」按钮

> 提示：点击粘性 Tab 会平滑滚动到对应区块；反向也成立——下滑时 Tab 会自动高亮跟随。

### 2. 新建家庭成员（v5 弹层）

1. 点击成员条最末的「+ 添加」圆形虚线按钮，弹出弹层；
2. 在「关系」步骤选择与本人的关系（16 项预置：本人 / 爸爸 / 妈妈 / 老公 / 老婆 / 儿子 / 女儿 / 哥哥 / 弟弟 / 姐姐 / 妹妹 / 爷爷 / 奶奶 / 外公 / 外婆 / 其他）；
3. 如选择「其他」，需填写自定义关系名（如「大儿子」「二儿子」）。系统会自动校验：
   - 不能与预置关系名重名；
   - 不能与该账号下已存在的任何成员关系名重复；
4. 点击「下一步」进入信息表单，填写：姓名（必填，2-20 字）/ 性别（必填）/ 出生日期（必填）/ 身高 / 体重 / 血型 / 手机号；
5. 点击「保存」，新成员立即出现在成员条最末（紧贴「+」按钮之前）。

### 3. 添加用药计划（含药品库联想）

1. 在主页用药计划 Tab 中点击「+ 添加用药」按钮，进入添加页；
2. 在「药品名称」输入框中输入药品关键字（如「阿」），300ms 后弹出药品库联想列表，每条显示：商品名（通用名）+ 规格 + 厂家 + 处方类型；
3. 点击列表中的药品，系统会自动带入药品名，并在下方提示「已从药品库选择」+ 处方类型 + 关联疾病标签；
4. 设置「用药剂量」「备注」；
5. 勾选用药时段（早晨 / 中午 / 晚上 / 睡前，可多选），并为每个时段设置提醒时间；
6. 点击「添加用药提醒」保存。
7. **保存成功后会自动返回上一页，新条目立即出现在列表中**（不需要手动刷新）。

> 本次修复了「添加用药保存后列表不刷新」的 P0 Bug。技术上通过 sessionStorage + visibilitychange 监听双保险，确保从添加页返回时列表必定刷新。

### 4. 设备绑定

主页中部「我的设备」区显示 10 项设备清单：

- **华为手环**（已接通）：点击即可进入绑定流程；
- 其余 9 项设备（小米手环 / Apple Watch / 华为手表 GT / 三诺血糖仪 / 鱼跃血压计 / 欧姆龙血压计 / 鱼跃血氧仪 / 小米体重秤 / 华为体脂秤）：点击会提示「该设备即将上线，敬请期待！上线后我们将通知您」，同时为您记录订阅，上线时自动通知。

### 5. 健康信息编辑

切换到 Tab 2「健康信息」，点击右上角「编辑 ›」，弹出编辑弹层：

- **既往病史**：10 项常见慢病多选（高血压 / 糖尿病 / 高血脂 / 冠心病 / 脑卒中 / 慢阻肺 / 哮喘 / 慢性肾病 / 甲状腺 / 痛风）；
- **过敏史**：药物过敏 / 食物过敏 / 其他过敏，标签化输入；
- **个人习惯**：抽烟（有/无）/ 饮酒（有/无）/ 运动频率（无/偶尔/经常）/ 饮食偏好（清淡/重口味/素食/其他）。

保存后，**主页 Tab 2 顶部会立即显示摘要胶囊**（如 🚭不吸烟 🩺高血压 ⚠️青霉素过敏）。

### 6. 共管与提醒配置

切换到 Tab 4「共管与提醒」：

- **共管家人列表**：跳转到完整的共管家人管理页；
- **邀请共管人**：通过手机号邀请家人共管；
- **漏打卡阈值**：1 / 2 / 3 / 5 / 7 天可选（默认 3 天）；
- **推送通道**：站内消息 / 微信小程序订阅消息 / 同时通知共管家人（每项独立开关）；
- **静默时段**：设置不推送的起止时间（建议 22:00–07:00）。

### 7. 健康日记（事件流）

切换到 Tab 5「健康事件」，点击「+ 写日记」按钮：

1. 选择日期、填写标题（如「感冒就诊」）、详情；
2. 可选标签：不适 / 复诊 / 用药调整 / 其他；
3. 点击「保存」后，日记自动出现在时间轴中。

时间轴支持按事件类型筛选：全部 / 日记 / 用药 / 异常 / 报告。

## 注意事项

1. **旧版下线**：旧版 `/health-profile` 路由已彻底下线，访问会显示 **404**。所有入口（积分任务、AI 对话侧栏、家庭页跳转）已自动切换到新版 `/health-profile-v2`；
2. **新建成员弹层中的「其他」关系名重复校验**：自定义关系名不能与预置 16 种关系重名，也不能与当前账号已存在的成员名重复。系统会给出明确提示，请按提示更换；
3. **药品库联想搜索**：本期上线时入库 **200+ 条核心慢病药**，覆盖 10 大慢病（高血压 / 糖尿病 / 高血脂 / 冠心病 / 心衰 / 哮喘 / 慢阻肺 / 甲状腺 / 痛风 / 慢性肾病）的常用药；如搜不到，可手动输入药品名继续录入；后续会每季度增量更新；
4. **拍照识药**：API 已上线（`/api/prd469/medication-library/ocr`），前端调用通用 OCR 接口后将文字回传后端做模糊匹配；
5. **设备绑定**：除华为手环外的设备状态为「敬请期待」，订阅后会在上线时通过站内消息通知您；
6. **微信小程序订阅消息**：需用户主动授权订阅一次，授权后才能接收漏打卡推送；
7. **静默时段**：仅影响主动推送（漏打卡提醒），到点用药打卡仍会正常提醒；
8. **新增 6 个数据表**：自动通过 SQLAlchemy `Base.metadata.create_all` 在容器启动时创建，无需手动执行迁移。
