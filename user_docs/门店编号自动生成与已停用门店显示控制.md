## 功能简介

本次更新为管理后台的 **门店管理** 模块带来两项功能增强：

1. **门店编号自动生成**：新建门店时，系统自动分配唯一编号（格式 `MD00001` ~ `MD99999`），无需手动输入，杜绝编号冲突和格式不规范问题。存量老门店已按创建时间统一重新编号。

2. **已停用门店显示控制**：门店列表默认仅显示营业中门店，通过右上角的「显示已停用」开关可查看全部门店（含已停用门店）。已停用门店整行置灰，视觉上与营业中门店明确区分。

## 使用说明

### 一、查看门店列表

1. 登录管理后台，进入左侧菜单 **门店管理**
2. 默认情况下，列表仅显示 **营业中** 的门店
3. 每个门店会显示系统自动分配的编号（如 `MD00001`）

### 二、切换显示已停用门店

1. 在门店列表页面右上角，找到 **「显示已停用」** 开关
2. **开关关闭（默认）**：仅显示营业中门店
3. **开关打开**：显示全部门店，包括已停用的门店
4. 已停用门店会以 **灰色** 样式展示（文字和背景均为浅灰色），排列在营业中门店之后
5. 开关状态切换后，列表会自动刷新

### 三、新建门店

1. 点击右上角 **「新建门店」** 按钮
2. 在弹窗中填写门店信息：
   - **门店名称**（必填）
   - **所属类别**（必填）
   - **联系人**（选填）
   - **联系电话**（选填）
   - **门店地址**（选填）
   - **经营范围**（选填）
3. 注意：**无需填写门店编号**，系统会自动分配
4. 点击 **「确定」** 保存，保存成功后列表自动刷新，可看到新门店及其自动分配的编号

### 四、编辑门店

1. 在门店列表中，点击对应门店的 **「编辑」** 按钮
2. 弹窗中会显示 **门店编码**（灰底只读，不可修改）
3. 修改其他需要调整的字段后，点击 **「确定」** 保存

> ⚠️ 已停用的门店无法编辑，需先将其启用后才能进行编辑操作。

### 五、启用/停用门店

1. 营业中门店：点击 **「停用」** 按钮，确认后门店变为已停用状态
2. 已停用门店（需先打开「显示已停用」开关）：点击 **「启用」** 按钮，确认后门店恢复营业中状态

### 六、搜索门店

- 使用顶部的 **类别筛选** 下拉框，可按门店类别过滤
- 搜索范围与「显示已停用」开关联动：
  - 开关关闭时：仅在营业中门店中搜索
  - 开关打开时：在所有门店中搜索（含已停用）

## 注意事项

1. **编号格式**：所有门店统一使用 `MD` + 5位数字 的编号格式（如 `MD00001`），前缀 `MD` 不可配置
2. **编号不可修改**：门店编号由系统自动生成后不可手动修改，确保编号的唯一性和规范性
3. **存量数据**：已有门店已按创建时间从早到晚自动重新编号，最早创建的门店为 `MD00001`
4. **编号上限**：系统最多支持 99,999 家门店编号（`MD00001` ~ `MD99999`）
5. **权限要求**：门店管理功能需要管理员权限才能操作
