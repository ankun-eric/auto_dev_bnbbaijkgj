## 一、本期做了什么

把原本"8 题答完只给一个体质名"的体验，升级为**一套完整的中医体质测评闭环**：

1. **6 屏沉浸式结果页**：体质名片 → 深度解读 → 个性化养生方案 → 专属套餐推荐 → 门店服务 → 分享卡。
2. **体质档案**：多次测评结果自动留存，可在"我的档案"里回看每一次的体质变化。
3. **加载过渡页**：提交测评后不再白屏等待，过渡到专题加载页（中医风动画 + 随机养生 Tips）。
4. **商业闭环接入**：
   - 体质自动匹配**养生套餐**（M1 气血双补 / M3 滋阴养肺 等），卡片可直接跳购买。
   - 一键领取 **AI 精准检测体验券**（广州门店）。
   - 艾灸 / 针灸**在线预约**入口直跳。
5. **分享卡（中医版 MBTI）**：一键生成海报分享给朋友 / 朋友圈，带体质名 + 人格化标签 + 雷达图。

覆盖三端：**H5 / 微信小程序 / Flutter APP**，全部同步上线。

## 三、用户使用流程

### 流程图（H5 视角）

```
首页「中医养生」
      │
      ▼
  [8 题答题卡]
      │  提交
      ▼
  [加载过渡页 /tcm/loading]  ←动态 Tips + 中医风动画
      │  结果返回
      ▼
 ┌─────── 6 屏结果页 /tcm/result/{id} ────────┐
 │ 屏 1：体质名片（名 + 人格化标签 + 雷达图）│
 │ 屏 2：深度解读（特征 / 成因 / 风险）      │
 │ 屏 3：个性化养生方案（食 / 眠 / 动 / 志）  │
 │ 屏 4：专属套餐推荐（可购买）              │
 │ 屏 5：门店服务（领券 / 预约）             │
 │ 屏 6：分享卡 + 免责声明                   │
 └────────────────────────────────────────────┘
      │
      ▼
 [我的档案 /tcm/archive]  ←任何时候回看历史
```

### 关键动作

1. **领取 AI 检测体验券**：屏 5 点击「领取体验券」→ 后端自动建券 + 发放（同一用户同一券只能领一次），领后按钮变灰显示「已领取」。
2. **买套餐**：屏 4 点套餐卡 → 跳商品详情（H5/小程序）或商品页（APP）。
3. **预约艾灸/针灸**：屏 5 点对应按钮 → 跳预约页，携带体质信息。
4. **分享**：
   - H5：屏 6 「保存分享海报」→ Canvas 生成 PNG → 长按保存到相册。
   - 小程序：右上角「…」→ 转发 / 分享到朋友圈。
   - APP：屏 6「分享」→ 系统分享面板。

## 五、已知限制（一期 → 二期规划）

- **内容静态**：9 种体质详细内容 / 人格化标签 / 套餐映射暂未入库，二期接入 CMS。
- **雷达图算法**：一期基于答题 JSON 做简化打分，二期按中医九体辨证算法重构。
- **小程序雷达图**：一期因原生 canvas 受限，降级为横向柱状图，二期接 `ec-canvas`。
- **套餐匹配**：按关键词匹配商品库 `Product.name/description`，未匹配时显示默认卡。
- **分享卡**：APP 端一期为系统分享，二期做专属海报生成。
