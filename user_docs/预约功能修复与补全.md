## 功能简介

本次更新修复并补全了「预约下单」核心功能，涵盖两大问题：

1. **日期选择器范围限制**：管理后台设置的「提前可预约天数」（advance_days）现在能正确限制三端（H5、小程序、Flutter APP）的日期选择范围。超出范围的日期将置灰不可选。

   - Flutter APP 订单详情新增预约信息展示
   - 后端新增预约日期必填校验和日期范围校验

## 使用说明

### 一、运营人员：设置预约商品

1. 登录 [管理后台](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 进入「商品管理」→ 选择商品 → 点击「编辑」
3. 在「预约与核销」Tab 中：
   - **预约模式**：选择「预约日期」或「预约时段」
   - **提前可预约天数**：输入数字（如 7），表示用户最远可预约从今天起的 7 个自然日
   - **时段配置**（预约时段模式）：设置各时段的开始时间、结束时间和容量
4. 保存商品设置

### 二、用户端：预约下单

#### H5 网页端

1. 打开 [H5 用户端](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)
2. 浏览并选择预约类商品，点击「立即预约」
3. 进入下单页面后：
   - **预约日期**：默认选中今天，可点击切换到其他日期（超出可预约范围的日期置灰不可选）
   - 页面顶部显示提示「最远可预约至 X月X日」
   - **预约时段**：选择商品配置的时段（如 09:00-10:00）或默认时段
   - **备注**：可选填预约备注信息
4. 确认信息后点击「提交订单」

#### Flutter APP（安卓/iOS）

1. 安装最新版 APP（见上方下载链接）
2. 打开 APP，浏览并选择预约类商品
3. 在下单页面：
   - 查看「预约信息」区域（仅预约类商品显示）
   - 点击日期选择器选择预约日期（默认今天）
   - 选择预约时段（高亮显示已选时段）
   - 填写预约备注（可选）
4. 点击提交订单

#### 微信小程序

1. 在微信开发者工具中导入小程序代码
2. 浏览商品并进入下单页
3. 在「预约信息」区域：
   - 日期选择器自动限制了可选范围（页面显示「最远可预约至 X月X日」）
   - 默认选中今天
   - 选择预约时间段
   - 填写备注（可选）
4. 提交订单

### 三、运营人员：查看订单预约信息

1. 登录 [管理后台](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 进入「订单管理」→ 点击某个订单查看详情
3. 在订单详情弹窗中，如果该订单是预约类订单，将显示独立的「预约信息」卡片，包含：
   - 预约日期
   - 预约时段
   - 预约状态（待核销/已完成/已取消）
   - 预约备注（如有）

### 四、用户端：查看订单预约信息

- **Flutter APP**：在「我的订单」中点击订单可查看预约日期、时段和备注信息
- **H5 商户端**：订单详情页展示预约时间和时段信息

## 注意事项

1. **日期范围计算规则**：设置 `advance_days = N` 时，可选日期为「今天 ~ 今天 + (N-1) 天」，共 N 个自然日（包含今天）。例如今天是 4 月 28 日，advance_days = 7，则可选 4 月 28 日至 5 月 4 日。

2. **预约日期为必选**：预约类商品下单时必须选择预约日期，未选日期将无法提交订单。

3. **默认选中今天**：进入下单页面时，预约日期默认选中今天，用户可自行更换。

4. **向后兼容**：如果商品的 advance_days 未设置或为 0，则不限制最远可选日期。

5. **后端校验**：即使前端绕过了日期限制，后端也会校验日期是否在合法范围内并返回错误提示。

6. **时区处理**：日期计算基于用户本地时区，确保跨时区用户的体验一致。
