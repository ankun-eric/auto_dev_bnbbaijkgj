> 版本：v1.0  
> 上线日期：2026-05-15  
> 适用端：H5 用户端 / 微信小程序 / APP（iOS·Android）/ 管理后台

---

## 一、功能介绍

「健康自查」是 AI 健康咨询的一个轻量化入口。用户在「AI 对话模式」首页点击「健康自查」胶囊按钮，会弹出一个三步式问卷抽屉：

1. 选择不舒服的身体部位
2. 选择对应部位的症状（多选）
3. 选择症状持续时间

提交后抽屉自动关闭，「自查卡片气泡」会插入到 AI 对话流中，AI 紧跟着给出专业的初步分析和建议。整个过程不需要离开对话页，体验连贯。

管理后台同步提供三处可配置项：
- **部位症状字典**：维护部位（如头部、胸部、腹部）和该部位下的症状列表
- **健康自查问卷模板**：勾选启用哪些部位、配置持续时间档位与默认 Prompt
- **功能按钮管理**：新增 `health_self_check` 类型的按钮，绑定模板与未选档案策略

## 三、终端用户操作步骤（H5 / 小程序 / APP 通用）

### 第 1 步：进入 AI 对话首页
打开 AI 对话模式首页 `/ai-home`，在底部的胶囊按钮区可看到「🩺 健康自查」按钮（首次部署会默认创建并启用）。

### 第 2 步：点击「健康自查」胶囊
点击胶囊后会从底部弹出健康自查抽屉。抽屉顶部以**只读**方式展示当前选择的咨询档案；如果按钮配置为「未选档案时强制 toast 提示」，则会 toast 提醒先在顶部选择档案。

### 第 3 步：依次完成三步问卷
1. **步骤 1：选择部位**  
   抽屉中以四列网格展示所有启用的部位（如 头部 / 眼部 / 耳鼻喉 / 胸部 / 腹部 …），点击即可选中（单选）。
2. **步骤 2：选择症状（多选）**  
   选中部位后，下方动态展示该部位的所有症状（如 头部 → 头痛 / 头晕 / 偏头痛 / 头胀 / 头皮发麻）。点击症状标签为选中/取消。至少需要选择一个。
3. **步骤 3：选择持续时间**  
   单选一档，如 `<1天` / `1-3天` / `3-7天` / `>1周` / `>1月`。

> 三步**没有强制顺序**，可来回切换。如有任一项未选，点击「开始 AI 分析」时会 toast 提示「请完成全部三项后再开始分析」，并把缺失项标题高亮为红色。

### 第 4 步：点击「开始 AI 分析」
- 抽屉立即关闭
- 主对话流中新增一条**用户侧自查卡片气泡**，展示「部位图标 + 名称 / 症状标签 / 持续时间 / 咨询人」
- 紧接着 AI 给出专业回答（流式输出），可继续以普通文本追问

### 第 5 步：「重新自查」（可选）
自查卡片气泡右上角有「重新自查」按钮。点击后会用上次的部位/症状/持续时间预填好抽屉，可二次修改后再次提交。

## 四、管理后台运营/医学编辑操作

### 4.1 维护「部位症状字典」
进入：`/admin/body-part-dict`
- **新增部位**：点击「新增部位」，填写图标（emoji 即可）、名称、症状列表（按回车追加多个）、排序值、启用状态。
- **编辑/停用**：列表行「编辑」、「停用 / 启用」。
- **删除**：仅当部位**未被任何模板引用**时才可删除；被引用时按钮置灰，建议先停用。

### 4.2 维护「健康自查问卷模板」
进入：`/admin/health-check-templates`
- 新建模板时填写模板名称、描述、勾选启用的部位（来自字典）、配置持续时间档位（至少 2 档）、编辑默认 Prompt。
- 右侧侧栏提供「占位符速查表」：`{档案信息}`、`{档案年龄}`、`{档案性别}`、`{档案既往病史}`、`{档案过敏史}`、`{部位}`、`{症状列表}`、`{持续时间}`。
- 保存时会校验 4 个核心占位符（档案信息 / 部位 / 症状列表 / 持续时间）是否齐全。
- 模板被任意按钮引用时不可直接删除，请先解绑或停用。

### 4.3 在「功能按钮管理」绑定模板
进入：`/admin/function-buttons`
- 新增按钮时「按钮类型」选择「健康自查（health_self_check）」。
- 展开三个专属字段：
  1. **关联问卷模板**（必填）：下拉选择已有模板
  2. **未选档案时的行为**：`使用默认档案` / `提交时再提示选择` / `强制 toast 先选档案`
  3. **Prompt 配置**：默认「继承模板默认 Prompt」，切到「按钮自定义」后可单独编辑 Prompt 文本（一键填入模板默认 Prompt）

## 五、注意事项

1. **Prompt 拼装在服务端完成**，前端不接触最终 Prompt 全文，保障安全。
2. 模板 / 按钮 / 字典任一项的修改对所有用户**实时生效**，不做版本快照。
3. 历史已发出的「自查卡片气泡」会按当时提交的数据展示，即使后续部位被停用、症状被改名也不会变。
4. 如出现「分析失败，请点击重试」提示，通常是 AI 服务一时不通，请稍后重试。
5. 小程序端建议使用基础库 2.20.0+。
6. 移动端抽屉默认高度 85vh，桌面端为居中浮层最大 80vh。
7. 角色权限：超级管理员可全量增删改，内容/医学编辑可编辑字典与模板，普通管理员只能编辑按钮配置。

## 六、常见问题

**Q：胶囊点击后没反应？**  
A：进入管理后台「功能按钮管理」，确认健康自查按钮处于「启用」状态且已正确绑定问卷模板。

**Q：抽屉打开后显示「该功能暂不可用，请联系管理员」？**  
A：模板可能已停用或被删除，请先在「健康自查问卷模板」恢复或重新关联。

**Q：症状列表里没有我想选的症状？**  
A：联系管理员，到「部位症状字典」对应部位添加新的症状即可，无需重发版本。

**Q：自查卡片气泡的「重新自查」打开后是空的？**  
A：通常是按钮已被删除导致预填失败，请刷新页面后重新从胶囊入口发起一次自查。
