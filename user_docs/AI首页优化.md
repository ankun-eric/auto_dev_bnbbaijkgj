## 功能简介

AI 首页有两个模式：**关怀模式**（给长辈用，暖橙风、无 AI 对话）和**标准模式**（蓝绿渐变、有 AI 对话）。本次优化集中处理了两个模式右上角「+」圆圈菜单与底部分享按钮：

1. **统一「分享好友」入口**：新增一个图标为 🎁、文案为「分享好友」的统一入口，点击后弹出原有分享面板（微信好友 / 朋友圈 / 复制链接 / 生成海报），分享出去的链接默认指向当前 AI 首页。两个模式里这一项的图标、文案、点击行为**完全一致**。
2. **去掉重复/无用入口**：删除了「+」圈里原来的「📤 立即分享」（功能合并进「分享好友」）和「🎁 邀请好友」（拉守护人功能，入口移到别处，功能本身保留）。
3. **关怀模式精简设置**：关怀模式没有 AI 对话，已删除只服务于 AI 对话的「💬 发起新对话」和「🔤 字体大小」；标准模式仍保留这两项。
4. **底部分享按钮整理**：关怀模式底部「分享好友」大按钮保留（已改名「分享好友」、图标换成 🎁）；标准模式不再单独放底部分享大按钮。
5. **欢迎区样式统一**：标准模式欢迎区版式/问候文案/头像与关怀模式统一，但配色仍保持标准模式的蓝绿渐变。

### 优化后两个模式「+」圈对照

| 关怀模式「+」圈（共 5 项） | 标准模式「+」圈（共 7 项） |
|---|---|
| 🔀 切换模式 | 💬 发起新对话 |
| 👑 会员中心 | 🔤 字体大小 |
| 📷 扫一扫 | 🔀 切换模式 |
| 🎁 分享好友 | 👑 会员中心 |
| ❓ 帮助与反馈 | 📷 扫一扫 |
| | 🎁 分享好友 |
| | ❓ 帮助与反馈 |

## 使用说明

### 一、打开 AI 首页
- **H5 网页版**：直接访问上方「AI 首页」链接。
- **App / 小程序**：打开应用后进入 AI 首页（首页/聊天入口）。

### 二、使用「分享好友」（两个模式一致）
1. 点击 AI 首页**右上角的「+」圆圈**，弹出更多菜单。
2. 在菜单中找到 **🎁 分享好友**，点击它。
3. 弹出分享面板，可选择 **微信好友 / 朋友圈 / 复制链接 / 生成海报** 等渠道。
4. 分享出去的链接默认指向当前 AI 首页，好友点开即可访问。

> 关怀模式下，除了「+」圈里的「🎁 分享好友」，**底部还保留了一个「🎁 分享好友」大按钮**，点击效果与菜单里的一致，方便长辈一眼找到。

### 三、关怀模式与标准模式切换
- 在「+」圈菜单点击 **🔀 切换模式**，或使用欢迎区右上角的模式切换胶囊，即可在关怀模式 / 标准模式之间切换。

### 四、找不到「邀请好友」了？
- 「邀请好友」（拉守护人 / 带专属邀请码）功能**依然存在**，只是不再放在 AI 首页的「+」圈里。它属于「守护」体系，请到对应的守护人 / 邀请入口使用。
