## 功能简介

本次操作为 **Bini Health 健康管理平台** 的重新部署，未涉及代码变更。项目包含以下主要功能模块：

- **H5 用户端**：面向终端用户的移动端健康管理页面，包含 AI 健康对话、健康档案、中医体质评估、健康计划、积分商城等功能
- **管理后台**：面向管理员的后台管理系统，包含用户管理、订单管理、专家管理、内容管理、AI 模型配置、短信管理、商家管理等功能
- **微信小程序**：整合了用户端和商家端功能，支持双身份切换、商家门店管理、订单核销等
- **安卓/iOS APP**：基于 Flutter 开发的移动端原生应用

## 使用说明

### H5 用户端使用

1. 打开浏览器（推荐使用手机浏览器）
2. 访问 [H5 用户端链接](https://101.33.205.59/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
3. 使用手机号 + 短信验证码登录
4. 登录后可使用 AI 健康对话、查看健康报告、管理健康档案等功能

### 管理后台使用

1. 打开浏览器（推荐使用电脑浏览器）
2. 访问 [管理后台链接](https://101.33.205.59/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
3. 使用管理员账号登录
4. 可管理用户、订单、专家、内容、AI 模型配置、短信配置、商家等

## 注意事项

1. 当前服务器仅配置了 HTTP（80 端口），建议在正式环境中配置 HTTPS 证书
2. 短信验证码登录需要在管理后台「系统管理 → 短信管理」中配置腾讯云短信参数
3. AI 健康对话功能需要在管理后台「AI 模型配置」中配置有效的 AI 模型 API 信息
4. iOS 端 IPA 使用无代码签名模式构建，如需分发到真实设备需配置 Apple Developer 证书
5. 微信小程序需要通过微信开发者工具导入体验，正式上线需在微信公众平台完成审核发布
