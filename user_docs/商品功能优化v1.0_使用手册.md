## 一、功能总览

本次升级同时覆盖 **管理后台** 与 **用户端 H5**，目标是「让商品维护更省心、让用户浏览更有卖点」。

| 模块 | 变化点 | 面向角色 |
|------|--------|---------|
| 管理后台 · 商品列表 | 删除冗余的「表单」按钮 | 运营 |
| 管理后台 · 商品编辑 | 移除「有效开始/结束日期」字段 | 运营 |
| 管理后台 · 商品编辑 | 新增「营销角标」多选字段 | 运营 |
| 用户端 · 服务列表 | 卡片删除销量与原价，新增卖点文案 | 顾客 |
| 用户端 · 服务列表 | 商品图左上角展示营销角标 | 顾客 |
| 用户端 · 商品详情页 | 头图左上角展示营销角标 | 顾客 |
| 用户端 · 首页 | 新增「热门推荐」横滑商品位，带角标 | 顾客 |

> 资深经理提示：历史上的「有效日期」字段及该列数据已**彻底清理**（不可回滚），如需限时活动能力请后续单独立项。

## 三、管理后台使用说明（运营视角）

### 3.1 商品列表 · 操作列精简

- 进入「商品体系 → 商品管理」后，操作列不再显示「表单」按钮。
- 预约表单仍然可在「商品体系 → 预约表单库」中独立维护；绑定关系请在商品编辑弹窗的「预约与核销」Tab 里完成。
- 其他操作（编辑 / 上下架 / 删除）位置和行为保持不变。

### 3.2 商品编辑 · 移除有效日期字段

- 编辑弹窗的「预约与核销」Tab 中已不再展示「有效开始日期」「有效结束日期」两个日期选择器。
- 下架开关仍然是控制商品是否接单的唯一入口。
- 如需停止销售，请直接使用列表的「下架」按钮即可。

### 3.3 商品编辑 · 新增营销角标多选

- 进入商品编辑弹窗后，在「基础信息」Tab 的「商品卖点」下方可见 **「营销角标」** 字段。
- 可勾选以下 4 个预置标签中的任意子集（支持多选）：

| 标签 | 颜色 | 建议场景 |
|------|------|---------|
| 限时 | 红色 | 限时活动专享商品 |
| 热销 | 橙色 | 销量领先商品 |
| 新品 | 蓝色 | 近期新上架商品 |
| 推荐 | 绿色 | 运营主推商品 |

- 用户端展示时**只展示 1 个角标**，优先级顺序为：**限时 > 热销 > 新品 > 推荐**。
- 不勾选任何标签时，用户端图片左上角**不展示任何角标**。

**操作步骤**：
1. 登录后台 → 商品体系 → 商品管理。
2. 选中某条商品点击「编辑」。
3. 在「基础信息」Tab 中找到「营销角标」，勾选需要展示的标签。
4. 点击右下角「保存」或「保存并上架」即可生效。

## 五、注意事项

1. **营销角标只在勾选后才出现**，未勾选则三处（列表/详情/首页推荐）都不显示，以保持页面干净。
2. 多个角标同时勾选时，按 `限时 > 热销 > 新品 > 推荐` 的优先级仅渲染最高优先级一个。
3. 「卖点」字段最多 100 字，单行省略展示，过长会自动截断，建议精炼到 15~25 字。
4. 「有效日期」字段及历史数据已**彻底清理**，老数据中配置过"仅 X 月 X 日~Y 月 Y 日在售"的商品，如今通过**上/下架开关**控制。
5. 首页「热门推荐」数据来自 `/api/products/hot-recommendations` 接口，取近 30 天销量 Top 商品，无需运营手动配置。

## 六、常见问题（FAQ）

**Q1：过去我在商品里配置过有效日期，现在还生效吗？**
A：字段已被彻底删除，不再生效。现在请用「下架」按钮临时停售即可。

**Q2：我勾选了「限时 + 热销」两个角标，为什么用户端只看到「限时」？**
A：这是产品设计，多角标场景下按优先级仅展示 1 个，避免卡片信息过载。要展示「热销」请只勾选「热销」。

**Q3：为什么我的商品在首页「热门推荐」里看不见？**
A：热门推荐按近 30 天销量排序取 Top，请确认该商品已上架且近期有成交。

**Q4：列表页为什么看不到销量了？**
A：按新设计，列表页强调「卖点 + 价格」，销量作为决策佐证保留在详情页展示。

**Q5：我的商品没有填卖点，那一行会显示什么？**
A：系统会自动用该商品绑定的「商品分类名」兜底（如"中医理疗""艾灸"等），确保卡片视觉一致。
