## 一、本次修复在做什么

进入 AI 对话首页后点击左上角 ☰ 按钮可拉出**左侧抽屉**，抽屉的左上角会展示你的「头像 + 账号名 + ID + 顶栏图标」。

修复前的问题：

- 头像独占第一行
- 账号名（如「账号 6399」）被挤到了头像下方第二行
- ID 胶囊（如「ID: 77890360」）又被挤到了第三行
- 头像右侧到铃铛/设置之间留下了一大段空白，整体视觉重心严重下沉

本次修复后：

- **头像 + 账号名 + ID + 顶栏图标全部在同一行水平排布**
- 账号名和 ID 上下两行紧贴在头像右侧（构成一个「名片块」）
- 顶栏铃铛（🔔）+ 设置（⚙）继续在最右侧，靠右对齐
- ID 胶囊改为**纯展示**，不再支持点击复制（不会再弹出「ID 已复制」Toast）

## 三、操作步骤

1. 在手机浏览器或微信内置浏览器打开 [AI 对话首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)。
2. 完成登录（如未登录会自动跳转登录页 `login`）。
3. 在 AI 对话首页**左上角点击 ☰** 按钮，左侧抽屉会从屏幕左侧滑入（占屏幕 85%）。
4. 观察抽屉**最顶部**的左上角：
   - 应该看到「**头像 │ 账号名 + ID 两行 │ 🔔 ⚙**」**整齐地排在同一水平线**上。
   - 账号名是大号黑色加粗字体；ID 在账号名正下方，是浅灰色胶囊。
   - 铃铛与设置按钮稳稳地停在最右边。
5. **试着点一下 ID 胶囊（如「ID: 77890360」）**：应该**不会**有任何反应——没有 Toast 弹出、没有任何跳转、没有复制行为。
6. 试着点击**铃铛**：应跳转到消息通知页。
7. 试着点击**设置（⚙）**：应跳转到 AI 设置页。

## 五、注意事项

1. **账号名超长会自动省略**：如果你的账号名很长（比如 20 个字以上），名片块会自动用「…」省略号截断，**不会撑破布局把图标挤掉**。
2. **ID 胶囊不再可点击复制**：之前点击 ID 会复制到剪贴板并出现「ID 已复制」Toast，本次修复**取消该功能**。如果你需要复制 ID，可以前往「[个人资料编辑页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/profile/edit)」中长按选中后复制。
3. **抽屉其他区块未受影响**：资产 4 并列（积分/优惠券/订单/收藏）、健康档案 + 我的设备入口、历史对话分组、左滑/⋯ 菜单/管理态批量删除等所有功能保持原状。
4. **多端表现**：本次修复仅作用于 H5 端（手机浏览器/微信内网页）。Flutter App、微信小程序如需对齐，会在后续版本同步处理。
5. **极小屏适配**：在 320px 极窄机型上，名片块会自动收缩，账号名优先省略，整体仍保持单行水平布局，不会发生折行。
