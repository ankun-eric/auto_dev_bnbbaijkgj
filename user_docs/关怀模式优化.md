## 二、详细操作步骤

### 1. 关怀模式首页

1. 进入小程序，切换到「关怀模式」。
2. 顶部一行从左到右依次为：
   - **☰ 历史**：查看历史对话记录（与标准模式一致）。
   - **宾尼小康** 标题 + **「宾尼小康 模式切换」** 胶囊：点击胶囊可切回标准模式。
   - **🎁 邀请**、**⋯ 更多**（会员中心 / 扫一扫 / 字体大小等）。
3. 中部绿色大卡显示问候语「我是宾尼小康，聊聊健康问题吧~」与今日用药提醒（24 小时制时间，如 20:00）。
4. 下方竖排功能卡片：用药提醒、健康记录、居家安全等。
5. **右下角悬浮红色 SOS 按钮**始终可见，周围有一圈圈往外扩散的光圈，点击立即进入紧急呼叫流程。

### 2. 用药提醒

1. 在首页点击「用药提醒」卡片。
2. 进入本人独立的用药提醒页，可看到「还有 N 次用药 / 下一次提醒 / 立即服用 / 今日时间线」。
3. 数据仅展示**本人**的用药计划，不混入家人数据。

### 3. 健康记录

1. 在首页点击「健康记录」卡片。
2. 进入今日健康记录页，显示血压、血糖、心率、血氧、睡眠等卡片。
3. 这些数据与「健康档案 → 本人 → 今天健康数据」完全一致；当天没有记录时显示「—」。

### 4. 紧急呼叫（SOS）

进入方式：首页右下角悬浮 SOS，或首页「紧急呼叫」卡片。

1. 页面中央是一个红色大 SOS 按钮，外圈有动态扩散的红色光圈（类似水波/雷达）。
2. **长按 SOS 按钮 3 秒**触发求助：长按过程中进度环会逐渐转满，转满 3 秒才真正发出求助（防止老人误触）。中途松手即取消。
3. 页面顶部「定位条」：
   - 定位中：灰色「定位中…」。
   - 定位成功：绿色「● 定位已就绪 · 具体地址」（显示中文地址，如「北京市朝阳区建国路 88 号」，不再显示精度米数）。
4. 下方为紧急联系人列表，每位联系人含头像首字、姓名、优先标签与绿色拨打按钮，点击即可一键拨打。

### 5. 紧急联系人维护

在紧急呼叫页右上角点击「管理」进入维护页：

1. **家庭住址**：填写后点「保存」。住址为空时会提示「请先填写家庭住址」，必填才能保存。
2. **新建联系人**：填写姓名、关系、电话。**电话必填**，为空会提示「请填写联系人电话」。
3. **分享我的位置**：
   - 点击「分享我的位置」按钮。
   - 选择「微信发给好友」。
   - 系统会分享**静态位置**（即发出那一刻的位置，更简单、更隐私可控）。

### 6. 对方查看分享的位置

1. 好友在微信中收到分享后，点击进入小程序。
2. 看到一个查看页面，包含：
   - **地图**：显示分享时的位置（坐标已转换为可读中文地址）。
   - **精简个人信息卡**：头像、姓名、年龄性别、家庭住址、慢性病/过敏（如有）、紧急联系人。
   - **一键拨打紧急联系人**按钮，方便好友快速联系。

## 三、注意事项

1. **数据范围**：关怀模式所有二级页面（用药提醒、健康记录等）只展示**当前登录长辈本人**的数据，不出现家人切换。
2. **定位权限**：紧急呼叫与位置分享需要授权定位。若提示定位失败，请在系统设置中开启定位权限后重试。
3. **长按求助**：SOS 必须长按满 3 秒才会真正发出，轻触或短按不会触发，避免误报。
4. **静态位置分享**：分享的是「点击那一刻」的位置，不是实时位置；如位置变化需要重新分享。
5. **分享链接有效性**：对方需在微信内打开小程序查看；位置查看页为公开只读页面，不展示二维码私密标识。
6. **地址解析**：坐标转中文地址优先使用高德地图服务，异常时自动切换备用服务，个别偏远坐标可能解析为概略地址。
