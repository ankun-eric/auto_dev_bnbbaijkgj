## 使用说明

### 一、体检报告识别完整流程

体检报告识别分为三个步骤：**上传报告 → 选择咨询对象 → AI 解读**

#### 第一步：上传体检报告

1. 打开 H5 页面，进入体检报告功能（底部导航 → AI 助手 → 体检报告）
2. 点击「相册」从手机相册选择体检报告图片，或点击「拍照」直接拍摄体检报告
3. 支持同时上传最多 **5 张**图片，每张不超过 20MB
4. 选择图片后可在预览区查看，点击图片右上角 × 可移除
5. 确认图片无误后，点击「开始识别」按钮进入下一步

#### 第二步：选择咨询对象（本次优化重点）

点击「开始识别」后，页面底部会弹出完整的咨询对象选择面板：

**① 选择成员**
- 默认选中「本人」，自动加载本人的健康档案
- 可点击其他家庭成员切换，系统会自动加载该成员的健康档案
- 选中的成员会高亮显示绿色边框

**② 查看/编辑健康档案**
选中成员后，下方展示该成员的完整健康档案，所有字段均可编辑：

| 字段 | 类型 | 是否必填 |
|------|------|----------|
| 姓名 | 文本输入 | ✅ 必填 |
| 出生日期 | 日期选择 | ✅ 必填 |
| 性别 | 单选（男/女） | ✅ 必填 |
| 身高 | 数字输入（cm） | 选填 |
| 体重 | 数字输入（kg） | 选填 |
| 既往病史 | 标签多选 + 自定义输入 | 选填 |
| 过敏史 | 标签多选 + 自定义输入 | 选填 |
| 家族遗传病史 | 标签多选 + 自定义输入 | 选填 |

> 💡 修改健康档案信息后，点击确认按钮会自动保存到数据库，其他模块（如健康自查）也能立即使用最新数据。

**③ 添加新家庭成员**
- 在成员列表底部点击「+ 添加家庭成员」
- 在弹出的面板中先选择与本人的关系（如爸爸、妈妈、配偶等）
- 填写新成员的基本信息（姓名、性别、出生日期为必填）
- 点击「确认添加」，新成员会自动保存并被选中

**④ 确认提交**
- 确认选中的咨询对象和健康档案信息无误后，点击「确认并开始识别」
- 系统会先保存修改的健康档案，然后开始上传图片进行 AI 识别

#### 第三步：AI 解读

- 上传完成后自动进入 AI 解读页面
- 页面顶部显示进度圈和百分比（居中）
- 下方显示 4 行阶段提示文字（**左对齐**，本次优化内容）：
  - 🔍 AI 正在分析您的指标…
  - 📊 正在评估各项指标的风险等级…
  - 📝 正在生成个性化健康建议…
  - 🧠 AI 正在深度分析中，请耐心等待…
- 分析完成后自动展示健康评分、指标分类详情和个性化建议

### 二、疾病史标签使用说明

在健康档案编辑区域，既往病史、过敏史、遗传病史三个字段支持：

1. **选择预设标签**：直接点击系统预设的常见标签即可选中/取消
2. **自定义输入**：点击「✚ 其它」按钮，在输入框中输入自定义内容，回车或点击确认添加
3. **删除已添加项**：预设标签再次点击取消选中；自定义标签点击标签上的 ✕ 删除

## 注意事项

1. **必填字段校验**：姓名、出生日期、性别为必填项，未填写时会有红色提示，无法提交
2. **图片限制**：单次最多上传 5 张图片，每张不超过 20MB，支持 JPG/PNG 等常见图片格式
3. **分析时间**：AI 分析通常在 30-60 秒内完成，若超过 60 秒系统会提示已转入后台处理
4. **网络要求**：上传和分析需要稳定的网络连接，建议在 WiFi 环境下使用
5. **数据同步**：在体检报告中修改的健康档案信息会实时同步，其他模块（如健康自查）可立即读取最新数据
6. **浏览器兼容**：适配 iOS/Android 主流浏览器及微信内置浏览器
