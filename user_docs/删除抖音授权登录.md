## 功能简介

本次更新移除了系统中所有与「抖音授权登录」相关的功能，包括：

- **后端 API**：注册设置接口（`/api/auth/register-settings` 和 `/api/admin/settings/register`）不再返回 `douyin_register_mode` 字段
- **管理后台**：系统设置 → 注册设置页面中，已移除「抖音端注册方式」配置选项
- **H5 用户端**：登录页面底部的「其他登录方式」区域，已移除抖音渠道入口，仅保留微信渠道
- **微信小程序**：登录页默认配置中已移除抖音相关字段

## 使用说明

### 对管理员的影响

1. 打开管理后台：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 使用管理员账号登录
3. 进入「系统设置」→ 点击「注册设置」选项卡
4. 可以看到注册设置中已不再包含「抖音端注册方式」选项
5. 现有可配置项包括：
   - 自助注册会员（开启/关闭）
   - 微信端注册方式（授权即会员 / 填写注册信息）
   - 注册页布局（上下结构 / 左右结构）
   - 会员信息补充提醒（开启/关闭）
   - 会员卡号生成规则（递增 / 随机）

### 对用户的影响

1. 打开 H5 用户端：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 在登录页面底部的「其他登录方式」区域，现在只显示微信渠道
3. 用户仍可通过手机号验证码方式正常登录/注册
4. 微信授权登录功能不受影响

---

## 注意事项

- 已注册的用户数据不受影响，所有用户账号、会员卡号、积分等信息均保持不变
- 如果此前通过抖音渠道注册的用户，其账号仍然有效，可继续通过手机号验证码方式登录
- 管理后台中此前配置的抖音端注册方式设置将被系统忽略，不再生效
- 本次变更不影响微信授权登录功能和短信验证码登录功能
