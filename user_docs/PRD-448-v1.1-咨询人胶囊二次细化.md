## 功能简介

本次更新对 AI 对话页面（AI 首页 + 对话详情页）顶部的「咨询人胶囊」做了第二轮细化打磨，主要解决两类问题：

1. **行为问题**：之前当咨询对象是「本人/自己」时，胶囊不显示（看起来像功能没生效）→ 本次修复后，本人会话也能看到胶囊。
2. **细节问题**：文案、字号、图标尺寸、箭头对齐等视觉细节做了贴近「晓医」参考样式的精修。

升级后的胶囊样式：

```
┌─────────────────────────────────────────────────┐
│ ┌─────────────────────────────────────────────┐ │
│ │ 👤  本次回答结合 妈妈 的档案              ▽ │ │  ← 14px 文字 / 16×16 图标
│ └─────────────────────────────────────────────┘ │
│                                                 │
│  AI 回答正文文字……                              │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 使用说明

### 一、查看本人会话的胶囊（核心修复点）

1. 打开 [AI 首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)，登录账号
2. 在咨询对象切换器中选择「本人」/「自己」
3. 向小康发送任意一条问题，等待 AI 回答
4. **预期效果**：AI 回答正文上方出现一条浅灰圆角胶囊，文案为：
   - 「**本次回答结合 本人 的档案**」（注意是固定写「本人」，不会显示真实姓名/账号昵称，避免在公共场合露出隐私）
5. 点击胶囊 → 展开成员档案的 7 项字段；再次点击 → 折叠回去

### 二、查看其他家人的胶囊

1. 在咨询对象切换器中选择任意家庭成员（如「妈妈」、「宝宝」、自定义"张三"）
2. 向小康提问 → AI 回答上方出现胶囊
3. 文案分别为：
   - 「本次回答结合 **妈妈** 的档案」
   - 「本次回答结合 **宝宝** 的档案」
   - 「本次回答结合 **张三** 的档案」（直接显示成员真实 name，不带书名号/引号）

### 三、未选定咨询对象时的兜底

- 如果还未选定咨询对象，或档案数据正在加载中，胶囊**整条不显示**（不会出现"加载中…"半成品胶囊或文案抖动）
- 等数据回来后胶囊会整体出现，避免视觉跳动

### 四、详情页（点开历史会话）的兼容

1. 进入「历史对话」→ 点开任一历史会话，自动跳转到 `/chat/{sessionId}` 详情页
2. 同样能看到与 AI 首页一致规格的胶囊
3. **后端配置兜底**：即使后端接口未下发 `profile_row_enabled` 字段，胶囊也默认显示（v1.0 时该情况会整体灰掉，本次已修复）

## 注意事项

1. **隐私保护**：本人会话固定显示"本人"二字，**不会泄露**真实姓名/手机号/账号昵称；公共场合截图分享更安心。
2. **整条不显示 ≠ 出错**：未选定档案、档案数据加载中、成员名为空时胶囊不渲染，是设计预期，而非 Bug。
3. **箭头位置**：折叠态箭头朝下、展开态朝上，**位置完全一致**（仅方向通过 transform 变化），用浏览器标尺/截图比对应能看到完全对齐。
4. **页面刷新**：升级后建议下拉刷新或强制刷新一次浏览器缓存，确保拉到最新前端 chunks。
5. **兼容性**：在 iOS Safari、Android Chrome、微信内置浏览器表现一致。
