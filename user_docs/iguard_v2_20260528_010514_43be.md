## 使用说明

### 1. 进入页面

打开 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/i-guard](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/i-guard) 或从「健康档案首页 → 我守护的人」入口进入。

### 2. 查看人数（Bug 1）

页面顶部统计栏显示「我守护的人 · **N** 人」，**N = 已绑定家人（不含本人）+ 全部未绑定邀请记录（含已拒绝/已过期）**。

### 3. 本人卡片：编辑档案（Bug 2）

点击本人卡片上的「编辑档案」按钮 → 弹出抽屉，与 Hero 区右上角「编辑」打开的抽屉完全一致，字段可编辑、可保存。

### 4. 其他人卡片：查看档案（Bug 3）

点击其他已绑定家人卡片上的「查看档案」按钮 → 弹出同款抽屉，但所有字段以**只读**模式呈现，右上角无「编辑」按钮，底部无「保存」按钮，标题为「查看档案」。

### 5. 发起邀请（Bug 4 / Bug 7）

三个入口现在统一调用同一个「新建邀请」抽屉：
1. 顶部右上角 **「+ 发起邀请」**（金色实心按钮）
2. AI 首页咨询人「新建 → 去邀请」
3. 未绑定记录卡片上的 **「再次邀请」**（再次邀请时姓名与关系会预填）

抽屉中输入对方称呼（可选）和关系（必填），点击「生成邀请二维码」即可。

### 6. 共享额度（Bug 5 ・ 原代付明细）

> ⚠️ 「代付明细」入口已重命名为「共享额度」，仅在你是主守护人的已绑定卡片上显示。

点击 **「共享额度」** → 抽屉自上而下展示：
- **额度概览**：已使用 X / 共 Y，剩余 Z 次
- **会员权益共享开关**：关闭后被守护人将无法使用你的会员权益
- **使用明细**：最近 20 条使用记录

### 7. 解除守护（Bug 6）

点击已绑定家人卡片的「解除守护」按钮：
- 弹出二次确认框：「将解除与 XXX 的守护关系，TA 的健康档案数据将完整保留。解除后您将无法再查看 TA 的档案与代付，且对方会收到解除通知。」
- 确认后接口调用 `DELETE /api/family/management/{id}`，列表自动刷新，对方收到通知。

### 8. 移除未绑定记录（Bug 8）

「移除」按钮的智能行为：

| 卡片状态 | 移除行为 |
|---|---|
| **pending 未过期** | 弹「该邀请仍在生效中」→ 确认后**级联**「取消邀请 + 移除」|
| **expired / rejected / cancelled** | 弹「邀请记录…」文案 → 确认后移除 |
| **孤儿档案**（无邀请，仅本地档案） | 弹「孤儿档案…」文案 → 确认后移除 |
| **active**（已绑定） / 绑定硬件 / 在途服药 | 按钮置灰，长按显示禁用原因 |

### 9. 邀请记录（Bug 9）

点击「邀请记录」 → 现在从底部滑出**抽屉**（不再是弹框），右上角有 **「×」** 关闭按钮，点空白处或下滑也可关闭。

### 10. 按钮配色规范（Bug 10）

页面所有按钮按四级规范统一配色：

| 等级 | 用途 | 视觉 |
|---|---|---|
| 主操作 Primary | + 发起邀请、再次邀请、生成邀请二维码 | 金色实心 (#FFB800) + 白字 + 阴影 |
| 次操作 Secondary | 共享额度、查看档案、编辑档案、邀请记录 | 白底 + 天蓝描边 (#0EA5E9) |
| 危险操作 Danger | 解除守护、移除 | 透明底 + 红字 (#EF4444) |
| 禁用态 Disabled | 不可移除的按钮 | 灰底 + 灰字 (#9CA3AF) + 长按提示原因 |

## 注意事项

1. **会员权益共享开关关闭后**：被守护人将无法使用你的会员权益；已开始的会话允许完成，下一次调用将拒绝。
2. **解除守护后**：双方都会收到通知；TA 的健康档案数据完整保留，但你不再能查看。
3. **pending 未过期邀请**不能直接移除，必须先取消邀请；新页面已为你自动级联处理。
4. **禁用态「移除」**按钮：长按可看到禁用原因（请先解除守护 / 请先解绑硬件设备 / 存在在途服药计划等）。
