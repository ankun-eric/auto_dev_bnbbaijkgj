## 功能简介

本次更新对健康档案"本人 Tab"的完善度判定逻辑进行了底层的简化和优化：

- **后端判定逻辑简化**：健康档案完善度判定从之前跨三表（health_profiles、family_members、users）取并集的方式，简化为仅判定 health_profiles 表的 name/gender/birthday 三个字段。
- **下线冗余接口**：`GET/POST /api/health/guide-status` 两个接口已下线，前端引导弹窗统一改用 `GET /api/health-profile/self` 接口的 needComplete 和 missingFields 字段判断。
- **频控逻辑迁移**：引导弹窗的频控（最多弹2次）从后端 guide_count 字段迁移到前端 localStorage 实现，减少后端状态维护。

**用户体验变化**：本次更新为用户无感更新，健康档案完善引导弹窗的行为和之前保持一致。

## 使用说明

### 健康档案完善引导

1. 打开宾尼健康应用，进入「健康档案」页面
2. 如果您的姓名、性别、出生日期三项中有缺失，系统会弹出引导弹窗提示您完善信息
3. 按引导填写完整后，弹窗不再显示
4. 引导弹窗最多弹2次（由浏览器/小程序 localStorage 控制），点击"跳过"也会减少剩余次数

### 验证更新是否生效

本次更新对用户无感知，验证方式如下：
- 访问 [https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/health-profile](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/health-profile) 查看健康档案页面
- 如果档案信息不完善，会收到引导弹窗
- 填写完善后刷新页面，弹窗不再显示
