## 功能简介

本次修复彻底解决了一个困扰用户的 H5 严重体验问题：

- **修复前**：在「居家安全设备」页面，点击顶部家庭成员头像列表末尾的「**+ 添加**」按钮，页面会**整页白屏**，只显示一行 `gateway ok` 的纯文本，用户无法继续任何操作，必须退出重进。
- **修复后**：点击「+ 添加」会**直接弹出「新增家庭成员」弹窗**（与「健康档案」页同名按钮的体验完全一致），填写资料保存后，新成员立即出现在所有用到家庭成员的页面（居家安全设备、健康档案等）。

同时还做了两项增强，让此类问题不再静默发生：

1. **全局兜底容错**：H5 后续遇到任何「接口路径走丢 / 网关兜底响应 / 非合法 JSON」的异常响应，会自动弹出友好的「网络异常，请稍后重试」提示，**绝不再出现整页白屏**。
2. **静默失败监控**：每次命中类似的兜底响应时，前端会异步上报一条日志到服务端（含 URL、用户、时间、页面路径等），方便运维主动发现同类问题。

## 使用说明

### 场景一：在「居家安全设备」页快速添加家人

1. 在 H5 中打开「居家安全设备」页：
   [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/home-safety/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/home-safety/)
2. 进入页面后，注意页面顶部的一排家庭成员圆形头像（如：本人 / 老婆 / 儿子 / …）。
3. 滚动到**头像列表末尾**，会看到一个虚线圈圈 ⊕，文字为「**添加**」。
4. **点击「+ 添加」按钮**。
5. 此时会弹出「**新增家庭成员**」表单弹窗（不再白屏），内容与「健康档案」页一致：
   - 必填项：成员关系（如妈妈、儿子）、姓名、性别、出生日期
   - 可选项：身高、体重、既往病史、过敏史
6. 填写完成，点击「保存」。
7. 弹窗自动关闭，页面顶部的家庭成员头像列表中立刻出现刚加的新成员。
8. 接下来您可以在该新成员名下绑定居家安全设备（紧急呼叫器、烟雾报警器、水位报警器等）。

### 场景二：在「健康档案」页查看刚加的家人（对照）

1. 进入「健康档案」页：
   [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/)
2. 在顶部家庭成员胶囊条中，您会看到刚才在「居家安全设备」页加的家人也出现在这里。
3. 两个入口对应的是**同一份家庭成员数据**，新增后全端可见。

### 场景三：异常情况下的友好提示

如果您在使用过程中遇到瞬时网络异常或后端短暂不可用：

- 您将看到一个简短的小提示：「**网络异常，请稍后重试**」
- 页面**仍然停留在当前界面**，不会跳走，也不会白屏。
- 稍后重试一次操作即可。

## 注意事项

1. **只加人，不绑设备**：在「居家安全设备」页点「+ 添加」加完家人后，**默认不会自动给新家人绑定任何设备**。设备绑定仍需您在分组卡片右上角点「+ 添加」按钮，单独完成（设备绑定流程未变）。
2. **关系唯一性约束**：同一类亲属关系（如「爸爸」「妈妈」）只能添加一位，添加重复关系会有提示。
3. **本人档案需先完善**：若您本人档案（姓名、性别、出生日期等）尚未完善，添加家人时会被引导先去完善本人资料。
4. **新成员立即全端可见**：新加的家人会同时出现在「居家安全设备」「健康档案」等所有使用家庭成员的功能中，无需手动同步。
5. **不需要重新登录或刷新缓存**：本次修复属于线上版本更新，您直接打开页面即可体验，无需清理浏览器缓存。
