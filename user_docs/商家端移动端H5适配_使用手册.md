## 一、功能介绍

**"商家端移动端 H5"** 是专为手机浏览器打造的商家管理入口，让您在**不开电脑、不进小程序**的情况下，也能随时随地完成：

- 登录商家账户（支持账号密码、手机短信验证码、微信授权）
- 查看工作台实时数据（今日订单、营业额、待核销等）
- 管理订单并查看订单详情
- **扫码核销**用户购买的商品/服务（核心功能）
- 查看简化版报表、对账单
- 修改门店常用设置、管理员工

整套界面采用仿小程序的底部 TabBar 风格，大按钮、大字号，手机单手也能轻松操作。

### 1.1 本期上线的 8 大页面（P0 + P1）

| 模块 | 功能概要 |
|---|---|
| 登录 | 账号密码 / 短信验证码 / 微信授权 |
| 工作台 | 今日概览、快捷入口、最近订单、60s 自动刷新 |
| 订单 | 列表（状态/时间/关键词筛选）+ 订单详情 |
| 核销 | **扫码核销（主）** + 手动输码（兜底） |
| 报表 | 汇总卡片 + 简化图表 |
| 对账 | 对账列表、确认对账、发起争议 |
| 门店设置 | 常用字段编辑（电话、营业时间、公告等） |
| 员工管理 | 员工列表、启停状态切换 |

### 1.2 本期不支持的页面（P2，请在 PC 操作）

发票管理、下载中心、财务明细、消息详情 —— 移动端打开后会显示"请使用电脑访问 PC 商家端"的引导页，并附带一键复制链接。

## 三、操作步骤

### 3.1 登录

1. 在手机浏览器打开登录链接。
2. 顶部有三个 Tab，按需切换：
   - **账号密码**：输入手机号/账号 + 密码。
   - **短信验证码**：输入手机号 → 点击"获取验证码" → 60s 内输入收到的 6 位验证码。
   - **微信授权**：仅在微信浏览器中可见，点击"微信一键登录"即可。
3. 点击"登录"按钮：
   - 若您只管理 1 家门店 —— 直接进入工作台。
   - 若您管理多家门店 —— 进入"选择门店"页，点击要进入的门店即可。

> ⚠️ 登录有效期 14 天，期间无需重复登录；退出后需要重新输入凭据。

### 3.2 工作台（首页）

- **顶部**：当前门店名 + 门店切换箭头（点击进入"选择门店"） + 消息红点
- **4 个概览卡**：今日订单数 / 今日核销数 / 今日营业额 / 待核销数
- **快捷操作**：核销 · 订单 · 报表 · 员工（按角色过滤）
- **最近 5 条订单**：点击直达订单详情
- **下拉刷新**：从上往下滑动即可立即刷新数据；每 60 秒页面会自动静默刷新。

### 3.3 订单列表与详情

1. 点击底部 TabBar "订单"进入列表。
2. 顶部可按以下条件筛选：
   - 状态：全部 / 待核销 / 已核销 / 已退款 / 已取消
   - 时间：今日 / 本周 / 本月 / 自定义
   - 关键词：订单号 / 商品名
3. 向下滚动自动加载更多订单。
4. 点击任一订单 → 进入订单详情：可查看商品信息、金额、用户信息（脱敏）、核销记录、附件等。
5. 详情底部操作：
   - **去核销**：仅"待核销"订单显示。
   - **申请退款**：点击后提示"请使用 PC 操作"。

### 3.4 扫码核销（核心功能）

> 🌟 这是商家移动端最高频的操作，建议熟悉。

1. 点击底部 TabBar "核销" 进入核销页。
2. 默认进入"扫一扫"Tab：
   - 首次使用会弹出"请允许使用摄像头"权限请求 —— 点击"允许"。
   - 将用户出示的核销二维码对准取景框，系统自动识别。
   - 识别成功 → 进入"核销确认"页 → 核对用户、商品、有效期信息 → 点击"确认核销"。
   - 核销成功显示绿色 ✓ + 提示，2 秒后自动返回扫码页，继续下一单。
3. 若扫码不可用（例如未授权摄像头、光线太暗），点击切换到"手动输码"Tab：
   - 让用户口述 6-8 位核销码，在输入框中输入。
   - 点击"确认核销"按钮即可。

> ⚠️ **非 HTTPS 环境**（例如 http:// 开头）下扫码功能不可用，页面会自动提示并引导到输码 Tab。

### 3.5 报表

- 顶部时间选择器：今日 / 近 7 日 / 近 30 日 / 自定义
- 4 个汇总卡片：营业额、订单数、核销数、客单价
- 2 个简化图表：营业额趋势折线图、订单状态饼图
- 需要复杂明细、多维分析、导出 Excel？页面底部有"去 PC 查看完整报表"入口。

### 3.6 对账

1. 进入"对账"页（财务或管理员可见），可按月份浏览对账单。
2. 状态包括：待确认 / 已确认 / 有争议 / 已结算。
3. 点击对账单进入详情：
   - **确认对账**：确认结算金额无误。
   - **发起争议**：若对金额有疑问，填写原因后提交。

### 3.7 门店设置

- 仅老板、店长可编辑。
- 移动端开放的常用字段：**联系电话、营业时间、门店公告**。
- 其他字段（门店名、地址、营业执照等）为只读，需用 PC 修改。

### 3.8 员工管理

- 员工卡片：头像 + 姓名 + 角色 + 状态（启 / 停）。
- **启停状态**：有权限者可一键切换。
- **新增员工**：移动端不支持，请使用 PC 操作（页面会引导）。

### 3.9 "我的"Tab

进入路径：底部 TabBar "我的"。

功能清单：
- 切换门店（多门店账号可见）
- 消息中心（只读列表）
- 复制 PC 版访问链接
- 修改密码（引导至 PC）
- 关于我们、版本信息
- **退出登录**（清除本机登录状态）

## 四、角色权限对照

不同角色登录后可见的 TabBar 与模块不同：

| 角色 | 底部 TabBar | 可访问模块 |
|---|---|---|
| 老板 / 店长 | 工作台 · 订单 · 核销 · 我的 | 全部 8 个页面 |
| 财务 | 工作台 · 订单 · 对账 · 我的 | 工作台 / 订单 / 报表 / 对账 / 门店设置（只读） / 我的 |
| 核销员 | 工作台 · 订单 · 核销 · 我的 | 工作台 / 订单 / 核销 / 我的 |

若访问越权页面，系统将自动跳转到"无权限"提示页。

## 五、常见问题（FAQ）

**Q1：为什么扫码按钮灰掉 / 点了没反应？**
A：请确认浏览器地址栏为 `https://` 开头，且已授权摄像头。如浏览器拒绝了权限，可前往手机系统的"浏览器权限设置"里重新开启；也可切换到"手动输码"继续操作。

**Q2：为什么切换门店后订单/报表没变化？**
A：部分页面会在 60 秒后自动刷新。您也可以在页面顶部向下拉动手动刷新。

**Q3：多人用同一账号在手机和电脑同时登录可以吗？**
A：可以，系统支持多端并行登录，互不影响。但小程序端的登录态与 H5 互相独立，请分别登录。

**Q4：为什么打开 `/merchant/invoice` 变成了"请用电脑访问"？**
A：发票、下载中心等页面对移动端交互要求较高，本期未做移动端适配。请点击提示页上的"复制 PC 链接"按钮，回到电脑上继续。

**Q5：如何在手机上强制看 PC 版？**
A：在 URL 后追加 `?desktop=1`，例如：

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/login?desktop=1](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/login?desktop=1)

**Q6：页面有点卡/加载慢？**
A：建议使用 4G/5G 或稳定 Wi-Fi。若首屏长时间不加载，可尝试：下拉刷新 → 清理浏览器缓存 → 重新登录。

## 六、注意事项

1. **HTTPS 必需**：扫码依赖摄像头 API，浏览器规定必须 HTTPS 才能调用。请始终使用 `https://` 开头的链接。
2. **登录安全**：请勿在公共设备上勾选"记住登录"；离开时请主动退出登录。
3. **手机号脱敏**：订单详情中的用户手机号默认中间 4 位打码，保护用户隐私。
4. **老机型兼容**：iOS 13 及以下、Android 5 及以下系统未列入支持目标；如遇问题建议升级系统或切换设备。
5. **数据一致性**：移动端与 PC 端共享同一套后端数据，任一端修改在另一端 60 秒内可见。
6. **网络波动**：扫码/核销时若出现"网络超时"，请不要重复扫码，先点击"重试"按钮，避免重复核销。
