## 功能简介

本次优化只做两件事，首页其它内容（轮播图/健康贴士、功能宫格、推荐问、聊天区、顶栏等）一律保持不变：

1. **标准模式欢迎区「换装」**：把标准模式首页顶部的欢迎区，改成与关怀模式欢迎区「长一样」的精致风格——同一套结构、版式、字号、时段问候语、大头像、模式切换胶囊；但做了**瘦身**：去掉了占地方的「今日用药提醒」白卡片，让欢迎区更紧凑，一屏即可看到「欢迎区 + 轮播图 + 大半个功能菜单」，功能触达更快。
2. **两种模式仅靠背景底色区分**：
   - **标准模式** = 蓝绿渐变背景（清爽专业）
   - **关怀模式** = 暖橙色背景（温暖亲切）

> 简单说：两个模式的欢迎区现在「同款不同色」——标准蓝绿、关怀暖橙，一眼就能分清自己在哪个模式。

## 使用说明

1. **打开标准模式首页**：访问 [AI 首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)（或在 App / 小程序中打开 AI 首页）。
2. **查看欢迎区**：页面顶部即为新的欢迎区——**蓝绿渐变背景**，左侧是大号时段问候语（如「早上好 ☀️」）和一行欢迎语，右侧是机器人头像，头像上方是「标准版 ▾」模式切换胶囊。
3. **确认瘦身效果**：标准模式欢迎区**不再显示**「今日用药提醒」卡片，欢迎区更紧凑，往下即可直接看到轮播图与功能菜单，无需大幅下滑。
4. **切换到关怀模式**：点击欢迎区右上角「标准版 ▾」胶囊 → 选择「关怀版」，或通过「⊕」更多菜单里的「切换模式」进入关怀模式。
5. **查看关怀模式首页**：访问 [关怀模式首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home)，欢迎区背景为**暖橙色**，其余内容（含「今日用药提醒」卡片）保持原样不变。
6. **切回标准模式**：在关怀模式欢迎区右上角「关怀版 ▾」胶囊中选择「标准版」即可切回。

### 体验步骤

1. **下载压缩包**：点击上方链接，将 zip 压缩包下载到本地电脑。
2. **解压文件**：将下载的 zip 文件解压到任意目录（记住解压后的文件夹路径）。
3. **下载微信开发者工具**：如尚未安装，请前往 [微信开发者工具官方下载页面](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装。
4. **打开微信开发者工具**：启动开发者工具，使用微信扫码登录。
5. **导入项目**：
   - 点击开发者工具首页的「导入项目」（或「+」号）。
   - 在「目录」栏点击浏览，选择第 2 步解压后的文件夹。
   - 「AppID」栏可填入项目的 AppID，或选择「测试号」进行体验。
   - 点击「导入」按钮。
6. **预览体验**：导入成功后，开发者工具会自动编译并在模拟器中展示小程序界面。进入「AI 首页」即可看到标准模式蓝绿欢迎区，切换到关怀版可看到暖橙欢迎区。
