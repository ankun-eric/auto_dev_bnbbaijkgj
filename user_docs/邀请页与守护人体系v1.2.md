## 一、功能介绍

「邀请页与守护人体系 v1.2」是宾尼小康对家庭健康守护功能的整体升级。本次升级把原本独立的「守护人体系」页融入【健康档案】，并新增「会员中心」「紧急 AI 呼叫」「主守护人代付」等关键能力，让你更轻松地照护家人健康。

### 核心新能力

| 能力 | 说明 |
| --- | --- |
| 我守护的人（直筒列表） | 一屏看完本人 + 所有被守护人，操作直达 |
| 守护管理抽屉 | 查看某位被守护人的全部守护人、转让主守护人、移除守护人 |
| 提醒设置抽屉 | 主守护人统管所有 AI 外呼提醒；普通守护人仅管自己 |
| 邀请记录抽屉 | "我发起的"+"别人邀请我的"一站式查看 |
| 我的 AI 外呼额度 | 一键跳转会员中心查看本月余量 |
| 会员中心 | 当前等级 / 本月配额 / 权益对比 / 套餐升级 |
| 紧急 AI 呼叫（原异常告警） | 串行外呼守护人，统一扣主守护人额度 |
| 主守护人代付开关 | 一键代付被守护人本人设置的 AI 外呼额度 |
| 紧急呼叫触发源管理 | 后台可启停 4 种内置触发源 + 自由扩展 |

## 三、操作步骤

### 3.1 普通用户：查看「我守护的人」

1. 微信扫码或浏览器打开 H5：<https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/i-guard>
2. 登录后看到直筒列表（无底部 Tab）：
   - **本人行**：头像 + 昵称，操作位「编辑档案」「邀请记录」「我的 AI 外呼额度」
   - **被守护人行**：头像 + 关系 + 角色角标（主守护人/普通守护人），操作位「查看档案」「提醒设置」「守护管理」

### 3.2 普通用户：设置 AI 外呼提醒

1. 在「我守护的人」列表点击某位被守护人的【提醒设置】
2. 抽屉打开后会显示：
   - 你本月剩余的 AI 外呼额度
   - 如果"代付"已开启，会显示"额度由 XXX 代付中"
3. 点击【新增提醒】 → 填写名称、时间 → 保存

> ⚠️ 默认规则：谁设置扣谁额度。被守护人本人设置时，若主守护人开启了"代付"，则改为扣主守护人额度。

### 3.3 主守护人：开启"代付被守护人 AI 外呼"

1. 在「我守护的人」列表点击该被守护人的【守护管理】
2. 抽屉中"本人行"会显示开关：【代付该被守护人的 AI 外呼额度】
3. 拨动开关 → 二次确认 → 立即生效
4. 被守护人在 App 中会看到醒目提示："您的 AI 外呼额度由 XXX 代付中"

> 注意：主守护人切换时，代付开关**不会自动继承**，新主守护人需重新开启。

### 3.4 主守护人转让（接收者同意流程）

1. 现任主守护人在【守护管理】抽屉中点击【转让主守护人】
2. 选择接收者 → 提交
3. 接收者会收到推送 + 短信，登录后在「待确认转让」入口点【同意】
4. 接收者同意后立即生效，三方收到结果通知
5. 接收者 24h 未确认或拒绝 → 转让作废，可无限次重发

### 3.5 被守护人「上帝视角」直接调整守护关系

1. 被守护人本人在【健康档案 → 我的守护人】列表中
2. 可直接：
   - 把某位普通守护人升为主守护人（原主守护人自动降为普通）
   - 直接移除某位守护人
3. 用于卧床老人不能操作 App 时由家人转让后，妈妈仍保留最终话语权

### 3.6 会员中心使用

1. 首页右上角三道杠菜单 → 【会员中心】（或直接打开 <https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center>）
2. 页面包含 4 大板块：
   - 我的当前等级卡片
   - 本月配额（3 个环形进度条：AI 外呼 / 紧急 AI 呼叫 / 守护他人上限）
   - 三档会员权益对比表
   - 升级套餐选购卡

> 配额每月 1 号自动重置；剩余 ≤2 次时**紧急 AI 呼叫**环显示黄色预警。

### 3.7 邀请记录查看与重发

1. 「我守护的人」本人行 → 点击【邀请记录】
2. 抽屉显示两类记录：
   - 我发起的邀请
   - 别人邀请我的邀请
3. 状态徽章含义：
   - **蓝色 待确认**：24h 有效期内
   - **绿色 已生效**：已被接受
   - **黄色 已过期**：24h 内未确认 → 可点【重新发送】
   - **灰色 已作废**：被覆盖或取消 → 也可点【重新发送】

### 3.8 运营管理后台：紧急呼叫触发源

1. 打开 <https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/emergency-sources>
2. 内置 4 种触发源（启停可改，**不可删**）：
   - 健康数据异常
   - 烟雾报警器
   - 水位报警器
   - 紧急呼叫器
3. 可新增自定义触发源（如"燃气报警器""门窗入侵传感器""跌倒检测手环"）：
   - 输入名称、描述
   - 配置启停、排序、适用设备类型
4. 自定义触发源可编辑、可删除

### 3.9 运营管理后台：守护关系管理（融合后）

1. 顶层"守护关系查询"已下线 → 已并入 <https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/family-management>
2. 列表新增列：
   - 角色（主/普通）
   - 优先级（串行外呼顺位）
   - 会员等级
   - 本月紧急 AI 呼叫额度剩余
   - 本月 AI 外呼额度剩余
3. 筛选器新增"仅看主守护人"、"仅看付费"等

## 四、关键业务规则

### 4.1 AI 外呼提醒扣额度规则

| 谁设置 | 扣谁 |
| --- | --- |
| 守护人 X 设置 | 扣 X 的额度 |
| 被守护人本人，代付**关闭** | 扣被守护人自己 |
| 被守护人本人，代付**开启** | 扣**主守护人**额度 |

- 外呼成功扣 1 次；失败（无人接、占线、关机）**不扣**
- 设置者额度耗尽 → 其设置的提醒自动暂停，下月 1 号自动恢复

### 4.2 紧急 AI 呼叫扣额度规则（v1.2 新）

- **统一扣主守护人**，无论最终是谁接通
- 串行外呼：主守护人无应答 → 拨第 2 顺位 → … 直到有人接通或全部尝试完毕
- 主守护人额度耗尽 → 立即降级为 **App 推送 + 短信**（永远免费、永远兜底）

### 4.3 主守护人操作规则

| 角色 | 移除自己 | 移除别人 |
| --- | --- | --- |
| 主守护人 | ❌ 必须先转让，转让后变普通才能退出 | ✅ 可移除任意普通守护人 |
| 普通守护人 | ✅ 可直接退出 | ❌ 不能 |

### 4.4 数量上限（运营后台自由配置）

- `max_managed`（守护他人上限）：**前端展示在会员中心**
- `max_guardians`（被绑定守护上限）：**前端不展示**，仅后台可见
- `-1` 表示不限次，前端展示为"不限"

## 五、注意事项

1. **「我的硬件」入口已从首页菜单删除**：健康档案里依然有入口，请去那里管理设备。
2. **「告警额度」改名为「我的 AI 外呼额度」**：所有出现"告警额度"的地方都已替换为新术语。
3. **"异常告警"统一改名为"紧急 AI 呼叫"**：含义不变，扣额度策略由"按设置者"改为"统一扣主守护人"。
4. **邀请凭证 24 小时有效，单次有效**：被预览不消耗，被确认后立即失效。
5. **付费会员到期后，主守护人身份保留**，但额度走免费档；额度用完降级为推送 + 短信。
6. **代付开关在主守护人切换时不自动继承**，新主守护人需要重新开启。
7. **App 推送 + 短信对所有守护人始终免费、始终发送**，不消耗任何额度。
8. **「会员中心 → 立即购买」当前为占位实现**，跳转模拟支付页，后续接入真实微信支付。

## 六、常见问题（FAQ）

**Q1：我是主守护人想退出，怎么操作？**
A：先在「守护管理」抽屉点【转让主守护人】并等接收者同意；转让生效后你变成普通守护人，再点【退出守护】。

**Q2：妈妈给自己设的提醒为什么扣到了我额度？**
A：你作为妈妈的主守护人开启了【代付被守护人 AI 外呼】开关。如不想代付，在「守护管理」抽屉关闭开关即可。

**Q3：紧急 AI 呼叫额度用完了会怎样？**
A：立即降级为 App 推送 + 短信，对所有守护人始终免费发送，不会失联。

**Q4：邀请链接被预览过还能用吗？**
A：可以，预览不消耗。只有任意一方点击"确认接受"后链接才失效。

**Q5：会员中心看不到「被绑定守护上限」？**
A：v1.2 起，"被绑定守护上限"（`max_guardians`）只在运营后台可见，前端仅展示"守护他人上限"（`max_managed`）。
