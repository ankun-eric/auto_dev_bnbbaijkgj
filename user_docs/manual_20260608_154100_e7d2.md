## 使用说明

### 入口位置

在「用药提醒」主页（`/ai-home/medication-reminder`），点击顶部 Banner 下方蓝色的「📋 历史记录」按钮，即可进入历史打卡记录页面。

### 日历操作

1. **切换月份**：点击日历顶部左右箭头（‹、›）可切换查看不同月份
2. **选中日期**：点击日历中的任意日期格子，下方会显示该天的用药打卡记录
3. **状态圆点**：每个日期格子右下角有小圆点标记当日打卡状态：
   - 🟢 绿色 = 全部打卡完成
   - 🟠 橙色 = 部分打卡
   - 🔴 红色 = 全部漏打
   - 无圆点 = 该天无用药计划

### 补打卡操作

1. 在日历中点击一个过去日期（昨天或前天）
2. 找到状态为「未打卡」的用药记录
3. 点击橙色的「补打卡」按钮
4. 在确认弹窗中点击「确认补打卡」
5. 补打卡成功后，卡片状态变为「已补」，日历圆点同步更新

**补打卡规则**：
- 仅限最近 3 天内的漏打卡记录（昨天、前天、大前天）
- 今天的用药请在主页打卡，不在这里补
- 超过 3 天的记录显示灰色「不可补」按钮

### 卡片状态说明

每条用药记录有四种显示状态：

| 状态 | 左侧色条 | 背景色 | 操作按钮 |
|------|----------|--------|----------|
| 已打卡 | 绿色 | 白色 | 「OK 完成」（可撤销） |
| 已补 | 橙色 | 浅黄色 | 「OK 完成」 |
| 未打卡（可补） | 红色 | 白色 | 橙色「补打卡」 |
| 未打卡（不可补） | 灰色 | 白色 | 灰色「不可补」 |

### 返回主页

点击页面左上角「← 返回」按钮，回到用药提醒主页。

## 注意事项

1. 补打卡有 3 天时限，超过后无法补打
2. 今天的用药请在主页「用药提醒」中打卡
3. 补打卡后日历圆点会同步刷新
4. 日历默认选中「昨天」，方便您查看昨天的漏打卡情况
5. 如果您看到了这个页面但找不到入口，请返回主页找蓝色「📋 历史记录」按钮
