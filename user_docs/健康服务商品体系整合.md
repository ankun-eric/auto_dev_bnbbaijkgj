## 功能简介

本次更新为 bini-health 健康管家系统实现了**「健康服务商品体系整合」**，将原有独立的"服务模块"和"积分商城"合并为统一的「健康服务」体系，打通了从浏览商品、下单支付、到店核销到评价收藏的全链路闭环。

### 本次主要新增功能

1. **统一商品体系** — 两级分类 + 统一商品管理 + 丰富版商品详情页
2. **统一订单体系** — 支持混合下单（到店+快递）、自动拆分履约、多种订单状态流转
3. **支付体系** — 微信支付 + 积分全额兑换
4. **到店核销** — 二维码 + 数字码双保险，支持套餐多次核销
5. **"我的"页面优化** — 统计栏（积分/优惠券/收藏）+ 订单快捷入口 + 菜单调整
6. **会员卡** — 动态二维码（60秒自动刷新）+ 卡片式设计
7. **优惠券体系** — 满减/折扣/代金/免费体验4种类型 + 领券中心
8. **收藏功能** — 商品收藏 + 知识收藏分Tab展示
9. **地址管理** — 收货地址增删改查（最多10个）
10. **评价体系** — 评分 + 文字 + 图片
11. **退款/售后** — 管理员审核退款流程
12. **自定义预约表单** — 多种字段类型 + 排班/自由选时间/随时到店模式
13. **管理后台** — 商品分类管理、商品管理、订单管理、优惠券管理、核销管理、进店记录、数据统计

## 使用说明

### 一、浏览健康服务商品

#### H5 端 / 小程序 / App

1. 打开应用首页，找到**「健康服务」**入口图标
2. 点击进入商品列表页
3. 顶部可切换**一级分类**（如口腔健康、营养管理、体检服务等）
4. 选择二级分类进一步筛选
5. 支持按**履约方式**（到店核销/快递发货）、**价格区间**、**积分可兑换**等条件筛选
6. 点击商品卡片进入详情页

### 二、查看商品详情

商品详情页包含以下模块（从上到下）：

1. **轮播图/视频** — 左右滑动查看商品图片和视频
2. **价格区域** — 售价 + 原价（划线价）+ 已售数量
3. **商品描述** — 富文本详情介绍
4. **常见问题 FAQ** — 点击展开/折叠查看
5. **适用门店** — 到店核销类商品显示可服务的门店列表
6. **用户评价** — 评分统计 + 评价列表，可筛选好评/中评/差评/有图
7. **底部操作栏** — 收藏按钮 + 购买按钮

### 三、购买商品

1. 在商品详情页点击**「立即购买」**
2. 进入下单页：
   - **快递商品**：选择或新增收货地址
   - **到店服务**：选择服务门店
3. 如商品配置了**预约表单**，需填写预约信息（姓名、手机号、预约时间等）
4. 选择可用的**优惠券**
5. 决定是否使用**积分抵扣**
6. 选择支付方式（微信支付 / 积分全额兑换）
7. 点击**「提交订单」**
8. 完成支付后，系统自动生成订单

### 四、查看和管理订单

1. 进入**「我的」**页面
2. 在**「我的订单」**区域，点击对应状态图标快速查看：
   - **待付款** — 未支付的订单，可取消或支付
   - **待收货** — 快递商品已发货，可查看物流或确认收货
   - **待使用** — 到店服务已购买，等待核销
   - **待评价** — 服务/商品已完成，等待评价
   - **退款/售后** — 退款申请进度
3. 点击**「全部订单」**查看完整订单列表

### 五、到店核销

1. 购买到店服务后，在订单详情中查看**核销码**
2. 核销码提供两种形式：
   - **二维码** — 出示给商家扫码
   - **数字码** — 商家手动输入核销
3. **套餐类商品**（如10次理疗套餐）：每次核销一次，订单显示进度"已使用 X/10 次"
4. 全部核销完成后订单进入"待评价"状态

### 六、会员卡

1. 在**「我的」**页面，点击右上角**二维码图标**
2. 进入会员卡页面，展示：
   - 用户头像、用户名、会员号
   - **动态二维码**（每60秒自动刷新）
   - 倒计时提示
3. 到店时出示给商家扫码，商家确认后自动触发**到店签到**，获得积分奖励
4. 网络断连时可点击**「手动刷新」**按钮

### 七、优惠券

1. 在**「我的」**页面，点击统计栏中的**「优惠券」**
2. 查看已领取的优惠券（未使用/已使用/已过期三个Tab）
3. 点击**「领更多券」**进入领券中心
4. 在领券中心浏览可领取的优惠券，点击**「领取」**

### 八、收藏

1. 在商品详情页底部点击**收藏按钮**收藏商品
2. 在**「我的」**页面点击统计栏中的**「收藏」**查看收藏列表
3. 收藏列表分两个Tab：**商品收藏** / **知识收藏**

### 九、地址管理

1. 在**「我的」**页面菜单中点击**「我的地址」**
2. 可新增、编辑、删除收货地址
3. 可设置默认地址
4. 最多保存10个地址

### 十、评价

1. 订单完成后（收到快递商品/完成到店核销），订单进入"待评价"状态
2. 在订单详情中点击**「去评价」**
3. 选择评分（1-5星）
4. 填写文字评价
5. 可上传评价图片
6. 点击**「提交评价」**

### 十一、退款/售后

1. 在订单详情中点击**「申请退款」**
2. 填写退款原因
3. 提交申请后等待管理员审核
4. 审核通过后退款将原路返回
5. 退款规则：
   - 到店服务：未核销可全额退款，已核销不可退
   - 快递商品：参照退货退款流程

## 管理后台操作指南

### 商品分类管理

1. 登录管理后台 → 侧边栏**「商品体系」**→**「商品分类」**
2. 支持两级分类管理（一级健康领域 + 二级具体分类）
3. 可新增、编辑、删除分类
4. 支持设置排序和上下架状态

### 商品管理

1. **「商品体系」**→**「商品管理」**
2. 新增商品时需填写：名称、分类、履约方式、价格、图片、描述、库存等
3. 可配置预约表单字段
4. 可设置积分兑换、推荐权重、待付款超时时间等

### 订单管理

1. **「商品体系」**→**「统一订单」**
2. 按状态筛选查看订单
3. 快递订单：可操作**发货**（填写快递公司+单号）
4. 退款申请：可**通过**或**拒绝**退款

### 优惠券管理

1. **「商品体系」**→**「优惠券管理」**
2. 创建优惠券：选择类型（满减/折扣/代金/免费体验）
3. 配置适用范围（全场/指定分类/指定商品）
4. 可手动发放给指定用户

## 注意事项

1. **登录要求**：购买商品、管理订单、收藏、领券等功能需要先登录
2. **管理后台权限**：商品管理、订单管理等功能需要管理员权限
3. **退款审核**：用户提交退款申请后，需等待有退款处理权限的管理员审核
4. **到店签到**：仅通过商家扫描用户会员码触发，用户不能主动签到

8. **套餐核销**：套餐类商品全部核销完才可评价
