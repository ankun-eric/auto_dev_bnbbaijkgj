## 功能简介

本次升级聚焦于**健康档案模块的体验重构**，让用户一眼识别"当前查看的是谁"、Hero 卡瘦身后突出三大高频入口、并将家庭成员的"邀请/守护/解绑/提醒设置"完整收口到 `/family` 列表页。

### 主要改进点

1. **标题简化**：顶部标题从「我的健康档案」改为「健康档案」，避免与底部导航语义重复。
2. **Hero 卡三入口快捷区**：底部新增 **「今日用药 · 我的设备 · 家庭成员」** 三张小卡，右上角橙色实心圆角标实时显示数量（数量为 0 时角标自动隐藏）。
   - 切换成员 Tab 时，"今日用药""我的设备"两个角标随当前成员实时联动；"家庭成员"角标始终显示用户名下成员总数。
3. **当前成员设备列表页（全新）**：路由 `/devices/member?member_id=xxx`。顶部显示"{成员称呼}的设备 · 共 X 台 · Y 台在线"，每行带在线状态 Pill 胶囊（在线 = 绿底 / 离线 = 灰底）。
4. **家庭成员列表页（激活 + 重做）**：原来跳回健康档案页的 `/family` 改为真正的列表页：
   - **守护状态标签**：已守护 → 浅蓝 Pill；未守护 → 浅橙 Pill；本人不显示标签。
   - **卡片底部操作条**：
     - 本人卡：`编辑 | 提醒设置`
     - 已守护家人：`编辑 | 解绑 | 提醒设置`
     - 未守护家人：`编辑 | 邀请 | 提醒设置`
   - 右上角 `+` 按钮：调起新增成员抽屉。
5. **提醒设置抽屉（全新）**：
   - 显示对方注册手机号（中间打码 `138****5678`）。
   - **AI 外呼提醒**：开关 + 4 档呼叫时机（准时拨打 / 延迟 5/10/15 分钟）。开关关闭时整个时机区置灰禁用，但选中值保留。
   - **超时通知守护者**：3 档触发时长（5/10/15 分钟，默认 5）。本人卡不显示该区块（本人没有守护者）。
   - 底部"查看提醒历史"入口，叠加打开历史抽屉，可返回。
6. **解绑抽屉（全新）**：严肃警告文案 + 短信验证码下发到**守护者本人手机**，6 位数字 + 60 秒倒计时。确认后档案保留、关系标签自动变为"未守护"、按钮自动变为"邀请"。
7. **后端配套**：FamilyMember 表新增 4 个字段（`avatar_color_index`、`ai_call_enabled`、`ai_call_timing`、`guardian_alert_minutes`），新增 7 个 API 端点专门服务于 V2 改造，且**全部 10 个验收测试用例服务器内 pytest 通过**。

## 使用说明

### 一、入口：进入健康档案

1. 在 H5 主页底部导航点击 **「档案」** 或直接访问 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)。
2. 页面顶部标题为 **「健康档案」**（不再含"我的"二字）。

### 二、切换查看的家庭成员

1. 顶部成员切换条横滚展示所有成员。
2. 切换成员后，下方 Hero 卡内容（含三入口角标数量）实时刷新。

### 三、使用 Hero 三入口快捷区

Hero 卡下方有三张小卡：

- **💊 今日用药**：右上橙色圆角标显示**当前成员今日总服用次数**（计划次数，不论吃没吃）。点击进入"今日用药"页。
- **🩺 我的设备**：右上橙色圆角标显示**当前成员名下设备总数**。点击进入新的"设备列表"页面。
- **👨‍👩‍👧 家庭成员**：右上橙色圆角标显示**用户名下成员总数（含本人）**，**不随成员 Tab 切换变化**。点击进入 `/family` 家庭成员列表。

> 💡 数量为 0 时角标自动隐藏。

### 四、家庭成员列表页

1. 进入路径：从健康档案页 Hero 卡的"家庭成员"快捷入口，或直接访问 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family)。
2. **每行卡片**：左侧圆形单字徽章（"我/爸/妈/娃/爱/兄/姐..."）+ 姓名 + 关系 + 右侧守护状态 Pill。
3. **本人永远在第一位**，其余按入档时间升序。
4. **右上角 `+` 按钮**：弹出"新增家庭成员"抽屉（与档案页 Tab 末尾 `+` 调起同一抽屉）。
5. **卡片底部操作条**（按角色不同）：

| 卡片类型 | 按钮组合 |
|----------|----------|
| 本人卡 | `编辑` &#124; `提醒设置` |
| 已守护家人 | `编辑` &#124; `解绑` &#124; `提醒设置` |
| 未守护家人 | `编辑` &#124; `邀请` &#124; `提醒设置` |

### 五、邀请家人加入守护

1. 在未守护成员卡上点击 **「邀请」** 按钮。
2. 自动跳转 `/family-invite?member_id=X` 邀请页（24 小时有效，复用原有页面）。
3. 把二维码或链接发给对方扫码/打开，对方注册并确认即可建立守护关系。

### 六、设置提醒规则

1. 在成员卡底部点击 **「提醒设置」**，从底部弹出抽屉。
2. **对方手机号**：只读展示（中间打码），来自对方注册手机号；若需更换需对方自行修改。
3. **AI 外呼提醒（提醒 TA 打卡）**：
   - 开关：默认关闭。
   - 呼叫时机：准时拨打 / 延迟 5 / 10 / 15 分钟。开关关闭时整组置灰禁用，但选中值保留。
4. **超时通知守护者**：
   - 触发时长：5（默认）/ 10 / 15 分钟。
   - 通道：仅 App / 小程序站内推送，强制接收。
   - 本人卡不显示此区块。
5. 点击底部 **「保存」**，配置即时生效。

> 💡 点击 **「查看提醒历史」** 可叠加打开历史抽屉，按时间倒序列表展示「AI 外呼」「超时通知」的下发记录。顶部返回箭头可回到上层提醒设置抽屉。

### 七、查看当前成员的设备列表

1. 在健康档案页 Hero 卡点击 **「🩺 我的设备」** 快捷入口。
2. 顶部展示 `{成员称呼}的设备 · 共 X 台 · Y 台在线`。
3. 每个设备一行：图标 + 设备名 + 在线状态 Pill + 最近上报时间。
4. **权限规则**：
   - 本人态：可管理（解绑、改名等，详情页操作）。
   - 其他成员态：只读，仅查看名称/型号/在线状态/最近上报。
5. 健康数据**不在列表页展示**，需进入设备详情页才能查看。

### 八、解除守护关系

1. 在已守护家人卡上点击 **「解绑」** 按钮，弹出严肃警告抽屉。
2. 点击 **「获取验证码」**，验证码会发送到**守护者本人的手机**。
3. 输入 6 位验证码，点击 **「确认解绑」**。
4. 解绑成功后：
   - Toast 提示 `已解除与 {成员称呼} 的守护关系`。
   - 该成员卡的标签从 `已守护` 变为 `未守护`，底部 `解绑` 按钮变为 `邀请`。
   - **档案保留**，下次还可重新邀请同一个人。

## 注意事项

1. **首次访问 V2 页面**：后端会自动迁移已有家庭成员的 `avatar_color_index`（按入档顺序循环 5 色），无需手动操作。
2. **未守护成员的提醒设置**：可正常进入并预设规则，但**真正生效需先通过邀请建立守护关系**。
3. **AI 外呼**：当前仅"提醒 TA 打卡"场景生效；具体语音内容与外呼系统配置一致。
4. **超时通知**：仅"App / 小程序站内推送"，不会发送短信或电话。
5. **解绑后**：相关的 AI 外呼定时任务和守护通知会立即关闭；如对方再次接受邀请，规则需重新配置。
6. **本人卡**：底部只有"编辑 / 提醒设置"两个按钮，**没有邀请/解绑入口**；提醒设置抽屉也不显示"超时通知守护者"区块。
7. **守护状态判定**：仅依据 `FamilyMember.member_user_id` 是否非空，不强制要求 `FamilyManagement.status='active'`。
8. **设备在线判定**：当前简化为"最近上报时间距今 ≤ 10 分钟视为在线"。如果设备长时间未上报数据会被显示为"离线"。
