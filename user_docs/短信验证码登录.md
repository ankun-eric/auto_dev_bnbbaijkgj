## 功能简介

本次更新为系统接入了**腾讯云短信服务**，使短信验证码登录功能真正可用。具体改动：

- **小程序端**：登录页面默认展示短信验证码登录（手机号 + 验证码），下方保留「微信一键登录」快捷入口
- **APP 端（安卓/iOS）**：已有的手机号验证码登录界面现已对接腾讯云短信服务，可真正收到短信验证码
- **H5 端**：已有的手机号验证码登录界面现已对接腾讯云短信服务，可真正收到短信验证码
- **三端统一账号**：同一手机号在小程序、APP、H5 任意端登录均为同一账户，数据完全互通

## 使用说明

### 短信验证码登录流程

1. 打开应用，进入登录页面
2. 在手机号输入框中输入 11 位手机号码
3. 点击「获取验证码」按钮
4. 等待手机收到 6 位数字验证码短信（签名：呃唉帮帮网络）
5. 在验证码输入框中填入收到的验证码
6. 勾选「已阅读并同意用户协议和隐私政策」
7. 点击「登录」或「登录/注册」按钮

### 首次登录（自动注册）

- 如果您的手机号从未在系统中注册过，验证码验证通过后将**自动为您创建账户并登录**
- 新用户首次登录后直接进入首页，无需强制填写额外信息
- 您可以随时在「个人中心」完善个人资料

### 小程序端特有功能

小程序端除了短信验证码登录外，还支持**微信一键登录**：
- 在登录页面下方找到「微信一键登录」按钮
- 点击后通过微信授权快速完成登录

## 注意事项

1. **验证码有效期**：每条验证码有效期为 **5 分钟**，请在收到短信后及时填写
2. **发送间隔**：同一手机号 **60 秒内只能发送一次**验证码，请耐心等待倒计时结束后再重新发送
3. **验证码格式**：验证码为 **6 位纯数字**，请确保完整输入
4. **短信来源**：验证码短信签名为「呃唉帮帮网络」，请注意识别
5. **账号统一**：小程序、APP、H5 三端使用同一账号体系，同一手机号登录的是同一个账户
6. **未收到短信**：如果长时间未收到验证码，请检查手机是否设置了短信拦截，或联系客服
