## 功能简介

本次 AI 首页优化（最终版）共做了 **4 处**用户可感知的改动，全部聚焦在「选择咨询人」与「底部输入区」两块：

1. **「选择咨询人」弹窗重构**：标题更直白、选中态高亮统一、右侧文字按钮"选择 / 已选择"一目了然。
2. **输入框 placeholder 动态化**：根据当前选中的咨询人，提示"问答已结合【XX】的健康档案~"，避免误以为 AI 没读档案。
3. **麦克风 / 键盘 圆形蓝底图标**：与「选中咨询人卡片」同款渐变蓝底，视觉锚点统一。
4. **「按住说话」按钮统一为 H5 聊天页同款样式**：天蓝实底 + 白字，整条输入栏外层透明，重心更聚焦。

## 操作步骤

### 一、切换「咨询人」（左下角胶囊）

1. 进入 AI 首页（[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)）。
2. 点击左下角的「为(XX)咨询 ⇆」胶囊。
3. 屏幕底部弹出**「选择咨询人」**弹窗：
   - 顶部标题为「**选择咨询人**」（不再是简略的「咨询人」）。
   - 当前选中的那条卡片为**蓝色渐变深色背景**（包括"本人"在内，统一规则）。
   - 其它未选中条目为**浅灰底**（即便是"本人"，未选中时也是浅灰，不再有"天生蓝底"特权）。
   - 每条卡片左侧主标题显示「**关系 · 姓名**」，例如：`儿子 · 张小明`、`本人 · 李雷`。如果该家庭成员的"关系"字段为空，会兜底为只显示姓名。
4. 卡片右侧的文字按钮：
   - **未选中条目** → 实心**蓝底白字"选择"按钮**，最醒目，点击即切换。
   - **当前选中条目** → 白底蓝字**"已选择"标签**（置灰禁用、无点击反馈，仅作状态指示）。
5. 点击任一未选中条目的"选择"按钮，弹窗关闭，左下角胶囊文案、输入框 placeholder、AI 回答的档案上下文都会同步切换到新咨询人。

### 二、查看输入框新 placeholder

切换不同咨询人后，输入框灰色提示文字（placeholder）会动态变化：

| 当前选中咨询人 | 输入框提示文案 |
|---|---|
| 本人 | `问答已结合【本人】的健康档案~` |
| 儿子 · 张小明 | `问答已结合【儿子】的健康档案~` |
| 女儿 · 张小红 | `问答已结合【女儿】的健康档案~` |
| 老婆 · 王芳 | `问答已结合【老婆】的健康档案~` |
| （关系字段缺失，仅有姓名 `张三`） | `问答已结合【张三】的健康档案~` |

> 这样可以一眼看到 AI 当前依据的是谁的档案，避免误以为没传上下文。

### 三、麦克风 / 键盘 切换

1. 输入框左侧有一个**圆形渐变蓝底按钮**（与"选中咨询人卡片"同款渐变 `#38BDF8 → #0284C7`）。
2. 默认显示**麦克风图标**（白色描边）—— 点击切到"语音输入"模式。
3. 进入语音模式后，同一位置变为**键盘图标**（白色描边）—— 点击切回键盘输入模式。
4. 这两个图标完全复用自 H5 聊天会话页（`/chat`）的官方 SVG，确保整个应用的图标风格一致。

### 四、按住说话（语音模式）

1. 点击麦克风圆底按钮 → 进入语音模式。
2. 输入框区域被替换为一条**「按住说话」**按钮：
   - 背景：**天蓝实底 `#0EA5E9`**
   - 文字：**白色「按住说话」**
   - 高度 40px，圆角 16px
3. 长按按钮即开始录音，向上滑出范围则提示「松开取消」。
4. 录音过程中按钮文案会变化：
   - 正常松开发送 → 显示「松开结束」
   - 滑到取消区域 → 显示「松开取消」（按钮变红色 `#d32f2f`）
5. 整条输入栏的**外层容器无背景色**（透明），沿用页面本身底色 `#F0F9FF`，让"按住说话"蓝色按钮自身成为视觉重心。

## 视觉锚点一致性总结

整个 AI 首页只有**一种主蓝**，所有蓝色元素彼此同源、风格统一：

| 元素 | 颜色 |
|---|---|
| 「选择咨询人」弹窗中选中卡片背景 | `linear-gradient(135deg, #38BDF8 0%, #0284C7 100%)` |
| 输入栏左侧麦克风/键盘 圆形按钮底色 | 同上（与选中卡片同源） |
| 「选择」按钮（未选中态，可点击） | 实心蓝 `#0284C7` |
| 「按住说话」按钮 | `#0EA5E9` 实底 + 白字 |

## 注意事项

1. **本次仅 H5 端有改动**，安卓 App、苹果 App、小程序、管理后台无任何变更，无需重新安装。
2. **必须登录后**才能看到 AI 首页和输入区；未登录会被引导到登录页。
3. **必须先建立"家庭成员"**才能切换咨询人，可在弹窗底部点击「+ 新增家庭成员」直接添加。
4. **placeholder 中的【XX】** 默认取"关系"字段（如：父亲、儿子、老婆）；如果某成员未设置关系，会自动兜底为姓名。
5. **iOS Safari / 微信内置浏览器** 上：首次使用麦克风按钮需手动允许麦克风权限，否则按住说话无法录音。
6. **AI 正在回答**期间，左下角的"切换咨询人"胶囊会变灰禁用并显示 loading 小转圈，此时点击会弹出 Toast 提示，请等回答完成再切换。

## 常见问题

**Q1：为什么点击"已选择"标签没反应？**
A：这是设计行为。"已选择"标签是**当前选中状态的指示**，非操作按钮，置灰禁用是为了和"选择"按钮形成视觉区分。要切换咨询人，请点击其它行的"**选择**"按钮。

**Q2：为什么本人卡片在弹窗里也是浅灰底（未选中时）？**
A：本次设计取消了"本人天生蓝底"的特权，统一规则为：**选中=蓝色渐变 / 未选中=浅灰**。这样能让用户立刻看出"哪个是当前选中的咨询人"，不会被多种深色背景干扰。

**Q3：placeholder 文案为什么有时显示【姓名】而不是【关系】？**
A：当某家庭成员的"关系"字段为空时（例如老数据未填写），placeholder 会**降级显示姓名**作为兜底。在「健康档案 → 家庭成员管理」里补齐关系字段后即可恢复正常。
