## 功能简介

本次修复了 H5 Web 客户端订单详情页中**预约信息卡片的展示位置**问题。

**修复前**：预约信息卡片虽已存在，但其位置不合理——核销码区域显示在商品信息上方，预约信息显示在商品卡片下方，不符合「买了什么 → 约了什么时间去哪里 → 到店凭码」的用户自然阅读流。

**修复后**：调整了订单详情页各卡片的展示顺序，使之符合主流平台（美团、抖音团购、支付宝口碑等）的设计惯例：

1. **商品信息卡片**（展示商品名称、价格、订单编号、支付信息等）
2. **预约信息卡片**（📅 预约时间蓝色高亮 + 📍 预约门店名称）
3. **核销码区域**（二维码 + 核销码 + 使用进度）

## 使用说明

### 查看订单预约信息

1. 打开 H5 Web 客户端：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)
2. 使用手机号 + 验证码登录
3. 点击底部导航栏的「我的」，进入个人中心
4. 点击「我的订单」或对应的订单状态Tab（如「待使用」）
5. 在订单列表中，点击一笔**到店核销类商品**的订单，进入订单详情页
6. 在订单详情页中，依次可看到：
   - **商品信息卡片**：展示商品图片、名称、数量、价格、订单编号、下单时间、支付信息和实付金额
   - **预约信息卡片**（如有预约数据）：
     - 📅 **预约时间**：日期 + 时间段，蓝色高亮显示（如 `2026-04-29 14:00-15:00`）
     - 📍 **预约门店**：门店名称，深色文字显示
   - **核销码区域**：展示核销二维码和数字核销码，可复制核销码

### 预约信息显示规则

- **有预约信息**：商品信息卡片下方会显示一个独立的「预约信息」白色圆角卡片
- **无预约信息**：（如纯物流配送商品）预约信息卡片区域自动隐藏，不显示任何占位提示

## 注意事项

1. **仅限 H5 Web 客户端**：本次修复仅涉及 H5 网页版，Flutter APP 端的订单详情页预约信息展示不受影响（已正常）
2. **预约数据来源**：预约信息展示的是用户下单时选择的预约时间和系统自动匹配的门店，数据由后端接口提供
3. **浏览器兼容性**：支持所有主流移动浏览器（微信内置浏览器、Safari、Chrome 等）
4. **界面自适应**：卡片布局在不同屏幕宽度下会自适应显示
