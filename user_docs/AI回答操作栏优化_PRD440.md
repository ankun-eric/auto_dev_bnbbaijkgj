## 使用说明

### 1. 找到 AI 回答下方的新操作栏

打开任意一条 AI 对话（首页对话或历史会话详情页），在 AI 回答正文的下方，您会看到全新设计的操作栏：

```
┌─────────────────────────────────────────────┐
│                                             │
│   AI 回答正文内容 ......                     │
│   AI 回答正文内容 ......                     │
│                                             │
│                          AI 生成仅供参考     │  ← 11px / 浅灰 / 靠右
│   - - - - - - - - - - - - - - - - - - - -   │  ← 1px 虚线分隔
│   📋 复制    ↗ 转发    🔊 语音播报           │  ← 渐变图标 / 左下一排
│                                             │
└─────────────────────────────────────────────┘
```

- **顶部右侧**：浅灰色文字「AI 生成仅供参考」，提醒用户 AI 内容仅供参考、不作为诊断依据
- **中间分隔**：一条横贯气泡的浅灰色虚线，把提示文字和操作图标分隔开
- **左下一排**：从左到右固定为 **复制 → 转发 → 语音播报** 三个图标，未触发时为浅灰色 `#999`，非常克制不打扰；点击或长按时图标变为蓝紫双色渐变（`#6a8dff → #b07cff`）高亮

### 2. 复制 AI 回答全文

- 点击最左侧的「**复制**」图标
- **Web / 桌面浏览器**：页面顶部居中会弹出「已复制」黑底白字 Toast，1.5 秒后自动消失
- **手机浏览器 / 小程序 / iOS / 安卓 App**：使用各自系统/平台的原生轻提示（小程序为微信 `wx.showToast`，App 为系统 SnackBar），不打扰当前对话流

### 3. 转发 AI 回答

- 点击中间的「**转发**」图标（「三点连线」分享形状）
- 弹出分享面板，**渠道列表与分享行为完全沿用之前的版本**（微信好友 / 朋友圈 / 复制链接 / 生成海报 等）
- **本次升级点**：弹窗内的渠道小图标也一并升级为方案 C 双色渐变风，与操作栏视觉完全统一

### 4. 语音播报 AI 回答

- 点击最右侧的「**语音播报**」图标
- 喇叭由浅灰色变为蓝紫渐变色，并立即开始 TTS 朗读
- **重点**：播报过程中，喇叭周围会出现 **三圈错位向外扩散的 Wi-Fi 弧线**（每秒一圈、每圈错位 333 毫秒），让您一眼就知道当前正在播报
- **想停止？再次点击同一个图标即可立即停止**，喇叭瞬间切回静态浅灰状态（无过渡动画，体感更利落）

## 注意事项

1. **语音播报底层 TTS 引擎沿用现网**：本次仅做前端视觉与交互改造，不影响语音内容、音色与速度
2. **转发弹窗与渠道列表沿用线上**：仅渠道小图标视觉同步升级为渐变风，分享行为完全不变
3. **图标状态切换无过渡动画**：起播 / 播报中 / 停止之间瞬切，体感更直接利落
4. **复制反馈与平台对齐**：Web 顶部 Toast「已复制」（1.5 秒自动消失）；移动端调用各平台系统原生轻提示，避免打断对话
5. **Wi-Fi 扩散动效仅在「播报中」状态可见**：未触发或停止后立即消失
6. **未触发态图标统一为浅灰 `#999`**：是有意设计的"弱提示"，避免操作栏抢占 AI 内容的视觉注意力
