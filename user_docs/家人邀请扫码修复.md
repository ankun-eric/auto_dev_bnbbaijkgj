## 功能简介

本次更新修复了**家人邀请扫码重复扫码**的问题。之前的版本中，被邀请方扫描邀请二维码后会打开摄像头扫码中转页，需要再次扫码才能进入授权页面。修复后，扫描二维码可直接进入授权确认页面，无需重复扫码。

### 修复要点

1. **二维码内容统一修正**：所有端生成的家人邀请二维码统一指向 H5 授权页面
2. **旧二维码兼容处理**：已发出的旧格式二维码仍可正常使用（自动重定向到授权页）
3. **微信环境智能跳转**：在微信中打开授权链接时，自动尝试拉起小程序
4. **多端扫码适配**：APP 内扫码和小程序内扫码都能正确识别并跳转到原生授权页面
5. **授权页面 UI 优化**：确认页简洁化，成功页详细化

## 使用说明

### 邀请方操作流程

1. 打开健康档案管理页面
2. 选择要邀请关联的家人档案
3. 点击「邀请本人关联」按钮
4. 系统自动生成包含邀请二维码的页面
5. 将二维码展示给被邀请方扫描，或点击「复制链接」将邀请链接发送给对方

### 被邀请方操作流程

#### 场景一：微信扫一扫

1. 打开微信，使用「扫一扫」功能扫描邀请二维码
2. 系统自动尝试拉起小程序的授权页面
3. 在授权页面查看邀请人信息
4. 点击「同意」按钮完成授权，或点击「拒绝」取消

#### 场景二：小程序内扫码

1. 在小程序首页点击右上角扫码图标
2. 扫描邀请二维码
3. 直接跳转到小程序原生授权页面（不出小程序）
4. 点击「同意」或「拒绝」

#### 场景三：APP 内扫码

1. 在 APP 首页点击右上角扫码图标
2. 扫描邀请二维码
3. APP 内直接打开原生授权页面（不跳外部浏览器）
4. 点击「同意」或「拒绝」

#### 场景四：手机相机 / 非微信浏览器扫码

1. 使用手机自带相机或第三方扫码工具扫描二维码
2. 自动打开 H5 授权页面
3. 如未登录，先完成手机号登录/注册
4. 登录后自动跳转到授权确认页面
5. 点击「同意」或「拒绝」

### 授权成功后

- 页面显示邀请人头像和姓名
- 显示关系说明："XXX 已成为你的家人"
- 可点击「打开小程序查看」跳转到小程序查看详情
- 或点击「稍后再说」关闭页面

## 注意事项

1. **已发出的旧二维码仍可使用**：之前生成的邀请二维码（指向 `/scan` 路径）仍然有效，系统会自动重定向到正确的授权页面
2. **邀请码有效期**：邀请码有一定有效期，过期后扫码会提示"该邀请已过期"
3. **一次邀请只能使用一次**：同一邀请码被接受或拒绝后将不可再次使用
4. **微信环境自动跳转**：在微信中打开授权链接时，系统会自动尝试拉起小程序。如果 3 秒内未成功跳转，将自动降级展示 H5 网页版授权页面
5. **未登录用户**：
   - 在微信小程序中：走微信快捷登录流程
   - 在 H5 网页中：需先完成手机号登录/注册
   - 在 APP 中：需先完成 APP 登录
