## 功能简介

在 AI 对话功能中，你可以把重要的历史对话「置顶」，让它一直显示在历史列表最上方，方便随时找到。

本次修复解决了一个老问题：

- **修复前**：在历史记录里置顶一条对话，看起来成功了，但只要切换到别的页面、刷新页面或重新进入 App，置顶就会**消失**（每次必现）。「取消置顶」也有同样的问题。
- **修复后**：置顶 / 取消置顶都会**真正保存到服务器**，无论怎么切换页面、刷新、重进 App，状态都会保留。万一遇到网络异常没保存成功，会给出「置顶失败，请重试」的提示，并把界面恢复成操作前的样子，不再「假装成功」误导你。

> 「删除对话」功能本来就是正常的，本次不受影响，仍可正常删除。

## 使用说明

### 一、置顶一条历史对话

1. 打开 **AI 对话首页**：[点此进入](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)
2. 点击页面**左上角的「三横杠」按钮**，打开左侧「历史记录」抽屉。
3. 在历史列表中找到想要置顶的对话。
4. 长按该对话，或点击其右侧的操作菜单，选择「**置顶**」。
5. 该对话会立即移动到列表最上方的「已置顶」分组，并提示「已置顶」。

### 二、验证置顶已真正保存

完成置顶后，可用以下任一方式验证（修复后置顶都应仍然存在）：

1. **关闭并重新打开侧边栏**：再次点开左上角「三横杠」，置顶对话仍在最上方。
2. **刷新页面 / 重进 App**：刷新浏览器或重新进入，置顶仍在。
3. **跳转其他功能页再返回**：先去「健康档案」「会员中心」等页面，再回到对话首页打开历史记录，置顶仍在。

### 三、取消置顶

1. 在历史记录中找到已置顶的对话（位于「已置顶」分组）。
2. 长按或打开操作菜单，选择「**取消置顶**」。
3. 该对话会回到普通分组，并提示「已取消置顶」。
4. 同样地，切换页面 / 刷新 / 重进后，取消置顶的状态也会正确保留。

### 四、置顶失败时的表现（网络异常场景）

- 如果因网络等原因没有保存成功，会弹出轻提示「**置顶失败，请重试**」（取消置顶则提示「取消置顶失败，请重试」）。
- 同时该对话的界面会**恢复成操作前的状态**，不会出现「看起来成功、实际没保存」的情况。此时稍后重试即可。

- 置顶有数量上限，达到上限时会提示「最多置顶 N 条对话」，需先取消部分置顶再操作。
- 历史记录中展示的是已归档的历史对话，正在进行中的当前对话不出现在历史列表里。
- 本次修复仅涉及 **H5 网页端**，已通过服务器自动部署生效，**无需下载或安装任何客户端**，直接用浏览器打开上面的链接即可体验。
