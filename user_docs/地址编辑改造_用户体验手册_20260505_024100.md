## 四、操作步骤（H5）

### 步骤 1：进入「我的地址」

打开 [我的地址](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/my-addresses)（需先登录）。

### 步骤 2：新增地址

1. 点击右上角「+ 新增」按钮。
2. 输入收货人姓名（≤20 字）和 11 位手机号（前端正则校验）。
3. 点击「所在地区」字段 → 底部弹出 `CascadePicker` 三列滚轮 → 滑动选择省 / 市 / 区，点「确认」。
4. 输入详细地址（≤80 字）。
5. 选择标签：家 / 公司 / 自定义（自定义 ≤6 个汉字）。
6. 切换「设为默认地址」。
7. 点击「保存」→ 后端 `POST /api/v2/user/addresses`，若坐标空则后端 2 秒内尝试高德 geocoding 兜底。

### 步骤 3：编辑 / 删除 / 设为默认

- 列表卡片右下方有「编辑」、「删除」、「设为默认」操作按钮。
- 默认地址自动置顶且显示红色「默认」Tag。
- 删除为软删除（`is_deleted=1`），不影响下游历史订单引用。

## 五、操作步骤（小程序）

1. 进入小程序「我」→「我的地址」。
2. 点击「+ 新增收货地址」。
3. 三种填写姿势：
   - **手动**：点击「所在地区」唤起底部 picker-view 三列联动。
   - **从微信导入**：点击右上角「微信导入」按钮，授权后自动回填收货人 / 手机 / 省市县 / 详细地址；旧版微信无 chooseAddress 时降级到 chooseLocation 选点 + 后端逆地理编码。
   - **混合**：导入后仍可继续修改任一字段。
4. 选择标签 → 设为默认 → 保存。

## 六、操作步骤（App）

1. 我的 → 我的地址 → 「+ 新增地址」。
2. 点击「📍 使用当前位置」：APP 申请定位权限 → `Geolocator.getCurrentPosition` 获取 GPS → 后端 `/api/v2/regions/reverse-geocode` 自动回填省市县 + 详细地址 + 经纬度。
3. 点「所在地区」也可手动滚轮选择。
4. 编辑标签、设为默认、保存。

## 七、关键技术与设计

1. **后端 v2 接口**：
   - `GET /api/v2/regions` — 行政区划数据（version + provinces[].cities[].districts[]）
   - `GET/POST/PUT/DELETE /api/v2/user/addresses` — 地址 CRUD
   - `PATCH /api/v2/user/addresses/{id}/default` — 设为默认
   - `POST /api/v2/regions/reverse-geocode` — 经纬度逆地理编码（高德兜底）
   - `GET /api/v2/app/version-check?platform=android|ios&currentVersion=x.y.z` — 版本检查

2. **数据库迁移**：
   - 启动期 lifespan 自动 `ALTER TABLE user_addresses ADD COLUMN ...`，加 10 个新列：`consignee_name`、`consignee_phone`、`province_code`、`city_code`、`district_code`、`detail`、`longitude`、`latitude`、`tag`、`is_deleted`。
   - 旧字段 `name / phone / street / province / city / district` 全部放宽为 `NULL`，保证下游 `unified_orders.shipping_address_id` / `service_address_id` 引用不破。
   - 一次性脚本 `backend/scripts/migrate_addresses_v2.py`：批量拆老地址（基于 regions.json 字符串匹配）。
   - 编辑时懒迁移：v2 API 收到 PUT 后自动补齐新字段。

3. **三端行政区划数据来源统一**：
   - 后端：`backend/app/data/regions.json`
   - H5：`h5-web/public/regions.json`（运行时通过 `resolveAssetUrl`，含 `BASE_PATH` 前缀）
   - 小程序：`miniprogram/data/regions.json`
   - Flutter：`flutter_app/assets/data/regions.json`（`pubspec.yaml` 已注册）
   - 完整版后续以版本发布形式更新；当前 MVP 覆盖 31 省 + 主要地级市 + 区县（直辖市三级展示）。

4. **隐私与安全**：
   - 列表展示手机号自动脱敏 `138****1234`。
   - 经纬度仅在后端持久化用于推荐与导航，不上报第三方。
   - 高德 Key 找不到时不阻断保存，仅日志告警。
