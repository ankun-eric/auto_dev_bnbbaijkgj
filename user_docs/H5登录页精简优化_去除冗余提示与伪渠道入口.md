## 功能简介

本次更新对 **H5 客户端登录页** 进行了视觉与交互精简，去除了两处冗余/无效元素，让登录流程更聚焦：

1. **删除登录卡片下方的灰色提示行**：「未注册手机号验证后将自动创建会员」、「当前未开放自助注册，仅支持已注册账号登录」这类文案不再展示
2. **删除登录页底部「其他登录方式」整块**：包含原微信渠道卡片以及点击后弹出的「暂不支持/请走手机号验证码」提示框
3. **保留邀请码识别行**：URL 携带 `?ref=xxx` 时，登录卡片中的绿色「🎉 已识别邀请码：xxx」提示仍按原逻辑显示

优化后的登录页只保留 **手机号 → 验证码 → 登录** 这一条主线，整体更紧凑、更聚焦。

## 优化前后对比

### 优化前

- 手机号 / 验证码输入框下方有一行灰色小字提示
- 登录按钮、协议勾选下方还有一个「其他登录方式」区块，包含微信卡片
- 点击微信卡片只会弹「暂未开放/请走手机号验证码」之类的提示框，给人「这是个坏功能」的体验

### 优化后

- 输入框下方无任何灰色提示行（无占位空白，登录按钮自然向上贴近）
- 登录卡片下方就是页面自然底部，无任何「其他登录方式」相关元素
- 只保留 LOGO、欢迎语、手机号输入、验证码输入、登录按钮、用户协议勾选 这 6 个核心元素
- 邀请码识别行（带 🎉 emoji 的绿色提示）仅在 URL 带 `ref` 参数时按原逻辑显示

## 操作步骤

1. **打开 H5 登录页**：在浏览器或微信中打开 [H5 登录页链接](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/login)
2. **输入手机号**：在「请输入手机号」输入框中填入 11 位手机号
3. **获取验证码**：点击右侧「获取验证码」按钮，系统下发短信，按钮进入 60 秒倒计时
4. **输入验证码**：在「请输入验证码」输入框中填入收到的 4~6 位验证码
5. **勾选协议**：勾选页面底部「我已阅读并同意《用户服务协议》和《隐私政策》」
6. **点击登录/注册**：点击绿色渐变圆角按钮（按钮文案根据后台开关显示「登录 / 注册」或「登录」）
7. **登录成功**：自动跳转到首页或健康指引页（根据是否新用户/是否已完善资料决定）

> 💡 如果 URL 中带有 `?ref=xxx`，验证码输入框下方会绿色高亮显示「🎉 已识别邀请码：xxx」，登录后会自动绑定推荐人。

## 注意事项

1. **登录核心流程不受影响**：手机号 + 短信验证码 + 协议勾选 + 登录这一条主路径完全不变，原有的图形验证码、短信风控、协议拦截逻辑都保持原样
2. **后端配置不受影响**：后台「自助注册开关」「微信注册策略」等业务能力本身依然有效，只是前端不再展示对应的灰色文案。如果未注册手机号尝试登录而后台关闭了自助注册，仍会通过接口返回的错误提示告知用户
3. **邀请码逻辑完全保留**：URL 中带 `?ref=xxx` 仍会被识别并随注册流程绑定推荐关系
4. **微信内打开**：微信浏览器中也不再展示微信渠道入口，所有用户统一走「手机号 + 验证码」登录通路
5. **横屏/平板**：横屏模式下登录卡片居中显示，下方无任何「其他登录方式」残留
6. **缓存清理**：如果旧版页面有缓存，请下拉刷新或清空浏览器缓存后再访问，即可看到新版精简登录页
