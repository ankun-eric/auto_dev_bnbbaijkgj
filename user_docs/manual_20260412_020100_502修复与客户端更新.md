## 功能简介

本次更新主要修复了以下问题：

1. **502 Bad Gateway 错误修复**：修复了通过正式域名 `newbb.bangbangvip.com` 访问项目时出现的 502 错误。根本原因是前端代理服务器的 nginx 配置直连测试服务器的 Docker 内部端口（8000/3000/3001），而这些端口未对外暴露。修复方案改为通过测试服务器的 Gateway Nginx（HTTPS 443）转发。

2. **客户端 API 地址修正**：统一将各端（Flutter、小程序、H5 Web）的 API 基础地址从测试服务器直连地址 `newbb.test.bangbangvip.com` 修正为正式域名 `newbb.bangbangvip.com`，确保客户端请求走正确的代理链路。

3. **静态文件下载修复**：修复了小程序 zip 文件通过正式域名无法下载（404）的问题，将 gateway 配置中 zip 文件的路由从本地文件系统改为反向代理到测试服务器。
