## 功能简介

本次操作完成了域名 `newbb.test.bangbangvip.com` 的 **SSL 安全证书安装**，将原有的自签名证书替换为 **Let's Encrypt** 签发的正规 SSL 证书，消除了浏览器访问时的安全告警。

### 变更内容

- **证书替换**：将自签名证书（CN=101.33.205.59）替换为 Let's Encrypt 签发的正规 SSL 证书（CN=newbb.test.bangbangvip.com）
- **HTTPS 安全连接**：采用 TLSv1.2 / TLSv1.3 协议，高强度加密算法，确保数据传输安全
- **自动续期**：配置了 certbot 自动续期机制和 renewal hook 脚本，证书到期前会自动续签，无需人工干预
- **HTTP → HTTPS 引导**：访问 HTTP 链接时会自动引导至 HTTPS 安全链接

## 使用说明

### 对终端用户的影响

1. **无需任何操作**：证书安装对终端用户完全透明，用户无需做任何配置或操作
2. **浏览器安全标识**：访问项目 URL 时，浏览器地址栏会显示 🔒 锁形图标，表示连接安全
3. **不再有告警**：之前访问时出现的"您的连接不是私密连接"或"此站点不安全"等告警已消除

用户可通过以下步骤验证证书是否生效：

1. 打开浏览器（Chrome / Firefox / Edge / Safari 等）
2. 访问 [https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
3. 查看浏览器地址栏，确认显示 🔒 锁形安全图标
4. 点击锁形图标可查看证书详情：
   - **颁发给**：newbb.test.bangbangvip.com
   - **颁发者**：Let's Encrypt (E8)

## 注意事项

1. **证书有效期**：Let's Encrypt 证书有效期为 90 天，系统已配置自动续期（certbot 定时任务），无需人工操作
2. **HTTP 访问**：通过 HTTP（非加密）方式访问项目路径时，系统仍然可以正常访问。如有特殊安全需求，可进一步配置强制 HTTPS 重定向
3. **证书续期 hook**：当证书自动续期时，系统会自动将新证书复制到 gateway-nginx 容器并重新加载 Nginx 配置，全程无需停机
4. **兼容性**：支持 TLSv1.2 和 TLSv1.3 协议，兼容所有主流浏览器和移动设备
