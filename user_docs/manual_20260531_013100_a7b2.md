## 功能简介

本次更新对四个健康指标（**血压 / 血糖 / 心率 / 血氧**）的卡片与详情页进行了统一改造，建立了一套可复用的「健康指标详情模板」，新增血氧（SpO₂）模块，并引入 AI 智能解读能力。

### 本次升级亮点

1. **新增血氧（SpO₂）模块**：与血压/血糖/心率并列，支持完整的录入、趋势、历史、AI 解读功能
2. **历史记录分流**：详情页只显示最近 5 条，右上角「全部 ›」入口跳转到独立的「全部历史页」
3. **全部历史页支持 4 项筛选**：日期范围 / 状态档位 / 测量场景 / 数据来源
4. **AI 解读全覆盖**：四指标全部支持「🤖 解读本次」与「📈 解读 7/30/90 天趋势」
5. **设备同步记录保护**：来自智能设备的记录不能改、不能删，确保原始数据完整性
6. **删除二次确认弹窗**：删除前回显记录关键信息（日期/时间/数值/状态），避免误删

## 使用说明

### 1. 进入指标详情页

打开 App / H5 后，从【健康档案】或【健康看板】点击对应指标卡片：
- 血压 → 血压详情页
- 血糖 → 血糖详情页
- 心率 → 心率详情页
- 血氧 → 血氧详情页（**新增**，位置排在睡眠卡片前）

### 2. 详情页布局（四指标统一）

从上到下依次为：
1. **主数值大卡**：最新一次测量数据 + 状态胶囊（如「正常」「轻度偏高」）
2. **大按钮区**：左侧「手工录入」（实心主色），右侧「绑定设备」（描边）
3. **趋势图**：支持 7 天 / 30 天 / 90 天切换
4. **🤖 AI 解读区**：
   - 「🤖 解读本次」：对最新一条数据进行解读
   - 「📈 解读 N 天趋势」：对当前选中的时间范围做趋势分析（N 跟随时间 Tab 动态变化）
5. **历史记录**：
   - 只显示最近 **5 条**
   - 右上角「全部 ›」入口跳转到全部历史页
   - 每行展示：时间 + 数值 + 单位 + 状态胶囊 + 场景标签 + 来源标签

### 3. 查看全部历史

点击详情页历史记录右上角「**全部 ›**」按钮，进入全部历史页：

- **顶部筛选区**（点击切换）：
  - 📅 日期范围：近 7 天 / 近 30 天 / 近 90 天 / 全部
  - 🩺 状态档位：偏低 / 正常 / 轻度偏高 / 中度偏高 / 严重偏高（按指标定义）
  - 📍 测量场景：晨起 / 睡前 / 空腹 / 餐后 2h 等
  - 📡 数据来源：手工录入 / 智能设备同步
- **顶部计数**：「共 X 条」，已筛选时显示「· 已筛选」
- **下拉滚动加载更多**：单次加载 20 条
- **排序规则**：按测量时间严格倒序

### 4. 修改记录（仅手工录入）

- **点击行** → 弹出编辑弹窗 → 修改完毕保存
- 仅手工录入的记录可以修改
- 修改字段（按指标不同）：
  - 血压：高压 / 低压 / 心率 / 测量时间 / 场景 / 备注
  - 血糖：血糖值 / 测量场景 / 时间 / 备注
  - 心率：心率值 / 时间 / 场景 / 备注
  - 血氧：血氧值 / 时间 / 场景 / 备注

### 5. 删除记录（仅手工录入）

- **左滑行** → 右侧滑出红色「删除」按钮
- 点击「删除」 → 弹出二次确认弹窗，回显记录关键信息：
  ```
  确认删除这条记录？
  📅 2026-05-30  08:30
  🩺 血压 125/82 mmHg（轻度偏高）
  ⚠️ 删除后无法恢复
  ```
- 点击「确认删除」（红色）完成删除

### 6. 设备同步记录（只读）

- 设备同步的记录**不能修改、不能删除**
- 点击设备记录 → 弹出只读详情弹窗，明确提示：
  > 🔒 该记录来自智能设备，数据已锁定。如需修正，请重新测量。
- 左滑设备记录 → 不出现删除按钮

### 7. AI 智能解读

在详情页的 AI 解读区：

- **🤖 解读本次{指标名}**：基于最新一次测量数据，给出状态判断、原因分析、就医建议
- **📈 解读 N 天趋势**：基于当前时间范围内的全部记录，给出平均值、波动幅度、达标率等总结
- 点击后弹出底部抽屉，AI 输出文案 5 秒内完成
- 抽屉底部提示：「⚠️ AI 建议仅供参考，不能替代医生诊断」

### 8. 血氧（SpO₂）模块（新增）

- 状态档位（参考医学标准）：
  - **正常**：≥ 95%（蓝色）
  - **偏低**：90% – 94%（黄色）
  - **较低**：85% – 89%（橙色）
  - **严重偏低**：< 85%（红色）
- 测量场景：静息 / 运动后 / 睡眠中 / 自定义
- 在首页中位置：**睡眠卡片之前**

## 注意事项

1. **历史数据完全兼容**：原有血压/血糖/心率的历史数据无任何丢失，可直接在新页面上查看
2. **设备同步记录无法修改**：如果数据有误，请重新测量录入（手工录入的覆盖会优先）
3. **AI 解读不能替代医生诊断**：所有 AI 建议都是参考性的，重大健康决策请咨询专业医生
4. **删除操作不可恢复**：删除后无法找回，务必通过二次确认弹窗仔细核对
5. **血氧需要兼容设备**：手工录入随时可用；设备同步需绑定支持血氧的智能手表 / 指夹式血氧仪
6. **筛选条件持久化**：在全部历史页切换筛选不会刷新页面，便于探索数据
