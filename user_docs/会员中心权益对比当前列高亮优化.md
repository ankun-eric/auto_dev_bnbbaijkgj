## 一、功能简介

在「会员中心 → 权益对比」表格里，系统会标出你**当前已开通的付费会员等级**所在的那一列。

**本次优化解决的问题：** 旧版用一个浮在表头上方、向外顶出的小气泡来标「当前」，结果被外层白色卡片的边框挡住了一部分，导致「当前」两个字显示不完整，既不美观也不清楚。

**本次采用的方案（紫金高级）：**

| 元素 | 优化后的表现 |
| --- | --- |
| **「当前」角标** | 改为**金色角标**（淡金底 + 金色描边 + 金色文字），**内嵌在列头格子里、贴着会员名称文字旁边**，不再浮出顶部，文字完整显示、不被任何边框遮挡 |
| **当前列底色** | 你所在的那一整列加上**淡紫色**底色高亮 |
| **列头顶部横条** | 当前列列头顶部加一条**紫色**横条，强化“这一整列是我的”视觉指引 |
| **整体气质** | 紫底配金标，与 App 现有紫色主题协调，显得高级、清晰、不突兀 |

## 二、操作步骤

1. 打开 [会员中心页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center)（如未登录，请先登录账号）。
2. 在会员中心页面中**向下滚动**，找到「权益对比」表格区域。
3. 观察表格列头：
   - 如果你**已开通某个付费会员**（如「尊享会员」），你所在的那一列会显示：
     - 列头上贴着会员名称旁的**金色「当前」角标**；
     - 整列带**淡紫色底色**；
     - 列头顶部有一条**紫色横条**。
   - 如果你**尚未开通任何付费会员**（免费用户），则全表所有列都不显示「当前」角标，也不做高亮，保持原始对比表样式。
4. 不同会员等级（如基础/尊享等）切换后，「当前」角标和高亮会自动落到你对应的那一列。

---

## 三、注意事项

- 本次仅为**展示样式优化**，不改变权益对比的内容、行列结构，也不涉及会员购买、支付等任何业务逻辑。
- 「当前」列的判定由系统根据你**实际开通的会员等级**自动完成，无需手动设置。
- 免费用户（未开通付费会员）看到的对比表不会有任何「当前」角标与高亮，这是正常表现。
- 在不同手机机型 / 不同屏幕宽度下，「当前」角标均完整显示在列头格子内部，不会溢出或被遮挡；若内容较宽，表格支持左右横向滑动查看。
- 小程序端的会员中心是通过内嵌网页加载本 H5 页面，因此本次优化在小程序中同样自动生效，无需更新小程序。
