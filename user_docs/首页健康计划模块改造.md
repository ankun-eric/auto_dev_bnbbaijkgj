## 功能简介

本次更新对 bini-health 健康管理应用的**首页健康计划模块**进行了全面改造，主要包括：

1. **三大健康计划分类体系**：用药提醒、健康打卡、自定义计划，覆盖用户主要健康管理场景
2. **首页今日待办汇总**：将所有进行中的健康任务汇总展示，支持快速打卡
3. **用药提醒**：设置每日用药时间和药品，到时推送提醒，支持多时段管理
4. **健康打卡**：自定义每日健康习惯打卡项，支持数值目标和提醒
5. **自定义计划**：基于 7 种预设模板（饮食、作息、康复、心理健康、戒断、健身、其它），支持管理员推荐 + AI 智能定制 + 用户自定义
6. **AI 个性化计划生成**：结合用户健康档案自动生成专属健康计划
7. **打卡统计**：今日进度、连续打卡天数、周/月完成率趋势图、各计划完成率排行
8. **后台管理**：推荐计划管理、模板分类管理、通用任务配置、用户打卡数据统计

## 使用说明

### 一、首页今日待办

1. 打开应用，进入首页
2. 在页面中部可以看到**「今日待办」**模块，显示当天所有待完成的健康任务
3. 待办按三大分类分组展示：
   - 💊 **用药提醒**：显示当天需要服用的药品
   - ✅ **健康打卡**：显示自定义的健康习惯打卡项
   - 📋 **自定义计划**：按模板类型（饮食、健身等）细分展示
4. **快速打卡**：
   - 无目标值的任务：点击左侧圆圈即可完成打卡
   - 有目标值的任务：点击后弹出输入框，填写实际完成值后提交
5. 已完成的任务会显示为打勾 + 置灰状态
6. 当天无待办任务的分类仍然显示但会置灰，标注"今日无待办"
7. 底部点击**「查看全部计划」**可进入健康计划主页

### 二、健康计划主页

1. 从首页点击「查看全部计划」或通过导航进入
2. 页面顶部有**「AI 为我生成计划」**按钮，点击后 AI 会根据你的健康档案生成个性化健康计划
3. 下方是三大分类卡片入口：
   - **用药提醒**：管理每日用药
   - **健康打卡**：管理健康习惯
   - **自定义计划**：浏览模板、创建计划
4. 每个卡片显示当前分类下的任务/计划数量
5. 底部可点击**「查看统计」**进入打卡统计页面

### 三、用药提醒

1. 进入用药提醒页面，点击右上角**「+ 添加」**按钮
2. 填写药品信息：
   - **药品名称**（必填）
   - **用药时段**（多选：早晨、中午、晚上、睡前）
   - **每个时段的提醒时间**（如 08:00、12:30）
   - **用药剂量**（选填，如"1片"）
   - **备注**（选填，如"饭后服用"）
3. 保存后，用药提醒按时段分组展示
4. 到达提醒时间后，App 端会收到推送通知
5. 可以对每条用药进行**编辑、删除、暂停/恢复**操作
6. 在用药提醒列表中可直接点击**打卡确认**

### 四、健康打卡

1. 进入健康打卡页面，点击**「+ 添加打卡项」**
2. 填写打卡项信息：
   - **打卡名称**（必填，如"喝水""冥想"）
   - **目标值**（选填，如 2000）
   - **目标单位**（选填，如"ml""分钟"）
   - **提醒时间**（选填）
   - **重复频率**（每天/工作日/自定义）
3. 保存后在列表页展示
4. 有目标值的打卡项会显示进度条
5. 打卡时可输入实际完成的数值

### 五、自定义计划

1. 进入自定义计划页面，展示 7 种模板分类卡片
2. 点击任意分类（如"健身计划"），进入模板详情页
3. 模板详情页从上到下依次为：
   - **「AI 为我定制」**按钮 —— AI 根据你的健康档案生成该分类下的个性化计划
   - **推荐计划列表** —— 管理员配置的推荐计划，可浏览并加入执行
   - **我的计划** —— 你已创建或加入的计划，显示执行进度
   - **「创建新计划」**按钮
4. 加入计划后，该计划的每日任务自动出现在首页「今日待办」中

### 六、AI 个性化计划

1. 首次完善健康档案后，系统会自动为你生成一份个性化健康计划
2. 在健康计划主页点击**「AI 为我生成计划」**可随时重新生成
3. 在每个模板分类下点击**「AI 为我定制」**可生成该分类专属的计划
4. 如果健康档案信息不完整，系统会提示先完善档案

### 七、打卡统计

1. 从首页或健康计划主页进入统计页面
2. 顶部显示**今日完成进度**环形图和**连续打卡天数**
3. 中部展示**周/月完成率趋势图**（可切换周视图和月视图）
4. 底部展示**各计划完成率排行**

### 八、后台管理（管理员）

管理员登录后台后，可进行以下管理操作：

1. **推荐计划管理**：为每种模板分类配置推荐计划，支持增删改查、上下架
2. **模板分类管理**：管理预设的计划模板分类，可增删改排序
3. **通用任务配置**：配置新用户首页默认展示的健康任务
4. **打卡数据统计**：查看所有用户的打卡统计和明细

## 注意事项

1. **健康档案**：使用 AI 生成计划功能前，请先完善个人健康档案（身高、体重、性别、生日等）
2. **通知权限**：使用用药提醒功能时，需授权 App 通知权限才能收到推送提醒
3. **小程序订阅消息**：微信小程序端需授权订阅消息才能收到用药提醒
4. **数据安全**：用药数据属于敏感健康信息，请勿在公共设备上登录
5. **打卡规则**：
   - 每个任务每天只能打卡一次（有目标值的可多次更新数值）
   - 连续打卡天数：当天完成至少 1 项任务即算打卡成功，中断 1 天即重置
6. **管理员权限**：推荐计划管理、模板分类管理等后台功能需要管理员账号登录
7. **H5 端**：H5 网页端不支持推送通知，仅做页面内任务展示
