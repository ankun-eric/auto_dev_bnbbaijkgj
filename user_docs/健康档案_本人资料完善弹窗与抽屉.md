## 一、功能介绍

为了让"用药提醒、报告解读、AI 问诊"等健康服务给您提供更精准的结果，系统会在您首次进入「健康档案」时，引导您完善**姓名、性别、出生日期**这三项基础资料。

本功能包含两部分：

1. **完善弹窗**：进入健康档案页（本人 Tab）后，若三项基础资料任一为空，会自动弹出。
2. **完善抽屉**：点击弹窗「去完善」按钮后，从屏幕底部滑出的轻量编辑面板，让您一步完成补全。

完成后，本人 Tab 的标签会自动从「本人」变成您填写的真实姓名（例如「张三」）。

## 三、操作步骤

### 步骤 1 · 进入健康档案

打开上方入口地址（H5 用户请直接点击链接 <https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/> ），默认会停留在「本人」Tab。

### 步骤 2 · 弹窗自动出现

页面加载完毕后 **延迟 0.5 秒**，若系统检测到您还没填写姓名、性别或出生日期中的任意一项，会弹出如下提示：

- **标题**：完善健康档案
- **正文**：为了给您提供更精准的健康服务，请先完善您的基本资料（姓名、性别、出生日期）。
- **主按钮**：去完善
- **次按钮**：稍后

> 同一次进入页面只弹一次。即使您来回切换「家庭成员 / 守护对象」Tab 再切回本人 Tab，也不会重复打扰。

### 步骤 3 · 点击「去完善」打开抽屉

抽屉从屏幕底部向上滑出，标题为 **「完善健康档案」**，内容如下：

1. **必填项（默认展开 + 红色星号 *）**
   - **姓名 ***：长度 1–20，不能为「本人」占位文案
   - **性别 ***：男 / 女
   - **出生日期 ***：不晚于今天，不早于 1900-01-01
2. **更多资料（默认折叠）**
   - 头像、手机号、身高、体重、慢病史、过敏史等
   - 点击「展开更多资料」可显示，全部为选填，不影响保存按钮状态

如果三项必填没填齐，**「保存」按钮会保持灰色不可点击**，且未填项下方会显示红字提示：
- 「请填写姓名」
- 「请选择性别」
- 「请选择出生日期」

### 步骤 4 · 保存

三项必填都填好后，「保存」按钮自动变蓝；点击后会：

1. 屏幕中央 Toast 提示「**保存成功**」
2. 抽屉自动关闭
3. 页面上方的「本人」Tab 标签自动更新为您填写的姓名（如「张三」）
4. **下次再进入健康档案不会再弹窗**

### 步骤 5 · 想稍后再填？

- 在弹窗上点「**稍后**」，弹窗关闭，您可以继续浏览。
- 在抽屉里点右上角「**×**」、下滑、或点击半透明遮罩，都会直接关闭抽屉且不保存。
- **下次进入健康档案页仍会弹出**，直到您完成补全为止。

## 六、注意事项

1. **关于「本人」占位**：未填写姓名时，本人 Tab 默认显示「本人」。系统会把「本人」当成空名处理，仍会触发弹窗。请填写您**真实的姓名**。
2. **出生日期范围**：必须在 `1900-01-01` 到「今天」之间。
3. **不允许部分保存**：三项必填必须**同时填齐**才能保存。若您只想补充其中一项，请先把其他两项也填上再保存。
4. **关闭抽屉不保存任何资料**：未点击「保存」之前的输入都不会留存。
5. **多端同步**：在任一端保存成功后，您在其他端再次进入「健康档案」，会自动同步最新姓名/性别/生日，且不再弹窗。
6. **隐私保护**：填写的资料仅用于您本人的健康服务，不会对外泄露。

## 七、常见问题（FAQ）

**Q1：我点了「稍后」，下次还会弹吗？**  
A：会。只要您还没把姓名/性别/出生日期补齐，下次进入「健康档案」就会再次弹出。这是为了保证健康服务质量，没有"今日不再提醒"。

**Q2：我之前已经填过姓名（不是「本人」），但还没填性别，是否会弹？**  
A：会。只要三项中有任意一项为空，就会触发弹窗。

**Q3：抽屉里的"更多资料"我必须填吗？**  
A：不必。展开后的所有字段都是选填，不影响保存按钮的可点状态。建议补全以获得更精准的健康服务。

**Q4：在家庭成员 Tab 也会弹这个吗？**  
A：不会。弹窗只在「本人 Tab」激活时触发。

**Q5：保存后多久能看到「本人」Tab 变成我的真实姓名？**  
A：立即。保存成功后会自动刷新档案数据，Tab 名同步更新。
