# 健康档案页面修复与邀请页面统一 — 用户体验手册

## 一、功能概述

本次更新对健康档案相关页面进行了全面优化，包括图标优化、数据显示修正、表单布局改善、邀请页面统一等多项改进，让您的健康管理体验更加顺畅。

---

|------|------|------|

## 三、功能详细说明与操作步骤

### 3.1 我守护的人入口优化

**操作路径**：打开健康档案首页

**变化说明**：
- 页面底部"我守护的人"入口图标由盾牌改为家庭三人头像，更直观地表达家庭管理含义
- 点击后仍然跳转到家庭成员列表

**访问地址**：[健康档案首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)

---

### 3.2 健康看板优化

**操作路径**：健康档案首页 → 健康看板

**变化说明**：
- 健康评分模块已暂时移除，因为现有评分规则缺乏权威依据
- 移除后页面其他模块自然填充，不会留空白
- 后续有权威评分标准时会重新上线

**访问地址**：[健康看板](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-dashboard)

---

### 3.3 用药计划优化

**操作路径**：健康档案首页 → 用药计划

**变化说明**：
1. **统计数字更简洁**：用药计划旁的数量显示去掉了括号，如原来的 `(2)` 现在显示为 `2`
2. **分类更准确**：
   - **服药中**：正在按计划服用的药品（已开始且未结束）
   - **未开始**：已创建但开始日期在未来的计划
   - **已结束**：已归档或已过期的计划
   - 长期用药的计划只要状态正常，始终显示在"服药中"

**访问地址**：[用药计划全部](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/medication-plans)

### 3.5 添加成员弹窗优化

**操作路径**：
- 路径一：我守护的人 → 右上角 "+" 按钮
- 路径二：AI 首页 → 选择咨询人 → 新建

**变化说明**：
1. **标题更简洁**：由"添加家庭成员"改为"添加成员"
2. **表单更紧凑**：每个字段的标签和输入框在同一行显示，节省空间
3. **出生日期更智能**：
   - 选择"爸爸"时，默认出生年份为您出生年份减25年
   - 选择"妈妈"时，默认出生年份为您出生年份减23年
   - 选择"儿子/女儿"时，默认出生年份为您出生年份加25年
   - 默认值仅供参考，您可以自由修改

**操作步骤**：
1. 点击"+"按钮打开添加成员弹窗
2. 输入成员名字
3. 选择性别
4. 选择与您的关系（系统会自动推荐出生日期）
5. 如需修改出生日期，点击日期选择器手动调整
6. 点击保存

### 3.8 邀请页面统一（重点更新）

**操作路径**：
- 邀请被守护人：健康档案首页 → 邀请横幅 → 去邀请
- 邀请守护我：守护我的人页面 → 邀请

**变化说明**：
两个邀请页面现在采用统一的卡片式设计：

#### 邀请 TA 成为被守护人
- **主题色**：蓝色渐变
- **功能标签**：📋 档案管理 · 💊 用药提醒 · 🔔 异常提醒
- **二维码**：居中展示，扫码即可接受邀请
- **绑定提示**：
  - 如果是从已有成员发起邀请，显示「将绑定到"XX"的档案」
  - 如果是新建邀请，显示「对方确认后将为 TA 新建档案」

#### 邀请 TA 守护我的健康
- **主题色**：绿色渐变
- **功能标签**：📋 档案管理 · 💊 用药提醒 · 🔔 异常提醒
- **固定提示**：「让 TA 随时关注我的健康」

**分享方式**（两个页面统一）：
1. **保存到本地**：将二维码保存为图片
2. **复制链接**：复制邀请链接到剪贴板
3. **转发微信好友**：通过微信分享给好友

**访问地址**：
- [邀请被守护人](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family-invite)
- [邀请守护我](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/my-guardians/invite)

**操作步骤**：
1. 进入邀请页面，系统自动生成邀请二维码
2. 选择分享方式：保存图片 / 复制链接 / 微信转发
3. 将邀请信息发送给对方
4. 对方扫码或点击链接后确认接受
5. 邀请有效期为 24 小时，过期需重新生成

### 3.9 邀请绑定与数据合并

**变化说明**：
- **已有档案绑定**：如果您已经为家人创建了本地档案（Tab），邀请成功后该档案会自动升级为与对方账号关联的档案
- **新建档案**：如果从"添加成员"入口发起邀请，对方确认后会自动创建一个新档案
- **数据合并**：绑定成功后，您之前代为录入的数据会与对方本人的数据自动合并
  - 只有一方录入的数据，直接保留
  - 双方同一天都录入了数据，以对方本人录入的为准（本人测量更准确）
  - 合并过程自动完成，无需手动操作

## 四、注意事项

1. **邀请有效期**：所有邀请链接/二维码有效期为 24 小时，过期后需要重新生成
2. **数据共享**：建立守护关系后，双方操作的是同一份数据，您代录入的数据对方也能看到
3. **出生日期默认值**：系统推荐的出生日期仅供参考，请根据实际情况调整
4. **小屏手机**：如果您使用较小屏幕的手机，编辑弹窗会自动调整为更紧凑的布局
5. **用药计划分类**：长期用药的计划只要状态正常，会一直显示在"服药中"分类中

## 五、常见问题

**Q：为什么健康评分消失了？**
A：因为现有评分规则缺乏临床权威依据，我们暂时移除了该功能。等有更科学的评分标准后会重新上线。

**Q：邀请链接过期了怎么办？**
A：重新进入邀请页面，系统会自动生成新的邀请二维码和链接。

**Q：对方已经是我的被守护人了，还能再邀请吗？**
A：系统会提示"TA 已是您守护的人"，无需重复邀请。

**Q：数据合并后，之前我代录入的数据还在吗？**
A：在的。如果对方在同一天也有录入，会优先保留对方本人的数据；其他日期的数据都会完整保留。
