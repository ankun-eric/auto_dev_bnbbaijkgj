## 功能简介

「数字安全绳」是面向独居老人的轻量守护功能：每天点一下"我今天平安"按钮即可完成签到；当连续 24 / 48 小时未签到时，系统自动向您预先设置的紧急联系人发送站内信预警，让远在天边的子女、邻居第一时间得知。

本轮 Bug 修复涉及 **4 个核心问题**：

1. **关怀模式入口缺失** — 现已新增绿色横卡 + 右下角"安全绳"悬浮球，紧贴 SOS 红球，双入口可见
2. **阈值选中态无视觉反馈** — 24/48 小时按钮选中时变成绿色填充 + 加粗 + 右上角白底绿勾 ✓
3. **紧急联系人添加后列表无记录** — 这是最严重的 Bug。**根因是后端原本在保存联系人时同步调用 SMTP 发送邮件，导致整个请求阻塞 → 网关超时 → 数据库事务被回滚**。本次彻底删除邮件通道，改用「手机号 + 站内信」方案：联系人必须填写 11 位手机号，且必须是 bini-health 已注册用户，保存后被设为联系人的那位 TA 会收到一条站内信"您被 XXX 设为紧急联系人"
4. **签到后顶部文案不刷新** — 签到成功后立即重新拉取状态，顶部 banner 立即变成"✅ 上次签到 MM-DD HH:mm"+"⏰ 下次签到截止 MM-DD HH:mm"两行

## 使用说明

### 操作流程一：进入数字安全绳功能页

**方式 A（推荐）**：

1. 用浏览器打开关怀模式首页（[care-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-home)）
2. 注意右下角，会看到两个圆形悬浮球紧贴着：
   - 🟢 **绿色"安全绳"球** — 点击即进入数字安全绳功能页
   - 🔴 **红色"SOS"球** — 紧急时长按 3 秒求救
3. 也可以下滑到页面里看到大块绿色"数字安全绳"横卡，同样点击进入

**方式 B**：直接打开 [care-safety-rope](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-safety-rope) 链接

### 操作流程二：每日签到（"我今天平安"）

1. 进入数字安全绳功能页
2. 点击中央大绿色按钮"我今天平安 ✋"
3. 浏览器会询问是否获取位置（建议允许，以便预警时让联系人看到最后位置）
4. 签到完成后，**顶部 banner 立即更新**：
   - ✅ 上次签到：MM-DD HH:mm
   - ⏰ 下次签到截止：MM-DD HH:mm

### 操作流程三：调整守护阈值

1. 在功能页中下滑找到"守护设置 → 超时阈值"
2. 24 小时 / 48 小时两个按钮二选一
3. **选中的那一项会变成绿色填充 + 加粗 + 右上角白底绿勾 ✓**，未选中的则是浅灰色
4. 点击即生效，无需"保存"

### 操作流程四：添加紧急联系人（核心修复）

1. 在功能页中下滑找到"紧急联系人 (n/3)"卡片
2. 点击右上角"+ 添加"按钮
3. 在弹出的表单中填写：
   - **姓名**（必填）— 如：张儿子
   - **手机号**（必填）— 11 位手机号，**必须是已经注册了 bini-health 的用户的手机号**
     - 输入满 11 位后系统会自动校验：
       - ✅ 绿色边框 + "✓ 已识别用户：张三" → 该手机号已注册，保存按钮可点
       - ❌ 红色边框 + "该手机号还未注册 bini-health，请先邀请 TA 注册" → 保存按钮置灰
   - **关系**（选填）— 7 个芯片单选：子女 / 配偶 / 父母 / 邻居 / 朋友 / 护工 / 其他
4. 点击"保存"
5. **联系人立即出现在下方列表中**（显示圆头像 + 姓名 + 关系标签 + 脱敏手机号 138****8888）
6. 被设为联系人的那位 TA 会同时收到一条站内信通知

> ⚠️ 表单中**不再有邮箱字段** — 国内场景不需要邮件，统一走站内信。

### 操作流程五：管理 SOS 紧急联系人

1. 在 [SOS 求助页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home/sos) 右上角点击"管理 ›"
2. 填写姓名 + 选择关系（**同样是 7 芯片单选：子女 / 配偶 / 父母 / 邻居 / 朋友 / 护工 / 其他**）+ 电话
3. 点击"+ 添加联系人"

## 注意事项

1. **首次进入需登录** — 数字安全绳功能要求登录后使用（未登录会跳转登录页）
2. **联系人手机号必须是注册用户** — 如果 TA 还没注册，请先邀请 TA 在 [首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/) 完成手机号注册
3. **联系人最多 3 位** — 超过会提示"紧急联系人最多 3 位"
4. **预警通知方式** — 不再发送邮件，统一走站内信，被通知的联系人需在 App 内的"消息中心"查看
5. **暂停守护** — 出差或长期外出时可点击"暂停守护"，可选 3 天 / 7 天 / 14 天 / 无限期；暂停期间不会触发预警
6. **位置授权** — 签到时建议允许位置，预警发出时联系人可以看到您的最后签到位置
7. **若提示"保存失败：未登录"** — 请刷新浏览器后重新登录
