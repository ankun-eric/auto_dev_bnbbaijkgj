## 功能简介

本次更新为「心率详情页」定义了清晰的**参考范围**与**状态判断**展示规则，让您一眼就能看懂自己当前的心率是正常、偏快还是偏慢。

主要变化：

1. **固定显示参考范围那行字**：页面恒定展示一行参考说明 —— **正常范围 60–100 次/分**（对所有用户统一显示，不区分成人 / 老年人）。
2. **新增状态标签**：根据「最近一次心率测量值」自动判断并展示 **正常 / 偏快 / 偏慢** 三档状态。
3. **无数据友好处理**：当本次测量没有心率数据时（例如只测了血压、没测心率），数值位置显示 `--`，**不显示**状态标签，并给出引导文案提示您去测量心率。

---

## 状态判断规则

以「最近一次心率测量值」为准，按下表三档判断：

| 心率测量值（次/分） | 状态标签 |
| --- | --- |
| 低于 60（< 60） | **偏慢** |
| 60 ～ 100（含 60 和 100） | **正常** |
| 高于 100（> 100） | **偏快** |

> 说明：边界值 60 和 100 都算「正常」。

## 操作步骤

1. **进入心率详情页**：在 H5 应用中，进入「健康档案 / 健康指标」，点击「心率」卡片，或直接访问上方「心率详情页」链接。
2. **查看当前值与状态**：
   - 页面顶部「当前值」卡片会显示您最近一次的心率数值（单位 bpm）。
   - 如果有数据，数值右侧会出现一个**状态胶囊**：
     - 蓝色「✓ 正常」：心率在 60～100 次/分之间；
     - 橙色「⚠ 偏快」：心率高于 100 次/分；
     - 橙色「⚠ 偏慢」：心率低于 60 次/分。
3. **查看参考范围**：当前值卡片底部固定显示一行参考说明「📋 正常范围 60–100 次/分」，作为您判断心率的依据。
4. **无数据时的引导**：如果当前没有心率记录，数值会显示「--」，状态标签不出现，并提示「暂无数据，点击下方『手工填写』录入心率」。此时您可点击下方「✎ 手工填写」录入一条心率数据，或「⌚ 绑定设备」自动同步。
5. **录入后查看效果**：录入一条心率后返回详情页，即可看到对应的数值与状态标签。

---

## 注意事项

- **统一标准、不分年龄**：参考范围 60–100 次/分 对所有用户一致显示，不随年龄变化，因此即使您未填写出生日期也能正常判断。
- **状态以最近一次测量为准**：状态标签反映的是您「最近一次」的心率测量值，不代表整体趋势。如需了解趋势，可查看页面中的「7 天趋势」与「AI 解读」。
- **仅供健康参考**：本规则采用医院体检中最通用的成人安静心率正常区间，状态标签仅作健康参考，**不能替代医生诊断**。如有持续偏快 / 偏慢或不适，请及时就医。
- **静息与运动的差异**：剧烈运动后心率自然升高属正常现象，建议在安静状态下测量以获得更具参考意义的结果。
