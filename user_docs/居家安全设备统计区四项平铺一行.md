## 功能简介

本次更新对「**健康档案 → 居家安全设备**」页面顶部的设备统计区进行了布局优化：

- **改造前**：统计区分为两行——上面一行是一个大的「已绑定总数」，下面一行并排显示 3 个分类（紧急呼叫器 / 烟雾报警器 / 水浸报警器），整体占用两行高度。
- **改造后**：把 **4 个统计项平铺到同一行**，依次为「已绑定总数、紧急呼叫器、烟雾报警器、水浸报警器」，数字在上、名称在下，项与项之间用一条浅灰竖线分隔，整体压缩为一行高度，**界面更紧凑、信息一目了然**。

四个统计数字分别保留对应的彩色标识，方便快速区分：

| 统计项 | 数字颜色 |
|--------|----------|
| 已绑定总数 | 蓝色 |
| 紧急呼叫器 | 红色 |
| 烟雾报警器 | 橙色 |
| 水浸报警器 | 黄色 |

> 说明：本次为**纯界面布局优化**，不改变任何数据含义与统计口径，也不影响设备绑定、设备列表、报警记录等其它功能。

## 使用说明

1. **打开页面**：在浏览器中访问 [项目首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/) 并登录，进入「健康档案」，点击「居家安全设备」入口；或直接打开 [居家安全设备页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/home-safety)。
2. **查看统计区**：页面顶部即为设备统计卡片。此时可看到 4 个统计项在**同一行**平铺：
   - 第 1 项「已绑定总数」（蓝色）——显示该成员名下**所有类型**设备的总数；
   - 第 2 项「紧急呼叫器」（红色）——紧急呼叫器数量；
   - 第 3 项「烟雾报警器」（橙色）——烟雾报警器数量；
   - 第 4 项「水浸报警器」（黄色）——水浸报警器数量。
3. **切换家庭成员**：在成员 Tab 间切换时，统计区的 4 个数字会自动刷新为当前所选成员的数据。
4. **窄屏适配**：在小屏手机等较窄的屏幕上，4 个统计项**始终保持一行不换行**，数字与文字会自动等比缩小以适应屏幕，不会变形或挤压。
5. **空数据提示**：当所选成员还没有绑定任何设备时，4 个数字均显示 `0`，统计区下方会出现「暂无设备，快去添加吧」的引导提示，点击下方的「+ 添加设备」按钮即可开始绑定。

## 注意事项

- **「已绑定总数」可能大于三类之和**：总数统计的是该成员名下**所有类型**的设备（包括门磁、燃气等未在统计区单独列出的类型）。因此当用户绑定了这些未单列的设备时，「已绑定总数」会大于「紧急呼叫器 + 烟雾报警器 + 水浸报警器」三类之和，这是**正常且正确**的，请勿误认为数据错误。
- **数据来源未变**：四个数字均来自后端真实统计字段，本次仅调整了展示布局，统计逻辑与口径完全保持不变。
- **名称口径**：统计区中第四项显示为「水浸报警器」，与系统及后端的正式名称保持一致（参考设计稿中曾写作「水位报警器」，为避免多端文案不一致，统一显示「水浸报警器」）。
- **其它功能不受影响**：本次改造仅涉及统计区布局，下方的「添加设备」按钮、设备分组列表、绑定/编辑弹窗、报警记录等功能均保持原样。
- **建议刷新页面**：如果你在更新前已打开过该页面，建议刷新一次浏览器以加载最新的界面布局。
