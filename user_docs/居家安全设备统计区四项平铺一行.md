## 一、功能简介

本次对「居家安全设备」页面顶部的设备统计区做了**布局压缩优化**：

- 改造前：上面一行显示「已绑定总数」大数字，下面再用一行显示三类设备分类计数，**共占两行**。
- 改造后：把「已绑定总数 + 三类设备」共 **4 项平铺在同一行**，数字在上、名称在下，项与项之间用浅灰竖线分隔，**整体高度从两行压缩为一行**，更紧凑、更省空间。

四项依次为：**已绑定总数、紧急呼叫器、烟雾报警器、水浸报警器**，并各自保留专属配色：

| 统计项 | 数字颜色 |
|--------|----------|
| 已绑定总数 | 蓝色 `#1F8FE6` |
| 紧急呼叫器 | 红色 `#E53935` |
| 烟雾报警器 | 橙色 `#FB8C00` |
| 水浸报警器 | 黄色 `#FBC02D` |

> 本次为**纯布局调整，未改动任何数据逻辑**，四项数字的取值与之前完全一致。

## 二、操作步骤

1. 打开 [居家安全设备页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/home-safety) 并登录。
2. 进入页面后，**顶部即为统计卡片区**，现在以「四项平铺一行」的形式展示：
   - 第一项「已绑定总数」（蓝色）：当前所选家庭成员名下绑定的**全部设备总数**。
   - 第二项「紧急呼叫器」（红色）、第三项「烟雾报警器」（橙色）、第四项「水浸报警器」（黄色）：各类型设备的数量。
3. 切换顶部家庭成员 Tab，统计区四项数字会随当前成员实时刷新。
4. 若当前成员尚未绑定任何设备，四项均显示 `0`，并在统计卡片下方显示「暂无设备，快去添加吧」的引导文案。

---

## 三、注意事项

1. **「已绑定总数」可能大于三类之和**：总数统计了该成员名下的**所有类型设备**（含门磁、燃气等未单列类型）。因此当用户绑定了未单列的设备时，「已绑定总数」会大于「紧急呼叫器 + 烟雾报警器 + 水浸报警器」之和，这是**正常且正确**的数据表现，并非计算错误。
2. **窄屏强制一行**：无论手机屏幕多窄，四项始终保持在同一行不换行；空间不足时数字与文字会自动等比缩小，不会挤压变形。
3. **文案口径**：第四项按现有系统口径显示为「**水浸报警器**」（参考稿中曾写作「水位报警器」，为与后端及其它页面文案保持一致，统一采用「水浸报警器」）。
4. **其它区域不变**：统计区下方的「添加设备」按钮、设备分组列表、绑定弹窗、报警记录等功能全部保持原样，本次未做任何改动。
