## 功能简介

本次更新把 **H5 网页端的关怀模式首页（宾尼小康）** 补齐到 V4 版本，并优化了紧急呼叫 SOS 页的"分享我的位置"入口。主要变化：

1. **顶栏与标准模式一致**：左上角菜单、右上角「🎁 邀请」和「⋯ 更多」，移除了原来会报错的「⊕ 圆圈加号」。
2. **模式胶囊文案更新**：顶部胶囊显示「宾尼小康 模式切换」，点击可切回标准模式。
3. **欢迎语更新**：改为「我是宾尼小康，聊聊健康问题吧~」。
4. **用药提醒直达本人用药页**：点击「用药提醒」卡片，直接进入本人独立的用药提醒列表（不再跳老的健康档案锚点）。
5. **居家安全文案更新**：标题「居家安全」，副标题「紧急呼叫、烟雾报警、水浸报警」。
6. **新增右下角悬浮 SOS**：红色圆形 + 扩散光圈动画，一点即进入紧急呼叫流程。
7. **「分享我的位置」挪到 SOS 页最底部**：从原来的「管理」弹窗里挪出来，放到 SOS 页最底部、联系人列表下方，长辈一眼可见、直接点；原管理弹窗里的旧入口已删除（避免双入口迷糊）。

> 首页主体排版（蓝绿大卡、竖排功能卡片）保持原样，未做改动。

## 使用说明

### 一、进入关怀模式首页

1. 用**手机浏览器**打开关怀首页链接（见上方「访问链接」表格）。
2. 页面顶部即为顶栏：
   - 左上角「☰」：进入个人中心 / 历史。
   - 中间标题：「宾尼小康」。
   - 右侧依次为：「宾尼小康 模式切换」胶囊、「🎁」邀请、「⋯」更多菜单。
3. 顶栏下方是蓝绿色欢迎区，显示时段问候、欢迎语「我是宾尼小康，聊聊健康问题吧~」以及今日用药提醒。

### 二、切换回标准模式

1. 点击顶部「宾尼小康 模式切换」胶囊。
2. 在弹出的下拉中选择「标准模式」，即可切回标准模式首页。

### 三、查看本人用药提醒

1. 在首页功能卡片中点击「用药提醒」。
2. 直接进入本人独立的用药提醒列表页，查看今日完整用药安排与打卡。

### 四、居家安全

1. 在首页功能卡片中点击「居家安全」。
2. 进入居家安全相关页面（紧急呼叫、烟雾报警、水浸报警等）。

### 五、紧急呼叫 SOS（两种入口）

- **入口一（悬浮按钮）**：首页右下角的红色「SOS」悬浮按钮（带扩散光圈动画），点击直接进入 SOS 页。
- **入口二（功能卡片）**：首页「紧急呼叫」卡片，点击进入 SOS 页。

在 SOS 页可以：
1. **长按 3 秒红色 SOS 大按钮**触发求助（进度环转满 3 秒才发出，防误触）。
2. 查看**定位状态条**（绿色「定位已就绪 · 具体中文地址」）。
3. 查看并一键拨打**紧急联系人**。
4. 一键拨打 **120 / 110**。
5. 页面**最底部**点击「📍 分享我的位置（微信发给好友）」分享当前静态位置。

### 六、维护紧急联系人 / 家庭住址

1. 在 SOS 页「紧急联系人」区域右上角点击「管理 ›」。
2. 在弹出的维护面板中：
   - **家庭住址必填**：住址为空时点保存会提示「请先填写家庭住址」，填写后才能保存。
   - **新建联系人电话必填**：电话为空时会提示「请填写联系人电话」，填写后才能添加。
3. 维护面板中**不再有**分享入口，分享请到 SOS 页最底部操作。

### 七、分享我的位置（给微信好友）

1. 在 SOS 页**最底部**点击「📍 分享我的位置（微信发给好友）」。
2. 系统会生成一个当前**静态位置**的分享链接：
   - 在微信内打开时，会触发「转发给好友」。
   - 在普通浏览器中，会调起系统分享或自动复制链接，可粘贴发送给微信好友。
3. 好友点开链接后，可看到「地图（中文地址）+ 精简信息卡」。

## 注意事项

1. 本次更新为 **H5 网页端**，用手机浏览器打开即可体验，无需下载安装任何 App。
2. **定位与分享**需要浏览器授予定位权限；若拒绝授权，定位条会显示「定位失败，请手动告知位置」。
3. **拨打电话**功能依赖手机的拨号能力，在桌面浏览器可能无法直接拨出。
4. 「分享我的位置」分享的是点击当下的**静态位置**，不是实时位置。
5. 家庭住址、联系人电话均为**必填项**，未填写无法保存，请按提示补全。
6. 如果页面显示仍是旧版本，请清除浏览器缓存或强制刷新后重试。
