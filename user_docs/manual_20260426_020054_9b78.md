## 功能简介

本次更新对**后台与 H5 登录页的图形验证码**进行了视觉规格升级，目的是解决之前用户反馈的"验证码字符极淡、几乎看不清"问题。

升级前用户在登录页看到的验证码呈现：

- 字符颜色极淡（淡灰色），几乎与背景融合
- 字符偏小，4 位字符要凑近才能勉强辨认
- 噪点和曲线干扰过强，比字符还显眼

升级后（v1.5 规格）：

- 字符**深色**（HSL 亮度 25%~45%、饱和度 40%~80%）
- 字符**放大**（字号 38px → 48px，占画布高度约 65%~70%）
- 字符**加 1px 同色系深色描边**，在任意背景下保持高对比度
- 干扰元素（曲线 + 噪点）**全部锁浅色**，退到背景层
- 整体观感：**字符强、干扰弱**，主次明确，一眼可辨

业务规则**完全不变**：

- 4 位字符（数字 2-9 + 大写字母去 OIL）
- 不区分大小写
- 5 分钟有效
- 一次性使用
- 同 IP/同手机号 5 分钟内 5 次失败 → 锁定 10 分钟

## 使用说明

### 1. PC 后台登录页验证码

1. **打开登录页**：在浏览器中访问 [PC 后台登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/login)
2. **观察验证码**：登录表单右侧（账号/密码下方）的图形验证码区域，会显示一张 160×60 像素的图片，包含 4 位深色字符
3. **辨认字符**：字符为深色（黑、深蓝、深紫、深绿等深色调），加 1px 描边，肉眼可一眼看清
4. **输入验证码**：将看到的 4 位字符输入到验证码输入框（不区分大小写）
5. **看不清时刷新**：点击验证码图片本身或旁边的"换一张"按钮，可立即刷新一张新的验证码
6. **提交登录**：填写好账号、密码、验证码后，点击「登录」按钮

### 2. H5 移动端登录页验证码

1. **手机浏览器打开**：在手机浏览器中访问 [H5 商家登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/m/login)
2. **观察验证码**：H5 登录页中验证码图片宽度按屏幕宽度 50% 等比缩放（120~220px），字符随之等比放大
3. **辨认字符**：升级后字符尺寸足够大、颜色足够深，手机上一眼可辨
4. **输入验证码**：将看到的 4 位字符输入到验证码输入框
5. **看不清时点击图片**：直接点击图片即可刷新

### 3. 商家 PC 登录页验证码

操作流程与 PC 后台登录页一致，访问 [商家 PC 登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/login) 即可。

## 验收要点（如何确认升级成功）

请按以下步骤验证升级生效：

1. **使用无痕窗口（隐身窗口）打开登录页**：避免浏览器缓存影响（Chrome：Ctrl+Shift+N；Edge：Ctrl+Shift+P；Safari：Command+Shift+N）
2. **观察验证码**：应能立刻看到 4 位**深色、清晰、占画布 65% 以上**的字符
3. **连续刷新 10 次**：每一次刷新都应满足上述视觉规格，不会出现淡灰色字符
4. **PC 与 H5 双端比对**：PC 后台与 H5 登录页都应呈现一致的清晰风格

如果验证码仍呈现淡灰、字符极小的旧版样子，请：

- 强制刷新页面（Ctrl+F5 / Command+Shift+R）
- 或更换无痕窗口重试
- 或清空浏览器缓存后重试

## 注意事项

| 项目 | 说明 |
|------|------|
| 字符大小写 | 验证码不区分大小写，输入小写字母也能通过 |
| 有效时间 | 单张验证码 5 分钟有效，超时需刷新 |
| 一次性使用 | 验证码提交（无论成功与否）后立即失效，下次登录需重新获取 |
| 失败锁定 | 同 IP 或同手机号 5 分钟内连续输错 5 次密码 → 自动锁定 10 分钟 |
| 限频保护 | 验证码生成接口每个 IP 每秒最多 5 次请求，超频会返回 429 错误 |
| 浏览器缓存 | 升级后如果仍看到旧版样式，请强制刷新或使用无痕窗口 |
| 部署范围 | 本次仅部署到测试服 `newbb.test.bangbangvip.com`，正式服待用户主动决定后再同步 |
