## 功能简介

本次修复了 AI 对话首页（ai-home）中**语音输入**和**TTS 语音播报**两项功能的 4 个 Bug，确保这两项功能在微信内置浏览器和各主流移动端浏览器中能够正常工作。

### 修复内容

| Bug | 严重程度 | 修复说明 |
|-----|----------|----------|
| 语音识别调用了错误的 API | 致命 | 录音完成后现在正确调用语音识别接口，而非 TTS 合成接口 |
| 录音格式检测和参数缺失 | 严重 | 自动检测浏览器支持的最佳音频格式，并传递完整的格式参数 |
| 麦克风权限拒绝后无引导 | 体验优化 | 新增权限预检流程，权限被拒绝时给出友好的引导提示 |
| TTS 播报不读取配置 | 严重 | 播报前先读取后台 TTS 配置，根据配置选择云端或本地播报方案 |

## 使用说明

### 一、语音输入功能

#### 操作步骤

1. **打开 AI 对话首页**：访问上方链接，进入 AI 健康助手页面
2. **点击麦克风图标**：在输入框左侧找到 🎤 麦克风按钮，点击后切换到语音输入模式
   - 首次使用时，会弹出授权确认弹窗，点击「去授权」后允许浏览器使用麦克风
   - 如果之前拒绝过权限，会提示您去系统设置中重新开启
3. **长按录音**：按住「按住说话」按钮开始录音
   - 录音时会显示声波动画，反映您的说话音量
   - **上滑取消**：录音过程中手指上滑超过一定距离可以取消本次录音
4. **松手发送**：松开按钮后，系统自动将语音识别为文字并发送给 AI
   - 识别过程中会显示「语音识别中...」提示
   - 识别完成后文字会自动发送，无需手动操作
5. **切回键盘**：点击左侧 ⌨️ 键盘按钮可切换回文字输入模式

### 二、TTS 语音播报功能

#### 操作步骤

1. **发送消息并等待 AI 回复**：在 AI 对话首页中输入问题并发送
2. **找到播报按钮**：在 AI 最新回复消息下方，可以看到「🔊 播报」按钮
3. **点击播报**：点击按钮，AI 回复的文字内容将被语音朗读出来
   - 系统会根据后台配置自动选择最佳播报方案（云端高质量语音或本地浏览器语音）
   - 微信浏览器中会优先使用云端语音方案，确保播报效果
4. **停止播报**：播报过程中按钮会变为「⏹ 停止播报」，点击即可停止

## 注意事项

1. **浏览器兼容性**
   - 推荐在微信内置浏览器、Chrome、Safari 中使用
   - 语音输入功能需要浏览器支持 MediaRecorder API
   - 首次使用语音功能需要授权麦克风权限

2. **麦克风权限**
   - iOS 用户：前往「设置 → Safari → 麦克风」或「设置 → 微信 → 麦克风」开启权限
   - Android 用户：前往「设置 → 应用管理 → 浏览器/微信 → 权限 → 麦克风」开启权限
   - 如果误点了「拒绝」，需要到系统设置中手动重新开启

3. **录音注意**
   - 录音时间不能太短（少于 1 秒），否则会提示录音时间太短
   - 请在安静环境中录音，以获得更好的识别效果
   - 录音最长支持 30 秒

4. **TTS 播报注意**
   - 播报音量取决于设备当前的媒体音量设置
   - 在微信浏览器中，建议确保微信的通知声音已开启
   - 播报过程中可随时点击停止
