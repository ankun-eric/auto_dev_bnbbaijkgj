## 功能简介

本次新增「底部导航Tab后台可配置」功能，使管理员可以在管理后台灵活配置 H5 端底部导航栏中间的 Tab 项（增删改查、排序、显隐控制），无需开发人员修改代码上线。

**核心能力：**
- "首页"和"我的"两个 Tab 始终固定在首尾，不可编辑/删除/隐藏
- 管理员可新增最多 3 个可配置 Tab（加上首页和我的，最多 5 个）
- 支持配置 Tab 的名称、图标、跳转路径、显示/隐藏状态
- 支持拖拽排序（可配置项之间）
- 配置修改后实时生效，用户刷新 H5 页面即可看到最新导航

## 使用说明

### 一、管理后台——底部导航配置

#### 1. 进入配置页面

1. 登录管理后台：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单中找到「首页配置」分组
3. 点击「底部导航配置」菜单项

#### 2. 查看导航项列表

进入页面后，可以看到当前所有底部 Tab 配置列表，包含：

| 列 | 说明 |
|----|------|
| 排序 | 上移/下移按钮（仅可配置项可操作） |
| Tab名称 | 显示在底部导航栏的文字 |
| 图标 | Emoji 预览 |
| 跳转路径 | 站内页面路径 |
| 显示状态 | 开关控制显示/隐藏 |
| 类型 | "固定"（蓝色标签）或"可配置"（绿色标签） |
| 操作 | 编辑、删除（仅可配置项有按钮） |

系统默认初始化 4 个 Tab：
- **首页**（固定）—— 始终在第一位
- **AI健康咨询**（可配置）
- **服务**（可配置）
- **我的**（固定）—— 始终在最后一位

#### 3. 新增导航项

1. 点击列表上方的「新增导航项」按钮
2. 在弹窗中填写：
   - **Tab名称**：最多 6 个字符，将显示在底部导航栏
   - **图标**：从预设图标库中选择一个图标
   - **跳转路径**：下拉选择常用页面路径，或手动输入自定义路径
   - **显示状态**：默认开启，关闭后该 Tab 在 H5 端隐藏
3. 点击「确定」保存

**预设页面路径：**

| 页面名称 | 路径 |
|----------|------|
| AI健康咨询 | /ai |
| 服务列表 | /services |
| 我的订单 | /orders |
| 健康档案 | /health-profile |
| 积分商城 | /points-mall |

**预设图标库：**

| 图标 | 名称 | 适用场景 |
|------|------|----------|
| 💬 | 咨询 | AI健康咨询 |
| 🏥 | 服务 | 服务列表 |
| 📋 | 订单 | 我的订单 |
| 📁 | 档案 | 健康档案 |
| 🛒 | 商城 | 积分商城 |
| ❤️ | 健康 | 健康相关 |
| 📊 | 报告 | 健康报告 |
| 🔔 | 消息 | 消息通知 |

> 注意：可配置 Tab 数量上限为 3 个。达到上限时，「新增导航项」按钮将置灰并提示"最多可配置3个导航项"。

#### 4. 编辑导航项

1. 在列表中找到需要编辑的可配置 Tab
2. 点击该行的「编辑」按钮
3. 修改名称、图标、路径或显示状态
4. 点击「确定」保存

> 注意：固定项（首页、我的）无法编辑。

#### 5. 调整排序

- 点击可配置 Tab 行的「上移」或「下移」按钮调整顺序
- 排序修改后自动保存
- 首页始终在第一位，我的始终在最后一位，不参与排序

#### 6. 删除导航项

1. 点击可配置 Tab 行的「删除」按钮
2. 在确认弹窗中点击「确认」
3. 删除后 H5 端立即生效

> 注意：固定项（首页、我的）无法删除。

#### 7. 控制显示/隐藏

- 通过可配置 Tab 行的显示状态开关，可以快速控制 Tab 在 H5 端是否显示
- 隐藏的 Tab 不会在 H5 底部导航栏中渲染
- 固定项的显示状态不可更改

### 二、H5 端——底部导航栏效果

H5 端的底部导航栏会根据管理后台的配置动态渲染：

| 场景 | 底部导航栏显示 |
|------|----------------|
| 3个可配置Tab全部显示 | 首页 \| Tab1 \| Tab2 \| Tab3 \| 我的（5个均分） |
| 隐藏了1个 | 首页 \| Tab1 \| Tab2 \| 我的（4个均分） |
| 隐藏了2个 | 首页 \| Tab1 \| 我的（3个均分） |
| 全部隐藏或删除 | 首页 \| 我的（2个均分） |

**体验步骤：**
1. 打开 H5 页面：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 查看底部导航栏，可以看到按后台配置显示的 Tab 项
3. 在管理后台修改配置后，刷新 H5 页面即可看到最新效果

## 注意事项

1. **权限要求**：只有管理员账号才能在管理后台配置底部导航
2. **实时生效**：配置修改后无需重新部署，用户刷新 H5 页面即可看到最新导航
3. **缓存机制**：H5 端对导航配置有 5 分钟的本地缓存，若希望立即看到效果可清除浏览器缓存后刷新
4. **数量限制**：可配置 Tab 数量上限为 3 个（加上首页和我的，最多 5 个 Tab）；下限为 0 个（底部只显示首页和我的）
5. **路径有效性**：管理员配置的跳转路径应确保对应页面存在，若路径不存在用户点击后将由前端 404 页面兜底
6. **异常兜底**：如接口请求失败，H5 端会降级显示默认导航（首页 + 我的），不会出现空白导航栏
