## 功能简介

在 **AI 对话模式首页（`/ai-home`）** 的输入框正上方，新增一排可横向滑动的「胶囊按钮」，用于一键发起常用咨询。

- **本次只在 AI 对话模式新增胶囊条**；菜单模式（九宫格）的胶囊条**保持现状不变**。
- 两边**共用同一份后台配置**：在管理后台「胶囊按钮配置」里改一处，AI 对话模式与菜单模式同时生效。
- **后端 0 改动**，复用菜单模式现有的胶囊按钮列表接口（`/api/function-buttons?is_enabled=true`）。

降低用户在 AI 对话模式下的发问门槛，让"健康咨询 / 用药提醒 / 体检报告……"等高频意图**一键触达**。

## 操作步骤

### 1. 进入 AI 对话模式

打开 H5 首页，进入 **AI 对话模式**（路径：`/ai-home`），可以直接点击下面的链接进入：

> [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/)

进入后，您会在屏幕底部输入框的**正上方**看到一排灰白色的圆角药丸按钮，每个按钮由「**小图标 + 文案**」组成（例如：📋 用药提醒、🩺 体检报告、💬 健康咨询）。

### 2. 点击胶囊一键发问

- **直接点击**任意胶囊：系统会立即以"**用户身份**"在对话流中自动发出该胶囊对应的问题文本（等同您手打了一遍这条问题并按下发送），AI 接着走正常回复流程。
- **胶囊本身不会消失**，可以反复点击。
- 点击时胶囊有**轻微按压反馈**（透明度/缩放）。

### 3. 横向滑动查看更多

当胶囊数量超过一屏宽度时：
- **左右滑动**胶囊条即可查看更多胶囊；
- 胶囊条**不换行、不挤压**，每个胶囊都保持完整可点击形态；
- 滑到两端会有轻微的回弹/渐隐效果。

### 4. 键盘联动（自动隐藏）

- 当您**点击输入框、键盘弹起、或正在打字**时，胶囊条会**自动隐藏**（避免遮挡键盘和输入区域）；
- **收起键盘**或**焦点离开输入框**后，胶囊条**自动恢复显示**；
- 整个过程无需手动开关，键盘体验流畅。

### 5. 数据为空时的处理

- 当后台返回的胶囊列表**为空**、**全部被禁用**、或者**接口异常**时，胶囊条**整体不渲染**；
- 输入框上方**完全空白，不留任何占位**，不影响输入体验。

## 注意事项

### 后台配置生效说明

- 在管理后台「胶囊按钮配置」中改动**启用/禁用、排序、文案、图标**，**刷新 AI 对话模式页面后立即生效**，无需发版。
- AI 对话模式与菜单模式（九宫格）始终展示**同一份配置**，不会出现两边不一致的情况。

### 与菜单模式的关系

- **菜单模式（九宫格）首页**的胶囊条**保持现状不变**，本次升级**不影响**菜单模式的任何展示与点击行为。
- 后续菜单模式下线时，AI 对话模式的胶囊条会承担同样的快捷入口职责，提前为下线做好平滑过渡。

### 跨端表现

| 端 | 表现 |
|---|---|
| H5（本次重点改造） | 在 `/ai-home` 输入框上方显示新胶囊条，支持横滑、键盘联动 |
| 小程序 | AI 对话页（pages/chat）原有胶囊条保持现状 |
| iOS / Android App | Flutter 聊天页原有 FunctionButtonsBar 保持现状 |
| Web 后台预览页 | 视觉与移动端一致 |

### 安全与可观测

- **可回滚**：胶囊条作为独立组件挂载，若后台「全部禁用」或前端关闭开关，AI 对话模式其它功能完全不受影响。
- **埋点上报**：胶囊条**曝光**与**点击**均已上报埋点（含胶囊 ID / 文案 / 端类型），便于后续菜单模式下线时评估真实使用情况。
- **接口静默降级**：接口异常时静默不显示（不弹错误提示），不阻塞主流程。

### 与"推荐问"区块的区别

- 「**推荐问**」（位于功能宫格下方、欢迎面板内）：常驻于对话顶部的欢迎区，**始终在文档流上方**，会随消息列表向下滚动而离开视野。
- 「**本次新增胶囊条**」：**贴在输入框正上方**，**始终可见**（除非键盘弹起），与输入框形成"两段式"独立块。

两者数据源不同（推荐问来自 `recommended_questions` 配置，胶囊条来自 `chat_function_buttons` 表 = 菜单模式同源），可以并存使用，互不干扰。

## 常见问题

**Q1：为什么我看不到胶囊条？**

A：可能的原因有三种：
1. 后台「胶囊按钮配置」中所有按钮都被禁用了，请联系管理员检查；
2. 您正在输入框中打字（焦点在 textarea 上），胶囊条会自动隐藏，把焦点移开即可恢复；
3. 网络异常导致接口未返回数据，刷新页面重试即可。

**Q2：点击胶囊后没反应？**

A：可能是该胶囊在后台未配置"预设问题文本"（即 `preset_prompt` 或 `auto_user_message` 字段为空）。请联系管理员在后台为该胶囊补充预设文本即可。

**Q3：我在管理后台改了胶囊文案，AI 对话模式不显示新文案？**

A：请刷新 `/ai-home` 页面（或重启浏览器 Tab），新配置会立即生效。AI 对话模式不会跨会话缓存胶囊配置。

**Q4：胶囊条会不会把输入框遮挡住？**

A：不会。胶囊条只占用约 36~44px 高度，与输入框是独立条带；并且在键盘弹起时会自动隐藏，避免任何遮挡。
