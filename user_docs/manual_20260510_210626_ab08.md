## 功能简介

本次更新修复了「**AI 模式首页（/ai-home）左上角 ☰ 唤出的个人中心抽屉，顶部头像 / 昵称 / VIP 会员号被页面顶栏「☰ 小康 ⋯」遮挡**」这一视觉 Bug。

修复后，无论您在手机 APP 内嵌、微信浏览器、手机系统浏览器还是电脑浏览器中打开 AI 模式首页，点击 ☰ 弹出的左侧抽屉**整条都会盖在「☰ 小康 ⋯」顶栏之上**，并且抽屉自身顶部会预留出"系统状态栏 + 外层 APP 标题栏"的安全区高度，确保**头像、昵称、VIP 会员编号 100% 完整可见**，不被任何顶栏遮挡。

本次修复仅涉及 H5 端样式调整（一个组件文件），不涉及任何后端接口、数据结构或其它端，无需用户重新安装任何客户端。

## 使用说明

### 操作步骤

1. **打开 AI 模式首页**：在浏览器或 APP 内打开 [AI 模式首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)
2. **登录账号**（若未登录）：先前往 [登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/login) 完成登录，再回到 AI 模式首页
3. **点击左上角 ☰**：屏幕左上角的"汉堡菜单"图标
4. **观察弹出的左侧抽屉顶部**：
   - ✅ **头像**：完整圆形显示，不再只有半截
   - ✅ **昵称**：完整文字显示，不再只露出顶部一条
   - ✅ **VIP 会员 NO.X**：完整可见，不再被顶栏盖住
5. **关闭抽屉**：点击右侧半透明遮罩区域，抽屉左滑收起，原页面顶栏「☰ 小康 ⋯」立即恢复正常显示

### 验证不同环境

为确保各环境效果一致，建议分别在以下场景下重复上述步骤验证：

| 环境 | 验证要点 |
|------|---------|
| 手机 APP 内嵌 | 抽屉顶部预留了系统状态栏 + APP 标题栏高度，头像不被刘海/灵动岛/APP 标题栏遮挡 |
| 微信浏览器 | 抽屉顶部预留了系统状态栏高度，头像完整显示 |
| 手机系统浏览器（Safari / Chrome） | 同上，异形屏不影响显示 |
| 电脑浏览器 | 抽屉宽度仍保持 70% / 最大 320px，顶部不被任何元素遮挡 |

## 注意事项

1. **本次为纯前端样式修复**，不涉及后端接口、数据结构、个人信息存储；无需重新登录、无需重新安装。
2. **抽屉的其他部分（宽度、底部、左右间距、滚动行为、左滑动画、点击遮罩关闭）维持现状**，仅修复顶部被遮挡这一个问题。
3. **页面顶栏（☰ 小康 ⋯）行为完全保持不变**，关闭抽屉后立即恢复显示，不会出现任何样式回退。
4. 若页面有缓存导致旧版本仍出现遮挡，请**强制刷新**（电脑端：Ctrl+F5 / Cmd+Shift+R；手机端：下拉刷新或清除浏览器缓存后重新打开）。
5. 微信小程序、安卓 APP、苹果 APP、Admin 后台**与本次修复无关**，无需任何更新。
