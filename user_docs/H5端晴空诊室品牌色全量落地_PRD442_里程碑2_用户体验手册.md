## 功能介绍

本次更新是 **「晴空诊室」品牌色全域迁移工程的里程碑 2**：把已沉淀的 PRD-442 设计 token（11 级天蓝色阶 + 5 大渐变 + 4 级阴影 + 8 图标种子集 + 8 条铁律）从设计系统正式带入 H5 端**全部业务页面**，让所有用户在使用 H5 时，都能体验到统一的「晴空诊室」视觉风格，告别此前混杂的旧绿色 / 旧紫色色板。

### 你能直观感受到的变化

1. **顶部导航条 / 主操作按钮 → 天蓝色**
   - 旧：偏黄绿色（如 `#52c41a`）；新：晴空蓝（brand-500 `#0EA5E9`）
   - 涉及登录页、首页、AI 主页、所有二级页面顶栏
2. **所有"主要按钮"渐变 → 天蓝渐变**
   - 旧：绿青双色渐变；新：从天蓝 brand-400 → brand-600 的同色系渐变（更高级、更克制）
3. **AI 头像 / 装饰圆 → 天蓝渐变**
   - 旧：绿青渐变；新：晴空诊室主蓝渐变
4. **Tab 选中态 / 列表标签 → 天蓝**
   - 「订单」、「优惠券」、「健康计划」等所有页面的选中态字体颜色 + 下划线统一切换为天蓝
5. **品牌名 → 「晴空诊室」**
   - 浏览器标签页、登录页、登陆页 hero 文案、各页面文案中的"宾尼小康 / 宾尼诊所 / 宾尼健康"字样统一替换为"晴空诊室"
   - 浏览器主题色（status bar）由 `#34C759` 改为 `#0EA5E9`
6. **错别字、null 判断 等 7 处低风险小修一并带上**
   - 顺手处理（不影响主功能）

### 不会受影响的范围

- 微信小程序：本次未改动 → **现有版本继续可用，无需更新**
- Flutter App（iOS/Android）：本次未改动 → **现有版本继续可用，无需更新**
- 后端 API：本次未改动 → 无任何接口变更
- 危险态色（如错误提示红色 `#ef4444`）：保留 PRD-442 §7 红线规范允许场景

## 注意事项

1. **缓存可能影响视觉效果**：如果你之前打开过本站，可能浏览器缓存了旧版静态资源。请按 `Ctrl + Shift + R`（电脑）或在手机微信中点击右上角"…"→"清除缓存"，强制刷新页面查看新色板。
2. **业务功能完全保持不变**：本次仅做视觉品牌色 + 品牌词替换，不影响任何业务流程、数据、API 接口。所有按钮、表单、跳转、下单、支付等功能继续正常工作。
3. **危险态保留红色**：根据 PRD-442 §7 红线规范，错误提示、删除按钮、危险操作等"危险态"色仍使用红色 `#ef4444`，未被天蓝化（这是有意为之，符合无障碍语义规范）。
4. **会员金徽保留 #d4af37**：会员相关的"金徽"色（如付费会员标识）保留，不在本次清退范围内。
5. **本次仅 H5 端有改动**：微信小程序与 Flutter App 仍是上一版本，**无需重新下载**。如需小程序 / 移动 App 的天蓝化改造，请等待后续里程碑 3 / 4。
6. **如发现任何展示异常**：请记录页面路径、操作步骤和截图，反馈给开发团队。本次改造覆盖 100+ 页面，理论上不影响业务逻辑，但若发现页面错位、白屏、文字读不清等问题，可能是 CSS 变量级联未覆盖到的边缘场景。
