## 功能简介

本次更新围绕「会员中心」补齐了两个关键体验：

1. **会员中心入口正式上线（多端统一）**
   - H5 顶部右上角「⋯」更多菜单首项 = **👑 会员中心**（金色文字 + 「新」角标 30 天），旧的「我的设备」入口完全移除。
   - H5 左侧抽屉的高频入口将「我的设备」替换为「👑 会员中心」金色卡片，与顶部菜单视觉一致。
   - 微信小程序、Android、iOS 三端的 AI 主页右上角也新增了「⋯」更多菜单，首项均为「👑 会员中心」，点击后跳转 / WebView 加载 H5 版会员中心，登录态自动透传。
   - 健康档案页内的「我的设备」入口保留不变，原有的设备管理路径不受影响。

2. **会员中心新增「权益对比」模块**
   - 位于「升级享更多权益」套餐区下方、「📦 我的订单 / 🎟️ 我的优惠券 / ❓ 常见问题」入口区上方。
   - 表格行 = 守护人上限 / AI 外呼提醒 / 紧急 AI 呼叫（与「我的会员权益」3 实卡同源）。
   - 表格列 = **普通会员** + **后台所有启用付费套餐**（按 `price_rank` 升序，年价优先）。
   - 列头颜色按档位自动渐变：紫（普通）→ 橙黄 → 橙红 → 深红。
   - 数值「-1」自动展示为「不限」。
   - 窄屏宽度不足时整张表 **可横向滑动**，不会破坏布局。

## 使用说明

### A. H5 端（手机浏览器）

1. 打开 [H5 主页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/) 并登录。
2. **方式一：从顶部「⋯」菜单进入** —— 点击右上角「⋯」按钮 → 在弹出菜单中点击金色 **「👑 会员中心」**（首次进入会看到右上角「新」角标）。
3. **方式二：从左侧抽屉进入** —— 点击左上角「☰」打开抽屉 → 在「健康档案 / 会员中心」快捷入口区域点击金色的「👑 会员中心」卡片。
4. 进入会员中心后向下滑动，依次会看到：
   1. 顶部用户信息卡（蓝紫渐变 + 金色皇冠 + 等级胶囊 + 有效期）。
   2. 「我的会员权益」4 格卡（守护人 / AI 外呼 / 紧急呼叫 / 占位）。
   3. 「升级享更多权益」套餐区，推荐档有金色「👑 推荐」角标。
   4. **新增**：「权益对比」表 —— 横向展示普通会员与所有启用付费套餐的差异，列头颜色按档位渐变；列数较多时可左右滑动查看。
   5. 「📦 我的订单 / 🎟️ 我的优惠券 / ❓ 常见问题」入口。

## 注意事项

- **H5 端缓存**：H5 端如未看到金色「👑 会员中心」菜单项，请清缓存或强制刷新（Ctrl/Cmd + Shift + R）。
- **「新」角标**：自上线起 30 天内（约 2026-06-25 之前）「会员中心」菜单项右侧会显示「新」角标，过期后自动消失。
- **小程序 web-view 域名**：正式发布前需在小程序后台「业务域名」中加入 `newbb.test.bangbangvip.com`，否则 web-view 会提示域名不在白名单。
- **App WebView 登录态**：Flutter / 小程序的 WebView 通过 query string 透传 token，H5 端会用拦截器自动消费；如出现未登录提示，请重新登录 App。
- **付费套餐数量**：「权益对比」表的列数取决于后台启用的付费套餐数。当列数较多时整表可横向滚动，不会破坏页面布局。
- **数据「-1」**：表格中显示「不限」表示该档套餐对应权益数量不限制。
- **健康档案的「我的设备」**：未受影响，仍可通过健康档案进入。
