## 功能简介

本次更新为 bini-health 系统新增了「商品门店绑定」、「订单通知增强」和「门店端功能增强」三大模块，形成"商品→门店→通知→处理"的完整业务闭环：

1. **商品绑定门店**：管理后台可为每个商品指定可售卖的门店，支持智能推荐
2. **订单通知矩阵**：新订单/预约到达时自动通知对应门店（站内消息+公众号推送）
3. **预约日历视图**：门店端新增月/日预约日历，含时段热力图
4. **订单处理操作**：门店可确认接单、调整预约时间、添加备注
5. **超时策略**：平台可配置订单超时自动处理规则
6. **公众号绑定**：门店员工可绑定公众号接收实时推送通知

## 使用说明

### 模块一：商品绑定门店（管理后台）

#### 1. 为商品选择适用门店

1. 登录管理后台：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 进入「商品系统」→「商品管理」
3. 点击某个商品的「编辑」按钮，打开编辑弹窗
4. 在「基本信息」Tab中，找到「适用门店」字段
5. 在下拉框中搜索并选择门店（支持多选）
6. 也可以点击「智能推荐」按钮，系统会根据商品分类自动推荐匹配的门店
7. 在推荐弹窗中，可逐个添加或一键全部添加
8. 未选择任何门店时，默认所有门店可售
9. 点击「保存」完成绑定

#### 2. 设置门店经营范围

1. 进入「门店管理」→「门店列表」
2. 点击某个门店的「编辑」按钮
3. 在弹窗中找到「经营范围」字段
4. 选择该门店可经营的商品分类（多选）
5. 保存后，新建商品时系统会根据经营范围智能推荐该门店

### 模块二：超时策略配置（管理后台）

1. 进入「系统设置」页面
2. 切换到「超时策略」Tab
3. 配置以下参数：
   - **催促提醒时间**：下单后多少分钟发送催促通知给门店（默认15分钟）
   - **超时时长**：下单后多少分钟视为超时（默认30分钟）
   - **超时处理方式**：
     - 自动取消订单并退款
     - 自动确认接单（默认视为已接）
     - 升级通知管理员（人工介入）
   - **预约提前提醒时长**：预约到期前多少小时提醒门店（默认2小时）
4. 点击「保存配置」

### 模块三：预约日历视图（门店端）

#### PC端

1. 登录门店PC端：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/dashboard/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/dashboard/)
2. 在左侧菜单中点击「预约日历」
3. **月视图**（默认）：
   - 日历上每天显示预约数量
   - 色块表示时段热力（绿色=低、橙色=中、红色=高）
   - 上午(8:00-12:00)、下午(12:00-18:00)、晚间(18:00-22:00) 三个时段
4. **日视图**：点击某一天，查看当天所有预约的详细列表
   - 显示预约时段、客户姓名、商品名称、预约状态
   - 可直接点击进入订单详情

#### H5移动端

1. 打开门店H5端：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/m/dashboard/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/m/dashboard/)
2. 在工作台点击「预约日历」入口
3. 操作方式与PC端相同，界面针对手机屏幕做了适配

### 模块四：订单处理操作（门店端）

#### 1. 确认接单

- 预约类订单下单后，状态为「待确认」
- 在门店端订单详情中，点击「确认接单」按钮
- 确认后订单状态变为「已确认」，系统自动通知用户

#### 2. 调整预约时间

- 在订单详情中，点击「调整预约时间」按钮
- 选择新的日期和时段
- 确认后，预约时间立即更新
- 系统自动推送通知告知用户新预约时间
- 调整历史记录在订单详情中可查看

#### 3. 添加门店备注

- 在订单详情底部，找到「门店备注」区域
- 输入备注内容，点击提交
- 支持多次添加备注，按时间倒序展示
- 备注仅门店端和管理后台可见，用户端不显示

### 模块五：公众号通知绑定（门店端）

#### PC端绑定

1. 登录门店PC端
2. 在左侧菜单中点击「公众号绑定」
3. 系统生成带参数二维码
4. 使用微信扫描二维码关注公众号
5. 绑定成功后页面显示「已绑定」状态

#### H5移动端绑定

1. 打开门店H5端，进入「我的」页面
2. 点击「公众号通知绑定」
3. 系统显示绑定二维码
4. 长按保存二维码，在微信中扫码关注
5. 绑定成功后自动显示「已绑定」

#### 解绑操作

- 在绑定页面点击「解除绑定」按钮即可取消

#### 绑定后可接收的通知

| 事件 | 通知内容 |
|------|----------|
| 新订单 | 订单编号、商品名、金额、下单时间 |
| 新预约 | 预约日期、时段、商品名、客户信息 |
| 退款申请 | 订单编号、退款金额、退款原因 |
| 预约即将到期提醒 | 预约时间、客户信息、商品名 |
| 核销完成确认 | 核销时间、订单编号、核销人 |
| 接单催促提醒 | 订单编号、已等待时长、截止时间 |

## 注意事项

1. **门店绑定**：未为商品选择任何门店时，默认所有门店可售。选择了特定门店后，只有被选中的门店才能接到该商品的订单
2. **接单确认**：仅预约类订单需要门店确认，普通购买类订单自动确认无需操作
3. **超时策略**：超时策略对所有预约类订单生效，建议根据实际业务情况合理配置催促时间和超时时长
4. **公众号推送**：需要先在管理后台配置微信公众号的 App ID 和 App Secret，绑定后才能收到推送通知
5. **权限控制**：门店端操作需要使用门店账号登录，不同角色（老板/店长/店员等）的操作权限可能不同
6. **并发安全**：确认接单操作使用了行级锁，多人同时操作同一订单时不会产生数据冲突
