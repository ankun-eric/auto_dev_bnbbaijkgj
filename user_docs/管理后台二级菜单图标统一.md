## 功能简介（本期修复内容）

本次修复**仅涉及管理后台 admin-web 的左侧侧边栏视觉效果**，对功能逻辑、数据、接口零影响。修复要点如下：

1. **统一视觉规范**
   - 一级菜单：保持「图标 + 文字」呈现，不做改动
   - 二级菜单：**全部改为纯文字**，不再显示任何图标
   - 折叠态弹出浮层：浮层内的二级菜单同样不显示图标
2. **从源头屏蔽**
   - 在菜单渲染层增加了 `stripChildrenIcons` 工具函数，会自动清除所有二级菜单项的 `icon` 字段。即便后续维护者手动给某个二级菜单配置了图标，也不会显示出来
3. **沉淀开发规范**
   - 新增 `admin-web/CONVENTIONS.md` 文档，把本规范写入项目长期约束，避免后续重复踩坑

## 使用说明

### 1. 登录管理后台

1. 在浏览器中打开 [管理后台登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/login)
2. 输入管理员账号密码登录
3. 登录成功后会进入数据概览页面，左侧出现侧边栏菜单

### 2. 体验侧边栏视觉一致性

**展开态侧边栏（默认展开）**：

- 任意点击一级菜单（如「AI 配置中心」「AI 咨询配置」「商品体系」「系统管理」），可以看到展开后的二级菜单**全部为纯文字**，没有图标
- 切换到其他一级菜单（如「健康计划管理」「内容管理」「积分体系」等），二级菜单同样是纯文字
- 整体侧边栏从顶到底，二级菜单风格完全一致，没有跳动感

**折叠态弹出浮层**：

1. 点击左侧顶部的折叠按钮（「☰」图标），把侧边栏切换到窄条状态
2. 鼠标悬停任意一级菜单（如「商品体系」「系统管理」），会弹出右侧浮层
3. 浮层内的二级菜单也都是纯文字，没有图标，与展开态保持一致

### 3. 一级菜单与下拉菜单的图标依旧保留

- **一级菜单**：所有一级菜单（数据概览、用户管理、AI 管理、商家管理、系统管理 等）头部仍然带有彩色图标
- **右上角头像下拉菜单**：点击右上角的管理员头像，弹出的下拉菜单中「个人信息」「修改密码」「退出登录」依然各自带有图标，未受本次修复影响

## 注意事项

1. **缓存刷新**：如果你之前访问过管理后台并被浏览器缓存，建议强制刷新（`Ctrl + F5` / `Cmd + Shift + R`）一次，确保看到最新版本的菜单效果
2. **业务功能不受影响**：本次修复**仅是视觉调整**，所有菜单的跳转、点击、搜索等功能均保持原状
3. **后续开发约束**：项目根目录下新增的 `admin-web/CONVENTIONS.md` 是开发规范文档，今后给侧边栏添加新菜单时请遵守其中"二级菜单禁止配置图标"的规定
4. **遗漏反馈**：如果在使用过程中发现仍有零星二级菜单带有图标（理论上已全部移除），请直接反馈对应菜单路径，单独修复即可
