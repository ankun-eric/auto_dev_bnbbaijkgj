## 功能简介

本次更新对 H5 搜索页面的**语音搜索功能**进行了全面的体验优化，包含以下四项改进：

1. **录音触发方式优化**：将原来需要长按 300 毫秒才能开始录音的操作，改为**单击即可开始录音**，大幅降低了使用门槛
2. **录音结束方式优化**：录音中不再显示"松开结束"的文字提示，改为醒目的**脉动呼吸大按钮**，按钮上显示"点我结束~"，用户操作更加明确
3. **声波动画增强**：声波竖条从原来的 5 根增加到 **7 根**，视觉效果更丰富，录音过程更生动
4. **全流程文案温暖化**：所有语音搜索相关的提示文案替换为温暖亲切的风格，让用户体验更加友好

## 使用说明

### 一、进入搜索页面

1. 打开 H5 用户端：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 在首页顶部点击搜索栏，进入搜索页面

### 二、使用语音搜索

1. **开始录音**：在搜索页面，点击搜索栏右侧的**绿色麦克风图标**（单击一下即可，不需要长按）
2. **录音中**：
   - 界面会弹出录音弹层，显示录音计时器（格式：Ns / 15s）
   - 中间区域展示 **7 根声波竖条**，实时反映您的说话音量
   - 下方有一个**白色圆形大按钮**（绿色边框，带脉动呼吸效果），按钮上显示"■ 点我结束~"
3. **结束录音**：点击脉动呼吸大按钮（"点我结束~"），录音将停止并进入语音识别
4. **等待识别**：系统会显示"我在认真听，马上就好~"，稍等片刻即可
5. **识别成功**：识别结果会自动填入搜索框，并显示"听到啦~ 2秒后帮你搜索"的倒计时提示
   - 如果想修改识别结果，可以直接编辑搜索框内容
   - 如果不想自动搜索，点击"先不搜了"取消倒计时

### 三、异常情况处理

- **录音时间太短**（不足 1 秒）：会提示"说的太快了，再试一次吧~"，点击"再来一次~"重新录音
- **语音识别失败**：会提示"没听清楚，再说一次好吗~"，点击"再来一次~"重新录音
- **未识别到内容**：会提示"好像没听到声音哦，再试试~"，点击"再来一次~"重新录音
- **麦克风权限被拒**：会提示"需要你允许使用麦克风才能听到你说话哦~"
  - 请在浏览器设置中允许网站使用麦克风权限，然后重新尝试

### 四、录音自动结束

- 录音最长时间为 **15 秒**，到达上限时会自动停止录音并进入识别
- 最后 3 秒计时器会变为红色提醒

## 注意事项

1. **浏览器兼容性**：语音搜索功能需要浏览器支持麦克风访问和 MediaRecorder API。推荐使用以下浏览器：
   - iPhone 微信内置浏览器
   - Safari（iOS）
   - Chrome（Android/iOS）
2. **麦克风权限**：首次使用语音搜索时，浏览器会弹出麦克风权限请求，请点击"允许"
3. **网络要求**：语音识别需要网络连接，请确保手机网络通畅
4. **安静环境**：在较安静的环境中使用语音搜索，识别准确率更高
5. **语音搜索按钮不显示**：如果搜索栏旁看不到麦克风图标，可能是浏览器不支持麦克风功能或 ASR 服务暂时不可用
