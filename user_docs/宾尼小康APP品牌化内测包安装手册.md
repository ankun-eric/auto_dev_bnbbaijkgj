### 2.2 Android 安装步骤

#### 第 1 步：下载 APK

- 用手机浏览器（Chrome、Edge、夸克等）打开上方下载链接
- 文件较大（80+ MB），请在 Wi-Fi 环境下载

#### 第 2 步：允许"安装未知来源应用"

由于此包未通过应用商店上架，首次安装时系统会弹出安全提示。授权方式因厂商而异：

| 厂商 / 系统 | 路径 |
|------------|------|
| 华为 EMUI / HarmonyOS | 设置 → 安全 → 更多安全设置 → 安装外部来源应用 → 选择浏览器并允许 |
| 小米 MIUI / HyperOS | 设置 → 隐私保护 → 特殊权限设置 → 安装未知应用 → 选择浏览器并允许 |
| OPPO ColorOS | 设置 → 系统应用 → 应用安装来源 → 选择对应应用 |
| vivo OriginOS / Funtouch | 设置 → 更多设置 → 权限管理 → 安装未知应用 |
| 三星 OneUI | 设置 → 应用 → 特殊访问权限 → 安装未知应用 |
| 原生 Android（Pixel） | 系统会在点击 APK 时直接弹窗，按提示授权即可 |

#### 第 3 步：点击 APK 安装

- 在文件管理器或下载列表中找到 APK
- 点击开始安装，按提示授权
- 安装完成后，桌面会出现「**宾尼小康**」图标，点击启动

## 三、首次使用注意事项

### 3.1 ⚠️ 与旧版本（开发期 APP）不可共存

由于包名已变更，从系统层面看，本次的内测版与之前安装过的「宾尼小康」开发版**是两个独立的应用**：

- **不能覆盖升级**：必须**先卸载旧版**，再安装本内测版
- **不互通本地数据**：旧版本 APP 中的本地缓存、登录状态、草稿等不会迁移到新版
- **服务端数据不受影响**：登录后从云端同步的数据（健康档案、订单、积分等）会自动恢复

**操作建议：**

1. 在桌面找到旧版「宾尼小康」图标，长按选择「卸载」
2. 按上文步骤安装本内测版
3. 启动后用原账号重新登录即可

### 3.2 安装失败的常见原因

| 问题现象 | 解决方法 |
|---------|---------|
| 下载到一半中断 | 重新点击链接续传；建议使用 Chrome / Edge / 夸克浏览器 |
| 提示"解析包时出现问题" | 文件未下载完整，删除后重新下载；确认文件大小为 80+ MB |
| 提示"已安装存在签名冲突的同名包" | 先卸载旧版本（开发期 `com.binihealth.app`） |
| 提示"应用未安装" | 检查是否给浏览器/文件管理器授予了"安装未知应用"权限 |
| 安装成功但无法启动 | 检查手机系统是否为 Android 7.0 或以上 |

## 六、关于本次品牌化改造的说明

### 6.1 为什么要改包名

应用商店（Google Play、Apple App Store、华为应用市场等）的核心规则之一是：

> **应用包名 / Bundle ID 一旦上架商店，永久不可更改。**

公司当前 APP 仍使用开发期占位标识 `com.binihealth.app`，必须在正式上架前完成"品牌标识收口"，避免后续陷入"想改改不动"的被动局面。本次将其统一收口为 **`com.benekang.app`**，与公司确定的品牌名 **Bene** 完全对齐。

### 6.2 为什么显示名和图标暂不改

- 显示名「宾尼小康」、应用图标当前都是"可逆的视觉资源"
- 等公司品牌设计部门提供正式的图标素材后，统一通过另一轮 UI 升级流程替换
- 本期目标是**先把"不可逆的包名"锁死**，"可逆的视觉"留到下一轮

### 6.3 后续节奏

| 时间窗口 | 事项 |
