# 健康档案首页 - 用户体验使用手册

## 功能简介

健康档案首页经过全新升级，从原来的多 Tab 切换模式改为**单页平铺折叠式布局**。您可以在一个页面内查看个人健康信息、用药计划、就医资料和今日健康数据，无需频繁切换标签页，操作更简单、更直观。

**访问地址**：[健康档案首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)

---

## 页面整体布局

页面从上到下依次排列以下板块：

1. **标题栏 + 成员切换条**（吸顶）
2. **Hero 卡片**（个人信息 + 健康信息标签）
3. **家庭成员 + 我的设备**（双卡片并排）
4. **预警提示条**（仅有预警时显示）
5. **用药计划**（可折叠/展开）
6. **就医资料**（可折叠/展开）
7. **今日健康数据**（用药提醒 + 体征数据）

## 操作指南

### 1. 切换家庭成员

在页面顶部的成员切换条中，左右滑动可查看所有家庭成员。点击任意成员头像，页面所有数据将自动切换为该成员的信息。

- **选中状态**：橙色背景 + 白色文字
- **守护中成员**：头像右上角有天蓝色"守护中"标记
- **添加成员**：点击最右侧的 "+" 按钮，即可添加新的家庭成员

### 2. 查看和编辑个人健康信息（Hero 卡片）

Hero 卡片展示当前成员的基本信息和健康标签：

- **基本信息**：头像、姓名、年龄、关系、性别
- **健康信息标签**：既往病史、过敏史、手术史、家族病史、个人习惯等，以彩色标签形式全量展示

**编辑方式**：
1. 点击 Hero 卡片右上角的编辑图标（笔形图标）
2. 在弹出的编辑面板中修改信息，分为两个区域：
   - **个人信息**：姓名、性别、生日、身高、体重、血型、关系
   - **健康信息**：既往病史、过敏史、手术史、家族病史、个人习惯
3. 修改完成后点击底部"保存"按钮

**未填写时的引导**：如果还没有填写健康信息，Hero 卡片下方会显示虚线引导区域，提示"尚未填写健康信息，点击补充"，点击即可直接进入编辑。

### 3. 家庭成员与设备管理

Hero 卡片下方有两张并排的快捷卡片：

| 卡片 | 显示内容 | 点击后跳转 |
|------|---------|-----------|
| 家庭成员 | 显示家庭成员人数 | [家庭成员管理页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family) |
| 我的设备 | 显示绑定设备数量 | [设备管理页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/devices) |

### 4. 健康预警提示

当检测到异常健康指标时，页面会自动显示一条橙红色渐变的预警提示条，显示预警摘要信息。点击提示条可查看详细预警内容。无异常时此区域不显示。

### 5. 用药计划

用药计划板块默认折叠，标题栏会显示药品总数。

**操作步骤**：
1. 点击"展开"按钮查看用药列表
2. 每条药品显示：药品名称、剂量、服药时间、今日打卡状态
3. 药品超过 5 条时，先显示前 5 条，点击"查看全部"可展开所有
4. 点击"+ 新增用药"可添加新的用药计划
5. 点击"收起"按钮可折叠列表

### 6. 就医资料

就医资料板块默认折叠，展开后显示四个分类卡片：

| 分类 | 说明 |
|------|------|
| 病例单 | 门诊/住院病历记录 |
| 体检报告 | 各类体检报告 |
| 药物 | 药物处方和说明 |
| 其他 | 其他医疗文件 |

**操作步骤**：
1. 点击"展开"按钮查看四个分类卡片
2. 点击任意分类卡片，底部弹出抽屉显示该分类下的资料列表
3. 每条资料显示：名称、上传日期、文件数、来源标签
4. 点击"上传新资料"可添加新文件

**上传限制**：
- 单次最多上传 9 个文件
- 图片文件不超过 10MB
- PDF 文件不超过 20MB

### 7. 今日健康数据

#### 用药提醒卡片
页面底部首先展示蓝色渐变的用药提醒卡片，显示今日用药完成进度（环形进度圈）。点击可跳转到 [用药提醒页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/medication-reminder)。

#### 体征数据
用药提醒下方是 2×2 网格的体征数据卡片：

| 体征 | 单位 | 说明 |
|------|------|------|
| 血压 | mmHg | 显示收缩压/舒张压 |
| 血糖 | mmol/L | 最近一次血糖值 |
| 心率 | bpm | 最近一次心率值 |
| 睡眠 | h | 昨晚睡眠时长 |

- 异常值会以黄色左边线高亮提示
- 点击任意体征卡片可查看历史记录

## 注意事项

1. **数据联动**：切换家庭成员后，页面所有板块数据会自动刷新，请耐心等待加载完成
2. **折叠状态**：用药计划和就医资料每次进入页面默认折叠，需手动展开查看
3. **编辑权限**：只有本人和有守护权限的家庭成员才能编辑健康信息
4. **数据安全**：所有健康数据受密码保护，请勿在公共设备上保持登录状态

## 常见问题

**Q: 为什么看不到预警提示条？**
A: 预警提示条仅在当前成员有异常健康指标时才会显示。如果没有异常，该区域不会出现。

**Q: 用药计划显示"暂无用药计划"怎么办？**
A: 点击"+ 新增用药"按钮添加您的用药计划，或联系您的医生获取用药建议。

**Q: 上传就医资料失败？**
A: 请检查文件大小是否超限（图片 ≤ 10MB，PDF ≤ 20MB），以及单次上传是否超过 9 个文件。

**Q: 健康信息标签没有更新？**
A: 编辑保存后标签会自动刷新。如未更新，请下拉刷新页面重试。
