### 二、订单管理

入口：左侧菜单 → 订单管理

#### 2.1 数据看板

页面顶部展示了6个核心指标卡片和4个数据图表：

**指标卡片**（一行6个）：
- 今日订单数：今天新增的订单总数
- 今日成交金额：今天已支付订单的总金额
- 本月订单数：当月累计订单总数
- 本月成交金额：当月已支付订单的总金额
- 订单总数：历史所有订单总数
- 累计总金额：历史所有已支付订单的总金额

**数据图表**（2×2布局）：
- **订单量趋势**（折线图）：展示每天的订单数量变化，支持切换近7天/近30天
- **成交金额趋势**（折线图）：展示每天的成交金额变化，支持切换近7天/近30天
- **服务分类订单占比**（饼图）：按服务项目统计订单数量占比
- **订单状态分布**（柱状图）：展示各状态的订单数量

#### 2.2 订单列表

- **Tab切换**：点击「全部」「待支付」「已支付」「服务中」「已完成」「已退款」Tab查看不同状态的订单
- **搜索**：在搜索框中输入订单号、用户昵称或手机号进行搜索
- **时间筛选**：使用日期范围选择器筛选特定时间段的订单
- **查看详情**：点击操作列的「详情」按钮，右侧弹出订单详情面板
- **状态操作**：
  - 点击「确认」将待确认订单标记为已确认
  - 点击「开始服务」将已确认订单标记为服务中
  - 点击「完成」将服务中的订单标记为已完成
  - 点击「取消」取消订单（需填写取消原因）
  - 点击「退款」处理退款（需填写退款原因和金额）

### 三、服务项目管理

入口：左侧菜单 → 服务管理 → 服务项目

#### 3.1 列表浏览与筛选

- **分类筛选**：顶部下拉框选择服务分类，快速过滤
- **名称搜索**：在搜索框中输入关键词，按服务项目名称搜索
- **筛选组合**：分类和搜索可以同时使用

#### 3.2 批量上下架

1. 勾选需要操作的服务项目（表格左侧复选框）
2. 点击顶部工具栏的「批量上架」或「批量下架」按钮
3. 在确认弹窗中确认操作

#### 3.3 排序管理

- 每个服务项目右侧有排序值输入框
- 输入新的排序数字后，按回车或点击其他地方即可自动保存
- 排序值越小越靠前

#### 3.4 图片管理

- 在新增/编辑服务项目的表单中，点击上传区域选择封面图片
- 支持 JPG/PNG 格式，建议尺寸 750×420px
- 图片将自动上传到腾讯云COS存储

---

### 四、内容管理（文章）

入口：左侧菜单 → 内容管理 → 文章管理

#### 4.1 文章列表

- **状态筛选**：顶部下拉框选择文章状态（全部/草稿/已发布/已下架）
- **分类筛选**：按文章分类过滤
- **搜索**：按标题关键词搜索
- **状态标识**：
  - 🟢 绿色 = 已发布
  - 🟡 橙色 = 草稿
  - ⚫ 灰色 = 已下架

#### 4.2 创建/编辑文章

1. 点击「新增文章」按钮
2. 填写以下信息：
   - **标题**：文章标题（必填）
   - **摘要**：文章简短描述
   - **分类**：选择文章分类
   - **标签**：输入标签后按回车添加，支持多个标签
   - **正文内容**：文章正文（支持HTML格式）
   - **封面图**：上传文章封面图片
   - **状态**：选择草稿或直接发布
3. 点击「确定」保存

#### 4.3 发布/下架操作

- 点击列表中的「发布」按钮将草稿文章发布
- 点击「下架」按钮将已发布文章下架
- 草稿状态的文章不会在用户端显示

#### 4.4 置顶功能

- 在列表中使用置顶开关，将文章设为置顶
- 置顶文章在用户端列表中优先展示
- 置顶文章在管理列表中以 📌 标识显示

---

### 五、视频管理

入口：左侧菜单 → 内容管理 → 视频管理

#### 5.1 统计概览

页面顶部展示3个统计卡片：
- 视频总数
- 总播放量
- 今日新增播放量

#### 5.2 视频列表

- **分类筛选**：按视频分类过滤
- **状态筛选**：按发布状态过滤
- **搜索**：按视频标题搜索
- **预览播放**：点击「播放」按钮在弹窗中预览视频

#### 5.3 创建/编辑视频

1. 点击「新增视频」按钮
2. 填写以下信息：
   - **标题**：视频标题（必填）
   - **时长**：格式为"分:秒"（如 05:30 表示5分30秒）
   - **分类**：选择视频分类
   - **描述**：视频简介
   - **封面图**：上传视频封面图片
   - **视频文件**：上传视频文件（支持MP4、MOV、AVI，最大500MB）
3. 点击「确定」保存

#### 5.4 发布/下架

- 点击「发布」按钮将草稿视频发布上线
- 点击「下架」按钮将已发布视频下架
- 可以按播放量排序查看热门视频

---

### 六、积分商城管理

入口：左侧菜单 → 积分体系 → 积分商城

#### 6.1 商品管理

**商品列表展示**：
- 商品名称、封面图、兑换所需积分、库存数量、状态、创建时间
- 库存 ≤ 10 时以橙色警告标识
- 库存 = 0 时显示红色「已兑完」标签

**新增商品**：
1. 点击「新增商品」按钮
2. 填写商品信息：名称、描述、兑换积分、库存数量、商品类型
3. 上传商品封面图（自动存储到COS）
4. 设置上架状态
5. 点击「确定」保存

**编辑商品**：点击操作列的「编辑」按钮修改商品信息

**删除商品**：点击操作列的「删除」按钮（软删除）

#### 6.2 上下架管理

- **单个操作**：点击操作列的「上架」/「下架」按钮
- **批量操作**：
  1. 勾选多个商品
  2. 点击顶部「批量上架」或「批量下架」按钮
  3. 确认操作
- 下架商品在用户端不显示

#### 6.3 兑换记录

1. 切换到「兑换记录」Tab页
2. 查看所有用户的积分兑换记录
3. 支持按时间范围筛选
4. 支持按用户昵称/手机号搜索
5. 点击「导出CSV」按钮下载兑换记录

#### 6.4 库存管理

- 编辑商品时可以修改库存数量
- 用户兑换时自动扣减库存
- 库存为0时，用户端该商品显示「已兑完」，不可兑换

### 七、文件上传说明（COS统一存储）

本次更新后，所有文件上传功能（包括头像、封面图、体检报告、视频文件等）均已统一接入腾讯云COS对象存储：

- 当COS配置已启用时，所有文件自动上传到腾讯云COS
- COS未启用时自动降级到本地存储
- 上传成功后返回完整的COS访问URL
- 管理后台可在「系统管理 → 存储配置」中查看和修改COS配置

## 注意事项

1. **管理后台需要管理员权限**：只有管理员账号才能登录管理后台
2. **订单状态操作不可逆**：确认订单、开始服务、完成订单等状态变更后不可撤回，请谨慎操作
3. **退款操作**：退款需要填写退款原因和退款金额，请确认信息正确后再提交
4. **文件上传大小限制**：
   - 图片文件：最大 10MB，支持 JPG/PNG/GIF/WEBP 格式
   - 视频文件：最大 500MB，支持 MP4/MOV/AVI 格式
   - 体检报告：最大 20MB，支持 JPG/PNG/PDF 格式
5. **批量操作**：批量上下架操作前会弹出确认对话框，请确认后再执行
6. **数据看板刷新**：订单数据看板的统计数据会在每次进入页面时自动刷新
7. **浏览器兼容性**：管理后台兼容 Chrome、Edge、Firefox 最新版本，建议使用 1920×1080 及以上分辨率
