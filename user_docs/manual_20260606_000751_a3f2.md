## 功能简介

本次更新对 AI 首页（标准模式 + 关怀模式）的欢迎区域进行了布局优化，主要变化如下：

1. **LOGO 位置左移**：机器人头像从原来右侧竖排大图（84px）移到了问候语左侧，改为精致的小圆头像（42px），与问候语水平排列，视觉更紧凑。
2. **模式切换色块移至右上角**：原本在右侧竖排的模式切换按钮移到了欢迎区域的右上角，采用同色系浅色底色块，醒目易找。
3. **标准模式新增用药提醒**：标准模式欢迎区下方新增了"用药提醒"卡片，显示今日还需要服用的药品数量，点击即可跳转查看详情，与关怀模式体验统一。
4. **色块颜色优化**：标准模式使用浅蓝底色块、关怀模式使用浅橙底色块，与各自的背景色协调搭配，不再撞色。
5. **问候语字号优化**：问候语字号调整为 16px，视觉更加精致。

## 使用说明

### 标准模式首页体验

1. 打开 [标准模式首页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/)，进入 AI 健康助手主页。
2. 在欢迎区域可以看到：
   - 左侧：**小圆头像** + **问候语**（如"早上好 ☀️"）+ **副标题**
   - 右上角：**浅蓝色切换色块**（👴 去长辈版 →）
   - 问候语下方：**用药提醒卡片**（如有未打卡用药，显示"💊 用药提醒：今天还有X种药需要服用"）
3. 点击右上角**"去长辈版 →"**色块，即可切换到关怀模式。
4. 点击用药提醒卡片，可直接跳转查看今日用药详情。

### 关怀模式首页体验

1. 打开 [关怀模式首页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/care-ai-home)，进入关怀版。
2. 在欢迎区域可以看到：
   - 左侧：**小圆头像** + **问候语** + **副标题** + **用药提醒**
   - 右上角：**浅橙色切换色块**（🧑 去标准版 →）
3. 点击右上角**"去标准版 →"**色块，即可切换回标准模式。

### 色块悬停效果

- 鼠标悬停在切换色块上时，色块会**上浮 2px** + 阴影加深
- 移动端点击色块有轻微缩放反馈
- 色块点击后直接切换模式，无需二次确认

## 注意事项

1. **浏览器兼容**：建议使用 Chrome、Safari、Edge 等现代浏览器访问，移动端微信内置浏览器也可正常使用。
2. **网络要求**：需要能够访问外网（HTTPS）。
3. **用药提醒**：标准模式的用药提醒仅在您有未打卡的用药计划时显示，如果没有用药计划则不显示该卡片。
4. **模式切换**：切换模式时会自动保存您的偏好，下次进入时将默认显示您上次选择的模式。
