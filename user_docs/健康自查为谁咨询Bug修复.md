## 功能简介

本次更新修复了 H5 端健康自查功能中的一个 Bug：

**修复前**：在健康自查页面完成三步流程（选择部位 → 描述症状 → 补充信息）后，点击"AI 智能分析"弹出"为谁咨询"弹窗时，虽然默认选中了"本人"，但本人的健康档案信息（出生日期、性别、身高、体重、既往病史、过敏史）未自动带出，表单显示为空白，用户需要再次手动点击"本人"才能加载档案。

**修复后**：弹窗打开时，默认选中"本人"的同时会自动加载本人的健康档案信息，用户无需任何额外操作即可看到已填写的档案数据。

## 使用说明

### 操作步骤

1. **进入健康自查页面**
   - 打开比尼健康 H5 页面，点击进入健康自查功能（或直接访问 [健康自查页面](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/symptom)）

2. **第一步：选择不适部位**
   - 从九宫格中选择身体不适的部位（头部、眼睛、耳鼻喉、胸部、腹部、腰背、四肢、皮肤、精神）
   - 选中后点击"下一步"

3. **第二步：描述症状**
   - 从常见症状标签中选择一个或多个匹配的症状
   - 可在文本框中补充详细描述（可选）
   - 点击"下一步"

4. **第三步：补充信息**
   - 选择症状持续时间
   - 查看症状摘要确认信息无误
   - 点击"AI 智能分析"

5. **弹出"为谁咨询"弹窗**
   - 弹窗打开后，默认选中"本人"
   - **本人的健康档案信息会自动填充**（出生日期、性别、身高、体重、既往病史、过敏史）
   - 如需修改档案信息，直接在表单中编辑即可
   - 如需为其他家庭成员咨询，点击对应成员即可切换，档案信息随之更新
   - 也可点击"添加家庭成员"新增成员

6. **确认并开始分析**
   - 确认档案信息无误后，点击"确认并开始分析"
   - 系统将跳转到 AI 聊天页面，AI 会根据您的症状和健康档案进行智能分析

## 注意事项

1. **登录要求**：使用健康自查功能需先登录账号，未登录时无法获取家庭成员列表和健康档案
2. **档案数据来源**：弹窗中自动带出的档案信息来自您之前填写和保存的健康档案，如果之前未填写过档案，则对应字段会显示为空
3. **档案可修改**：在弹窗中修改的档案信息，在点击"确认并开始分析"后会自动保存
4. **切换成员**：点击不同家庭成员时，档案表单会自动切换为该成员的数据
5. **新增成员**：通过"添加家庭成员"按钮添加的新成员，添加成功后会自动选中并加载其档案
