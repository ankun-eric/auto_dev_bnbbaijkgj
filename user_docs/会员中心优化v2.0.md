## 功能简介

本次「会员中心优化 v2.0」对会员中心进行了从入口、视觉、数据源到购买闭环的全面升级：

1. **入口前置**：AI 主页右上角"更多"菜单第一项替换为「👑 会员中心」，原扫一扫/字体大小/分享依次后移
2. **视觉升级**：会员中心采用「蓝紫主调 + 金色点缀」全新设计——头部蓝紫渐变用户卡 + 金色皇冠 + 金色等级胶囊
3. **权益卡片化**：原数据写死改为「3 实卡 + 1 占位卡」：守护人数量 / AI 外呼提醒 / 紧急 AI 呼叫 / ✨ 更多权益敬请期待
4. **套餐动态化**：会员套餐区从后台「付费会员套餐管理」实时拉取，月价/年价任一为空时不展示对应按钮，推荐套餐带金色描边
5. **购买闭环**：新增完整的下单→支付→开通流程，支持续费时长追加 + 跨套餐升级，**禁止降级**
6. **订单标记**：所有会员费订单 `fulfillment_type='virtual'` + `membership_plan_id` 填充，商品名格式统一为「【会员费】{套餐名}·{月卡/年卡}」
7. **管理后台**：用户详情新增"会员信息卡片"，支持后台手动延期/降级/重置额度
8. **定时任务**：每日 03:00 自动到期降级；每日 10:00 提前 7/3/1 天到期提醒（接口已就绪，可由 cron 触发）

## 使用说明

### 一、从 AI 主页进入会员中心

1. 在 H5 端首页（AI 主页）右上角点击「⋯」更多按钮
2. 菜单的**第一项**就是「👑 会员中心」，点击即可进入

### 二、查看我的会员权益

进入会员中心后，从上到下依次为：

1. **用户信息卡**（蓝紫渐变背景）
    - 头像 + 金色/白色皇冠（付费会员金冠、普通会员白冠）
    - 昵称
    - 金色胶囊形等级标签：普通会员 / 健康会员 / 尊享会员 / …
    - 有效期：长期（普通会员） 或 `2026-06-25`（付费会员，仅展示日期）

2. **我的会员权益**（白色卡片，悬浮在头部下边缘）
    - 顶部「查看全部 ›」入口
    - 2×2 网格的 4 张卡片：
        - 守护人数量（人）
        - AI 外呼提醒（次/月）
        - 紧急 AI 呼叫（次/月）
        - ✨ 更多权益 / 敬请期待（虚线占位卡，永远在末尾）

### 三、购买/续费/升级会员

1. 滚动到「升级享更多权益」区
2. 推荐套餐头部带金色「👑 推荐」角标，整卡有金色描边
3. 每个套餐显示：套餐名 / 权益概览 / 月卡按钮（蓝紫描边）+ 年卡按钮（蓝紫/金色实心）
4. 点击对应按钮：
    - 普通用户 → 「立即开通」
    - 已是同套餐 → 按钮变灰显示「当前套餐 / 已开通」
    - 已购更高套餐 → 当前套餐之下的低档套餐**整卡隐藏**（不允许降级）
    - 已购低档套餐 → 高档套餐按钮变为「升级月卡/升级年卡」
5. 点击后弹出"开通成功"提示，页面自动刷新，等级、有效期、权益数据均按新套餐立即生效

### 四、查看会员订单（金色尊享卡片）

1. 会员中心底部点击「📦 我的订单」入口
2. 列表顶部有 4 个 Tab：**全部 / 会员费 / 商品 / 积分**
3. 默认进入「会员费」Tab，所有会员费订单使用**金色渐变卡片**展示
4. 已过期会员订单会以略降饱和度的「哑光金」展示

### 五、续费规则

- **同套餐续费**：剩余时长 + 新购周期天数 = 新到期日（不损失旧时长）
- **跨套餐升级**：立即生效，旧时长作废，重新计算到期日
- **跨套餐降级**：系统拒绝，提示"请等当前套餐到期"

### 六、管理员后台操作

1. 登录 [管理后台](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin)
2. 「会员管理 → 付费会员套餐」可新增/编辑/启用停用/删除套餐
    - 月价、年价任一为空，前端用户端就不会展示对应购买按钮
    - 启用"推荐"开关后，前端套餐卡显示金色描边和"推荐"角标
3. 「会员管理 → 免费会员额度」可全局调整普通会员的 4 项配额
4. 用户管理详情页可手动延长会员期、降级、重置额度（新接口已就绪）

## 注意事项

1. **本次为模拟支付**：当前实现为 `simulate=true` 模式，调用 `POST /api/member/order/{id}/pay` 后直接置为支付成功，**未对接真实的微信/支付宝支付通道**。真实通道需在管理后台「支付配置」菜单完成商户号、AppID、密钥配置后才可启用
2. **支付通道按端写死**：小程序=微信 JSAPI；H5=支付宝 H5；App=微信+支付宝双通道二选一（仅前端 UI 占位，未对接 SDK）
3. **占位卡永远在末尾**：「✨ 更多权益 敬请期待」卡片永远是 4 格的最后一格，未来新增权益会从前面顺序填入
4. **降级被拒绝**：在有效期内不能购买比当前套餐档次更低的套餐
5. **定时任务**：到期降级和到期提醒任务对应接口为 `/api/member/_internal/cron/expire` 和 `/api/member/_internal/cron/remind`，可由系统 crontab 每日定时触发

## 后端接口清单

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/member/center` | 会员中心一次性聚合（当前等级 + 权益卡 + 套餐列表 + 排名表） |
| GET | `/api/member/plans` | 启用中的付费套餐列表 |
| POST | `/api/member/order` | 创建会员购买订单（仅 fulfillment_type=virtual） |
| POST | `/api/member/order/{id}/pay` | 发起支付（模拟支付直接 grant） |
| GET | `/api/member/orders` | 我的订单（支持 tab=all/membership/product/points） |
| GET | `/api/admin/users/{id}/membership` | 后台用户会员卡片信息 |
| POST | `/api/admin/users/{id}/membership/adjust` | 后台手动延期/降级/重置 |
| POST | `/api/member/_internal/cron/expire` | 到期降级任务（cron 调用） |
| POST | `/api/member/_internal/cron/remind` | 到期提醒任务（cron 调用） |
