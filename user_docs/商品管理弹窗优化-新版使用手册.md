## 三、弹窗 5 大 Tab 详解

弹窗左侧 Tab，点击可在各分组间切换；任一 Tab 有必填缺失时，Tab 名后会显示**红色小圆点**提醒。

### Tab 1 · 基础信息

最核心的字段区，按"从左到右、从上到下"的顺序填写：

1. **产品条码**（可选，最多 10 个）
   - 输入框中输入条码（1–30 字符），按回车生成一个 Tag；Tag 后点 × 可删除。
   - 条码超过 10 个时无法继续添加。
   - 输入框右侧「实例」按钮可弹出示例图，帮助理解填写规范。

2. **商品名称**（必填，≤50 字）

3. **商品分类** / **履约类型**（必填）

4. **商品规格**（必填，**新增**）
   - 默认"统一规格"：使用商品级售价/原价/库存字段。
   - 选"多规格"：商品级售价/库存**自动隐藏**，下方出现「规格管理」区域，详见 §4。

5. **售价 / 原价 / 库存**
   - 统一规格：这里直接填写数字（售价 ≥0，原价 ≥ 售价，库存整数 ≥0）。
   - 多规格：这三个字段隐藏，库存改为只读「总库存：N」（自动 = Σ 各规格库存）。

6. **商品图片**（必填至少 1 张，≤15 张，单张 ≤5M）
   - 格式：jpg / bmp / png / gif，建议尺寸 750×600。
   - **第 1 张自动成为封面**，左上角有「封面图」角标。
   - 支持**按住拖拽排序**，松手生效；第 1 张自动变成封面。
   - 缩略图右上角 × 可删除。

7. **主图视频**（可选，≤1 个，≤100M，≤3 分钟）
   - 格式：mp4 / mov。
   - 上传后 H5 详情页头部的「视频」Tab 会自动出现。

8. **商品卖点**（可选，≤100 字，**新增**）
   - 超出 100 字会被禁止输入。
   - 右侧「实例」按钮弹出示例文案供参考。

9. **商品描述**（可选，富文本，**改造**）
   - 支持：粗体 / 斜体 / 下划线 / 删除线、H1-H3 标题、有序/无序列表、左中右对齐、文字颜色与背景色（16 色）、字号 12-32、引用块、表格、超链接、**图片上传**、**视频外链/上传**、撤销/重做、清除格式、**全屏编辑**。
   - 图片 / 视频上传后会自动走 CDN 生成外链 URL。
   - 外链视频支持 B 站、腾讯视频、优酷、YouTube 的 iframe 白名单嵌入。

### Tab 2 · 标签设置
- 症状标签 / 适用体质：从老版本平移，交互与规则不变。

### Tab 3 · 积分设置
- 积分相关字段（是否可用积分兑换、积分抵扣规则等）平移至此。

### Tab 4 · 预约与核销设置
- 预约时段、核销次数、核销方式等字段平移至此。

### Tab 5 · 排序与权重设置
- 排序值、推荐权重等字段平移至此。

## 四、多规格（SKU）管理

### 4.1 开启多规格
- 在「基础信息」Tab 将「商品规格」切换为**多规格**。
- 商品级的售价/原价/库存会自动隐藏，并在下方出现**规格管理**表格区，首次切换时自动预置 1 条空白规格行。
- 反向切换回"统一规格"会提示"已录入的规格数据将被清空"，请先确认再继续。

### 4.2 规格表字段

| 列 | 必填 | 规则 |
|---|---|---|
| 规格名称 | ✅ | 1-20 字，同商品内不重复 |
| 售价（元） | ✅ | ≥0，保留 2 位小数 |
| 原价（元） | ❌ | 若填则必须 ≥ 售价 |
| 库存 | ✅ | 整数，≥0 |
| 默认规格 | ✅ | 整表只能勾 1 条，前端详情页默认选中此行 |
| 启用/停用 | - | 停用后前端详情页不展示该规格，但历史订单保留 |
| 操作 | - | 删除按钮；表格下方可「+ 新增规格」 |

- 支持**按住行拖动**调整顺序，前端详情页的展示顺序即为此处顺序。
- 至少保留 1 条规格行，不可全部删除。
- 若删除了当前"默认规格"那一行，系统会自动把第一行设为默认。

### 4.3 规格被订单引用后的特殊规则
- **规格名称 / 售价 / 默认规格**：置灰锁定，鼠标悬停提示"该规格已有订单，不可修改"。
- **删除按钮**：置灰不可点击，悬停提示"请改为停用"。
- **库存 / 原价 / 停用开关**：依然可修改，方便运营做价格微调或下架旧规格。

## 五、强校验：保存 vs. 保存并上架

弹窗底部有三个按钮：

| 按钮 | 行为 |
|---|---|
| 取消 | 有未保存改动时会二次确认 |
| **保存**（宽校验） | 仅校验商品名称、分类、履约类型、规格必填、原价≥售价 |
| **保存并上架**（强校验） | 在宽校验基础上，追加：至少 1 张图片、统一规格库存 >0、多规格至少 1 条库存 >0 且有默认规格 |

任一校验不通过时：
- 系统会**自动切换到错误字段所在的 Tab**。
- 失败字段会红框 + 文案提示。
- 弹窗不关闭，修正后可重试。

> 图片/视频上传中点击保存会提示"请等待上传完成再保存"。

## 六、老数据兼容（无感升级）

打开一个旧商品的编辑弹窗：

- 无条码 → 条码字段留空。
- 老只有单价/库存 → 规格模式自动置为"统一规格"，老字段平移。
- 老「视频链接」→ 回填到「主图视频」字段。
- 纯文本描述 → 自动装入富文本编辑器，`\n` 转换为 `<br/>`。
- 首次保存后，后端按新结构写回，老字段保持兼容读取。

**运营无需手动做任何迁移动作**。

## 八、常见问题

**Q1：条码已满 10 个，怎么办？**
A：先删除不需要的条码 Tag，再添加新的。

**Q2：切到多规格后，为什么售价/库存字段消失？**
A：多规格模式下，售价/库存下沉到规格表；商品级的总库存会自动按各规格求和只读展示。

**Q3：想删除一条规格但删除按钮置灰？**
A：该规格已有订单引用，为保护已下单用户权益，系统不允许删除。可改为「停用」：停用后前端不再展示，历史订单不受影响。

**Q4：富文本插入的图片/视频保存到哪？**
A：均上传到后端 CDN，返回外链 URL 嵌入到富文本 HTML 中保存。最终保存的字段是 HTML 字符串。

**Q5：上传视频提示失败？**
A：请检查是否 ≤100M、格式是否 mp4/mov、时长是否 ≤3 分钟。

**Q6：「保存并上架」点击后自动跳到别的 Tab 是怎么回事？**
A：那个 Tab 里有强校验未通过的字段，红框即为失败位置，请补齐后重试。

**Q7：老商品打开弹窗会不会报错？**
A：不会。系统已做无感兼容，老字段会自动装载到新版结构；首次保存会按新结构写回。

## 九、注意事项

1. **切换规格模式会清数据**：从"多规格"回切"统一规格"会清除已录入的规格行，请谨慎操作。
2. **封面图 = 第 1 张图**：通过拖拽排序来换封面，不需要单独的"设为封面"按钮。
3. **已被订单引用的规格**：编辑/删除受限，这是为了保护已下单用户权益。
4. **富文本 XSS 防护**：后端保存富文本时会白名单过滤；请勿在 HTML 中注入 `<script>` 等危险标签。
5. **上传未完成禁止保存**：避免图片/视频上传未完成就提交导致数据缺失。
