## 功能简介

本次更新在用户端「首页 → 服务」页面的左侧分类列表最前方新增了一个「🔥 推荐」虚拟分类。该分类聚合展示所有被运营标记为「推荐」营销角标的已上架商品，让优质商品获得优先曝光入口，帮助用户快速发现平台精选推荐内容。

### 核心特点

- **推荐分类位于分类列表第一位**，名称前带 🔥 火焰图标
- **默认选中推荐分类**：进入服务页面时，若存在推荐商品，自动选中推荐分类
- **跨分类聚合展示**：推荐商品同时出现在原所属分类和推荐分类中
- **自动隐藏**：当没有任何推荐商品时，推荐分类不显示，页面表现与之前完全一致
- **实时生效**：运营在管理后台修改商品的推荐角标后，用户下次刷新即可看到变化
- **三端一致**：H5 网页端、Flutter APP 端、微信小程序端体验完全一致

## 使用说明

### 在 H5 网页端使用

1. **进入服务页面**：打开 [H5 前端](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)，在底部导航栏点击「服务」进入服务页面

2. **查看推荐分类**：
   - 左侧分类列表的第一位会显示「🔥 推荐」分类
   - 进入页面时默认选中推荐分类，右侧展示推荐商品列表

3. **浏览推荐商品**：
   - 推荐分类下展示所有被标记为「推荐」的已上架商品
   - 商品按推荐权重从高到低排序
   - 点击商品卡片可进入商品详情页

4. **切换分类**：
   - 点击左侧其他分类（如减脂瘦身、中医调理等）可切换查看对应分类的商品
   - 推荐商品同时出现在其原所属分类中

### 在 Flutter APP 端使用

1. **安装最新版 APP**：下载并安装最新的 APK 安装包
2. **进入健康服务**：在底部导航栏点击「服务」或「健康服务」
3. **查看推荐分类**：顶部 Tab 栏第一项为「🔥 推荐」，默认选中
4. **浏览和切换**：操作方式与 H5 端一致

### 在微信小程序端使用

1. **导入最新代码**：将小程序最新代码导入微信开发者工具
2. **进入服务页面**：在底部 Tab 栏点击「服务」
3. **查看推荐分类**：顶部横向分类栏第一项为「🔥 推荐」，默认选中
4. **浏览和切换**：操作方式与 H5 端一致

## 注意事项

1. **推荐分类的显示/隐藏**：推荐分类只有在系统中存在被标记为「推荐」的已上架商品时才会显示。如果没有推荐商品，分类列表与之前完全一致
2. **运营管理**：运营人员可在管理后台的商品编辑页面，通过「营销角标」功能勾选/取消「推荐」来控制哪些商品出现在推荐分类中，无需额外学习新操作
3. **跨分类展示**：被标记为推荐的商品会同时出现在推荐分类和其原所属分类中，这是设计上的聚合视图
4. **排序规则**：推荐分类下的商品按推荐权重从高到低排列，权重相同时按创建时间倒序
5. **管理后台无需改动**：本次功能完全复用已有的「营销角标 → 推荐」机制
