## 功能简介

本次更新对管理后台「首页配置 → 首页菜单管理」中的 Emoji 图标选择器进行了大幅扩充优化，具体包括：

1. **新增 7 个图标分类**：在原有 7 个分类基础上，新增「科技数码」「人物表情」「交通工具」「建筑地标」「天气时间」「办公学习」「金融商务」共 7 个分类，分类总数从 7 个扩展到 14 个
2. **大幅扩充中文标签映射库**：从原有 260+ 条提升至 500+ 条，搜索"机器人""电脑""汽车""学校"等常见关键词均可命中对应 Emoji
3. **现有分类 Emoji 补全**：为「运动健身」「食物饮料」「自然植物」「动物」等现有分类补充了遗漏的常用 Emoji
4. **分类标签栏 UI 适配**：14 个分类标签支持自动换行显示，确保所有分类可见可点击

## 使用说明

### 1. 进入首页菜单管理

1. 打开管理后台：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 使用管理员账号登录
3. 在左侧导航中找到「首页配置」→「首页菜单管理」，点击进入

### 2. 新增或编辑菜单时选择 Emoji 图标

1. 点击「新增菜单」按钮，或点击已有菜单的「编辑」按钮
2. 在弹出的表单中，确保「图标类型」选择为「Emoji」
3. 点击「选择图标」按钮，打开 Emoji 选择面板

### 3. 使用扩充后的 Emoji 选择面板

#### 3.1 浏览分类

- 面板中间区域展示 **14 个分类标签**，自动换行为两行显示
- 点击任意分类标签可切换查看该分类下的所有 Emoji
- **完整分类列表**：
  - 第一行：医疗健康、运动健身、食物饮料、自然植物、活动物品、符号标志、动物
  - 第二行：科技数码、人物表情、交通工具、建筑地标、天气时间、办公学习、金融商务

#### 3.2 搜索 Emoji

- 在面板顶部的搜索框中输入中文关键词
- 支持的高频搜索示例：

| 搜索关键词 | 可命中的 Emoji |
|-----------|---------------|
| 机器人 | 🤖 |
| AI / 人工智能 | 🤖 |
| 电脑 | 🖥️ 💻 |
| 手机 | 📱 |
| 汽车 / 开车 | 🚗 |
| 飞机 / 出差 | 🛩️ |
| 学校 / 教育 | 🏫 🎓 |
| 银行 / 金融 | 🏦 💰 💳 |
| 办公 | 💼 🖥️ 📝 |
| 游戏 | 🎮 🕹️ |
| 天气 | ☀️ 🌧️ 🌤️ |
| 时间 | ⏰ ⏳ 🕐 |
| 火箭 / 冲刺 | 🚀 |
| 老人 / 养老 | 🧓 |
| 宝宝 / 婴儿 | 👶 |

#### 3.3 智能推荐

- 在表单中输入菜单名称后，面板会根据名称自动推荐匹配的 Emoji
- 推荐逻辑与之前保持不变，基于 40+ 组关键词映射

#### 3.4 选中并确认

- 点击任意 Emoji 图标即可选中（选中项会以蓝色边框高亮显示）
- 面板底部会显示「已选择：[所选Emoji]」
- 点击「确定」按钮完成选择

### 4. 保存菜单

- 完成图标选择及其他字段填写后，点击弹窗的「确定」按钮保存菜单

## 注意事项

1. **已有菜单不受影响**：本次优化不会改变任何已有菜单的图标，所有已配置的菜单图标保持不变
2. **Emoji 显示风格**：不同操作系统（Windows、macOS、iOS、Android）的 Emoji 显示风格可能略有不同，这是系统原生渲染效果，属于正常现象
3. **较新 Emoji 兼容性**：部分较新的 Emoji（如 🛰️ 卫星、🪙 硬币等）在较旧的操作系统上可能显示为方框，属于系统版本限制
4. **Emoji 跨分类出现**：部分 Emoji 可能同时出现在多个分类中（如 📱 同时在「活动物品」和「科技数码」中），方便从不同维度查找
5. **搜索去重**：通过搜索找到的 Emoji 结果会自动去重，同一个 Emoji 只会显示一次
6. **权限要求**：使用此功能需要管理员权限
