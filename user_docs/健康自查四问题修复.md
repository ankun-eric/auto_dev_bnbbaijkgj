## 功能简介

本次集中修复用户在 AI 对话页「健康自查」流程中反馈的 4 个体验问题：

1. **结果卡片布局问题**：标题太小、字段堆叠、「查看详情」按钮无反应。
2. **AI 追问内容空泛**：点击"居家如何处理"等 chip 时，AI 回答与本次问卷症状无关。
3. **Q5 严重程度选项重复**：原 10 档每两档共用同一文案。
4. **Q6 题干与提示语完全重复**。

修复后，您将获得以下体验改进：

- ✅ **统一的通用结果卡**：白底圆角卡片，顶部色块 + 标题 + 主结论 + 4 个关键字段（部位 / 主症状 / 持续时间 / 严重程度）+ 可点击的「查看详情 ›」按钮。
- ✅ **真正能跳转的详情页**：点击「查看详情 ›」打开全新的 6 区块详情页：答题记录、AI 解读、居家建议、就医警示（红框）、推荐商品、底部 CTA。
- ✅ **结构化的 AI 追问**：AI 追问回答固定包含【针对性建议】【何时需就医】【注意事项】三段，并自然引用本次问卷的关键症状（如"头部·搏动性头痛·7/10·4-7 天"）。
- ✅ **Q5 严重程度 5 档单值**：轻微 / 轻度 / 中度 / 重度 / 剧烈，每档文案不重复。
- ✅ **Q6 题干 / 提示语分离**：题干"还有什么想补充告诉小白的吗？（选填）"，提示语"例如：受凉后加重 / 每天下午 3 点准时出现 / 服用布洛芬后缓解"。
- ✅ **后台可配置 AI 追问字段**：运营在管理后台「问卷模板」编辑页可自定义 AI 追问要注入的关键字段名（默认：部位 / 症状 / 严重程度 / 持续时间）。

## 使用说明

### 1. 用户侧：完成一次健康自查

1. 打开 H5 用户端首页：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)
2. 登录后进入「AI 健康管家对话」首页。
3. 在底部菜单 / 胶囊条中找到「健康自查」按钮，点击打开问卷抽屉。
4. 按 6 步顺序填写：
   - **Q1 部位**（多选 + 图标网格）
   - **Q2 症状**（多选，根据部位自动过滤可选项）
   - **Q3 持续时间**（单选：今天 / 1-3 天 / 4-7 天 / 1-4 周 / 1 个月以上）
   - **Q4 症状性质**（单选 + Emoji 标签：搏动 / 钝痛 / 烧灼 / 刺痛 / 麻木 / 酸痛 / 紧绷 / 瘙痒 / 跳过）
   - **Q5 严重程度** ✨ 修复后：滑块 5 档单值（1 轻微 / 3 轻度 / 5 中度 / 7 重度 / 10 剧烈）
   - **Q6 备注** ✨ 修复后：题干为"还有什么想补充告诉小白的吗？（选填）"，输入框 placeholder 是填写示例
5. 提交后回到对话页，自动出现"健康自查"结果卡：
   - 🩺 健康自查 · 完成时间
   - 主结论（如"头部·搏动性头痛"）
   - 4 个关键字段：部位 / 主症状 / 持续时间 / 严重程度
   - 「查看详情 ›」按钮 ✨ 修复后：可点击跳转

### 2. 用户侧：查看详情

1. 在结果卡上点击「查看详情 ›」按钮。
2. 进入全新的 **健康自查结果详情页**（路径 `/health-self-check/result/{id}`），自上而下 6 个区块：
   - **顶部摘要条**（天蓝渐变）：AI 一句话结论 + 完成时间
   - **B1 您的答题记录**：按题号顺序列出 Q&A
   - **B2 AI 解读**：基于关键症状的整体说明 + 📌 关键症状高亮框
   - **B3 居家处理建议**：5 条可执行建议（休息 / 饮食 / 监测 / 用药等）
   - **B4 ⚠️ 出现以下情况请立即就医**（红色框）：红线信号清单
   - **B5 为您推荐**：横滑商品卡（按本次自查动态匹配）
   - **B6 底部固定 CTA**：「重新填写」「找医生咨询」两个按钮

### 3. 用户侧：AI 追问 chip 体验改进

✨ 修复后，结果卡下方的 chips 行点击任意一条（"居家如何处理"/"注意事项"/"是否需要就医"）后，AI 必定输出三段式回答：

```
本次回答结合您的档案。关于您关心的「居家如何处理」，结合本次自查关键信息：
📌 部位：头部；症状：搏动性头痛；严重程度：7/10；持续时间：4-7 天

【针对性建议】
1. 缓解症状：根据症状性质选择对应方式（如热敷/冷敷、休息、抬高患处）；
2. 调整作息：今日先减少高强度活动，保证充足睡眠；
3. 简单饮食干预：多饮温水、清淡饮食，避免酒精与咖啡因；
4. 监测变化：每 2-4 小时给症状重新打分（0-10），观察变化趋势；
5. 必要药物：在医生既往建议范围内可选择常用 OTC 药品，剂量勿超说明书。

【何时需就医】
- 居家观察 24-48 小时无改善或反而加重 → 建议就诊；
- 出现剧烈疼痛、意识模糊、持续呕吐、明显出血等急性表现 → 立即就医；
- 老人 / 儿童 / 孕妇 / 慢病患者出现新症状 → 不要拖延，尽早就诊。

【注意事项】
- 不要自行联合多种处方药，避免相互作用；
- 居家期间避免独居高风险行为（如开车、操作机械）；
- 如确诊基础病，请按医嘱继续既有治疗方案；
- 自查结果仅供参考，不能替代医生诊断。
```

不再出现"请查看上方卡片"这种空泛话术。

### 4. 运营侧：配置 AI 追问关键字段（可选）

1. 进入管理后台：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 左侧菜单 → **问卷模板**。
3. 找到「健康自查」模板，点击「编辑」。
4. 滚到"问卷完成后体验配置"区块，找到新增的 **AI 追问关键字段**。
5. 多选/输入字段名（默认 `部位 / 症状 / 严重程度 / 持续时间`），保存即可生效。
6. 用户下次完成问卷时，AI 追问 prompt 中只会注入这些字段对应的答案值。

## 注意事项

1. **历史数据**：本次修复采用 D2 方案，**首次启动后已删除全部旧的健康自查答卷与题目**，重新 seed 一份干净的 6 题模板。本次修复部署后这一步只执行一次，后续重启不会再清空。
2. **AI 追问无外部依赖**：当前 AI 追问采用结构化模板渲染，不依赖外部大模型 API，回答确定性高、响应快；后续可平滑切换到 function calling 接入真实 AI。
3. **自查结果仅供参考**：详情页底部已默认提示"不能替代医生诊断"，请运营在推广文案中保持一致口径。
4. **语音输入 Q6 暂未实现**：本次按需求暂不开发，将进入下次迭代待办（TODO-1）。
5. **回退方案**：如生产环境部署后需要回滚，可回退代码至本次 commit 前的版本；但 D2 已删除的历史 answer/result **不可恢复**，需提前确认是否允许丢失。
