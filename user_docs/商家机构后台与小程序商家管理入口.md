## 一、功能介绍

本次上线了覆盖**自营分店**与**合作机构**（体检、家政、其他）的完整"商家经营闭环"，同时在三个端补齐入口：

1. **小程序 C 端「我的」** 新增「商家管理」菜单（仅商家身份账号可见，自动识别，点击即可进入商家工作台）。
2. **PC Web 商家/机构后台** 提供独立子路径 `/merchant`，面向老板 / 店长 / 核销员 / 财务四档角色，支持订单、核销、报表、对账、发票、导出与打印。
3. **平台 Admin** 新增「机构类别管理」和「对账单管理」两个菜单，用于维护机构类型与月度对账。

所有新功能的访问入口均以 **`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27`** 为根路径。

## 三、小程序端：「商家管理」入口使用

### 3.1 显示规则

- 非商家账号：「我的」页看不到此菜单项。
- 商家账号：在「我的」页"功能服务"区块内看到 **🏪 商家管理** 菜单，与"我的订单"并列。
- 多门店商家：菜单右侧会显示 `N家门店` 徽标。

### 3.2 操作步骤

1. 打开小程序 → 底部 Tab 切到**「我的」**。
2. 点击 **🏪 商家管理**。
3. 系统实时校验商家身份：
   - 仅绑定 1 家门店：直接进入该门店的商家工作台。
   - 绑定多家门店：弹出门店选择页，选中任意门店进入工作台。
4. 在商家端「我的」顶部可点击 **「切回用户端」** 返回 C 端首页。

### 3.3 注意事项

- 如短时间内平台管理员取消了您的商家身份，再次点击会被提示"您当前不是商家账号"并自动隐藏菜单。
- 如您是刚新建、尚未绑定任何门店的商家账号，请联系平台客服协助开通门店归属。

## 四、PC Web 商家/机构后台使用

### 4.1 登录

1. 浏览器访问：https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/login/
2. 输入**手机号 + 密码** 或 **手机号 + 短信验证码** 登录。
3. 多门店账号进入门店选择页，选择当前工作门店。

### 4.2 角色与权限

| 角色 | 工作台 | 订单 | 核销 | 报表 | 对账 | 发票 | 员工 | 门店设置 |
|---|---|---|---|---|---|---|---|---|
| 老板 | ✅ | ✅ | ✅ | ✅（跨店合并） | ✅ | ✅ | ✅ | ✅ |
| 店长 | ✅ | ✅ | ✅ | ✅（本店） | ❌ | ❌ | ✅（本店） | ✅（本店） |
| 核销员 | ✅ | 只读 | ✅（本人） | ❌ | ❌ | ❌ | ❌ | ❌ |
| 财务 | ✅ | 只读 | ✅ | ✅ | ✅ | ✅ | ❌ | ❌ |

> **注意**：权限由平台管理员在 Admin 端统一配置，商家端只能查看不能修改，如需调整请联系平台客服。

### 4.3 工作台首页

登录后默认进入 `/merchant/dashboard/`，头部展示机构名、当前门店切换器与核心指标卡片：

- 今日订单数
- 今日核销数
- 本月 GMV
- 待对账金额
- 待处理附件订单数
- 未读消息

### 4.4 订单管理（附件上传）

1. 左侧菜单选择 **订单管理**。
2. 顶部筛选区支持按**订单号 / 用户手机号 / 状态 / 日期范围** 过滤。
3. 点击操作列**「附件」**打开侧边抽屉：
   - **上传附件**：选择图片（JPG/PNG/WebP）或 PDF。
   - **下载/删除**：对已有附件进行管理。
4. **限制**：
   - 单文件 ≤ **20 MB**。
   - 单订单最多 **5 个**附件。
   - 仅图片与 PDF 格式。
5. 订单附件仅作为附加信息，**不影响订单状态流转**；核销前后均可上传。

> **机构端不可修改订单状态**：取消、改期、退款等变更类操作由平台客服统一处理。

### 4.5 核销记录

- 可按时间范围、门店、核销员、用户手机号等条件筛选。
- 支持一键导出 Excel。

### 4.6 报表分析

1. 顶部切换**时间维度**：日 / 周 / 月。
2. 顶部切换**对账维度**：机构维度（合并）/ 门店维度（分店）。
3. 展示内容：
   - 订单趋势
   - GMV（订单金额 / 实际收款 / 已结算 / 待结算）
   - TOP 商品
   - 新增用户 / 复购率

### 4.7 对账结算

1. 平台每月 **1 号 00:00** 自动生成上月对账单（机构 + 门店双视图）。
2. 商家在 **对账结算** 列表中可：
   - **查看详情**：下载 PDF / 下载明细 Excel / 打印。
   - **确认对账单**：核对无误后点击「确认」。
   - **发起异议**：填写异议原因，平台客服介入处理。
3. 平台财务完成**线下打款**后，上传打款凭证，状态自动变为「已结清」。

### 4.8 发票信息

商家可在 **发票信息** 页维护：
- 开票抬头、税号、开户行 / 账号、注册地址 / 电话、收票地址、邮箱。

### 4.9 员工与权限

- 只读视图：展示所有成员的手机号（脱敏）、昵称、门店、角色、状态。
- 需要增删改权限请联系平台客服。

### 4.10 下载中心

1. 列表或报表页点击 **导出** 会在此页生成导出任务。
2. 任务状态：排队 → 处理中 → 已完成 → 可下载（有效期 7 天）。
3. **限制**：
   - 单次导出时间范围 ≤ **1 年**。
   - 同一机构**每分钟最多发起 1 次**导出任务。

### 4.11 消息中心

- 接收平台通知：用户下单、客服改期、到店签到、对账单出账 / 结清。
- 不发送短信。如已关注平台公众号并完成 UnionID 绑定，会同步推送模板消息。

## 五、平台 Admin 侧新增功能

### 5.1 机构类别管理

入口：`/admin/merchant-categories/`

功能：
- 查看、新增、编辑、禁用/启用机构类别。
- 字段：编码（`self_store` / `medical` / `homeservice` / `other` 等）、显示名、图标、允许附件类型、附件标签、排序、状态。
- 小程序商家端工作台会根据类别显示不同的标题和附件标签。

### 5.2 对账单管理

入口：`/admin/admin-settlements/`

功能：
- **生成月度对账单**：选择年月，系统汇总该月所有机构的订单并生成对账单。
- **上传打款凭证**：选中对账单 → 上传图片/PDF 作为打款凭证 → 状态自动置为"已结清"。
- **异议处理**：协助机构方解决异议账单，修订后重新出账。

## 八、常见问题

**Q1：我是商家，但小程序「我的」没看到商家管理菜单？**
A：1) 确认账号是否已被平台管理员配置为商家成员；2) 下拉刷新「我的」页触发异步身份校验；3) 仍无请联系平台客服。

**Q2：登录 PC 后台提示"非商家账号无法登录"？**
A：商家后台仅允许商家/机构员工登录；如需平台管理请访问 `/admin/`。

**Q3：附件上传失败？**
A：检查文件大小是否超过 20MB、格式是否为图片或 PDF、附件数是否已达 5 个上限。

**Q4：为什么导出一直在"处理中"？**
A：大数据量导出会异步处理，完成后会在「下载中心」显示为"已完成"并推送站内消息。如需要重试可在任务列表中点击对应按钮。

**Q5：对账单金额有疑问怎么办？**
A：在对账单详情页点击「发起异议」，填写原因后平台客服会在 3 个工作日内跟进。

**Q6：PC 后台可以用手机打开吗？**
A：可以基础查看，但主推 PC 端（1366×768 起），建议使用 Chrome / Edge / Safari 最新版。
