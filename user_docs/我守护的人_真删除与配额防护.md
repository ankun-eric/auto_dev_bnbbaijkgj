# 我守护的人 · 真删除与配额防护 用户体验手册

> 版本：v1.0（bugfix-guardian-list-consistency-v1）  
> 更新日期：2026-05-29  
> 适用范围：H5 端、AI 首页

---

## 一、本次更新做了什么

本次更新围绕"我守护的人"列表与"AI 首页选择咨询人"列表，解决了**列表口径不一致**、**移除不彻底**、**邀请漏洞**、**配额可被刷**等多个体验问题。

### 1. 两个列表口径一致了

| 入口 | 显示内容 |
|---|---|
| 健康档案 → 我守护的人 | 本人 + 已绑定 + 邀请中 + 代管档案 + 「对方已退出」灰标 |
| AI 首页 → 左下角"选择咨询人" | 本人 + 已绑定 + 代管档案 + 「对方已退出」灰标（不包含"邀请中"） |

被你移除 / 拒绝 / 过期 / 彻底删除的对象，**两个列表都看不到**。

### 2. 「移除」改名为「彻底删除」，且真正物理删除

之前的"移除"只是把数据隐藏，本次起改名为**「彻底删除」**：
- 永久删除该家人的健康档案、健康报告、AI 对话记录、用药提醒、紧急联系人引用等共 **8 类数据**；
- 删除前会展示数据量清单，二次确认后才会真正执行；
- **不可恢复**，请务必谨慎操作。

### 3. 邀请时姓名必填了

之前直接发起邀请时姓名可以留空，会产生"无名"档案。现在：
- 在「+新增 → 去邀请」表单里，**姓名为必填项**（红色 \*），留空无法提交；
- 后端 422 拦截兜底，姓名空格也会被识别为空。

### 4. 配额公式更严了（防刷）

| 操作 | 配额变化 |
|---|---|
| 创建一条"暂未响应"的邀请 | 占用 1 个名额 |
| 邀请自然过期（24 小时无响应） | 释放 1 个名额 |
| 撤回未接受的邀请 | 释放 1 个名额 |
| 彻底删除档案 | 释放 1 个名额 |

配额上限按你的会员等级动态计算（免费用户默认 5 位守护人）。

### 5. 频次防护（每日上限 5 次）

- 创建邀请：5 次 / 自然日
- 解除守护：5 次 / 自然日
- 彻底删除：5 次 / 自然日

超出后会给出友好提示，请明日再试或联系客服。

### 6. 「对方已退出」状态保留

如果你守护的人在 App 里主动取消了被守护关系，你这边卡片不会消失，而是变成**灰色卡片 + 「对方已退出」标签**，提供两个按钮：
- **「重新邀请」**：复用原档案 ID 重新发起邀请，对方接受即可恢复关系；
- **「彻底删除」**：走真删流程，永久清理该家人在你账户中的全部数据。

## 二、操作步骤（H5 端）

### 入口

或 AI 首页：

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)

### 步骤 1：新增并邀请守护对象

1. 在「我守护的人」页面，点击右上角「+ 邀请家人」按钮；
2. 在弹出抽屉中：
   - **姓名**（必填）：填写对方的真实姓名或称呼，例如"张妈妈"；
   - **关系**（必填）：从下拉中选择，如"母亲"；
   - **手机号**（可选）：如填写，邀请短信会直接发到对方手机；
3. 点击「发起邀请」生成邀请码。**留空姓名无法提交**。

### 步骤 2：彻底删除一个家人档案

> 此动作不可恢复，请先确认对方已不需要被守护。

1. 找到该家人的卡片（必须是**非"已绑定"状态**：邀请已撤回 / 对方已退出 / 已拒绝 等）；
2. 点击卡片下方的「**彻底删除**」按钮（红色文字）；
3. 系统会弹出**影响清单**：
   ```
   ⚠️ 彻底删除「张妈妈」
   
   此操作不可恢复，将永久删除以下数据：
   • 健康档案 1 份
   • 健康报告 12 份
   • AI 对话历史 28 条
   • 用药提醒 5 条
   • 紧急联系人引用 2 处
   
   确定继续？
   ```
4. 仔细核对数字后，点击「**确定彻底删除**」；
5. 如果当前正在和对方的 AI 对话中，对话会自动关闭并跳回 AI 首页。

### 步骤 3：对方主动退出后的处理

1. 卡片自动变成灰色 + 「对方已退出」标签；
2. 在两个列表（守护人列表 + AI 选择咨询人）中都仍可见，AI 对话历史保留；
3. 可选两种处理方式：
   - **「重新邀请」**：弹窗确认后会以原 member_id 重新发起邀请，对方接受即可恢复关系；
   - **「彻底删除」**：清理你账户中的全部相关数据。

## 三、注意事项

### 安全提示

1. **彻底删除不可恢复**：删除前请务必查看数据量清单，确认无误后再操作。建议在删除前先备份关键报告 PDF。
2. **删除前置条件**：以下情况无法直接彻底删除，需先解决后再试：
   - 守护关系仍处于"已绑定"状态 → 请先解除守护关系；
   - 对方账户名下有绑定的硬件设备 → 请先在「设备管理」里解绑；
   - 还有未撤回的"邀请中"邀请 → 请先撤回未接受的邀请；
   - 该家人名下有"进行中"的服药提醒 → 请先终止用药计划。
3. **频次限制**：单日操作上限为 5 次，避免账号异常时被恶意刷接口；正常用户极少触达。

### 配额规则

- "暂未响应"的邀请也会占用配额，请勿故意刷邀请；
- 邀请 24 小时后未被接受会自动过期，对应的名额会自动释放；
- 想"快速腾出名额"的最快方式是：撤回未接受的邀请 → 配额 -1。

### 数据迁移

- 历史上 `nickname` 为空的"暂未响应"邀请已批量回填为"待激活成员"占位字符；
- 历史"已移除"的家人卡片在新版本中**默认不再显示**，但数据仍保留在后台（可由运维灰度清理）。

## 四、常见问题（FAQ）

**Q1：我之前移除过的人现在还能看到吗？**  
A：升级后不会再显示。如需彻底删除请联系客服由运维处理。

**Q2：彻底删除后还能找回吗？**  
A：**不能**。这是物理删除，建议删除前先导出/备份关键数据。

**Q3：对方退出守护后，AI 对话历史会消失吗？**  
A：不会。卡片保留为灰色，对话历史也保留，直到你主动「彻底删除」。

**Q4：为什么我点击「彻底删除」提示"请先解除守护关系"？**  
A：处于"已绑定（active）"状态时不能直接彻底删除，需先解除守护关系，目的是防止误删正在使用中的家人档案。

**Q5：今天点了 5 次邀请就不能再点了？**  
A：是的，5 次是每日上限，第二天自然日 0 点自动重置。如有特殊情况请联系客服。
