## 使用说明

### 一、首页城市定位（H5 / 小程序 / App）

1. **打开应用首页**，在搜索栏左侧可以看到城市名称显示区域
2. **首次进入**时，应用会请求您的位置权限：
   - **允许授权**：应用将自动获取您的当前城市并显示（如"北京"）
   - **拒绝授权**：显示"定位"文字，您仍可手动选择城市
3. 城市名称右侧有一个**下拉箭头 ▼**，表示可以点击切换城市

### 二、切换城市

1. **点击首页搜索栏左侧的城市区域**（如"北京 ▼"），进入城市选择页面
2. 城市选择页面从上到下依次为：
   - **搜索框**：输入城市名称可实时搜索（支持中文和拼音）
   - **GPS 定位城市**：显示您当前定位的城市，点击可快速选择
   - **最近访问**：显示您最近选择过的城市（最多 3 个），点击可快速切换
   - **热门城市**：显示热门城市列表（北京、上海、广州、深圳等 12 个城市），点击可选择
   - **全国城市列表**：按拼音首字母 A-Z 排列所有城市
3. **右侧字母索引条**：点击任意字母可快速跳转到对应分组
4. **选择城市后**，自动返回首页，搜索栏左侧更新为您选择的城市名称

### 三、管理后台 — 热门城市管理（仅管理员）

1. **登录管理后台**：访问管理后台地址并登录
2. 在左侧菜单栏找到**"城市管理"**菜单，点击进入
3. 页面显示当前配置的**热门城市列表**
4. **添加热门城市**：点击页面顶部"添加热门城市"按钮，在弹窗中搜索城市并添加
5. **调整排序**：使用"上移"和"下移"按钮调整热门城市的显示顺序，完成后点击"保存排序"
6. **移除热门城市**：点击操作列的"移除"按钮即可取消某城市的热门标记

## 注意事项

1. **位置权限**：城市定位功能需要您的位置权限。如果拒绝授权，不影响其他功能使用，但无法自动定位城市，需手动选择
2. **定位精度**：定位精度为城市级别，不会获取您的精确地址
3. **城市切换**：用户手动选择的城市优先级高于 GPS 定位城市，下次打开应用仍显示您手动选择的城市
4. **最近访问**：最近访问城市列表最多保留 3 条记录，按时间倒序排列
5. **H5 端**：在 HTTPS 环境下才能使用浏览器定位功能
6. **小程序端**：首次使用定位功能时，微信会弹出位置授权确认框
7. **管理后台**：热门城市建议控制在 8-16 个，确保城市选择页视觉美观
