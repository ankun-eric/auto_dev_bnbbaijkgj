## 使用说明

### 功能一：体检报告 AI 解读

1. 打开 H5 页面或 APP，进入首页
2. 点击「体检报告」功能入口
3. 点击「相册」或「拍照」上传体检报告图片（支持 JPG/PNG 格式，最多 5 张）
4. 上传成功后，系统自动进行 OCR 识别 + AI 智能解读
5. 解读完成后展示：
   - **健康评分**：综合评分及评级
   - **指标分析**：按分类展示各项指标，标注异常项
   - **详细建议**：针对异常指标提供饮食、运动、生活习惯等个性化建议

### 功能二：健康自查（症状持续时间必选）

1. 进入「健康自查」页面
2. **第一步**：选择不适部位（如头部、胸部、腹部等）
3. **第二步**：选择具体症状（可多选）
4. **第三步**：**必须选择症状持续时间**（今天刚开始 / 1-3天 / 一周内 / 一个月内 / 超过一个月），未选择将提示"请选择症状持续时间"
5. 点击「AI智能分析」按钮，选择咨询人
6. 填写咨询人信息后确认，AI 将根据症状进行智能分析

### 功能三：咨询人信息必填（全成员）

1. 在健康自查的咨询人选择弹窗中，选择任意家庭成员
2. 无论选择"本人"还是其他家庭成员，姓名、性别、出生日期三个字段都**必须填写**
3. 各字段旁显示红色 `*` 必填标识
4. 如已有成员档案信息，系统自动带入，无需重复填写
5. 未填写完整时点击确认，系统提示"请填写完整的必填信息"

## 注意事项

1. **网络要求**：使用体检报告解读功能需要网络连接，AI 解读需要一定处理时间（通常 10-30 秒）
2. **图片要求**：上传的体检报告图片请尽量清晰、完整，模糊或不完整的图片可能导致 OCR 识别失败
3. **文件大小**：单张图片不超过 20MB，最多同时上传 5 张
4. **个人信息**：填写的咨询人健康档案信息会被保存，下次使用时自动带入
5. **免责声明**：AI 解读结果仅供健康参考，不构成医疗诊断或治疗建议，如有异常请及时就医
