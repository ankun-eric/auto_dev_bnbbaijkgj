## 三、操作步骤

### 1. 进入健康档案 v2

1. 在浏览器中打开：  
   `https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile-v2`
2. 顶部 Hero 卡片即时展示「既往病史 / 过敏史 / 家族遗传 / 长期用药」**四格健康摘要**。

### 2. 编辑基本信息（M2 新增）

1. 点击 Hero 卡片右上角 **「编辑基本信息」** 按钮。
2. 在弹层中可修改：姓名、性别、生日、身高、体重、血型。
3. 点击「保存」，Hero 卡片即时刷新。

### 3. 新增/编辑用药计划（M4 重做）

1. 在档案中切换至 **「用药计划」** Tab。
2. 顶部「**今日用药 / 全部用药计划**」分段切换，可只看今日待提醒的药物。
3. 点击右上角 **「+ 添加用药」** 进入添加页。
4. 在添加页可设置：
   - **药品名称、剂量**（也可点击「📷 拍照识药」自动识别）
   - **每日次数** 1～6 次（次数变化时，时间点会自动配齐）
   - **服药时间点**：支持自定义时间，可逐条增删，弹出时间选择器选择
   - **开始日期**（默认今日）/ **结束日期**，或勾选「长期服用」
   - **用药提醒开关**（默认开启）
   - **关联疾病标签**（高血压、糖尿病、高血脂等多选，可自定义）
5. 点击「保存」即可写入计划，并按时间点生成提醒。

### 4. 编辑家族病史（M6 新增）

1. 切换至 **「健康信息」** Tab，点击「编辑」。
2. 在弹层中下滑至 **「家族病史」** 区。
3. 点击 **「+ 添加家族病史」**：
   - 选择关系（父亲、母亲、兄弟、姐妹等）
   - 选择疾病（糖尿病、高血压、心脏病等预设，或自定义输入）
   - 备注（可选，如"55 岁确诊"）
4. 不需要的条目，点击右侧「🗑 删除」即可。
5. 慢病栏目新增 **「确诊年份」**，可填写如 2018。

### 5. 编辑手术史（M6 新增）

1. 同样在 **「健康信息」** Tab 的编辑弹层中找到 **「手术史」** 区。
2. 点击 **「+ 添加手术」**：填写手术名称、时间（年月）、备注。
3. 支持多条，点击「🗑 删除」可移除。

### 6. 上传病历卡 + OCR 解析（M8 新增）

1. 切换至 **「健康事件」** Tab。
2. 点击右上角 **「📷 病历卡」** 按钮，从相册或拍照选取一张病历照片。
3. 系统将：
   - 自动 OCR 识别图片文字
   - 解析出：医院名称、就诊科室、诊断、就诊日期、医生姓名、处方
   - 在「病历卡列表」中横向滚动展示该卡片
   - 自动生成一条「上传病历」类型的时间轴事件
4. 点击病历卡 → 查看详情（图片预览 + 解析字段），可执行 **删除** 操作。

## 四、注意事项

1. **图片要求**：拍照识药 / 病历卡上传支持 JPG/PNG/HEIC，建议 **正面、清晰、文字不反光**，识别准确率更高。
2. **隐私保护**：所有病历、药品、家族病史均按 **家庭成员维度隔离存储**，仅本人/共管家人可见。
3. **数据同步**：保存后无需刷新页面，列表即时更新；如发现未同步，可下拉刷新。
4. **OCR 解析仅供参考**：自动识别的医院/诊断/医生等字段可能存在偏差，建议在详情页核对，必要时手动修改。
5. **提醒开关**：用药提醒默认开启。如不想接收提醒，可在添加页关闭"用药提醒"开关，已加入计划但不发送通知。
6. **长期服用**：勾选后无需填写结束日期，系统会持续提醒直至手动归档。
7. **首次使用**：若是新建档案账号，Hero 四格摘要的数值会显示为 0，待你逐步录入既往病史/过敏史/家族遗传/长期用药后即时累加。

## 五、常见问题（FAQ）

**Q1：拍照识药识别错误怎么办？**  
A：识别后名称会自动填入「药品名称」输入框，可手动修改后再保存。

**Q2：自定义服药时间点最多几个？**  
A：每日次数最多支持 6 次，对应最多 6 个时间点。

**Q3：病历卡能否替换图片？**  
A：当前版本不支持替换，建议删除原卡后重新上传。

**Q4：家族病史与既往病史有何区别？**  
A：**既往病史**记录本人确诊；**家族病史**记录直系亲属遗传/确诊情况，用于辅助医生评估遗传风险。

**Q5：编辑基本信息后哪些功能会同步更新？**  
A：身高/体重/生日变化会影响首页 BMI、年龄相关健康指标的展示，建议保持准确。
