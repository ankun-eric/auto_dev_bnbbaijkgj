## 功能简介

本次更新修复了「宾尼小康」AI 健康管家平台中的两个 Bug：

1. **体检报告 AI 解读失败**：用户在 H5 网页端上传体检报告图片后，AI 解读总是失败，显示"分析失败，请重试"。
2. **后台 OCR 测试按钮无响应**：管理员在后台 OCR 配置页面点击"测试 OCR"按钮后，选择图片点击"开始识别"无任何结果返回。

---

## 修复详情

### Bug 1：体检报告 AI 解读失败

**修复内容：**
- AI 模型调用超时从 60 秒延长至 120 秒，避免复杂体检报告解读因超时而失败
- 新增自动重试机制（最多 3 次尝试，间隔 2 秒），网络波动时自动重试
- 优化错误处理：AI 调用失败时明确抛出异常，而非静默返回错误字符串
- 增加详细的后端日志记录，便于问题排查
- 前端显示更具体的错误提示（超时/连接失败/其他错误）

### Bug 2：后台 OCR 测试按钮无响应

**修复内容：**
- 修正前端上传文件字段名（从 `file` 改为 `files`），与后端 API 参数名保持一致
- 修复后"测试 OCR"按钮能正常发送请求并显示识别结果

## 使用说明

### 体检报告 AI 解读（面向普通用户）

1. 打开 H5 网页端首页：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 登录账号后，点击首页的「体检报告」入口
3. 点击上传，选择一张体检报告图片（支持 JPG、PNG 格式，或 PDF 文件）
4. 图片上传成功后，系统自动跳转到结果页面，显示"AI 正在分析..."动画
5. 等待 10-30 秒后，页面将展示完整的结构化解读结果，包括：
   - **健康评分**：0-100 分的综合健康评分，含评级和评语
   - **检测项目统计**：总项数、异常项数、优秀项数
   - **分类指标详细解读**：按分类（如血常规、肝功能等）展示各项指标的检测值、参考范围、风险等级及详细建议

### 后台 OCR 测试（面向管理员）

1. 登录管理后台：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 进入「OCR 识别配置」页面
3. 选择需要测试的厂商 Tab（如百度云）
4. 点击「测试 OCR」按钮，打开测试对话框
5. 选择一张包含文字的图片，点击「开始识别」
6. 等待几秒后，对话框中将显示 OCR 识别出的文字结果
7. 如识别失败，将显示明确的错误提示信息

## 注意事项

1. **AI 解读依赖 AI 模型服务**：体检报告 AI 解读功能需要后台正确配置 AI 模型（API Key、模型地址等）。如 AI 解读仍然失败，请管理员检查后台 AI 模型配置。
2. **OCR 测试依赖 OCR 服务**：OCR 测试功能需要后台正确配置 OCR 厂商参数（百度云/腾讯云/阿里云的 API Key 等）。
3. **图片清晰度**：上传的体检报告图片应尽量清晰、文字可辨认，模糊的图片可能导致 OCR 识别失败。
4. **文件大小限制**：单个上传文件不超过 20MB。
5. **重试机制**：如遇网络波动，系统会自动重试最多 3 次（每次间隔 2 秒），总超时最长约 6 分钟。
