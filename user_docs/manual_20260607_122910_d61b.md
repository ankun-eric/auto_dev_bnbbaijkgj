## 功能简介

本次对「益智乐园」页面（`/brain-game`）进行了以下优化：

1. **游戏卡片横向布局**：首页三张游戏卡片（数学游戏、排行榜、组队挑战）由原来的纵向大卡片改为横向紧凑布局——图标在左、文字在右，三张卡片在手机上一屏内即可全部看到，浏览更快捷。

2. **删除「随便选一个」按钮**：省市区选择弹窗中去除「随便选一个」按钮，简化操作流程，用户仅需正常从四级菜单中选择居住地即可。

3. **统一入口**：删除旧的静态 HTML 文件 `brain-game.html`，现在访问益智乐园统一使用 `/brain-game` 路径。

## 使用说明

### 如何进入益智乐园

1. 打开宾尼健康首页
2. 在长辈版首页找到「乐龄游戏」卡片（图标 🧩，紫色渐变），点击即可进入益智乐园

### 如何玩数学游戏

1. 进入益智乐园后，点击「数学游戏」卡片
2. 选择难度等级：基础训练、进阶训练或挑战训练
3. 在限定时间内完成算术题，得分自动统计
4. 答题结束后可查看得分和排名，并可邀请好友来挑战

### 如何查看排行榜

1. 进入益智乐园后，点击「排行榜」卡片
2. 先选择您所在的街道（点击右上角「选择街道」按钮）
3. 选择后可查看街道、区域、城市三个维度的排名

### 如何组队挑战

1. 进入益智乐园后，点击「组队挑战」卡片
2. 创建挑战或输入挑战编号加入好友的队伍
3. 组队完成后开始答题 PK

- 首次使用需先选择居住街道，才能参与排行榜
- 数学游戏答题有时间限制，请在规定时间内完成
- 组队挑战需要与好友分享挑战编号，好友加入后方可开始
