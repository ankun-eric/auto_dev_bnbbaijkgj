## 功能简介

本次更新对会员中心页中「邀请家人」入口做了一次视觉与信息升级。原先一个普通按钮的入口被替换为一张**视觉权益感更强的渐变大卡片**，让你在会员中心首屏就能：

- **看懂**：当前是哪一档套餐、可以管理几位家人；
- **看清**：当前已经管理了几位、还剩几位名额；
- **点击**：通过白色大按钮"邀请家人"一步进入既有邀请流程；
- **感知上限**：达到套餐上限时按钮自动禁用，并直接告知"升级套餐"路径，避免邀请失败才发现达上限的尴尬。

> 邀请流程本身（弹窗 / 邀请码 / 链接发送等）**完全保持不变**，仅入口卡片视觉与交互升级。

## 使用说明

### 第 1 步：进入会员中心

在 App / 微信小程序 / 浏览器中打开 [会员中心](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center/) 页面。

### 第 2 步：找到邀请家人入口卡片

在页面顶部「用户信息卡（蓝紫渐变）」之后，会看到一张**蓝色渐变大卡片**，左侧带有家人手拉手剪影图标。这就是新版"邀请家人入口卡片"。

卡片上自上而下依次展示：

1. **主标题**：套餐名 + 权益短语，例如 `家庭版 · 可管理 10 位家人`；不限档显示为 `尊享版 · 不限家人数`；
2. **用量行**：例如 `已管理 3 / 上限 10`（含本人口径）；不限档展示为 `已管理 3 人 · 不限上限`；
3. **辅助说明**：固定文案 `邀请家人，共享健康守护`；
4. **白色"邀请家人"大按钮**：右下角胶囊形按钮。

### 第 3 步：邀请家人（正常态）

只要你当前管理人数 **未达套餐上限**：

- 点击右下角的 **"邀请家人"** 白色大按钮，会跳转到既有的「邀请家人」流程页（输入手机号 / 生成邀请码 / 分享链接 / 微信扫码等），按既有流程完成邀请即可。
- 如果你点击了卡片其他区域（非按钮、非链接），效果与点击按钮等价。

### 第 4 步：达上限态（已管理 ≥ 上限）

当你已邀请的家人数（含你本人）达到套餐上限时，卡片会自动切换到"达上限态"：

- **"邀请家人" 按钮变灰禁用**：点击无任何反应（不会报错、不会弹 toast，按设计保持沉默）；
- **用量行下方新增黄色警示**：`⚠ 已达上限，请升级套餐`；
- **"升级套餐" 四字带白色下划线**：可单独点击。

点击 **"升级套餐"** 链接：

- 如果当前页面下方有「升级享更多权益」区，会自动平滑滚动定位到该区，挑选合适档位升级；
- 升级成功后回到本页，卡片会刷新为最新档位与配额，按钮恢复可点击。

### 第 5 步：不限档套餐永远可邀请

如果你购买的是**不限档套餐**（例如尊享版/不限家人数），卡片永远展示为正常态：

- 主标题：`尊享版 · 不限家人数`
- 用量行：`已管理 X 人 · 不限上限`
- 主按钮永远可点击；不会出现"达上限"警示。

## 注意事项

1. **数据口径（含本人）**：用量数据 `已管理 X / 上限 Y` 都是「含你本人」口径。即如果你的套餐 `max_managed = 10`，则你和其他 9 位家人合计不超过 10 人。这与会员中心页面其他区域（家庭守护成员配额、健康档案配额）的口径完全一致，保证全产品统一。
2. **登录后才显示真实数据**：未登录或会话失效时，卡片可能展示为兜底文案（如"会员套餐 · 可管理家人"），并不会让页面出错。
3. **接口零新增**：本次需求**没有任何新增后端接口**，卡片所需的 `plan_name`、`max_managed`、`max_managed_used` 等数据全部来自现有 `/api/member/center` 与 `/api/family/member/quota`，不会引入额外网络开销。
4. **会员中心其他区域不变**：用户信息卡、4 格权益、套餐对比表、底部入口区等其他模块外观与交互**与改造前完全一致**，本次升级仅替换"邀请家人入口"位置的视觉。
5. **微信小程序自动同步**：微信小程序的"会员中心"页面由 web-view 加载本 H5 页面，因此你**无需重新下载小程序**即可立即看到新卡片。
6. **达上限态的容错**：如果由于历史脏数据导致 `已管理 > 上限`，卡片同样按"达上限态"渲染，不会崩溃。
7. **网络异常**：当 `/api/family/member/quota` 接口短暂不可达时，卡片会用 `/api/member/center.current.max_managed` 作为兜底上限值，并把 `已管理` 临时显示为 0；恢复后下次进入页面会自动校准。
