## 功能简介

本次修复了 **H5 Web 客户端订单详情页预约信息缺失**的 Bug。

修复前，用户在 H5 网页版查看订单详情时，无法看到预约信息（预约时间、预约门店等）。修复后，订单详情页在商品信息卡片下方新增了**预约信息卡片**，清晰展示预约时间和预约门店名称。

---

## 使用说明

### 查看订单预约信息

1. **打开 H5 客户端**：在浏览器中访问 [H5 首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)

2. **进入订单列表**：点击底部导航栏的「订单」或进入「我的订单」页面

3. **查看订单详情**：点击任意一个到店核销类订单（即购买时选择了预约的商品），进入订单详情页

4. **查看预约信息**：在商品信息卡片下方，可以看到新增的**预约信息卡片**，其中包含：
   - 📅 **预约时间**：显示您下单时选择的预约日期和时间段（蓝色高亮显示）
   - 📍 **预约门店**：显示该订单对应的门店名称

### 预约信息卡片展示效果

```
┌─────────────────────────────────────┐
│  预约信息                            │
│                                     │
│  📅  预约时间   2026-04-29 14:00-15:00  │  ← 蓝色高亮
│  📍  预约门店   xxx健康中心（朝阳店）    │  ← 正常深色
└─────────────────────────────────────┘
```

### 页面布局说明

预约信息卡片位于订单详情页的以下位置：

1. 订单状态栏
2. 核销码区域（待使用状态下显示）
3. 商品信息卡片
4. **📅 预约信息卡片** ← 新增位置
5. 物流信息（如有）
6. 备注信息（如有）
7. 订单进度

---

## 注意事项

1. **预约信息仅在有预约数据时显示**：如果订单是纯物流配送商品（无需预约），订单详情页不会显示预约信息卡片

2. **预约门店信息**：只有当订单关联了门店时，才会显示预约门店一行。部分订单可能只有预约时间而无关联门店

3. **其他客户端不受影响**：本次修复仅涉及 H5 Web 客户端。Flutter APP 端的订单详情页预约信息展示一直正常，无需更新

4. **时间格式**：预约时间以「年-月-日 时间段」格式显示，例如「2026-04-29 14:00-15:00」
