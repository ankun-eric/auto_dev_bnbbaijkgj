## 功能简介

在此之前，健康指标的「全部历史记录」页里，**点击一整行手工录入的记录**就会弹出一个「修改 / 删除」操作面板。这是一种和其他页面不一样的"特殊待遇"。

本次更新去掉了这个特殊处理，让交互回归通用做法：

- **点整行不再弹菜单**：手工录入的记录，点击整行不会再有任何弹窗反应。
- **操作入口收敛到「⋯」**：要修改或删除某条记录，请点击该行**右侧的「⋯」三点按钮**。
- **两端一致**：H5 端与微信小程序端表现完全一致（小程序的指标历史页本质就是加载这同一个 H5 页面，所以改一处、两端自动同步）。

因为 H5 历史记录页是**血糖 / 血压 / 心率 / 血氧通用同一个页面**，所以这几个指标的历史记录都会一起统一为"只点「⋯」操作"。

## 操作步骤

### 一、查看历史记录

1. 打开 H5 健康首页：[点此进入](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)。
2. 进入某位家庭成员的健康档案，打开**血糖**（或血压、心率、血氧）的指标详情页。
3. 点击「全部历史 / 查看更多」进入全部历史记录页。

### 二、修改或删除一条记录（新方式）

1. 在历史列表中找到要操作的**手工录入**记录。
2. 点击该行**右侧的「⋯」三点按钮**（不要再点整行——点整行现在没有反应）。
3. 在弹出的操作面板中选择：
   - **修改**：打开编辑弹窗，调整数值/时段后点击「保存」。
   - **删除**：会弹出二次确认窗，确认后该记录被删除，列表自动刷新。

### 三、查看设备同步记录（保持不变）

- 对于**智能设备同步**的记录（只读、不可编辑），点击整行**仍会弹出只读详情**供查看，这一行为没有改变。
- 设备同步记录是锁定的，如需修正请重新测量。

| 设备同步记录（只读） | 弹只读详情查看 | **保持原样**，仍弹只读详情 |
| 右侧「⋯」三点按钮 | 点开 → 修改 / 删除 | **保持不变**，仍是操作入口 |

## 注意事项

- **操作入口已统一到「⋯」**：如果您习惯了点整行弹菜单，请改为点击右侧的「⋯」按钮，功能完全一致，只是入口位置更规范。
- **修改 / 删除功能完全保留**：包括删除前的二次确认弹窗，均未改动。
- **设备同步记录仍只读**：来自智能设备的记录无法修改或删除，点击整行仅展示详情。
- **影响范围**：血糖、血压、心率、血氧四个指标的历史记录页共用同一页面，本次统一调整对它们同时生效。
- **微信小程序无需单独更新**：小程序的指标历史页是加载 H5 页面的"空壳"，H5 更新后小程序端自动同步，无需重新安装或导入小程序。
