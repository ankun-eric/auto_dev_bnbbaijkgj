## 功能简介

本次更新包含 1 个 Bug 修复和 2 个功能优化，涉及 H5 移动端、微信小程序、Flutter 客户端三端同步改动：

### Bug 修复：进度动画与 AI 接口不同步

**问题**：用户提交体检报告后，进度动画是纯前端定时器模拟的，与后端 AI 接口响应完全脱钩。当 AI 响应超过动画时长时，用户看到"分析完成"但实际接口还未返回，造成页面"卡住"的感知。

**修复方案**：
- 将固定定时器动画替换为**渐近百分比进度条**，进度永远不会自行到达 100%
- 增加 **60 秒超时保护**，超时后提示用户已转入后台处理
- 增加**错误分类提示**，根据不同错误类型显示对应提示信息

### 优化 1：上传时选择咨询对象

用户可以在上传体检报告时选择这份报告是为谁做的（本人、爸爸、妈妈等家庭成员），AI 解读时会使用对应家庭成员的健康档案信息。

### 优化 2：报告列表显示咨询对象标签

在体检报告历史列表中，每条报告卡片上显示彩色的咨询对象标签，让用户一眼区分每份报告是谁的。

## 使用说明

### 一、上传体检报告并选择咨询对象

1. 打开 H5 移动端，进入**体检报告**页面
2. 页面顶部会显示 3 步步骤条：**上传报告** → **选择对象** → **AI 解读**
3. 点击**从相册选择**或**拍照**上传体检报告图片（最多 5 张，每张不超过 20MB）
4. 上传图片后，点击**开始识别**按钮
5. 此时会弹出**选择咨询对象**弹窗（步骤条切换到"选择对象"）
6. 在弹窗中选择这份报告是谁的：
   - 可选择"本人"或已添加的家庭成员
   - 如需添加新的家庭成员，点击弹窗中的"添加家庭成员"
7. 选择完成后，系统自动开始上传和识别（步骤条切换到"AI 解读"）

#### 各端交互差异

| 端 | 选择咨询对象的交互方式 |
|---|---|
| **H5 移动端** | 底部弹窗 Popup |
| **微信小程序** | 页内横向滑动选择 |
| **Flutter 客户端** | 整页步骤式选择 |

### 二、AI 解读进度显示

上传完成后进入 AI 解读页面，会看到**渐近百分比进度条**：

| 阶段 | 进度范围 | 提示文案 |
|------|---------|---------|
| 阶段 1 | 0% → 30% | 🔍 AI 正在分析您的指标… |
| 阶段 2 | 30% → 60% | 📊 正在评估各项指标的风险等级… |
| 阶段 3 | 60% → 90% | 📝 正在生成个性化健康建议… |
| 阶段 4 | 90% → 99% | 🧠 AI 正在深度分析中，请耐心等待… |
| 完成 | 100% | ✅ 分析完成！ |

- 进度条会**逐渐放慢**，越接近 99% 越慢，但不会自行到达 100%
- 只有 AI 分析真正完成后，进度才会跳到 100% 并展示结果
- 如果 **60 秒后**仍未完成，会提示"已转入后台处理"，您可以稍后在报告列表中查看

### 三、查看报告列表中的咨询对象标签

1. 在体检报告页面下方的**历史记录**列表中
2. 每条报告卡片的**标题行右侧**会显示彩色标签
3. 标签颜色含义：

| 关系 | 颜色 | 示例 |
|------|------|------|
| 本人 | 🔵 蓝色 | 自己的报告 |
| 爸爸 | 🟠 橙色 | 为爸爸上传的报告 |
| 妈妈 | 🩷 粉色 | 为妈妈上传的报告 |
| 配偶 | 🟣 紫色 | 为配偶上传的报告 |
| 子女 | 🩵 青色 | 为子女上传的报告 |
| 其他 | ⚪ 灰色 | 其他家庭成员 |

4. 历史报告（更新前上传的报告）默认显示为"本人"

### 四、错误处理

如果 AI 分析过程中出现错误，系统会根据错误类型显示不同的提示：

| 错误类型 | 提示信息 |
|---------|---------|
| 网络异常 | 网络连接异常，请检查网络后重试 |
| 服务器繁忙 | 服务器繁忙，请稍后重试 |
| 报告不存在 | 报告不存在或已被删除 |
| 其他错误 | 分析失败，请重试 |

出现错误时可以：
- 点击**「重新分析」**按钮重试
- 点击**「返回报告列表」**按钮回到报告列表页

## 注意事项

1. **家庭成员管理**：如果您还没有添加家庭成员，可以在选择咨询对象时点击"添加家庭成员"进行添加
2. **AI 解读时间**：AI 分析时间取决于报告复杂度和服务器负载，通常在 5-30 秒内完成
3. **超时处理**：如果超过 60 秒未完成，系统会将分析转入后台，您可以稍后在报告列表中查看结果
4. **向下兼容**：更新前上传的历史报告不受影响，在列表中会默认显示"本人"标签
5. **图片要求**：上传的体检报告图片应清晰可读，支持 JPG/PNG 格式，每张不超过 20MB
