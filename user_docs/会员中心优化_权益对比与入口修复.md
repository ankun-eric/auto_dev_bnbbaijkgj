## 功能简介

本次更新围绕「会员中心」做了 4 处关键优化，让运营配置即时生效、消除入口 404、统一品牌话术：

1. **权益对比表「免费会员」列数值动态化**：会员中心底部的「权益对比」表格中，"免费会员"列的"守护人上限 / AI 外呼提醒 / 紧急 AI 呼叫"三个数值，现在**实时从管理后台「免费会员额度配置」读取**。运营改完即时生效，付费用户和免费用户看到的数值都是一致的全局额度。
2. **全端统一改称「免费会员」**：参考主流健康类产品（华为终端云 / 苹果 Care 等以"免费版"区分付费档）的命名风格，把全代码库内的"普通会员"统一改为"免费会员"，让用户更直观理解"我现在是免费档，付费可解锁更多守护额度"。
3. **修复「我的订单」入口 404**：会员中心底部「我的订单」点击不再跳到 404 页，正确跳转到 H5 已有的「统一订单列表页」 `/unified-orders`。
4. **修复「我的优惠券」入口 404 + 加"回会员中心"快捷入口**：会员中心底部「我的优惠券」点击不再跳到 404，跳转到 `/my-coupons`；同时在优惠券页面顶部导航栏右侧、空状态页中央各增加一个「回会员中心」入口，方便用户来回切换。

## 使用说明

### 一、查看权益对比表「免费会员」列实时数值（H5 用户端）

1. 用浏览器打开 [会员中心](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center) 并登录
2. 向下滚动到「权益对比」模块
3. 表头第一列显示**「免费会员」**（不再是"普通会员"）
4. 三行的"免费会员"列数值（守护人上限 / AI 外呼提醒 / 紧急 AI 呼叫）来自管理后台配置，不再随登录用户档位变化

### 二、运营修改免费额度并立刻生效（管理后台）

1. 用管理员账号登录 [管理后台 - 免费会员额度配置](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/membership/free-quota)
2. 修改「守护人上限 / AI 外呼提醒 / 紧急 AI 呼叫 / 被守护上限」任一字段
3. 点击「保存」，页面将出现"已保存：所有免费会员的额度立即按新配置生效"
4. 在用户端（[会员中心](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center)）刷新页面，「权益对比」表的"免费会员"列数值即时变成新值（无需任何缓存清理）

### 三、从会员中心进入「我的订单」

1. 打开 [会员中心](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center)
2. 滚动到页面底部的入口区
3. 点击「📦 我的订单」
4. 页面跳转到 [统一订单列表页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/unified-orders)（不再 404）

### 四、从会员中心进入「我的优惠券」并返回

1. 打开 [会员中心](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center)
2. 滚动到页面底部的入口区
3. 点击「🎫 我的优惠券」
4. 页面跳转到 [我的优惠券](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/my-coupons)（不再 404）
5. **回到会员中心的两种方式**：
   - **方式一**：点击页面右上角导航栏的「👑 会员」按钮（与"兑换码"并列）
   - **方式二（仅无券时可见）**：在空状态页中点击中央的「去会员中心看看套餐」按钮
