## 一、功能介绍

### 1.1 我们做了什么？

之前登录页的图形验证码字符太小、容易看错，常常导致登录失败。本次升级把它**整体替换为"滑块拼图验证"**：你只需要把右边的拼图块拖到左边图片的缺口位置，就能完成验证。

**升级后的优势：**
- ✅ **再也不用辨认模糊小字符**：拼图块清晰可见，缺口位置一目了然
- ✅ **手机/电脑统一体验**：手机上可触摸拖动，电脑上可鼠标拖动
- ✅ **更安全**：在阻挡撞库脚本的同时，正常用户操作丝滑流畅
- ✅ **「换一张」永远在线**：图片不喜欢？一键换一张

## 二、操作步骤

### 2.1 商家端 PC 后台登录（电脑端）

1. 用电脑浏览器打开 [商家端 PC 登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/login)
2. 输入手机号、密码
3. 在密码下方会看到一张 **320×160** 的滑块图：
   - 图片左侧某个位置有一个**镂空缺口**
   - 图片下方有一个**滑块条**，左端是**拼图块**
4. **按住鼠标**拖动拼图块向右移动，把它对准图片中的缺口
5. 松开鼠标，系统自动校验：
   - ✅ 校验通过 → 显示"验证成功"
   - ❌ 校验失败 → 滑块自动回到起点，提示"验证失败，请重试"，可重新拖动或点「换一张」
6. 验证通过后，点击「登录」按钮即可

### 2.2 商家端 H5 登录（手机端）

1. 用手机浏览器打开 [商家端 H5 登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/merchant/m/login)
2. 输入手机号、密码
3. 滑块图自动适配为 **280×140**，方便手指操作
4. **手指按住**滑块上的拼图块，**水平向右拖动**，把它对准图片中的缺口
5. 松开手指，系统自动校验
6. 验证通过后点「登录」

### 2.3 平台管理后台登录（仅管理员）

1. 用电脑浏览器打开 [管理后台登录页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/login)
2. 输入手机号、密码
3. 完成滑块验证（操作与商家 PC 后台一致）
4. 点击「登 录」

### 2.4 滑块界面元素说明

| 元素 | 说明 |
| --- | --- |
| 背景图（带缺口） | 一张随机抽取的小图，图中有一个深色镂空区域 |
| 拼图块 | 与缺口形状完全相同的小块，初始位于图片下方滑条的最左端 |
| 滑动条 | 拼图块拖动的轨道，长度等于图片宽度 |
| 提示文字 | 默认显示"按住滑块，拖动完成拼图"；操作过程中会变成"松开手指/鼠标进行验证"等 |
| 「换一张」按钮 | 滑块右上角的小图标按钮，随时可点击换一张新图 |

## 三、常见状态与提示

| 出现的提示 | 含义 | 建议操作 |
| --- | --- | --- |
| **验证成功** ✅ | 拼图位置正确、轨迹合理，已自动给登录请求附上验证凭证 | 直接点「登录」按钮即可 |
| **位置不对，请重试** | 拼图块没对准缺口（误差 > 5 像素） | 滑块自动归位，再拖一次，看准缺口位置 |
| **轨迹异常** | 拖动过快/过慢、过于死板（如纯水平直线） | 用正常的速度自然拖动即可，不要"瞬移" |
| **验证已过期，请重新拖动滑块** | 滑过验证后超过 5 分钟才点击登录，凭证已失效 | 重新滑一次即可 |
| **验证太多次，请稍后再试**（带倒计时） | 同一 IP 在 60 秒内连续失败 ≥ 3 次，已被锁定 60 秒 | 等待倒计时结束（屏幕会显示剩余秒数），自动解锁后再操作 |

## 四、注意事项

### 4.1 操作建议
- **不要太快**：滑动总耗时低于 200 毫秒会被判为机器脚本
- **不要太慢**：超过 8 秒未松手也会失败，建议 1～3 秒内完成
- **手指/鼠标不要完全水平直线**：自然拖动会有轻微上下抖动，这是正常的；纯直线反而会被判为脚本
- **拖到缺口正中间松手**：允许 ±5 像素的误差，看准位置再松开

### 4.2 失败排查
| 现象 | 可能原因 | 解决办法 |
| --- | --- | --- |
| 一直显示"位置不对" | 缺口看不清/拖偏了 | 点击右上角「换一张」按钮换一张更清晰的图 |
| 拖动后没反应 | 网络异常 | 检查网络后点「换一张」重试 |
| 突然全部禁用且显示倒计时 | 触发了风控锁定 | 等倒计时结束（最长 60 秒），自动恢复 |
| 滑过验证再点登录提示"已过期" | 滑完后超过 5 分钟才登录 | 重新拖一次滑块 |

### 4.3 跨端使用须知
- 商家 H5、商家 PC、管理后台**互相独立**：在一个端拖完滑块，**不能**直接拿到另一个端登录
- 用户端 H5 暂未升级，仍使用**原来的字符验证码**，不会看到滑块界面

### 4.4 安全提示
- 滑块验证不会收集任何个人隐私，仅记录拖动轨迹用于辨别真人
- 一次验证凭证只能登录一次，登录失败后需重新拖动滑块
- 凭证有效期 5 分钟，超时自动作废
