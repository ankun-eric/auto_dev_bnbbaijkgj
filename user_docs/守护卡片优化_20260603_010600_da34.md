## 功能简介

本次对「健康档案」中的两张守护卡片做了 4 项安全性与易用性优化：

1. **解绑高危操作补上二次确认**：「家庭成员」卡片的「解除守护」此前点击直接执行；「守护我的人」卡片的「解除」已有弹框。本次将两者**统一使用同一套提醒文案**，避免误触。
2. **发邀请必填「名字」**：在「守护我的人」发邀请页，关系选择下方新增名字输入框（必填），保证邀请信息完整。
3. **被守护上限校验**：保存邀请前先校验你能被几个人守护（`max_managed_by`，免费会员默认 **3**）。已达上限时弹「升级提示框」，提供「去升级」按钮跳转会员页。
4. **邀请二维码可找回**：「守护我的人」列表中，邀请中的卡片在「取消邀请」按钮**后面**新增「查看邀请码」按钮，可随时重新打开该邀请的二维码页面。

## 使用说明

### 1) 「家庭成员」卡片 - 解除守护（二次确认）

1. 打开 H5 / 小程序，进入「健康档案 → 家庭成员」。
2. 点击某位家人卡片进入「守护详情」页。
3. 滑到底部点击「解除守护」按钮。
4. 系统会弹出统一二次确认框：
   - 标题：**解除守护**
   - 内容：**解除后对方将无法查看您的健康数据，确定解除吗？**
   - 按钮：**取消** / **确定**
5. 点击「确定」才真正执行解除；点击「取消」或弹框外区域则取消操作。

### 2) 「守护我的人」卡片 - 解除（统一文案）

1. 进入「健康档案 → 守护我的人」。
2. 在某位守护者卡片右侧点击「解除」按钮。
3. 弹出与上方完全相同的二次确认框，文案统一。

### 3) 发邀请：必填名字 + 上限校验

1. 进入「守护我的人」页面，底部点击「邀请别人守护我」。
2. 在新邀请页：
   - **第一步**：在「请选择 TA 与您的关系」中选择一种关系（爸爸、妈妈、老公、老婆、儿子、女儿、哥哥姐姐弟弟妹妹、爷爷奶奶外公外婆、朋友、其他）。如选「其他」需在下方输入框补充关系名称（1~8 字）。
   - **第二步**：在「名字」输入框填写对方的名字（必填，去掉首尾空格后非空即可）。
   - **第三步**：点击「保存」按钮。
3. 系统会先校验被守护上限：
   - **未达上限** → 立即生成邀请二维码，可保存到本地、复制链接、或转发微信好友。
   - **已达上限** → 弹出升级提示框：
     - 标题：**已达被守护上限**
     - 内容会显示当前已被守护人数（含邀请中）与上限值
     - 按钮：**稍后再说** / **去升级**
   - 点击「去升级」会跳转到会员页面进行升级；升级后即可获得更高被守护上限。

### 4) 查看邀请码（找回已发邀请的二维码）

1. 进入「守护我的人」列表。
2. 找到状态为「**邀请中**」的卡片（带橙色「待确认」小标签）。
3. 卡片右侧依次有两个按钮：
   - 「取消邀请」：取消该邀请并释放名额。
   - 「**查看邀请码**」：点击重新打开该邀请的二维码页面，方便再次保存到本地、复制链接或转发好友。

## 注意事项

- **被守护上限默认值**：免费会员可被 **3 人**守护（包含「邀请中」状态）。付费会员的上限以你当前会员套餐为准。
- **关系 + 名字均填写后**才能点击「保存」生成二维码；任一为空都会保留按钮置灰状态。
- **取消邀请会立即释放名额**：取消后被守护人数会回降 1，可再次发起新邀请。
- 「查看邀请码」按钮**仅对邀请中状态的卡片可见**，已生效的守护者卡片右侧只有「解除」按钮。
- 解除守护是**不可恢复操作**，请慎重确认。如对方仍需要守护你，需要重新发邀请走完整流程。
