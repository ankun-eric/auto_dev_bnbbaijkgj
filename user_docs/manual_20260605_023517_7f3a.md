## 功能简介

本次改版对关怀模式首页（care-ai-home）进行了以下优化：

1. **数字安全绳功能整合**：从旧版 care-home 页面搬入数字安全绳，放在功能卡片区第 1 位，长辈可第一时间看到并签到平安
2. **悬浮球布局优化**：右下角悬浮球改为竖排排列，绿色安全绳球在上、红色 SOS 球在下，双入口更易触达
3. **模式切换入口简化**：模式切换改为欢迎区右上角半透明白色胶囊按钮，简洁直观
4. **旧版页面废弃**：care-home 页面已移除，访问时会自动导向新版

## 使用说明

### 1. 进入关怀模式

首次使用可通过 [模式选择页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/welcome-mode) 选择「关怀模式」，之后会自动跳转到关怀版首页。

### 2. 功能卡片区

关怀版首页主体为 6 张大字整行功能卡片：

| 排序 | 卡片名称 | 点击进入 |
|------|---------|---------|
| 第1位 | 🪢 数字安全绳 | 数字安全绳签到页（每天点一下"我今天平安"） |
| 第2位 | 💊 用药提醒 | 今日完整用药提醒列表 |
| 第3位 | 📈 健康记录 | 血压、血糖、心率、血氧、睡眠记录 |
| 第4位 | 🛡️ 居家安全 | 紧急呼叫、烟雾报警、水浸报警 |
| 第5位 | 🆘 紧急呼叫 | 一键 SOS 求助、联系家人与急救 |
| 第6位 | 🪪 个人信息卡 | 身份与健康名片，便于出示与求助 |

### 3. 数字安全绳

数字安全绳位于卡片区第 1 位（绿色横卡），点击进入后可进行：
- **每日签到**：点击"我今天平安"按钮签到
- **设置签到阈值**：设定每天最晚签到时间
- **紧急联系人**：超过阈值时间未签到，自动通知亲友

同时，右下角悬浮球区新增绿色安全绳球（🪢 图标，位于红色 SOS 球上方），可快速跳转数字安全绳页面。

### 4. 模式切换

在关怀版首页的欢迎区（暖橙渐变区域）右上角，有一个半透明白色胶囊按钮，显示「去标准版 ?」。点击即可快速切换到标准模式。

切换到标准模式后，标准版首页欢迎区右上角同样会显示「去长辈版 ?」按钮，可随时切回关怀模式。

### 5. 悬浮球区

右下角固定显示两个悬浮球（竖排排列）：

| 位置 | 悬浮球 | 样式 | 功能 |
|------|--------|------|------|
| 上方 | 🪢 安全绳球 | 绿色渐变圆形 | 快速跳转数字安全绳 |
| 下方 | SOS 球 | 红色渐变圆形 + 扩散光圈 | 紧急呼叫求救 |

两个球之间间距 14px，互不遮挡，方便长辈准确点击。

## 注意事项

1. 关怀模式与标准模式使用同一后端账号体系，切换模式不会丢失数据
2. 数字安全绳签到功能需要先设置签到阈值和紧急联系人
3. SOS 紧急呼叫功能需要在系统设置中授予通知权限
4. 旧版 care-home 页面已废弃，原有链接会自动导向新页面
5. 如遇到任何使用问题，可通过首页「⊕ 更多菜单 → 帮助与反馈」提交反馈
