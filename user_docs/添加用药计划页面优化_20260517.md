## 功能简介

本次对「添加用药计划」页面做了大幅交互优化，**核心目标：让中老年用户尽量"选择"而不是"手工填"**。所有可选字段都改为点选/滚轮/卡片，参考晓医、阿福等主流医疗 App 的简洁风格。

主要改动概览：

| 字段 | 改造前 | 改造后 |
|------|--------|--------|
| 药品名称 | 纯手工输入 | 输入 ≥ 2 字自动联想下拉（最多 6 条），支持手工兜底 |
| 用药时间 | 单一时间选择 | 抽屉式：4 个频次卡片 + 单行时间卡片（仅显示时间数字） |
| 每次剂量 | 两栏下拉 | 两栏滚轮，数量新增 ¼、½ 两个特殊值 |
| 服用周期 | 开始/截止日期 + 长期开关 | 起始日固定"今天" + 持续天数滚轮（默认 4 天，范围 1–90），右侧自动显示结束日期，移除"长期"开关 |
| 服用时机 | 多选 | 5 个圆角标签**单选**：饭前 / 饭后 / 空腹 / 随餐 / 睡前 |
| 备注 | 多行文本 | 单行输入框，最多 30 字，实时字数显示 |
| 提醒开关 | 页内开关 | **本页移除**，默认开启（在系统设置中心管控） |

## 操作步骤

### 第 1 步：打开"新增用药计划"页面

进入路径（任选其一）：

- 健康档案 → 用药摘要卡右上角「+ 新增」
- 用药提醒打卡页（[/ai-home/medication-reminder](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/medication-reminder)）→ 右下浮动「+」
- 用药计划列表页 → 右下浮动「+」
- 直接打开 [新增用药计划页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/medication-plans/new)

### 第 2 步：填写药品名称（必填）

- 在「药品名称」输入框中输入药品关键词（如"阿莫"）。
- **输入到第 2 个字开始**自动弹出最多 6 条联想下拉。
- 点击下拉项 → 完整药品名一键填入。
- 如果联想没有匹配项，可以**直接手工输入**保存，系统不强制必须选库内药品。

### 第 3 步：设置用药时间（必填）

点击「用药时间」字段 → 弹出底部抽屉。

1. **顶部 4 个频次卡片**：单选 `1次/天`、`2次/天`、`3次/天`、`4次/天`。
2. **下方时间卡片**：根据频次动态显示对应数量的时间卡片，**强制单行**。
   - 默认时间点：
     - 1 次/天 → `08:00`
     - 2 次/天 → `08:00 / 18:00`
     - 3 次/天 → `08:00 / 12:00 / 18:00`
     - 4 次/天 → `08:00 / 12:00 / 18:00 / 22:00`
   - 卡片**只显示时间数字**，无"早/中/晚/睡前"等文字，无图标，无序号。
3. **点击任意一张时间卡片** → 弹出底部"时:分"滚轮，滚到目标时间后点「确定」即可应用，「取消」则不修改。
4. 调整完成后点抽屉底部「确定」关闭抽屉。

### 第 4 步：设置每次剂量（必填）

点击「每次剂量」字段 → 弹出底部抽屉，左右两栏滚轮：

- **数量轮**：包含 `¼`、`½`、`1`、`2` … `500` 等常见数量。
- **单位轮**：包含 `片`、`粒`、`袋`、`支`、`瓶`、`贴`、`g`、`mL`、`mg`、`μg`。
- 滚到目标值后点「确定」。

### 第 5 步：设置服用周期（必填）

- **开始日**固定为「今天」（不可修改，仅展示）。
- 点击 `持续 X 天` 按钮 → 弹出底部滚轮，范围 **1–90 天**，默认 4 天。
- 滚到目标天数后点「确定」，右侧自动显示 `至 X月X日`。

### 第 6 步：选择服用时机（必填）

下方一行**圆角标签单选**：饭前 / 饭后 / 空腹 / 随餐 / 睡前。

- 一行排不下会自动换行（最多 2 行）。
- 点击高亮选中，必须有一项被选中。

### 第 7 步：填写备注（可选）

- 单行输入框，最多 30 字。
- 右侧实时显示字数 `X/30`。
- 超过 30 字会自动截断，无法继续输入。

### 第 8 步：点击「保存」

- **保存按钮始终可点击**（不做置灰）。
- 点击时统一校验，校验失败会按顺序 Toast 提示首个缺失项（如「请先填写药品名称」）。
- 保存成功后 Toast 提示「添加成功」并**返回上一页**。

## 注意事项

1. **联想触发阈值**：输入 1 个字不弹下拉，**≥ 2 个字**才开始联想。
2. **未匹配可手动保存**：药品库中没有匹配项时，可继续手工输入药品名保存。
3. **频次切换的时间点保留规则**：
   - 例如从 3 次/天切到 4 次/天，前 3 个时间点会**保留**，新增的第 4 个填入默认值；
   - 从 2 次/天切回 1 次/天，**恢复**默认 `08:00`（不保留之前的微调，避免歧义）。
4. **持续天数硬限**：滚轮最大滚到 90 天，本期不做"长期/到期续期提醒"。
5. **服用时机为单选**：与改造前的多选不同，本次按 PRD 改为**单选**。
6. **备注超 30 字会直接禁止输入**，无 Toast 弹窗。
7. **提醒开关已移除本页**：所有用药计划默认开启提醒，按"用药时间"中每个时间点推送；如需统一控制可前往系统设置中心。
8. **支持编辑模式**：在用药计划列表点击单条进入详情页，使用同一表单组件可编辑已有的用药计划。

## 常见问题

**Q1：药品名称联想为什么不出来？**
A：请确认已输入 ≥ 2 个字符；如果联想接口超时或失败，会静默不弹下拉，您可以继续手工输入并保存。

**Q2：为什么没有"长期"开关了？**
A：本期参考晓医、阿福做法，**最大 90 天**即可覆盖绝大多数常规疗程；长期慢性病用药可在到期前重新创建。后续会迭代到期续期提醒。

**Q3：保存后跳转到哪？**
A：默认返回上一页。如果您从用药计划列表进入，会回到列表；如果从抽屉进入，会回到来源页。

**Q4：可以选择过去的开始日期吗？**
A：本次「新增用药计划」页面的开始日固定为**今天**。如需补录过去的服药记录，可使用"用药计划列表 → 单条编辑"页支持的过去日期补录功能。
