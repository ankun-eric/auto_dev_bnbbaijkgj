# 咨询人信息收起/展开功能 — 用户体验使用手册

## 1. 功能简介

在使用 **AI报告解读（体检报告）**、**用药参考（拍照识药）**、**健康自查** 三个核心功能时，选择咨询人后，系统会展示该成员的健康档案信息。

为减少信息视觉负担，页面默认只展示 **核心信息**（姓名、出生日期、性别），其余详细信息（身高、体重、病史等）收起在可展开区域中，用户可随时展开查看和编辑。

## 3. 操作步骤

### 3.1 进入选择咨询人步骤

1. 打开上述任一功能页面
2. 完成前置步骤后，进入"选择咨询人"步骤
3. 从家庭成员列表中选择一位咨询人

### 3.2 查看默认展示信息（收起状态）

选择咨询人后，页面默认展示以下三项核心信息：

- **姓名** — 可直接点击输入框修改
- **出生日期** — 点击后弹出日期选择器修改
- **性别** — 点击后切换性别选项

在这些字段下方，会显示一条横线分隔按钮：

- `更多信息（已完善）▼` — 表示该成员的关键档案信息已填写完整（绿色标注）
- `更多信息（待完善）▼` — 表示还有关键信息未填写（橙色标注）

### 3.3 展开查看更多信息

点击 `更多信息` 按钮，页面会以平滑动画展开以下内容：

- **身高（cm）** — 可直接输入数字
- **体重（kg）** — 可直接输入数字
- **既往病史** — 可选择或输入病史标签
- **过敏史** — 可选择或输入过敏信息
- **家族遗传病史** — 可选择或输入遗传病信息

展开后，分隔线按钮变为：

- `收起信息（已完善）▲` 或 `收起信息（待完善）▲`

点击即可收起详细信息。

### 3.4 编辑信息

无论信息处于收起还是展开状态，所有可见字段均可直接编辑：

1. 点击需要修改的字段
2. 输入或选择新的值
3. 修改会暂存在当前页面，**不会立即保存**

### 3.5 提交咨询并保存

点击页面底部的 **"开始咨询"** 按钮时：

1. 系统会自动校验必填字段（姓名、性别、出生日期）
2. 校验通过后，修改的档案信息会自动保存到该成员的健康档案
3. 页面弹出提示：`健康档案信息已同步更新`

## 4. 档案完善度说明

### 4.1 完善度判断标准

系统根据以下 **5 项字段** 判断档案完善度：

| 序号 | 字段 | 说明 |
|------|------|------|
| 1 | 姓名 | 必填项 |
| 2 | 性别 | 必填项 |
| 3 | 出生日期 | 必填项 |
| 4 | 身高 | 完善度指标，非提交必填 |
| 5 | 体重 | 完善度指标，非提交必填 |

- **5项全部填写** → 显示 **已完善**（绿色）
- **任意一项为空** → 显示 **待完善**（橙色）

### 4.2 完善度与提交的关系

- 显示"待完善"**不影响提交咨询**
- 只有姓名、性别、出生日期三项为**提交必填**
- 身高和体重未填时可正常提交，但建议填写以提高 AI 分析准确性

## 5. 注意事项

### 5.1 关于状态重置

- 每次进入"选择咨询人"步骤，信息默认为**收起状态**
- 切换不同咨询人时，展开状态会**自动收起**
- 切换咨询人时，之前对某位成员的未保存修改会**自动丢弃**

### 5.2 关于校验提示

- 如果必填字段未填写，点击"开始咨询"时会：
  - 在对应字段下方显示红色提示文字（如"请输入姓名"）
  - 弹出全局提示：`请填写完整的必填信息`
- 补充填写必填字段后，即可正常提交

### 5.3 关于数据保存

- 所有修改仅在点击"开始咨询"且校验通过后才会保存
- 不点击提交直接离开页面，修改**不会保存**
- 保存成功后，该成员在其他功能页面的健康档案也会同步更新

## 7. 常见问题

**Q: 为什么显示"待完善"但我可以正常提交？**
A: "待完善"仅表示身高或体重未填写，不影响提交。建议填写完整以获得更准确的AI分析结果。

**Q: 我修改了信息但没有点提交，信息会保存吗？**
A: 不会。所有修改只有在点击"开始咨询"并通过校验后才会保存到健康档案。

**Q: 切换咨询人后之前的修改去哪了？**
A: 切换咨询人时，之前未保存的修改会自动丢弃。如需保留修改，请先点击"开始咨询"提交。

**Q: 收起状态下点击提交，展开区的信息会一起保存吗？**
A: 会的。无论信息是否处于展开状态，提交时都会保存所有字段的当前值。
