## 功能简介

本次更新对系统的消息管理模块进行了全面拆分与增强，主要变更如下：

1. **短信管理增强**：在原有腾讯云短信支持的基础上，新增阿里云短信支持，增加短信模板管理功能（增删改查），发送记录和测试发送均支持按服务商筛选。
2. **微信推送管理独立**：从原"推送管理"中拆分为独立菜单模块，功能与原有一致。
3. **邮件通知管理独立**：从原"推送管理"中拆分为独立菜单模块，新增邮件发送记录查询和测试发送功能。
4. **删除推送管理**：原"系统设置 → 推送管理"标签页已删除，三个模块各自独立管理。

## 使用说明

### 一、短信管理

**入口**：管理后台左侧菜单 → 系统管理 → 短信管理

短信管理包含4个标签页：

#### 1.1 短信配置

- 页面上方为**腾讯云短信配置**区块，下方为**阿里云短信配置**区块
- 每个区块有独立的**启用开关**（Switch），同一时间只能启用一家服务商
- 切换服务商时系统会弹出确认提示："切换服务商将禁用当前已启用的服务商，是否继续？"
- **腾讯云配置字段**：SecretID、SecretKey（密码输入）、SDK AppID、短信签名
- **阿里云配置字段**：AccessKey ID、AccessKey Secret（密码输入）、短信签名
- 密码类字段回显时脱敏显示，重新输入将覆盖原值
- 填写完成后点击对应区块的"保存"按钮

#### 1.2 短信模板

- 展示所有短信模板的列表，包含模板名称、服务商、模板ID、内容、签名、用途场景、变量说明、状态等信息
- **筛选功能**：
  - 按服务商筛选（全部/腾讯云/阿里云）
  - 按用途场景筛选（全部/验证码/通知/营销/其他）
  - 按模板名称搜索
- **操作功能**：
  - 点击"新增模板"按钮，在弹窗中填写模板信息并保存
  - 点击"编辑"可修改已有模板
  - 点击"删除"可删除模板（需二次确认）

#### 1.3 发送记录

- 展示所有短信发送记录，包含发送时间、手机号（脱敏显示）、验证码、服务商、发送状态、失败原因、是否为测试发送
- **筛选功能**：
  - 按手机号搜索
  - 按服务商筛选（全部/腾讯云/阿里云）
  - 按发送状态筛选（全部/成功/失败）

#### 1.4 测试发送

- 选择**服务商**（腾讯云/阿里云），可测试任意服务商的短信发送（不受"启用"开关限制）
- 输入**手机号**
- 可选填**模板ID**（留空则使用配置中的默认模板ID）
- 点击"发送测试短信"按钮，系统会实时反馈发送结果（成功/失败及原因）
- 测试发送的记录会写入发送记录中，标记为"测试"

### 二、微信推送管理

**入口**：管理后台左侧菜单 → 系统管理 → 微信推送管理

- 页面提供微信推送的配置管理
- **配置字段**：
  - 启用微信推送（开关）
  - 微信AppID
  - 微信AppSecret（密码输入，回显时脱敏）
  - 订单通知模板ID（可选）
  - 服务通知模板ID（可选）
- 填写完成后点击"保存配置"按钮

---

### 三、邮件通知管理

**入口**：管理后台左侧菜单 → 系统管理 → 邮件通知管理

邮件通知管理包含3个标签页：

#### 3.1 邮件配置

- **配置字段**：
  - 启用邮件通知（开关）
  - SMTP服务器（如 smtp.qq.com）
  - SMTP端口（如 465）
  - SMTP用户名（发件邮箱地址）
  - SMTP密码（邮箱授权码，密码输入，回显时脱敏）
- 填写完成后点击"保存配置"按钮

#### 3.2 发送记录

- 展示所有邮件发送记录，包含发送时间、收件人、邮件主题、发送状态、失败原因、是否为测试邮件
- **筛选功能**：
  - 按收件人邮箱搜索
  - 按发送状态筛选（全部/成功/失败）

#### 3.3 测试发送

- 输入**收件人邮箱**
- 输入**邮件主题**（默认为"测试邮件"）
- 可选填**邮件内容**（默认为"这是一封测试邮件，用于验证邮件配置是否正确。"）
- 发送前系统会校验邮件配置是否已完成，未配置则提示"请先完成邮件服务器配置"
- 点击"发送测试邮件"按钮，系统会实时反馈发送结果

## 注意事项

1. **权限要求**：所有消息管理模块均需要管理员权限才能访问，非管理员用户无法看到相关菜单。
2. **服务商互斥**：短信管理中，腾讯云和阿里云同一时间只能启用一家，启用其中一家时另一家将自动禁用。
3. **密钥安全**：SecretKey、AccessKey Secret、AppSecret、SMTP密码等敏感信息在API返回时均会脱敏处理，已设置的密钥不会明文展示。
4. **测试发送**：短信测试发送不受"启用"开关限制，方便您分别测试两家服务商的连通性；但正式业务发送验证码时，仅使用当前已启用的服务商。
5. **邮件配置**：使用QQ邮箱等免费邮箱时，SMTP密码通常为邮箱的"授权码"而非登录密码，请从邮箱设置中获取。
6. **数据迁移**：升级后原"推送管理"中的数据已自动迁移到对应的独立模块，无需手动操作。
