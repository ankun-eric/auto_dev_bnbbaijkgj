## 3. 平台运营 / 商家运营 — 操作流程

### 3.1 看「优惠券类型」是啥意思

1. 进入 **商品体系 → 优惠券**。
2. 点 **新增优惠券**（或编辑已有券）。
3. 在「优惠券类型」字段右侧点击 **"?"** 图标。
4. 弹窗按卡片列出：
   - **满减券**：满 X 减 Y（固定金额抵扣）
   - **折扣券**：全场 N 折
   - **满折券**：满 X 元后再打 N 折
   - **免费试用券**：指定商品 / 分类整单减免（建议搭配 scope=product/category）

### 3.2 指定「适用分类」

1. 「适用范围」选 **指定分类**。
2. 出现分类选择器，点开下拉，可看到完整分类树（一级 / 二级 / 三级）。
3. 支持：
   - **关键词搜索**：直接输入分类名称即可定位。
   - **多选**：勾选多个父子分类自动去重。
   - **按 ID 回填**：编辑历史券时，已选分类会回填为标签；分类被删除会标红 `已删除`。
4. 点击「确定」即生效，下方实时统计条会刷新「该范围下当前 X 件在售商品」。

### 3.3 指定「适用商品」

1. 「适用范围」选 **指定商品**。
2. 点击「**+ 添加商品**」打开弹窗。
3. 弹窗能力：
   - **3 个 Tab**：全部 / 快递商品 / 到店服务（虚拟商品本期不纳入）。
   - **关键词搜索**：按商品名搜。
   - **分类筛选**：左侧分类筛选条直接收窄。
   - **跨页保留**：翻页 / 切 Tab / 切分类，已勾选不会丢。
   - **顶部计数**：实时显示"已选 X / 上限 N"（默认 N=300，可在 `system_configs.coupon_scope_max_products` 调整）。
   - **接近上限提示**：超过 80% 时变橙；触顶时禁止再勾，并提示**「建议改用『指定分类』模式」**。
4. 点「确定」回到表单，已选商品以**带图卡片**形式显示，鼠标悬停可单独删除。
5. 历史券的字符串 `1,2,3` 形态 `scope_ids` 自动转为数组并回显商品详情。

### 3.4 设置「排除商品」（新功能）

> 适用场景：满减券想除掉"高客单 / 已亏损 SKU"，免费试用想除掉指定 SPU。

1. 不论 scope 选 `all` 还是 `category`，都会出现「**排除商品**」按钮。
2. 弹窗与"指定商品"相同；上限默认 N=200（`system_configs.coupon_exclude_max_products` 可调）。
3. 已被选为「适用商品」的 SKU 不能再选为「排除」（会自动过滤）。
4. 排除生效后：
   - 用户下单时，命中 `exclude_ids` 的商品**小计不计入优惠门槛**（满 100 减 10 → 排除商品 60 元，购物车其它商品需 ≥100 才能用券）。
   - 命中 `exclude_ids` 的商品**也不享受折扣**（按比例分摊时不参与）。

### 3.5 看「实时统计条」

表单最下方会出现一行紫色提示：

- scope=all：`本优惠券将作用于全店 X 件在售商品（已排除 Y 件）`
- scope=category：`所选 N 个分类共 X 件在售商品（已排除 Y 件）`
- scope=product：`已选 X 件商品`

便于运营立刻判断"这张券能覆盖多少 SKU"。

### 3.6 保存

- 「免费试用券」 + scope=all 时，保存前会弹**警告确认**："该券将免单全店所有商品，请确认无误"。
- 「指定商品/分类」必须至少有 1 个 ID，否则提示"请添加至少 1 个商品/分类"。

## 5. 接口清单（仅 admin / 后端工程师参考）

| Method | Path | 说明 |
|---|---|---|
| GET | `/api/admin/coupons/type-descriptions` | 4 种类型说明（"?"弹窗数据源） |
| GET | `/api/admin/coupons/scope-limits` | 当前上限配置 |
| GET | `/api/admin/coupons/category-tree` | 分类树形选择器数据源 |
| GET | `/api/admin/coupons/categories-by-ids?ids=...` | 按 ID 批量查分类（编辑回填） |
| GET | `/api/admin/coupons/product-picker?...` | 商品弹窗数据源（搜索/分页/批量回显） |
| GET | `/api/admin/coupons/category-product-count?category_ids=...` | 分类下商品数（统计条） |
| GET | `/api/admin/coupons/active-product-count` | 全店在售商品数（scope=all 统计条） |

数据库变更：`coupons.exclude_ids` (JSON, nullable)；`system_configs` 新增 2 项。
