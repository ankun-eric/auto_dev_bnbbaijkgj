## 一、功能简介

本次围绕 **品牌一致性 + 头像兜底** 两条主线，对 AI 对话首页（ai-home）做了 6 项优化：

| 编号 | 优化点 | 用户可感知效果 |
| ---- | ---- | ---- |
| R1 | 顶部固定栏背景改主色 | 顶部栏由"白底 → 全站品牌主色（天蓝色）"，与下方内容明显区分，增强品牌识别度。文字"小康"、☰、⋯ 图标保持黑色不变，对比清晰可读 |
| R2 | "小康"标题靠左 + 间距收紧 | 标题位置整体左移，与左侧 ☰ 按钮间距从 8px 压缩到 **4px**，视觉更紧凑，但保留必要的呼吸感 |
| R3 | 修复欢迎区头像裂图 BUG | 当后台返回的头像 URL 失效、字段为空、加载失败时，欢迎区头像不再显示破图（裂图）小图标 |
| R4 | AI 消息小头像改读后台 | AI 回答消息上方的小头像（28×28）由原来写死的"🌿 树叶 emoji"，改为读取与欢迎区相同的「AI 对话模式首页配置 - 欢迎区 - 头像」字段，后台改一处即可全屏生效 |
| R5 | 三场景兜底 | 接口失败 / 后台字段为空 / 图片加载失败 —— 全部统一显示默认"宾尼小康"卡通图，杜绝裂图与空白 |
| R6 | 加载占位 | 头像加载期间立即先显示默认 LOGO（无白屏），后台头像加载完成后通过 200ms CSS opacity 过渡平滑切换（无闪烁） |

## 二、操作步骤（终端用户）

### 1. 体验「顶部栏主色 + 标题靠左」

1. 打开 [AI 对话首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/)（如未登录，先在登录页登录）
2. 观察页面顶部固定栏：
   - **背景颜色** 为天蓝色主色（不再是白色）
   - **左侧 ☰** 汉堡菜单按钮：黑色
   - **"小康"标题**：黑色，紧贴 ☰ 右侧 4px 间距
   - **右侧 ⋯** 更多菜单按钮：黑色
3. 上下滚动消息列表 —— 顶部栏保持固定不动，依然是主色背景（不会跟随滚动）

### 2. 体验「头像兜底」（三场景演示）

#### 场景 A：后台正常配置头像（http(s) URL）
1. 进入「商家后台 → AI 对话模式首页配置 → 欢迎区 → 头像」上传一张头像图
2. 在用户端 H5 刷新 AI 首页 → 欢迎区大头像 + AI 消息小头像 同步显示该后台头像
3. 加载过程中先看到默认"宾尼小康"图，加载完成后平滑切换为后台头像（200ms 过渡，无闪烁）

#### 场景 B：后台头像字段为空
1. 在后台清空头像字段（或不上传）
2. 在用户端 H5 刷新 AI 首页 → 欢迎区头像 + AI 消息小头像 均显示默认"宾尼小康"图（不再有裂图、空白或破图）

#### 场景 C：后台返回 URL 但 404 / 加载失败
1. 后台配置的头像 URL 已失效（如图片被删除、URL 拼错、CDN 失联）
2. 在用户端 H5 刷新 AI 首页 → 浏览器尝试加载失败后，自动 onerror 切换为默认"宾尼小康"图（无裂图）

#### 场景 D：断网 / 接口请求失败
1. 在浏览器 DevTools 中模拟断网或将 `/api/ai-home-config` 改为返回 500
2. 刷新 AI 首页 → 欢迎区与 AI 消息小头像 均显示默认"宾尼小康"图（页面骨架完整不崩塌）

### 3. 体验「AI 消息小头像同步后台」

1. 进入 AI 首页后，发送一条消息（如"帮我推荐一些低脂食物"）
2. 等待 AI 回答出现
3. 观察 AI 回答消息上方左侧的小头像（28×28 圆形）：
   - 如果后台配置了图片 → 显示该图片（与欢迎区大头像同源）
   - 如果未配置 / 失败 → 显示默认"宾尼小康"图
4. 同一会话内的多条 AI 回答 + Loading 占位卡片中的小头像，**全部统一**为该头像（不再出现 🌿 emoji 与后台头像并存的混乱情况）

## 三、注意事项

1. **后台头像字段说明**：本次共用「AI 对话模式首页配置 - 欢迎区 - 头像」**同一字段**，后端无新增字段。后台修改后，用户**下次进入页面**即可同步更新（H5 端有 5 分钟本地缓存，缓存失效后自动拉取新配置）。
2. **支持的图片类型**：
   - 完整 URL：`https://example.com/avatar.png` 或 `http://...`
   - 站内相对路径：`/images/xxx.png`（自动拼接 basePath 前缀）
   - emoji 字符（如 🌿、🤖）：直接渲染为 emoji 头像（兼容旧数据）
3. **默认图说明**：默认头像存储于服务器 `/images/default-ai-avatar.png`（约 60KB，蓝色背景"宾尼小康"卡通机器人），无需用户操作，前端组件自动加载。
4. **顶部栏文字 / 图标颜色**：本次只改了顶部栏背景色，文字"小康"、左侧 ☰、右侧 ⋯ 三处图标颜色**保持原黑色**（`THEME.textPrimary`）不变，确保与主色背景的对比度可读性。
5. **不影响其他端**：本次仅 H5 前端代码改动，后端 / Admin 后台 / 微信小程序 / Flutter App 均零改动，老 App / 老分享链接 / 老对接全部"零感知"，无需任何兼容操作。
