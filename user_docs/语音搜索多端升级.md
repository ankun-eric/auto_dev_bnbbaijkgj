## 功能简介

本次更新将语音搜索功能从 H5 端**全面升级并扩展到三端**（H5 网页端、微信小程序、Flutter App），同时将交互方式升级为**长按录音**模式，提供更接近微信语音消息的自然操作体验。

### 核心升级点

- **H5 网页端**：原"语音搜索功能即将上线"占位文案替换为完整可用的语音搜索，支持浏览器麦克风录音 + 腾讯云 ASR 语音识别
- **微信小程序**：新增搜索页面和语音搜索功能，使用微信同声传译插件实现语音识别
- **Flutter App**：新增搜索页面和语音搜索功能，使用 record 插件录音 + 后端 ASR 识别
- **后端**：新增 `POST /api/search/asr/recognize` 语音识别接口，支持多格式音频上传

## 使用说明

### 一、进入搜索页面

**H5 网页端**：
1. 打开 [H5 首页](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 点击顶部搜索框，进入搜索页面

**微信小程序**：
1. 在小程序首页，点击顶部搜索框
2. 自动跳转到搜索页面

**Flutter App**：
1. 在 App 首页，点击顶部搜索框
2. 自动跳转到搜索页面

### 二、使用语音搜索

#### 步骤 1：长按麦克风按钮

在搜索输入框右侧找到 🎤 麦克风图标按钮，**长按不放**（需按住超过 0.3 秒）。

> 首次使用时，系统会请求麦克风权限，请点击「允许」。

#### 步骤 2：说出搜索内容

长按后会出现全屏半透明遮罩，中央显示声波动画和录音时长。**对着手机/电脑说出您想搜索的内容**，例如"感冒了怎么办"。

- 声波动画会跟随您说话的音量实时波动
- 录音时长显示在声波上方（如 "3s / 15s"）
- 剩余 3 秒时，数字会变为红色提醒
- 最长录音 15 秒，到时间自动停止

#### 步骤 3：松手识别

说完后**松开手指**，系统自动进入识别状态（显示"正在识别..."）。

#### 步骤 4：自动搜索

识别完成后：
- 遮罩消失，识别出的文字自动填入搜索框
- 搜索框下方出现提示：**"2秒后自动搜索，点击可取消"**
- 2 秒后自动跳转到搜索结果页

#### 可选操作

- **修改文字**：在搜索框中直接编辑识别结果，倒计时会重置为 2 秒
- **取消自动搜索**：点击"点击可取消"，搜索不会自动触发，由您决定何时搜索
- **立即搜索**：直接点击搜索按钮或按回车，立即搜索

### 三、搜索结果查看

搜索结果页支持按类型筛选：
- **全部**：展示所有匹配结果
- **文章**：健康资讯文章
- **视频**：健康视频内容
- **服务**：健康服务项目
- **积分商品**：积分商城商品

### 四、其他搜索功能

- **热门搜索**：搜索页展示热门搜索关键词，点击直接搜索
- **搜索历史**：登录后显示最近搜索记录，支持删除和清空
- **搜索联想**：输入文字时实时展示联想建议

## 注意事项

### 权限说明

- **麦克风权限**：首次使用语音搜索时，系统会弹出权限请求，请点击「允许」
- 如果之前拒绝了麦克风权限：
  - H5 网页端：需要在浏览器设置中重新开启麦克风权限
  - 微信小程序：前往「手机设置 → 微信 → 麦克风」开启
  - Flutter App：前往「手机设置 → 应用管理 → 比尼健康 → 权限」开启

### 浏览器兼容性（H5 端）

- 推荐使用 Chrome 70+、Safari 14.5+ 或微信内置浏览器
- **必须使用 HTTPS** 协议访问（HTTP 下无法获取麦克风权限）
- 部分旧版浏览器不支持语音搜索，麦克风按钮会自动隐藏

### 语音搜索限制

- 单次录音最长 **15 秒**
- 录音时间不足 **1 秒** 时不会发起识别（提示"说话时间太短"）
- 长按 **0.3 秒以上** 才会开始录音（短按不触发）
- 建议在安静环境中使用，提高识别准确率

### ASR 服务说明

- 语音识别服务由管理员在后台配置，如麦克风按钮不显示，可能是管理员尚未启用 ASR 服务
- 管理员可在「管理后台 → 搜索管理 → 语音配置」页面启用和配置腾讯云 ASR 服务
