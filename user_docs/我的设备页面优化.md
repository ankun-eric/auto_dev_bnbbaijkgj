## 功能简介

本次将健康档案「我的设备」页面**全量重做**：

1. **修复 404**：AI-home 左上角抽屉里的「我的设备」原本跳到不存在的 `/my-devices`，现已统一到 `/devices`，点击不再 404。
2. **品牌分区平铺**：「支持设备列表」按 5 个品牌（宾尼 / 华为 / 小米 / 苹果 / 其他）平铺呈现，共 17 款设备，宾尼自有 7 项全部可绑、华为手环可绑、其他敬请期待。
3. **完整的绑定/解绑流程**：
   - 绑定抽屉：填 SN（支持扫码）、可填别名、必选使用人（默认本人，支持选择家庭成员）。
   - 唯一类设备每人仅可绑 1 台（如智能手表、血压计、血糖仪）；可多绑类设备允许同一账户绑多台（如烟雾报警器、水位报警器、紧急呼叫器）。
   - 解绑前二次确认，明确告知"新数据将停止上传、历史数据保留"。
4. **同 SN 多账户共享**：同一台设备（同一 SN）允许在多个账户之间共享绑定，适合家庭场景。
5. **卡片右上角 ✎ 编辑别名 / 使用人**：SN 与设备品类不可改（需要改请先解绑重绑）。
6. **整页视觉**：严格采用 AI-home 同款天蓝色阶（主色 `#0EA5E9` / 渐变 `#38BDF8 → #0284C7`），与其他健康档案页面风格一致。

## 操作步骤

### 入口

- **路径 A**（推荐）：在 AI 对话主页（[ai-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)）左上角点击「☰」打开侧边抽屉，点击「📱 我的设备」入口。
- **路径 B**：直接打开页面 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/devices/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/devices/)。

### 绑定一台新设备

1. 进入「我的设备」页面，向下滚动到「支持设备列表」。
2. 选择需要绑定的品牌（如「宾尼」），在卡片中找到目标设备（如「宾尼智能手表」）。
3. 点击右侧蓝色实心「绑定」按钮，从底部弹起的抽屉中填写：
   - **SN 码**（必填）：可手动输入，也可点击右侧 📷 扫码图标。微信内打开会调起微信扫一扫；普通 Chrome / Safari 会调起摄像头权限并提示输入或粘贴。
   - **别名**（选填，最长 20 字）：例如「爸爸的手表」，便于多设备区分。
   - **使用人**（必选，默认本人）：点击该项后会弹出与 AI 对话页一致的「选择咨询人」面板，可选「本人」或任意家庭成员；底部还有「+ 添加家庭成员」入口。
4. 点击底部蓝色渐变「确认绑定」按钮，成功后会自动回到列表，新设备卡片立即出现在页面顶部「我的设备」区域。

### 解绑设备

1. 在「我的设备」区域找到要解绑的卡片，点击卡片底部红色描边「解绑」按钮。
2. 弹出确认对话框，提示：
   > 解绑后，该设备将不再上传新数据到你的健康档案。
   > 已上传的历史数据将保留，可在健康档案中查看。
3. 确认无误后点击红色「确认解绑」按钮，列表立即更新，「支持设备列表」对应按钮恢复为蓝色「绑定」态。

### 编辑设备别名 / 使用人

1. 在「我的设备」卡片右上角点击 ✎ 图标。
2. 在弹出的编辑抽屉中修改：
   - **别名**：可清空或修改（最长 20 字）。
   - **使用人**：点击后弹出选择面板，可在「本人」与各家庭成员之间切换。
3. 点击「保存」按钮，卡片信息立即更新。
4. **注意**：SN 码和设备品类不可在编辑中修改；如需更换设备，请先解绑后重新绑定。

### 空状态体验

如果当前账户尚未绑定任何设备，「我的设备」区域会显示一张引导卡片：

- 顶部："还没有绑定任何设备"
- 底部蓝色渐变按钮：「立即绑定 ↓」，点击会平滑滚动到下方「支持设备列表」。

### 按钮状态说明

「支持设备列表」中每个设备右侧按钮共 4 种状态：

| 按钮 | 颜色 | 含义 |
|------|------|------|
| 绑定 | 蓝色实心 | 可立即点击进入绑定抽屉 |
| 继续绑定 | 蓝色描边 | 可多绑类设备且已绑过至少 1 台，仍可继续绑定 |
| 已绑定 | 灰色 | 唯一类设备已绑定，需先解绑才能换设备 |
| 敬请期待 | 浅灰 | 该型号尚未接通，点击无响应 |

## 注意事项

1. **SN 输入校验**：留空时前端会阻断；后端只做存在性校验，具体 SN 的硬件层匹配以设备厂商接入为准。
2. **同 SN 多账户共享**：家庭中多人可绑定同一 SN（如夫妻共用同一台血压计），所有绑定人都能在自己的健康档案中看到该设备上传的数据。
3. **唯一类 vs 可多绑**：智能手表 / 睡眠检测垫 / 血压计 / 血糖仪 / 体脂秤 / 体重秤 / 手环 / Apple Watch / 血氧仪 为唯一类；烟雾报警器 / 水位报警器 / 紧急呼叫器 为可多绑类。
4. **微信扫一扫**：在微信内浏览器中扫码会调起微信扫一扫接口；如未配置 wx-jssdk 域名白名单或权限被拒，会自动降级为摄像头权限 + 手动输入。
5. **解绑保留历史数据**：解绑只是停止新数据上传，已上传的健康监测数据会一直保留在你的健康档案中。
