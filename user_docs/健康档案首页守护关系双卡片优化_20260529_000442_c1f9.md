## 功能简介

本次优化对健康档案首页的两张并排卡片「守护我的人」「我守护的人」进行了文案统一与升级会员引导改造：

1. **统一文案口径**：两张卡片副标题统一为「守护者/守护对象：X 人（上限 Y 人）」格式，让用户一眼看清当前关系数 X 与会员等级允许的上限 Y。
2. **新增升级会员引导**：卡片右侧根据会员等级和饱和状态动态展示【升级会员】按钮，鼓励用户通过升级解锁更多守护名额。
3. **饱和状态视觉警示**：X = Y 时副标题变红字、按钮变红色高亮（普通会员）或置灰显示【达上限】（顶级会员），让用户清晰感知"已经满了"。
4. **邀请超额硬约束 + Toast 提示**：用户在详情页发起邀请且 X 已等于 Y 时，后端校验拦截，前端用 Toast 提示「守护者/我守护的人已达上限，请先升级会员或解绑现有守护者/守护对象」。
5. **「守护我的人」详情页新增【取消邀请】按钮**：邀请中（待确认）的条目可一键取消，名额立即释放。
6. **家人 Tab 文案同步只读**：「守护 TA 的人」「TA 守护的人」卡片文案同步新口径，但**所有按钮一律不显示**，纯只读视图。
7. **返回首页自动刷新**：在详情页执行任何写操作后返回健康档案首页，会强制重新拉取接口，保证 X 数字实时准确。

## 使用说明

### 一、查看双卡片当前状态（本人 Tab）

1. 打开 H5，登录账号
2. 进入「健康档案」首页
3. 在顶部成员 Tab 中选中「本人」
4. 在页面中部可以看到两张并排卡片：
   - 左侧：**我守护的人** —— 副标题「守护对象：X 人（上限 Y 人）」
   - 右侧：**守护我的人** —— 副标题「守护者：X 人（上限 Y 人）」
5. 副标题颜色和按钮形态会根据您当前的会员等级和饱和状态自动变化（见下方"视觉规则"）

### 二、视觉规则一览（本人 Tab）

| 场景 | 副标题颜色 | 按钮文案 | 按钮样式 | 是否可点 |
|---|---|---|---|---|
| 普通会员，未满（X < Y） | 黑字 | 升级会员 | 蓝色 | 可点 |
| 普通会员，已满（X = Y） | 红字 | 升级会员 | 红色高亮 | 可点 |
| 顶级会员，未满（X < Y） | 黑字 | （无按钮） | — | — |
| 顶级会员，已满（X = Y） | 红字 | 达上限 | 灰色置灰 | 不可点 |

> 业务硬约束：**X 永远 ≤ Y**，系统永远不会出现 X > Y 的情况。

### 三、点击【升级会员】

1. 在双卡片中任意一张点击【升级会员】按钮
2. 系统会跳转到「会员中心」首页
3. 选择您想升级到的套餐进行支付即可
4. 升级成功后返回健康档案首页，卡片的上限 Y 会自动更新

### 四、邀请超额 Toast 拦截

#### 场景 1：守护者已达上限（被别人守护）

1. 在「守护我的人」卡片点击进入详情页
2. 点击【邀请别人守护我】，选择关系生成邀请二维码
3. 如果当前 X 已等于 Y，将弹出 Toast 提示：
   > **守护者已达上限（X/Y），请先升级会员或解绑现有守护者**
4. Toast 自动消失，您可以选择：
   - 升级会员（跳转会员中心）
   - 解绑现有守护者（在详情页点【解除】）
   - 取消已发出的"邀请中"邀请（在详情页点【取消邀请】）

#### 场景 2：守护对象已达上限（我守护别人）

1. 在「我守护的人」卡片点击进入详情页
2. 点击邀请按钮新建邀请
3. 如果当前 X 已等于 Y，将弹出 Toast 提示：
   > **我守护的人已达上限（X/Y），请先升级会员或解绑现有守护对象**

### 五、「守护我的人」详情页【取消邀请】（新功能）

1. 在「守护我的人」卡片点击进入详情页
2. 列表中"邀请中"的条目右侧显示橙色【取消邀请】按钮
3. 点击【取消邀请】，弹出二次确认弹窗
4. 确认后，该邀请立即失效，名额释放
5. 返回首页，卡片 X 实时 -1

### 六、家人 Tab 只读视图

1. 在顶部成员 Tab 中选中任意一位家人
2. 此时双卡片标题变为：
   - 左侧「TA 守护的人」（即该家人作为主守护人守护了几个人）
   - 右侧「守护 TA 的人」（即谁在守护该家人）
3. 副标题：「守护对象：X 人（上限 Y 人）」/「守护者：X 人（上限 Y 人）」
4. **所有按钮全部不显示**（包括升级会员、达上限、取消邀请、解绑、邀请等）
5. 点击右侧「守护 TA 的人」卡片可弹出只读详情列表，仅可查看，不可操作

### 七、邀请态（待确认）管理规则

- 邀请中（待确认）状态**永久有效**，不设自动失效时间，名额一直占用 Y 直至：
  - 对方主动扫码确认 → 转为已绑定
  - 我主动点【取消邀请】 → 释放名额
- 「守护我的人」「我守护的人」两个详情页都新增/已有【取消邀请】按钮，规则完全对称

## 注意事项

1. **X 口径**
   - 「守护我的人」X = 已绑定（active）+ 邀请中（pending 反向邀请）合计；已解绑/已取消的不计入
   - 「我守护的人」X = 按现有规则不变（已绑定非本人档案数）

2. **Y 来源**
   - 「守护我的人」Y = 会员等级配置中的「被管理上限」字段（`max_managed_by`）
   - 「我守护的人」Y = 会员等级配置中的「管理上限」字段（`max_managed`）
   - 当前所有套餐 Y 默认为 3，未来通过后台管理可调整

3. **顶级会员判定**
   - 系统会自动判断：当前生效套餐的 `max_managed_by` 是否为所有 active 套餐中最大值 → 是 = 顶级会员
   - 若套餐的 `max_managed_by ≥ 9999` 或为 -1 → 视为无上限（也算顶级）

4. **数据实时性**
   - 在详情页执行解绑/确认/取消邀请/发出邀请等任何写操作后，返回首页时会**强制重新拉取接口**
   - 不依赖前端缓存，不需要您手动下拉刷新

5. **错误码**
   - 邀请超额时，后端会返回结构化错误码：
     - `GUARDIAN_LIMIT_REACHED`（守护我的人侧）
     - `WARD_LIMIT_REACHED`（我守护的人侧）
   - 前端会自动识别并展示对应 Toast

## 常见问题

**Q1：为什么我升级会员后卡片上限没变？**
A：请退出健康档案页面，重新进入，或下拉刷新页面。系统在页面可见时会自动重新拉取接口。

**Q2：邀请发出去了但对方还没扫码，我能取消吗？**
A：可以。在「守护我的人」/「我守护的人」详情页找到对应"邀请中"条目，点击【取消邀请】即可。

**Q3：家人 Tab 下为什么我看不到【升级会员】按钮？**
A：家人 Tab 是只读视图，所有按钮都不显示。要升级自己的会员，请先切回「本人 Tab」。

**Q4：顶级会员卡片上为什么没有【升级会员】按钮？**
A：顶级会员已经是最高等级，无需升级，所以未满状态不显示按钮；已满状态会显示置灰的【达上限】按钮提示您。
