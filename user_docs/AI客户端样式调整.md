## 功能简介

本次更新对 H5 客户端进行了全量视觉样式调整，统一对齐「方案A · 晴空诊室」设计稿标准，涵盖 **13 个页面/组件群的样式对齐** 和 **12 项新增/改造组件**，为用户带来更一致、更精致的视觉体验。

### 主要变更概览

- **品牌色系统一**：全面采用天蓝色品牌色阶（#F0F9FF ~ #0C4A6E），页面背景、按钮、气泡、卡片等全部对齐设计稿
- **渐变体系升级**：顶栏、主按钮、用户消息气泡、会员卡等区域均使用统一的品牌渐变
- **圆角与阴影规范化**：所有卡片、按钮、输入框的圆角和阴影统一为设计规范值
- **7 个全新组件**：欢迎区、语音输入面板、快捷操作面板、历史会话提示、中断态提示、隐私确认弹层、预约时间选择页
- **2 个新页面**：预约时间选择页、收货地址管理页

## 注意事项

1. **浏览器兼容性**：推荐使用 Chrome、Safari 或微信内置浏览器访问
2. **屏幕适配**：本次优化适配 320px ~ 428px 宽度范围的主流移动设备
3. **安全区适配**：顶部和底部已适配 iPhone 刘海屏和底部安全区
4. **动画效果**：所有过渡动画为 200~300ms，骨架屏闪烁动画 1.4s 循环
5. **预约页和地址页**：当前使用 mock 数据展示，后续将对接真实后端接口
6. **隐私弹层**：当前仅完成 UI 组件，触发时机将在后续需求中定义
