## 功能简介

### 数字安全绳（独居守护）

在关怀模式下，用户可以：

1. **设置超时阈值**：选择 24 小时或 48 小时，超过设定时间未签到时系统自动通知紧急联系人
2. **每日签到**：点击"我今天平安"按钮签到，顶部横幅会显示"✅ 上次签到时间"和"⏰ 下次签到截止时间"
3. **紧急联系人**：添加 1-3 位紧急联系人（只需姓名+手机号，系统自动校验手机号是否已注册）
4. **入口可见**：关怀模式首页有绿色"🪢 数字安全绳"横卡和右下角绿色悬浮球

## 使用说明

### 进入数字安全绳

1. 打开 [关怀模式首页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/care-home)
2. 在页面中找到绿色卡片"🪢 数字安全绳"（或点击右下角绿色悬浮球"🪢安全绳"）
3. 进入数字安全绳功能页

### 设置阈值

- 在功能页顶部，点击"24小时"或"48小时"切换超时阈值
- 选中项会有绿色填充 + 白底绿勾 ✓ 的视觉效果

### 每日签到

- 点击"我今天平安"大按钮完成签到
- 签到后顶部横幅会显示上次签到时间和下次签到截止时间

### 添加紧急联系人

- 点击"+ 添加"按钮
- 输入联系人姓名和手机号
- 系统自动校验手机号是否为已注册用户
- 保存成功后列表立即显示记录
