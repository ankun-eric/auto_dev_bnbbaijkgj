## 功能简介

本次升级把 **AI 对话页中所有"功能引导卡片"**（如「健康自查」「中医体质测评」「AI 食谱」「运动方案」等点击可跳转到功能页的卡片）的**视觉样式统一升级为新版**，主要改进：

1. **视觉风格更现代**：圆角更柔和（16px）、采用品牌天空蓝渐变（`#38BDF8 → #0284C7`），与 AI 首页输入区蓝色锚点完全同源。
2. **信息层级更清晰**：主标题（18px 加粗）/ 副标题（13px 灰）/ 按钮副说明（12px 浅灰）/ 主按钮（44px 高、渐变蓝、白字 16 加粗）— 层级一目了然。
3. **新增「按钮副说明文字」展示位**：后台已有的 `按钮副说明文字` 字段，现在会以**灰色小字**展示在按钮正上方，运营可向用户传递更细致的引导信息（如「约 6 道题，2 分钟完成」）。
4. **整卡可点击**：用户点击卡片任意位置都能跳转，按钮也保留独立的高亮主操作区，体验更顺手。
5. **后台「卡片预览」浮层**：在编辑功能按钮时，可点击右上角「预览效果」按钮，在 **375×667 真机框**中 **1:1 实时**查看 H5 真机效果，**改字段即看效果**，无需保存即可预览。
6. **历史会话兼容**：用户回看历史对话时，旧卡片也以**新样式**渲染（一刀切刷新，体验统一）。

> 本次改造**不引入新字段**、**不引入版本号**，后台数据结构完全不变。如需回滚，可一键回滚前端代码。

## 使用说明

### 一、H5 用户：体验新版卡片

1. **打开 AI 对话首页**：在浏览器中访问 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)
2. **触发功能引导卡片**（任一方式）：
   - 在底部宫格中点击「健康自查」「中医体质测评」「AI 食谱」「报告解读」等任一功能按钮
   - 或在对话框中输入「我想做体质测评」「帮我看下报告」等关键词
3. **查看新版卡片**：AI 气泡会显示**新版功能引导卡片**——
   - 顶部：彩色图标（默认 📋，或后台配置的 Emoji / 图片 / 封面图）
   - 中部：**主标题**（如「中医体质测评」）+ **副标题**（如「基于王琦国标 36 题，2 分钟出报告」）
   - 底部：**按钮副说明文字**（如「约 36 道题，2 分钟完成」）+ **主操作按钮**（如「开始测评」）
4. **点击体验**：点击卡片任意位置（整卡可点击）或点击主按钮，立即进入对应功能页或抽屉。

### 二、运营/管理员：在后台预览卡片效果

1. **登录管理后台**：访问 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/function-buttons](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/function-buttons)
2. **新增或编辑功能按钮**：点击列表中任意按钮的「编辑」操作，或点击右上角「新增功能按钮」。
3. **填写卡片字段**：在弹出的表单中填写：
   - **按钮图标（Emoji）**：如 🩺、🌿、💊
   - **卡片标题**：如「中医体质测评」
   - **卡片副标题**（可选）：如「基于王琦国标 36 题」
   - **按钮副说明文字**（可选）：如「约 36 道题，2 分钟完成」— **这是本次新版卡片重点呈现的字段**
4. **打开「预览效果」浮层**：点击表单标题右侧的 **「预览效果」** 按钮（蓝色幽灵按钮 + 👁 图标）。
5. **查看 375×667 真机预览**：弹出的浮层中央展示**带圆角和状态栏的真机外框**，内部以 H5 真实样式 **1:1 实时渲染**当前编辑中的卡片效果。
6. **实时联动**：在浮层打开期间，你可以切换回表单修改任意字段（主标题/副标题/按钮副说明/图标/封面图），预览浮层会**实时刷新**，无需保存即可查看。
7. **关闭预览**：点击遮罩或右上角 ✕ 即可关闭浮层，继续编辑或保存。

### 三、历史对话回看

- 打开任意历史对话，曾经的功能卡片现在会**自动以新版样式渲染**（一刀切刷新，无需任何用户操作）。
- 旧卡片数据无需迁移、无需重建会话。

## 注意事项

1. **图标三选一**：在「卡片图标」字段中可选择「使用默认」「选择 Emoji」「上传图片 URL」三种模式。预览浮层会根据当前模式自动渲染。
2. **封面图优先级最高**：如果同时配置了「卡片封面图 URL」和「图标 Emoji」，**新版卡片会优先展示封面图**（占据卡片顶部 140px 高的横幅），图标位会被覆盖。
3. **历史数据字段不全的兼容**：旧卡片如缺少「按钮副说明文字」字段，新样式中该位置会自动留空，不影响布局。
4. **整卡可点击与按钮防双触发**：卡片任意位置都能点击跳转，主按钮点击不会触发双重事件（已用 `stopPropagation` 防护）。
5. **像素级一致**：新样式严格遵守 UI 标注稿（字号、行高、间距、圆角、色值完全一致）。如发现真机效果与后台预览不一致，可在浮层中对照检查后保存。
6. **回滚预案**：本次改造无数据库变更，如线上出现严重问题，可一键 Git 回滚前端代码（无字段级回滚成本）。
7. **浏览器兼容**：iOS Safari、微信内置浏览器、Android Chrome 均已验证表现一致。
