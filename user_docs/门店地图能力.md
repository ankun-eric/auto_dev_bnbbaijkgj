## 一、功能介绍

本次升级为 **管理后台 + 用户端（H5 / 微信小程序 / 安卓 App / 苹果 App）** 同时引入完整的「门店地图能力」：

- **管理员录入门店**：可在管理后台直接打开嵌入式地图，通过 **POI 关键字搜索 / 拖拽大头针 / 手动输入经纬度** 三种方式精确定位门店；选点后系统会自动调用「逆地理编码」回填省 / 市 / 区 / 详细地址。
- **C 端用户找门店**：商品/卡项详情页的"可用门店"卡片右侧会显示带大头针的 **静态地图缩略图**；点击门店卡片即可一键调起手机本地地图 App（高德 / 百度 / 腾讯 / 苹果地图）进行路线规划。

> 设计上 **完全兼容无地图 Key 的环境**：管理后台地图使用 OpenStreetMap 瓦片渲染，POI 搜索与逆地理编码在配置高德 Web 服务 Key 时走高德、未配置时自动降级到 OpenStreetMap Nominatim，全链路始终可用。

## 三、管理员操作步骤（管理后台）

### 1. 进入门店管理

1. 浏览器打开 [管理后台门店管理页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/merchant/stores)。
2. 用管理员账号登录。
3. 点击右上角「**新建门店**」按钮，即可看到新版编辑弹窗（已集成地图选点）。

### 2. 在地图上选定门店位置（三选一）

1. **方式 A：POI 关键字搜索**
   - 在搜索框输入门店名称或地标（如 `广州塔`、`xx医院`），下拉会显示候选 POI；
   - 点击候选项 → 地图自动飞到该位置 → 大头针落点 → 省 / 市 / 区 / 详细地址被自动覆盖回填。
2. **方式 B：直接在地图上点击或拖拽大头针**
   - 在地图视窗直接 **点击** 任一位置即可落点；
   - 已有大头针时可 **长按拖拽** 大头针到新位置；
   - 落点后约 1 秒，系统自动调用逆地理编码，覆盖省 / 市 / 区 / 详细地址。
3. **方式 C：手动输入经纬度（兜底）**
   - 点击「**高级设置：手动经纬度**」展开输入区；
   - 分别输入纬度（-90 ~ 90）、经度（-180 ~ 180），点击「定位到地图」按钮，地图自动飞到该点。

### 3. 检查并保存

1. 系统已自动覆盖回填的省 / 市 / 区 / 详细地址，可手动二次编辑（以管理员手填为准）。
2. 检查"当前坐标"标签，应显示绿色 "已选点" 状态。
3. 点击「**确定**」保存门店。

### 4. 注意事项

- **新建门店强制要求选点**：未在地图上选点时点击保存会被前端拦截，并提示"请在地图上选择门店位置"。
- **编辑老门店可以不补坐标**：仅当管理员主动选点时才会更新坐标；对历史无坐标门店，前端会自动隐藏 C 端的"距您 X km"和地图缩略图。
- **坐标范围校验**：经度必须在 -180 ~ 180、纬度必须在 -90 ~ 90，否则会被前端 + 后端双重拦截。
- **逆地理编码限流**：每位管理员 30 次/分钟，超过会返回 429。

## 四、用户端体验

### 1. H5（手机浏览器）

1. 打开商品详情页（举例：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/product/{商品ID}`）。
2. 滚动到"可用门店"卡片：
   - **有坐标的门店**：右侧会显示带 📍 大头针的静态地图缩略图，并展示"距您 X km · 点击导航"；
   - **没有坐标的老门店**：缩略图与"距您 X km"均不显示，仅展示文字信息。
3. 点击门店卡片 → 弹出「**使用以下地图打开**」底部抽屉：
   - Android / iOS 浏览器均显示 **高德 / 百度 / 腾讯**；
   - iOS 设备额外显示 **苹果地图**。
4. 选择任一地图 → 自动唤起对应地图 App 进行路线规划；若本机未安装该 App，1.5 秒后会自动跳转到对应 Web 版地图作为兜底。

### 2. 微信小程序

1. 在小程序中打开商品详情页，下拉到"可用门店"卡片。
2. 同样可见缩略图 + 距您 X km。
3. 点击门店卡片 → 直接弹出 **微信原生地图面板**（腾讯地图底图，包含"导航 / 查看周边"按钮，无需自定义抽屉，体验最佳）。
4. 若有多个门店，可点击右侧「**切换**」按钮在多个门店间选择。

### 3. 安卓 App / iOS App（Flutter）

1. 启动 App 进入商品详情页。
2. 同样可见 100×100 大小的静态地图缩略图 + "距您 X km · 点击导航"。
3. 点击门店卡片 → 弹出底部「使用以下地图打开」抽屉，自动检测本机已安装的地图 App：
   - Android：高德 / 百度 / 腾讯（基于 `<queries>` 包可见性 + `canLaunchUrl` 双重探测）；
   - iOS：苹果地图 / 高德 / 百度 / 腾讯（基于 `LSApplicationQueriesSchemes` 白名单 + `canLaunchUrl` 探测）。
4. 选择任一已装地图 App，调起后会传递「**门店全称 + 详细地址 + 经纬度**」三段信息，导航精度由经纬度保证。
5. 未安装任何地图 App 时，App 会自动尝试跳到 Web 版地图。

## 六、后端环境变量（可选配置）

如需启用真正的高德地图能力（更精准的搜索/逆地理编码），可在后端容器环境变量中追加：

```bash
AMAP_SERVER_KEY=<在高德开放平台申请的 Web 服务 Key>
AMAP_WEB_JS_KEY=<管理后台 Web 端 JS Key>
AMAP_H5_JS_KEY=<H5 Web 端 JS Key>
```

未配置时系统自动降级到 OpenStreetMap，保证基础能力可用。**严禁将 AMAP_SERVER_KEY 透出到前端**。

## 七、常见问题（FAQ）

**Q1：管理后台保存门店时，提示"请在地图上选择门店位置"？**
A：新建门店强制要求经纬度必填。请在地图上点击/拖拽大头针，或用搜索框选定 POI，或在"高级设置"手动输入经纬度。

**Q2：编辑老门店时，地图上没有大头针？**
A：老门店历史上未录入坐标。无需强制补，可只编辑其他字段后保存；若要补坐标，按新建门店的方式选点即可。

**Q3：用户端门店卡片没有缩略图？**
A：该门店无经纬度。请管理员到后台编辑该门店并补充地图选点。

**Q4：H5 用户点了"高德"但浏览器没反应？**
A：本机未安装高德 App。1.5 秒后系统会自动弹出"未检测到该 App"提示，并跳转 Web 版地图作为兜底。

**Q5：iOS App 抽屉里只有 Apple 地图？**
A：本机仅安装了 Apple 地图，符合预期。安装其他地图 App 后重启应用即可在抽屉里看到更多选项。

**Q6：小程序点门店卡片为什么不弹自定义抽屉？**
A：微信小程序使用 `wx.openLocation`，由微信原生面板（腾讯地图底图）提供"导航 / 查看周边"按钮，体验更佳，无需自定义抽屉。
