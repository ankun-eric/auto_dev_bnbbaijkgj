## 二、操作步骤

### 1. 进入健康档案 v2

1. 在浏览器中打开主入口链接：[健康档案 v2](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile-v2/)
2. 首次进入会要求登录，登录后将看到顶部 Hero 卡片 + 5 个粘性 Tab。

### 2. 查看 Hero 四格健康摘要（M2 新）

进入页面后即可看到 Hero 卡片底部新增的 4 个统计指标：
- **既往病史**：本人当前记录的慢病/既往病史条数
- **过敏史**：药物 / 食物 / 其他过敏汇总
- **家族遗传**：家族病史条数
- **长期用药**：当前用药计划中标记为「长期服用」的种类数

### 3. 编辑基本信息（M2 新）

1. 点击 Hero 卡片右上角 **「✎ 编辑基本信息」** 按钮
2. 在弹层中修改：**姓名 / 性别 / 生日 / 身高 / 体重 / 血型**
3. 点击「保存」，Hero 卡片即时刷新

### 4. 新增/编辑用药计划（M4 重做）

1. 点击 Tab 切换至 **「用药计划」**
2. 顶部 **「今日用药 / 全部用药计划」** 分段切换，可只看今日待提醒
3. 点击右上角 **「+ 添加用药」** 进入添加页
4. 在添加页可设置：
   - **药品名称、剂量**（输入框联想匹配药品库；可点击 **「📷 拍照识药」** 自动识别）
   - **每日次数** 1～6 次（次数变化时，时间点自动配齐）
   - **服药时间点**：支持自定义时间点逐条增删，弹出时间选择器
   - **开始日期**（默认今日）/ **结束日期** / 或勾选 **「长期服用」**
   - **用药提醒开关**（默认开启，关闭后不再推送）
   - **关联疾病标签**（高血压、糖尿病、高血脂 等多选，可自定义）
5. 点击「保存」即可写入计划，并按时间点自动生成提醒

### 5. 编辑家族病史（M6 新）

1. Tab 切换至 **「健康信息」**，点击右上角「编辑」
2. 在弹层中下滑至 **「家族病史」** 区
3. 点击 **「+ 添加家族病史」**：
   - 选择关系（父亲、母亲、兄弟、姐妹、爷爷、奶奶……）
   - 选择疾病（糖尿病、高血压、心脏病、肿瘤等预设，或自定义输入）
   - 备注（可选，如 "55 岁确诊"）
4. 不需要的条目，点击右侧「🗑 删除」即可
5. 慢病栏目中可填写 **确诊年份**（如 2018）

### 6. 编辑手术史（M6 新）

1. 同样在 **「健康信息」** 编辑弹层中找到 **「手术史」** 区
2. 点击 **「+ 添加手术」**：填写手术名称、时间（年月）、备注
3. 支持多条，点击「🗑 删除」可移除

### 7. 上传病历卡 + OCR 自动解析（M8 新）

1. Tab 切换至 **「健康事件」**，点击 **「+ 上传病历卡」**
2. 选择/拍摄一张纸质病历照片
3. 系统自动 OCR 识别并解析以下 6 个字段（保存为病历卡）：
   - 医院名称
   - 科室
   - 就诊日期
   - 医生
   - 诊断
   - 处方
4. 病历卡保存成功后，**自动生成一条事件入时间轴**
5. 在时间轴上可查看 / 删除该病历卡，或下拉刷新

## 三、注意事项

| 项 | 说明 |
|----|------|
| 登录后才能用 | 健康档案、用药计划、病历卡所有写接口都需登录态 |
| 时间点设置 | 建议至少保留 1 个时间点，否则提醒不会触发 |
| 拍照识药 | 接入了后端 OCR 接口，识别准确率取决于药盒清晰度 |
| 病历卡 OCR | 当前 OCR 解析器基于关键字段匹配（医院 / 科室 / 医生 / 诊断 / 处方 / 就诊日期），字段未识别时不报错，仅留空 |
| 旧路由 | `/health-profile` 已完全下线，访问会返回 404，请改用 `/health-profile-v2` |
| 浏览器要求 | 推荐使用最新版微信内置浏览器、Chrome、Safari |
| 数据库迁移 | 已通过 `schema_sync` 自动追加 `medication_reminders` 7 个新列 + 自动创建 `medical_record_cards` 表，无需手动操作 |
