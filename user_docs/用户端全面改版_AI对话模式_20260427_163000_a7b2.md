## 功能简介

本次小康健康用户端全面改版，核心变化如下：

### 架构革新
- **取消底部四Tab导航**，默认直接进入AI对话页面

- **保留菜单模式（过渡期）**：Admin后台可切换回传统四Tab模式，便于平稳过渡

## 使用说明

### 一、AI对话模式（用户端核心体验）

#### 1. 进入AI对话页

打开用户端链接 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)，系统默认进入AI对话模式。

**首次访问（欢迎态）**：
- 页面显示LOGO头像 + 时段问候语（早上好/中午好/下午好/晚上好）
- 下方展示轮播图和3个功能卡片
- 底部有快捷标签栏

**有对话记录时（对话态）**：
- 显示AI消息气泡（左对齐，蓝紫色背景）和用户消息（右对齐，白色背景）
- AI回答带有"AI生成 仅供参考"提示

#### 2. 输入栏交互

| 状态 | 左侧 | 中间 | 右侧 |
|------|------|------|------|
| 默认态 | 🎤语音按钮 | 输入框（"问问小康"） | 📷拍照按钮 |
| 聚焦态 | 🎤语音按钮 | 输入框（光标闪烁） | 📷拍照按钮 |
| 输入文字态 | 🎤语音按钮 | 输入框（有文字） | ➤发送按钮 |

- 聚焦时自动显示**咨询人行**（胶囊标签，点击可切换咨询人）和**推荐问题卡片**
- 推荐问题以#标签形式横向滚动，点击即可发送

#### 3. 功能按钮直达

在对话页底部的快捷标签栏，可以一键跳转到：

| 按钮 | 功能 | 跳转目标 |
|------|------|---------|
| 看报告 | 上传体检报告获取AI解读 | 体检报告页 |
| 查用药 | 查询用药参考/拍药盒识药 | 用药参考页 |
| 看档案 | 查看健康档案（四子Tab） | 健康档案页 |
| 查订单 | 查看全部订单 | 统一订单列表 |
| 找专家 | 查找专家服务 | 服务页-专家Tab |
| 找服务 | 浏览健康服务 | 服务页 |

#### 4. 侧边栏「我的」

点击左上角 **☰** 图标，侧边栏从左侧滑出：
- **顶部**：显示头像、昵称、VIP会员编号
- **快捷入口**：消息中心（带未读数）、优惠券（带可用数）
- **订单状态**：待付款/待使用/待评价/退款 各显示数字，可点击"全部订单"
- **历史对话**：按时间分组（今天/昨天/近7天/更早），置顶对话显示📌
- **底部**：设置入口 + 管理按钮

#### 5. 右上角···菜单

点击右上角 **···** 图标：
- **扫一扫**：打开扫码功能
- **字体大小**：调节AI对话字体大小
- **立即分享**：弹出分享面板（微信/朋友圈/复制链接/海报）

#### 6. 选择咨询人

输入框聚焦时，显示当前咨询人胶囊标签。点击可打开**选择咨询人弹窗**：
- 显示家庭成员列表（头像+关系+姓名）
- 点击选中后切换对话上下文
- 支持"添加新成员"

#### 7. 历史对话管理

在侧边栏底部点击"管理"按钮进入管理模式：
- 对话列表左侧出现多选框
- 底部操作栏：清空全部 / 删除选中
- 长按对话条目可：置顶/取消置顶 | 删除
- 删除含附件的对话时，会弹窗确认"仅删除对话"或"连同报告一起删除"

### 二、健康档案页

通过功能按钮"看档案"跳转，包含四个子Tab：

| 子Tab | 内容 | 操作方式 |
|-------|------|---------|
| 健康数据 | 身高/体重/腰围/BMI/血压/心率/血糖/体温 | 底部半屏弹窗编辑 |
| 运动睡眠 | 今日步数 + 近7日趋势图 + 睡眠记录 | 手动录入 |
| 就医资料 | 病历/体检报告/药物/其它 | 上传文件（支持拍照/相册/文件） |
| 健康信息 | 慢性病/既往病/过敏/家族遗传/生活习惯 | 表单选择 + 自定义输入 |

### 三、专家预约流程

1. 点击"找专家"按钮 → 进入服务页
2. 选择专家 → 进入专家详情页
3. 点击底部**「预约」**按钮 → 跳转至关联商品详情页
4. 选择服务规格（如30分钟/60分钟） → 确认订单 → 支付
5. 支付完成后可在"查订单"中查看

> 注意：如果专家尚未关联商品，"预约"按钮会置灰显示"暂未开放预约"

### 四、设置页

从侧边栏底部"设置"入口进入，包含：

| 分组 | 设置项 |
|------|--------|
| 个人信息 | 头像/昵称/性别/生日/手机号 |
| 通知设置 | 消息推送开关/声音提醒开关 |
| 隐私设置 | 健康数据分享范围/对话记录保存时长 |
| 语音设置 | AI语音播报开关/语速调节/音色选择 |
| 存储 | 清除缓存 |
| 关于 | 版本号/用户协议/隐私政策/意见反馈 |
| 操作 | 退出登录 |

### 五、账号安全

从设置页"手机号"项进入：
- 查看当前绑定手机号（脱敏显示，如 138****8888）
- 更换手机号
- 修改/设置登录密码
- 绑定/解绑微信
- 账号注销（需验证码二次确认，7天冷静期）

### 六、意见反馈

从设置页"关于"→"意见反馈"进入：
1. 选择反馈类型：功能建议 / 内容反馈 / Bug报告 / 其他
2. 输入详细描述（最多500字）
3. 可上传截图（最多4张）
4. 可选填联系方式
5. 点击"提交"

## Admin 管理后台新增功能

### 一、页面风格配置

路径：首页设置 → 页面风格

- **AI对话模式**（推荐，默认）：用户端取消底部Tab，首屏为AI对话
- **菜单模式**（过渡期保留）：保留传统四Tab底部导航

切换后即时生效，用户端下次加载使用新模式。

### 二、视频客服配置

路径：AI咨询配置 → 视频客服

可配置项：
- 视频客服总开关
- 坐席服务URL
- 服务时段（开始～结束时间）
- 排队上限
- 欢迎问候语 / 等待提示语 / 非服务时段提示
- 超时时长

### 三、专家关联商品

路径：专家管理 → 编辑专家

- 新增"关联商品"下拉选择
- 选择商品后，用户端专家详情页底部将显示"预约"按钮
- 用户点击"预约"后进入商品详情→下单→支付全链路

## 注意事项

1. **登录要求**：使用AI对话、管理档案、预约专家等功能需要先登录
2. **页面风格切换**：仅Admin后台可切换，用户端无切换入口
3. **专家预约**：需Admin先为专家关联商品，且商品为"已上架"状态
4. **对话置顶**：最多可置顶5条对话
5. **历史对话删除**：删除含附件的对话时请注意选择是否同时删除附件
6. **清空全部对话**：需输入确认文字方可执行，不可恢复
7. **账号注销**：提交注销后有7天冷静期，期间可取消
8. **色彩更新**：全站主色调已统一为蓝紫色 #5B6CFF
