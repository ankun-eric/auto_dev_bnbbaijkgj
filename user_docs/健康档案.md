## 一、功能介绍

**「健康档案」** 是您在 bini-health 平台上记录、管理和追踪家庭成员健康信息的核心模块。本次升级实现了 **H5 / 微信小程序 / Flutter APP** 三端 UI 与数据完全一致，并把入口地址统一回简洁的 `/health-profile`。

### 1.1 本次更新亮点

| 升级点 | 升级前 | 升级后 |
|---|---|---|
| **H5 入口地址** | 必须访问 `/health-profile-v2`，主路径是 404 | 主入口 `/health-profile` 直接进入，路径干净无后缀 |
| **小程序界面** | 旧版独立 UI，模块字段较少 | 全新 UI，与 H5 完全一致，支持成员切换、Hero 卡、5 大模块 Tab |
| **APP 界面** | 旧版独立 UI | 全新 UI，与 H5 完全一致 |
| **数据互通** | 三端各管各的 | 任一端保存，其他端实时同步 |

### 1.2 五大模块速览

- **今日数据**：当天的血压、血糖、体温、心率、SpO2 等关键指标
- **健康信息**：基础病、过敏史、家族病史、习惯偏好（标签 + 自定义）
- **用药计划**：当前在服药品、提醒时间、剂量
- **关怀与提醒**：定时关怀消息、复查提醒
- **健康事件**：最近一段时间的就诊、检查、住院记录时间轴

## 三、操作步骤

### 3.1 切换家庭成员

页面顶部有 **成员横向滑动栏**，每个圆形头像代表一个家庭成员。点击切换即可查看/编辑该成员的健康档案。

- "本人" 始终在最左边
- 其他成员按关系（父母 / 配偶 / 子女）排列，下面显示昵称
- 当前选中的成员有蓝色描边和小铃铛标识

### 3.2 编辑 Hero 卡片（核心信息）

Hero 卡片位于成员栏下方，包含 **姓名、性别、年龄、身高、体重、BMI**。

操作：
1. 点击右上角 **"编辑"** 按钮
2. 在弹出表单中修改字段（数字字段会限制为合理范围）
3. 点击 **"保存"** → BMI 自动重算

### 3.3 切换 Tab 查看各模块

Hero 下方是 **5 个粘性 Tab**，从左到右依次为：

1. **今日数据** — 看今天的体征指标
2. **健康信息** — 基础病、过敏史、习惯偏好
3. **用药计划** — 在服药品和提醒
4. **关怀与提醒** — 自动关怀消息
5. **健康事件** — 时间轴事件流

### 3.4 健康信息 Tab：管理疾病/过敏/习惯标签

- **预设标签**：常见疾病、常见过敏原、健康习惯均以可点击的灰色标签显示，点击变成蓝色高亮即为选中
- **自定义标签**：输入框输入自定义内容 → 回车或点击 "+" 添加
- **删除**：点击已选标签上的 × 即可移除
- 修改后自动保存，无需点 "提交"

### 3.5 用药计划 Tab

- 显示当前生效的服药方案：药品名 / 剂量 / 频次 / 提醒时间
- 点击行右侧的箭头查看历史记录与备注
- 由 **医生端 / 后台运营** 端配置，本端为只读展示

### 3.6 健康事件 Tab

按时间倒序显示就诊 / 检查 / 住院记录。点击某一条进入详情页查看完整记录。

## 四、注意事项

### 4.1 数据安全

- 所有数据走 HTTPS 加密传输，存储在受访问控制保护的数据库
- 切换成员前请确认当前账号有权限查看，未授权成员会提示 "无权限"

### 4.2 老用户特别说明

- **旧版数据自动回填**，您之前在小程序或 APP 填过的疾病/过敏/用药信息都会原样出现
- **不需要重新登录**
- 字段可能换了位置，但都还在，找不到时请按 5 个 Tab 逐个看

### 4.3 浏览器/平台兼容性

- **H5**：Chrome / Safari / Edge / 微信内置浏览器（iOS+Android）均已测试
- **小程序**：微信 8.0.30+，iOS 15+ / Android 10+ 主流机型
- **APP**：iOS 14+，Android 10+

### 4.4 性能建议

- 首屏加载耗时一般 < 1.5s（4G 网络）。如出现长时间白屏，请检查网络
- 字段较多时（如疾病预设标签 30+），首次加载可能略慢；之后会本地缓存

### 4.5 常见问题（FAQ）

**Q1：我以前的书签 `/health-profile-v2` 打不开了？**
A：页面已升级，主入口统一为 `/health-profile`，内容完全一致。请重新保存书签：
`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile`

**Q2：小程序/APP 界面变了，是不是数据没了？**
A：本次为统一升级，所有数据完整保留，请放心使用。

**Q3：找不到某个字段（比如某项过敏原）？**
A：所有原有字段都还在，只是位置可能调整。请在 "健康信息" Tab 的预设标签或自定义标签中查找。

**Q4：保存后另一端没同步？**
A：请下拉刷新另一端页面。若仍未同步，请联系客服。
