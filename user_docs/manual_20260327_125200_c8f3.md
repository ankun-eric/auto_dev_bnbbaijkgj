## 功能简介

「宾尼小康」是一款AI驱动的一站式健康管理与健康服务平台，覆盖APP（iOS/Android）、微信小程序、H5网页端、Web管理后台和核销小程序。主要功能包括：

### 🤖 AI智能健康服务
- **智能健康问答**：支持文字输入与AI对话，获取个性化健康建议
- **体检报告解读**：上传体检报告（拍照或PDF），AI自动分析指标并给出建议
- **症状自查**：引导式问诊，AI分析可能的疾病方向和就医建议
- **中医辨证**：舌诊拍照、面诊拍照、体质测评，AI综合分析给出养生方案
- **药物查询**：搜索药名查看药品信息，AI分析用药安全，支持拍照识药

### 📋 健康档案管理
- 记录个人基础信息、过敏史、既往病史、用药记录、就诊记录
- 体检指标趋势追踪，支持历史数据对比
- 数据自动关联AI问诊，提供个性化建议

### 👨‍👩‍👧‍👦 家庭健康中心
- 添加家庭成员，每位成员独立健康档案
- 可替家庭成员进行AI问诊
- 紧急SOS一键求助功能

### 🏥 健康服务预约
- 5大服务品类：健康食品、口腔服务、体检服务、专家咨询、养老服务
- 在线预约专家咨询，查看排班时间
- 订单全流程管理（下单→支付→履约→评价）

### 🎯 健康计划与打卡
- AI根据健康档案自动生成个性化健康计划
- 每日健康任务打卡，完成可获得积分奖励
- 定期生成健康行为报告

### 💎 积分与会员体系
- 签到、任务、邀请等多种方式获取积分
- 积分可兑换商品或抵扣订单
- 4级会员等级（普通→银卡→金卡→钻石），享受不同折扣权益

### 📚 健康知识科普
- 丰富的健康科普文章和短视频内容
- 评论互动，收藏分享
- AI个性化推荐

### 💬 智能客服
- AI客服自动回复常见问题
- 需要时可转接人工客服

## 使用说明

### 一、注册与登录

1. 打开应用/H5页面/小程序
2. 在登录页面输入手机号
3. 点击「获取验证码」获取短信验证码
4. 输入验证码后点击「登录」
5. 首次登录自动注册账号

### 二、AI健康问答

1. 在首页点击「AI问诊」或底部导航「AI问诊」Tab
2. 选择问诊类型（健康问答/症状自查/中医辨证/药物查询）
3. 点击「新建对话」开始与AI交流
4. 在对话框中输入您的健康问题
5. AI会结合您的健康档案给出个性化回答

### 三、体检报告解读

1. 在首页点击「体检报告」
2. 点击「上传报告」，选择拍照或从相册选择
3. 系统自动识别并分析报告中的指标
4. 查看AI解读结果，了解各项指标含义
5. 解读结果自动归档至健康档案

### 四、中医辨证

1. 在首页点击「中医辨证」
2. 依次完成：
   - 舌诊：拍摄舌头照片
   - 面诊：拍摄面部照片
   - 体质测评：回答九种体质问卷
3. AI综合分析给出辨证结果和养生方案

### 五、健康档案

1. 在「我的」页面点击「健康档案」
2. 填写基础信息（身高/体重/血型等）
3. 可添加过敏史、既往病史、用药记录等
4. 所有信息在AI问诊时自动关联

### 六、家庭成员管理

1. 在「我的」页面点击「家庭成员」
2. 点击「添加成员」，填写姓名和关系
3. 每位成员拥有独立的健康档案
4. 可切换成员进行AI问诊

### 七、服务预约

1. 在底部导航点击「服务」Tab
2. 选择服务分类（健康食品/口腔/体检/咨询/养老）
3. 点击感兴趣的服务查看详情
4. 点击「立即预约」或「购买」完成下单
5. 线下服务到店后出示核销码即可

### 八、积分中心

1. 在「我的」页面点击「积分中心」
2. 每日点击「签到」获取积分
3. 完成健康任务打卡获取更多积分
4. 进入「积分商城」兑换商品或权益

### 九、Web管理后台使用（管理员）

1. 访问管理后台链接
2. 使用管理员账号登录（默认: 手机号 13800000000，密码 admin123）
3. 在左侧菜单中可管理：
   - **数据概览**：查看平台运营数据
   - **用户管理**：查看/管理用户
   - **AI模型配置**：配置AI大模型参数（baseURL/模型名/API Key）
   - **服务管理**：管理服务分类和项目
   - **订单管理**：处理订单和退款
   - **内容管理**：发布/管理文章和视频
   - **积分体系**：配置积分规则和商城
   - **客服工作台**：在线客服对话管理
   - **系统设置**：基础配置/推送/协议管理

### 十、核销小程序使用（门店人员）

1. 下载并导入核销小程序到微信开发者工具
2. 使用商家账号登录
3. 点击「扫码核销」
4. 扫描用户订单的核销码
5. 确认核销完成

## 注意事项

1. **AI功能需配置**：AI问答、体检解读等AI功能需要管理员在后台配置AI大模型参数后才能正常使用
2. **登录安全**：请勿将登录验证码告知他人
3. **健康建议仅供参考**：AI给出的健康建议仅供参考，不能替代专业医疗诊断，如有不适请及时就医
4. **数据安全**：平台采用加密传输和存储，保障您的健康数据安全
5. **管理员默认账号**：手机号 13800000000，密码 admin123，首次登录后建议修改密码
6. **小程序体验**：需要安装微信开发者工具导入代码包后体验
7. **安卓安装**：需要在手机设置中允许安装未知来源应用
8. **iOS安装**：需要使用第三方工具侧载安装IPA文件
9. **网络要求**：使用AI功能和在线服务需要良好的网络连接
