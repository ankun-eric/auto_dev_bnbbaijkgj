## 功能简介

本次更新为 bini-health 项目新增了 **AI对话记录查询功能**，覆盖管理后台和用户端（H5/小程序/Flutter App），主要包括：

1. **管理后台**：管理员可查看所有用户的 AI 对话记录，支持多维度筛选、分页、详情查看和导出
2. **用户端历史会话**：用户可查看和管理自己的 AI 聊天历史，支持时间分组、置顶、重命名、删除
3. **会话分享**：用户可生成分享链接，他人无需登录即可查看对话内容
4. **数据保留**：用户删除的记录实行软删除，管理员端仍然可见

## 一、管理后台 — AI对话记录管理

### 入口路径

管理后台 → 左侧菜单「AI对话记录」

### 1. 对话记录列表

进入「AI对话记录」页面后，显示所有用户的 AI 对话列表，包含以下信息：

| 列名 | 说明 |
|------|------|
| 用户 | 发起对话的用户昵称 |
| 对话标题 | 会话标题（默认为首条消息摘要） |
| AI模型 | 使用的 AI 模型名称 |
| 消息轮次 | 该会话中的消息数量 |
| 创建时间 | 会话创建时间 |
| 最后活跃 | 最后一条消息时间 |
| 状态 | 正常 / 已删除（用户端软删除标记） |
| 操作 | 查看详情、导出 |

#### 筛选与搜索

页面顶部提供以下筛选条件：

- **用户搜索**：输入用户昵称或手机号进行模糊搜索
- **时间范围**：通过日期选择器筛选创建时间范围
- **关键词**：搜索对话内容中包含的关键词
- **AI模型**：下拉选择使用的 AI 模型进行筛选
- **消息轮次范围**：设置最小/最大消息数量进行筛选

点击「搜索」按钮应用筛选条件，点击「重置」清除所有筛选条件。

#### 导出功能

在列表页面，每条记录的操作栏有「导出」按钮，支持两种格式：

- **CSV 格式**：纯文本格式，适合数据分析
- **Excel 格式（XLSX）**：带格式的电子表格，适合归档和阅读

导出内容包含完整的对话消息、发送时间、角色（用户/AI）等信息。

### 2. 对话详情页

点击列表中的「查看」按钮，进入对话详情页，包含以下内容：

#### 基础信息区

以描述列表形式展示：
- 用户昵称、手机号
- 会话标题
- 使用的 AI 模型
- 创建时间、最后活跃时间
- 消息总数
- 设备信息
- IP 地址及归属地
- 状态（是否被用户删除）

#### 对话消息区

以聊天气泡形式展示完整对话内容：

- **用户消息**：蓝色气泡，右侧对齐，显示用户头像
- **AI回复**：灰色气泡，左侧对齐，显示 AI 头像
- 每条消息下方显示发送时间
- AI 消息额外显示元数据：响应耗时（毫秒）、Token 消耗（Prompt/Completion）

## 二、用户端 — 对话历史管理

### 适用平台

H5 网页端、微信小程序、Flutter App 均支持以下功能，交互方式根据平台特性有所不同。

### 1. 打开历史会话侧边栏

#### H5 端
- 在 AI 首页点击左上角菜单图标（三条横线），或点击「查看全部」
- 在对话页面点击顶部导航栏的菜单图标
- 侧边栏从左侧滑出（移动端为抽屉样式）

#### 小程序端
- 在 AI 首页点击「历史对话」按钮
- 在对话页面点击顶部的菜单图标
- 抽屉从左侧滑出

#### Flutter App 端
- 在 AI 首页点击左上角菜单图标
- 在对话页面点击 AppBar 中的历史记录图标
- 侧边栏以 Drawer 形式展开

### 2. 历史会话列表

侧边栏中按时间分组展示历史对话：

| 分组 | 说明 |
|------|------|
| 今天 | 当天创建的对话 |
| 昨天 | 昨天创建的对话 |
| 最近7天 | 最近一周内的对话 |
| 最近30天 | 最近一个月内的对话 |
| 更早 | 超过30天前的对话 |

每条会话显示标题（首条消息摘要）和最后活跃时间。已置顶的会话在列表顶部显示，并带有置顶标记。

### 3. 会话操作

**点击会话**：跳转到该对话页面，继续聊天。

**长按 / 右键会话**（弹出操作菜单）：

| 操作 | 说明 |
|------|------|
| 重命名 | 修改会话标题，弹出输入框填写新名称 |
| 置顶 / 取消置顶 | 将会话固定在列表顶部或取消固定 |
| 分享 | 生成分享链接，可复制或转发 |
| 删除 | 软删除该会话，用户端不再显示（管理员仍可见） |

#### 各平台操作方式差异

| 平台 | 触发方式 |
|------|---------|
| H5 | 长按会话项弹出 ActionSheet |
| 小程序 | 长按会话项弹出 wx.showActionSheet |
| Flutter App | 长按会话项弹出 BottomSheet |

### 4. 新建对话

侧边栏顶部有「新建对话」按钮，点击后关闭侧边栏并跳转到新的空白对话页面。

## 三、会话分享功能

### 生成分享链接

1. 在历史会话侧边栏中，长按某条会话
2. 选择「分享」操作
3. 系统自动生成唯一分享链接
4. 链接自动复制到剪贴板（小程序端弹出提示）

### 查看分享内容

分享链接格式：`https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/shared/chat/{token}`

- 任何人打开链接即可查看对话内容，**无需登录**
- 页面为只读模式，不可编辑或继续对话
- 页面顶部显示品牌信息和对话标题
- 底部显示免责声明："以上内容由 AI 生成，仅供参考"

## 四、API 接口说明

### 管理员接口（需管理员 Token）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/chat-sessions` | 获取所有对话列表（支持筛选分页） |
| GET | `/api/admin/chat-sessions/{id}` | 获取对话详情（含消息列表） |
| GET | `/api/admin/chat-sessions/{id}/export` | 导出对话（支持 csv/xlsx） |

#### 列表查询参数

| 参数 | 类型 | 说明 |
|------|------|------|
| user_search | string | 用户昵称/手机号模糊搜索 |
| start_date | string | 开始日期（YYYY-MM-DD） |
| end_date | string | 结束日期（YYYY-MM-DD） |
| keyword | string | 对话内容关键词 |
| model_name | string | AI 模型名称 |
| min_messages | int | 最小消息数 |
| max_messages | int | 最大消息数 |
| page | int | 页码（默认1） |
| page_size | int | 每页条数（默认20） |

### 用户接口（需用户 Token）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/chat-sessions` | 获取当前用户的对话列表 |
| GET | `/api/chat-sessions/{id}` | 获取对话详情 |
| PUT | `/api/chat-sessions/{id}` | 重命名对话标题 |
| PUT | `/api/chat-sessions/{id}/pin` | 置顶 / 取消置顶 |
| DELETE | `/api/chat-sessions/{id}` | 删除对话（软删除） |
| POST | `/api/chat-sessions/{id}/share` | 生成分享链接 |

### 公开接口（无需认证）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/shared/chat/{share_token}` | 查看分享的对话内容 |

## 五、注意事项

1. **数据保留策略**：用户删除的对话记录仅在用户端隐藏，管理后台仍可查看和导出，状态标记为「已删除」
2. **分享链接安全**：分享链接包含随机 Token，不可猜测；但任何拥有链接的人均可查看内容，请注意隐私保护
3. **导出权限**：仅管理员可导出对话记录，用户端不支持导出
4. **消息元数据**：AI 回复的响应时间和 Token 消耗仅在管理后台详情页展示，用户端不可见
5. **默认排序**：所有对话列表默认按最后活跃时间倒序排列，置顶的对话始终排在最前
