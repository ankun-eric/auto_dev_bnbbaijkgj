## 使用说明

### 一、查看「数字一致性」修复效果（H5）

1. **登录账号**：用任意已登录账号打开「会员中心」页面
2. **观察两数字**：
   - 顶部蓝色「邀请家人」大卡片右侧的「已管理 X / 上限 Y」
   - 下方紫色「本月配额」卡第三格「家庭成员 X/Y 人 · 已管理 X 人」
   - **两处的 X 必须完全相等**（修复前可能不一致）
3. **新增 / 删除家人验证同步增减**：进入「健康档案 → 档案列表」，新建或删除一个家人档案后，返回会员中心，两处数字应同步变化

### 二、查看「健康档案列表卡片」新文案

1. 打开「健康档案」首页
2. 切换到「本人」Tab
3. 找到「健康档案列表」入口卡片：
   - 副标题统一为 **`已管理 X / 上限 Y`** 格式
   - 不限档套餐时显示 `已管理 X / 上限 不限`

### 三、查看「守护我的人卡片」新文案

1. 打开「健康档案」首页（本人 Tab）
2. 找到「守护我的人」入口卡片
3. 副标题统一为 **`守护者 X / 上限 Y`** 格式
   - 这里的 X 是「别人来守护我的人数」**不含本人**
   - 不限档时显示 `守护者 X / 上限 不限`

### 四、口径速查表（一图记牢谁含本人、谁不含）

| 卡片 / 数字 | 含义 | 是否含本人 | 文案 |
| --- | --- | --- | --- |
| 会员中心 · 蓝卡片 | 我管理的家庭档案数 | ✅ 含本人 | `已管理 X / 上限 Y` |
| 会员中心 · 配额卡家庭成员 | 我管理的家庭档案数 | ✅ 含本人（修复后） | `X / Y` + `已管理 X 人` |
| 健康档案 · 档案列表卡片 | 我管理的家庭档案数 | ✅ 含本人 | `已管理 X / 上限 Y` |
| 健康档案 · 守护我的人卡片 | 别人来守护我的人数 | ❌ 不含本人 | `守护者 X / 上限 Y` |

> **记忆口诀**：凡是"我管的档案"都含本人；唯独"守护我的人"不含本人（因为那是统计别人，不是统计自己）。

6. **预览体验**：导入成功后，开发者工具会自动编译并在模拟器中展示小程序界面
7. **重点查看**：进入「健康档案」页面 → 本人 Tab，确认双卡片副标题展示为：

## 注意事项

1. **缓存清理**：浏览器若仍显示旧文案，请 `Ctrl + F5` 强制刷新或清除浏览器缓存
2. **真值验证**：若您账号下确实有 3 个家庭档案（含本人），现在两个数字必须都是 3；如发现仍不一致，可联系开发组排查 DB 是否残留脏数据
3. **小程序版本**：本次小程序 zip 包对应代码版本 `20260531_184756_d2ec`，旧版本请下载并替换
4. **不限档套餐**：付费"不限"档套餐的所有卡片右侧 Y 均显示为 `不限`，X 仍正常显示当前数量
5. **守护我的人 X 不含本人**：这是因为该卡片统计的是"别人来守护我的人数"，与"我管理的家庭档案"语义不同
