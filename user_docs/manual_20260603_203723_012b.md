## 本次变更概要

本次为运维层面的基础设施升级，**不涉及业务功能变更**，用户无需任何操作。

### 变更 1：代码仓库迁移

- **原仓库**：GitHub（`github.com/ankun-eric/auto_dev_bnbbaijkgj`）
- **新仓库**：阿里云 Codeup（`codeup.aliyun.com/6a05a6159b7ce0afb00c035e/...`）
- GitHub 仓库仍保留，**仅用于 APP 打包环节**（通过 GitHub Actions 构建安卓 APK 和 iOS IPA）
- 日常开发迭代和代码管理统一使用 Codeup，访问更快更稳定

### 变更 2：部署方案升级

- **原方案**：子路径部署（通过宿主机端口映射 + 旧 Nginx 配置，已废弃）
- **新方案**：子域名部署（每个项目拥有独立域名，通过 gateway-nginx 按域名路由）
- 优势：部署更稳定、项目间隔离更好、不易出现路由冲突
- 旧部署配置已清理，服务器资源已释放

### 变更 3：Docker 基础镜像迁移

- 所有 Dockerfile 的 FROM 镜像从 Docker Hub 公共镜像迁移至阿里云 ACR 私有仓库
- 构建速度更快（国内网络环境），拉取更稳定
- 镜像版本：Python 3.12-slim / Node.js 20-alpine
