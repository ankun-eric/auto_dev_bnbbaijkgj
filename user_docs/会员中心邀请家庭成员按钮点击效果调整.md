## 功能简介

本次更新只调整了**一处交互效果**：

- 在「会员中心」页面里那张蓝色卡片上的**「邀请家庭成员」按钮**，点击后的效果，调整为与「健康档案 → 家庭成员 → 去邀请」**完全一致**。
- 调整后，点击该按钮会**在当前页面弹出一个半屏浮层**完成邀请家庭成员的操作，**不再跳转到新的页面**，全程不离开会员中心。

> 本次**只改了点击效果**，卡片的外观、高度、布局、按钮文字（仍为「邀请家庭成员」）、按钮位置、以及「已管理 X / 上限 Y」的数量信息**都保持原样、没有任何变化**。「家庭守护」卡片也未受任何影响。

## 使用说明

1. **进入会员中心**：在浏览器打开 [会员中心页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/member-center)（如未登录，请先按提示完成登录）。
2. **找到蓝色卡片**：页面顶部下方有一张蓝色渐变卡片，左侧显示套餐名与「已管理 X / 上限 Y」，右侧是白色圆角的「邀请家庭成员」按钮。
3. **点击「邀请家庭成员」**：点击后，页面底部会**弹起一个半屏浮层**（添加成员表单），而不是跳转到新页面。
4. **在浮层内填写并保存**：
   - 选择「成员关系」，填写姓名、性别、出生日期等必填项（可展开「其他（选填）」填身高、体重、既往病史、过敏史）。
   - 点击底部「保存」即可完成添加。
5. **关闭后仍在会员中心**：保存成功（或点击浮层外的灰色遮罩、左上角返回箭头关闭）后，您**仍然停留在会员中心页面**；新增成功后，卡片上的「已管理」人数会自动 +1。

> 小贴士：这套半屏浮层与「健康档案 → 家庭成员 → 去邀请」点出来的浮层是**同一套**，外观与操作方式完全相同，您在两处的体验保持一致。

## 注意事项

- **达到上限时**：若家庭成员数量已达套餐上限，「邀请家庭成员」按钮会处于禁用状态，点击不会弹出浮层。此时可在卡片提示处点击「升级套餐」获取更多名额。
- **需先完善本人档案**：若您本人的档案尚未完善（缺少出生日期等），打开浮层时会先提示「请先完善您的个人档案」，按提示完成后即可继续添加家庭成员。
- **登录要求**：邀请家庭成员需要登录后操作；未登录会先跳转登录。
- **本次无需更新任何客户端**：本次仅调整了网页端（H5）的一处按钮点击效果，**没有改动微信小程序、安卓、苹果等任何客户端**，无需下载或重新安装任何安装包。
