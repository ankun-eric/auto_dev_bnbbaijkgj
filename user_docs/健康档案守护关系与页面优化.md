8. **卡片简化** — 去掉副标题，只保留主标题 + 数量 + 图标
9. **守护者列表页** — 展示守护者头像、昵称、守护时间、权限等详细信息

## 使用说明

### 一、健康档案首页（改造后）

1. 打开 H5 页面，登录后进入**健康档案**页面
2. 页面顶部为健康概览卡片（Hero 区域），下方为两张并排卡片：
   - **「我守护的人」**（橙色图标）— 显示您正在守护的家人数量，点击进入家庭成员列表
   - **「守护我的人」**（绿色图标）— 显示正在守护您的人数量，点击进入守护者列表
3. 再往下依次为：健康看板入口、提醒管理入口、**我的设备入口**（紫色图标）

### 二、查看「守护我的人」

1. 在健康档案首页点击**「守护我的人」**绿色卡片
2. 进入守护者列表页，可查看每位守护者的：
   - 头像和昵称
   - 守护开始时间
   - 权限范围（默认：全部健康信息）
   - 最近查看时间
3. 如果尚无守护者，页面会显示引导文案和邀请按钮

### 三、邀请别人守护我（反向邀请）

1. 在「守护我的人」列表页，点击底部的**「邀请别人守护我」**按钮
2. 系统自动生成一个邀请链接和二维码
3. 将链接通过微信等方式分享给您希望守护您的人
4. 对方打开链接后注册/登录，点击**「同意守护」**即可建立守护关系
5. 守护关系建立后，对方会自动出现在您的「守护我的人」列表中

**邀请规则说明：**
- 每个邀请链接最多 **3 人** 使用
- 邀请链接有效期 **24 小时**，过期后需重新生成
- 已是守护者的人再次点击链接会提示"你已经在守护 TA 了"
- 不能守护自己

### 四、解除守护关系

1. 在「守护我的人」列表页，点击某个守护者卡片上的**「解除守护」**按钮
2. 弹出确认弹窗，内容为："解除后，对方将无法查看您的健康数据。确定要解除吗？"
3. 点击**「确认解除」**后，守护关系立即终止
4. 该守护者将从您的列表中移除，对方也无法再查看您的健康数据

### 五、设备成员维度过滤

1. 在健康档案首页切换不同家庭成员 Tab
2. 点击**「我的设备」**入口进入设备页面
3. 设备列表会自动按当前选择的成员进行过滤：
   - **本人 Tab** → 显示账号下全部设备
   - **其他成员 Tab**（如"爸爸"）→ 仅显示绑定到该成员的设备
4. 如果该成员下没有绑定设备，显示"暂无绑定设备"空状态

### 六、用药计划（竞态修复）

1. 从健康档案首页点击用药计划区域的"全部"进入列表页
2. 列表页会先显示加载状态，待数据准备就绪后一次性展示正确数据
3. 不再出现之前的"数量先闪烁再修正"的问题

## 注意事项

1. **守护者权限**：守护者默认可查看被守护人的全部健康信息（健康数据、用药提醒、健康看板、设备数据等），不做细粒度权限控制
2. **数据同步**：守护者和被守护人操作的是同一份健康档案数据，双方的修改互相可见
3. **守护者人数上限**：每个被守护人最多 3 个守护者
4. **双向守护**：A 守护 B 和 B 守护 A 可同时存在，是两条独立的关系
5. **解除守护**：被守护人可单方面解除，无需守护者同意
6. **反向邀请**：邀请链接 24 小时有效，最多 3 人使用
7. **自动创建档案**：接受反向邀请后，系统自动在守护者名下创建被守护人的家庭成员档案，关系名称默认使用被守护人的昵称
