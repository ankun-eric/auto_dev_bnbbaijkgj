## 功能简介

本次修复了**会员等级管理**菜单的报错问题。修复前，管理后台中的「会员等级配置」页面因前后端数据字段不匹配而崩溃，表格无法正常渲染。修复后，以下功能恢复正常：

- **查看会员等级列表**：正确显示等级名称、图标、积分范围、折扣、权益、标签颜色、会员人数
- **新增会员等级**：支持自定义等级名称、图标（Emoji）、积分范围、折扣比例、权益说明、标签颜色
- **编辑会员等级**：点击「编辑」可修改已有等级的所有属性
- **删除会员等级**：点击「删除」可移除不需要的等级

## 使用说明

### 1. 登录管理后台

1. 打开管理后台链接：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 使用管理员账号登录（默认账号：admin / 密码：admin123）

### 2. 进入会员等级管理

1. 登录后，在左侧导航菜单中找到「积分管理」分组
2. 点击「会员等级」菜单项，即可进入会员等级配置页面

### 3. 查看会员等级列表

进入页面后，会自动加载当前系统中的所有会员等级信息，以表格形式展示：

| 列名 | 说明 |
|------|------|
| ID | 等级编号 |
| 等级名称 | 等级名称和 Emoji 图标，带颜色标签 |
| 积分范围 | 该等级的最低积分 ~ 最高积分 |
| 折扣 | 该等级享受的折扣（如 9折、8.5折） |
| 权益 | 该等级包含的会员权益说明 |
| 会员人数 | 当前处于该积分范围内的会员数量 |
| 操作 | 编辑 / 删除按钮 |

### 4. 新增会员等级

1. 点击页面右上角的「新增等级」按钮
2. 在弹出的表单中填写以下信息：
   - **等级名称**：如「黄金会员」
   - **等级图标**：输入一个 Emoji 表情，如 🥇
   - **最低积分**：该等级的起始积分
   - **最高积分**：该等级的结束积分
   - **折扣**：输入整数，如 90 表示 9 折
   - **标签颜色**：选择颜色标签
   - **会员权益**：输入权益说明，多项用逗号分隔
3. 填写完毕后，点击「确定」即可创建

### 5. 编辑会员等级

1. 在等级列表中找到需要修改的等级
2. 点击对应行的「编辑」按钮
3. 在弹出的表单中修改需要更新的信息
4. 点击「确定」保存修改

### 6. 删除会员等级

1. 在等级列表中找到需要删除的等级
2. 点击对应行的「删除」按钮
3. 在确认弹窗中点击「确定」完成删除

## 注意事项

1. **管理员权限**：会员等级管理功能仅管理员可操作，需使用管理员账号登录
2. **积分范围不要重叠**：不同等级的积分范围应该连续且不重叠，避免会员被划分到多个等级
3. **折扣数值含义**：折扣字段输入 90 表示 9 折，100 表示无折扣（原价），85 表示 8.5 折
4. **会员人数统计**：「会员人数」列显示的是当前积分处于该等级区间的用户数量，会实时统计
5. **删除需谨慎**：删除会员等级后不可恢复，请确认后再操作
