## 功能简介

本次优化围绕「**严重性提示更直观、数据来源更透明、趋势分析更专业**」三条主线，对健康档案血压模块的两处入口做了升级：

1. **首页小卡片**（健康档案首页 → 今日健康数据 → 血压卡）  
   新增右上角「严重性胶囊」和底部「时间 · 来源」一行，让用户一眼看清最新一次测量结果的严重程度和来源。

2. **血压详情页**（点击血压小卡片进入）  
   - 顶部主数值大卡保持原样不变  
   - 主卡和趋势图之间，新增并排大按钮：**手工录入（实心主色）** + **绑定设备（描边白底）**  
   - 趋势图重做：仅保留「日 / 周」两档，默认选中「周」；新增范围带、平均值平滑连线、参考线 SBP=140 / DBP=90、点击数据点弹窗、空状态插画引导。

## 使用说明

### 一、查看首页血压小卡片

1. 打开 [健康档案首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/)（如未登录会先跳转到登录页，登录后自动回到本页）。
2. 滚动至「**今日健康数据**」区域，可在 2x2 卡片网格中看到血压卡（左上角第一张）。
3. 卡片元素自上而下依次为：
   - 顶部行：左侧 `💓 血压` 标题；**右侧严重性胶囊**（如：`正常` 蓝色 / `轻度偏高` 黄色 / `严重偏高` 橙色）。
   - 中部：超大字号血压值，如 `142/92`，单位 `mmHg`。
   - 底部：分隔细线 + 一行小字 `今日 08:30 · 欧姆龙血压计·自动同步` 或 `昨日 21:15 · 手工录入`。
4. 时间显示规则：
   - 当天 → `今日 HH:mm`
   - 昨天 → `昨日 HH:mm`
   - 2~7 天前 → `X 天前`
   - 超过 7 天 → `MM-dd`
5. 点击卡片跳转至**血压详情页**。

### 二、血压详情页

详情页从上到下分为四个区块：

#### 1）顶部主卡片（保持原样）
- 大字号主数值 + `mmHg`
- 「时间 · 来源」一行
- 居中胶囊状态标签（与主卡片同色调）

#### 2）操作按钮区（新位置：主卡片下方、趋势图上方）
- **手工录入**（实心主色）：高频操作，点击弹出录入弹窗，可填写收缩压 / 舒张压 / 时段（晨起 / 午间 / 晚间）→ 保存。
- **绑定设备**（描边白底）：次要操作，本期点击仅 Toast 「即将上线」（设备绑定通道开通后会跳转）。

#### 3）最近 7 天趋势

- 标题左侧：`最近 7 天趋势`（切换到日视图时显示「今日趋势」）。
- 标题右侧：紧凑型分段控件，仅 `日` / `周` 两档；**默认选中「周」**。
- 周视图：
  - X 轴：周一 / 周二 / … / 今日（最右一格高亮）。
  - Y 轴固定 `40 - 200 mmHg`（每 40 一格，刻度 40 / 80 / 120 / 160 / 200）。
  - 每天的呈现：
    - 范围带：当日所有测量的 `min - max` 浅色色带（收缩压深蓝、舒张压浅蓝）。
    - 平均值平滑连线：贝塞尔曲线连接每天的均值。
    - 数据点：在每天平均值位置画圆点，**按当日均值档位染色**（蓝 / 黄 / 橙）。
  - 参考线：两条横向浅灰虚线，SBP = 140 mmHg、DBP = 90 mmHg，右侧标注。
  - 异常天背景：连续偏高 / 严重档位的日子，曲线下方背景浅黄色带。
- 日视图：
  - X 轴：`00:00 - 24:00`。
  - 当日所有测量点按时间散点 + 平滑连线（双线：收缩压 / 舒张压）。

#### 4）数据点点击弹窗

- 点击趋势图任一数据点，从底部弹出「血压详情」抽屉。
- 内容包括：
  - 测量日期 + 时间
  - 收缩压 + 舒张压（mmHg）
  - 严重性档位（高亮胶囊）
  - 测量时段（晨起 / 上午 / 下午 / 睡前 等，如有）
  - 来源（手工录入 / `{设备名}·自动同步`）
  - 周视图模式下，如果当日有多次测量，会注明「当日共 N 次测量，此处显示当日均值」。
- 点击「知道了」按钮或弹窗外区域关闭。

### 三、空状态

- 当所选周期（日 / 周）内没有任何测量记录时，趋势图区域显示插画 + 提示：
  - 周视图：`本周还没有测量记录`
  - 日视图：`今日还没有测量记录`
  - 副文案：`点击上方"手工录入"或"绑定设备"开始记录吧`
- 提供两个快捷按钮：「立即录入」「绑定设备」（点击行为等同上方按钮区）。

## 严重性档位规则（沿用现有，仅明示配色）

| 档位 | 收缩压 (mmHg) | 舒张压 (mmHg) | 颜色 |
|---|---|---|---|
| 偏低 | < 90 | < 60 | 橙 |
| 正常 | 90 ~ 119 | 60 ~ 79 | 蓝 |
| 轻度偏高 | 120 ~ 139 | 80 ~ 89 | 黄 |
| 中度偏高 | 140 ~ 159 | 90 ~ 99 | 黄 |
| 严重偏高 | ≥ 160 | ≥ 100 | 橙 |

> 判定原则：收缩压与舒张压「或」关系——任一项落入更高档位即按更高档位定级。

## 注意事项

1. **登录态**：首次访问会跳转到登录页，登录后才能看到「健康档案首页」中的血压小卡片。
2. **数据为空时**：
   - 首页小卡片：主数值显示 `—`，无胶囊，无底部「时间·来源」行。
   - 详情页趋势图：显示空状态插画 + 双按钮。
3. **手工录入**：收缩压、舒张压均为必填数字字段；时段（晨起 / 午间 / 晚间）非必填。
4. **绑定设备**：本期为占位通道，点击只显示 Toast；后续版本会接入欧姆龙血压计、华为 Watch、小米手环等设备的真实绑定流程。
5. **趋势图维度调整**：本次仅 **血压 Tab** 的维度被收敛为「日 / 周」，**心率、血糖、血氧、睡眠** 等其他指标 Tab 维度保持原日 / 周 / 月 / 年四档不变。
6. **顶部主卡片不变**：本次升级**没有改变**详情页顶部主卡片的样式（大数值 + 同步信息行 + 状态胶囊），用户视觉迁移成本极低。
