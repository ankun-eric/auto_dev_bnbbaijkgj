## 功能简介

本次升级聚焦在 **AI 对话页拍照识药结果卡片** 与 **用药计划模块** 的深度联动，让咨询师 / 用户在 AI 对话中识别出某种药品后，**无需跳页**即可完成：

1. **一键加入用药计划** —— 抽屉内填写完整用药信息后保存；
2. **快速查看当前咨询人的用药计划** —— 抽屉内就地展开 / 编辑 / 删除任意记录；
3. **「是否已加入」状态可视化** —— 同一药品在当前咨询人名下、有效期内的用药计划中已存在时，卡片按钮自动变为「✓ 已加入用药计划」，避免重复添加。

## 使用说明

### 一、识药卡片新增按钮

在 AI 对话页拍照识药后，识别成功的药品卡片底部现在并排显示两个主按钮：

- 🔵 **+ 加入用药计划**（主色实心）：当前咨询人未加入该药品时显示；点击直接弹出「加入用药计划」抽屉
- ✅ **✓ 已加入用药计划**（灰色描边）：当前咨询人有效期内已有该药记录时显示；点击会弹出二次确认 Dialog「该药已在用药计划中，是否再次加入 / 修改？」
- 📋 **查看用药计划**：始终可点击，弹出「查看用药计划」抽屉

> 💡 **状态自动同步**：识药结果列表展示时会批量批准 / 刷新所有卡片的「已加入」状态；切换咨询人（顶部胶囊「本人 / 家人」）后也会自动重新计算。

### 二、加入用药计划抽屉

点击「+ 加入用药计划」打开抽屉，**与新增用药计划页面 100% 一致的全表单**：

| 字段 | 说明 |
|------|------|
| **药品名称** | 必填；自动预填识药结果中的药品名，可编辑 |
| **用药时间 / 每日次数** | 必填；选 1~4 次，自动生成默认时间点（08:00 / 20:00 等） |
| **每次剂量** | 必填；¼/½/1/2/... 与片/粒/mL/支 等单位组合 |
| **服用周期** | 必填；开始日期 + 结束日期，或勾选「长期服用」 |
| **服用时机** | 必填；饭前 / 饭后 / 空腹 / 随餐 / 睡前 5 选 1 |
| **备注** | 选填；最多 30 字 |

**抽屉底部**：

- **取消** 按钮：直接关闭抽屉，不保存
- **保存** 按钮：调用新增用药计划接口
  - ✅ 保存成功 → 关闭抽屉 → 识药卡片按钮立即变为「✓ 已加入用药计划」+ Toast「已加入用药计划」
  - ❌ 保存失败 → 抽屉不关闭，顶部红色 Toast 提示错误原因（可重试）

抽屉标题：
- **本人态**：「加入用药计划」
- **家庭咨询人态**：「为 {昵称} 加入用药计划」

### 三、查看用药计划抽屉

点击「📋 查看用药计划」打开抽屉，沿用 `/ai-home/medication-plans` 列表页样式，但承载形式改为抽屉弹出，**不再跳页**。

#### 列表项交互

| 操作 | 行为 |
|------|------|
| 点击列表项主体 | 在抽屉内就地展开 / 折叠该条计划的全部明细字段（通用名、剂量、频次、周期、时机、备注、关联疾病） |
| 编辑 | 列表项原地切换为表单态（字段可编辑），底部出现「保存修改 / 取消」 |
| 删除 | 弹出二次确认 Dialog「确定删除该用药计划？」→ 确认 → 调删除接口 → 移除 + Toast「已删除」 |

#### 空态

当前咨询人名下无任何用药计划时，抽屉内显示：

- 💊 大图标
- 文案：「您 / 该咨询人 暂无用药计划」
- **去新增** 按钮 → 关闭「查看」抽屉，自动打开「加入用药计划」抽屉（双抽屉切换，全程不开新页面）

### 四、咨询人切换联动

AI 对话页顶部「咨询对象」胶囊切换时（本人 ↔ 家人 A ↔ 家人 B），识药卡片上的「✓ 已加入用药计划」状态会**自动重新计算**：

- 同一种药品，本人下已加入，但家人 A 名下未加入 → 切到家人 A 后按钮恢复为「+ 加入用药计划」
- 反之亦然

### 五、典型流程（端到端示例）

1. 在 AI 对话首页点击 **拍照** → 拍摄药品包装盒
2. AI 识别成功 → 识药卡片显示，底部出现「+ 加入用药计划」与「📋 查看用药计划」
3. 点击 **+ 加入用药计划** → 抽屉打开 → 药品名已自动填好
4. 选择剂量「1 片 / 每日 2 次」、用药时间「08:00 / 20:00」、服用周期「2026-05-18 ~ 长期」、服用时机「饭后」→ 点击 **保存**
5. 抽屉关闭 → 卡片按钮变为「✓ 已加入用药计划」+ Toast 提示
6. 点击 **📋 查看用药计划** → 抽屉打开 → 列表中可看到刚刚添加的记录
7. 点击列表项展开 → 点击 **编辑** → 修改剂量 → **保存修改** → 抽屉内列表刷新

## 注意事项

1. **匹配为宽松模糊匹配**：识药结果是「阿莫西林胶囊」，已加入的计划是「阿莫西林克拉维酸钾片」（通用名「阿莫西林」），仍会判定为「已加入」。复方制剂、剂型不同的同名药可能产生误判，必要时点击「+ 加入用药计划」走二次确认即可。
2. **咨询人维度隔离**：每个咨询人的用药计划相互独立；同一种药给本人加入过，并不代表给家人 A 也加入了。
3. **有效期判定口径**：用药计划的「服用周期」包含今天（或为「长期」）时才算「已加入」。计划自然结束或被手动归档后，会自动从「已加入」状态退出。
4. **抽屉操作不影响 AI 对话上下文**：保存 / 取消 / 关闭抽屉均不会清空当前会话，可继续提问。
5. **网络异常处理**：所有抽屉接口失败时**抽屉都不会关闭**，请重试或检查网络。
