## 功能简介

本次更新修正了 H5 移动端全站操作提示（Toast）的样式，将原本的大尺寸 antd-mobile 默认弹框升级为**微信风格紧凑带小图标**样式，同时新增了 **loading 加载状态**支持。

### 修正前 vs 修正后

| 对比项 | 修正前（v1.0） | 修正后（v2.0） |
|--------|---------------|---------------|
| 成功提示 | 大尺寸弹框 + 大勾号图标（约 120px 高） | 紧凑弹框 + 24px 小勾号（约 80px 高） |
| 失败提示 | 大尺寸弹框 + 大叉号图标 | 紧凑弹框 + 24px 小叉号 |
| 加载提示 | 不支持 | 新增：紧凑弹框 + 旋转动画 + 透明遮罩 |
| 整体风格 | antd-mobile 原生大图标模式 | 微信风格紧凑带小图标 |

## 使用说明

### 1. 成功操作提示

当您在任何页面执行成功操作（如打卡、收藏、保存、修改设备信息等）后：

1. 屏幕正中央会浮出一个紧凑的深色半透明圆角矩形
2. 上方显示一个白色勾号小图标（✓）
3. 下方显示简洁的操作结果文字（如"已保存"、"已收藏"）
4. **1.5 秒后自动淡出消失**
5. Toast 显示期间不遮挡页面内容，您可继续操作

### 2. 失败操作提示

当操作失败时：

1. 同样在屏幕正中央浮出紧凑弹框
2. 上方显示一个白色叉号小图标（✕）
3. 下方显示失败原因文字（如"保存失败"、"网络异常"）
4. **1.5 秒后自动淡出消失**

### 3. 加载等待提示（新增功能）

当系统正在处理异步操作时（如提交表单、发送请求等）：

1. 屏幕正中央浮出紧凑弹框
2. 上方显示一个白色旋转加载动画（◎）
3. 下方显示等待文字（如"提交中..."、"加载中..."）
4. 同时会覆盖一层全屏透明遮罩，**阻止您重复操作**
5. 加载完成后弹框自动消失
6. 如果超过 30 秒仍未完成，系统会自动关闭 Toast 并移除遮罩，防止页面卡死

## 注意事项

1. **样式统一**：全站所有操作提示均已统一为微信风格紧凑样式，无需任何设置
2. **自动消失**：成功和失败提示 1.5 秒后自动消失，无需手动关闭
3. **加载防重复**：加载提示显示期间页面无法点击，防止重复提交
4. **超时保护**：加载提示超过 30 秒自动关闭，不会导致页面卡死
5. **兼容性**：支持微信内置浏览器、支付宝内置浏览器、Safari、Chrome 等主流手机浏览器
