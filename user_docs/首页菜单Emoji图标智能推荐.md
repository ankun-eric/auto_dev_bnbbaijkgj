## 功能简介

本次更新在管理后台「首页菜单管理」模块的新增/编辑弹窗中，新增了 **Emoji 图标智能推荐** 功能：

- 管理员填写菜单名称后，系统**自动根据关键词**推荐 3～5 个相关 Emoji 图标
- 点击推荐的 Emoji 即可一键选中作为菜单图标，无需手动搜索或上传图片
- Emoji 字符直接存储在数据库，前端各端（H5、App、小程序）无需任何改动即可渲染展示
- 已有图片 URL 类型的菜单图标保持兼容，不受影响

## 使用说明

### 进入首页菜单管理

1. 使用管理员账号登录管理后台：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单中找到 **「首页配置」→「菜单管理」**
3. 点击进入首页菜单管理页面

### 新增菜单（使用 Emoji 推荐）

1. 点击页面右上角的 **「新增菜单」** 按钮，弹出新增菜单对话框
2. 在 **「菜单名称」** 输入框中填写菜单名称（如"体检预约"）
3. 确认 **「图标类型」** 选择为 **「Emoji」**
4. 填写菜单名称后，**推荐图标区域** 会自动出现在浅蓝色背景框中，展示 3～5 个相关 Emoji

   例如：填写"体检预约"后，推荐图标会自动显示：🏥 🩺 💊 🔬 📋

5. **点击**任意一个推荐的 Emoji 图标，图标即被选中（选中后图标周围出现绿色边框高亮）
6. 选中后，**当前图标预览** 区域会以大号展示已选中的图标
7. 继续填写其他信息（跳转类型、跳转地址、排序等）
8. 点击 **「确定」** 保存菜单

### 编辑菜单（修改 Emoji 图标）

1. 在菜单列表中找到要编辑的菜单，点击 **「编辑」** 按钮
2. 弹窗中图标类型若已是 Emoji，推荐区域会根据当前菜单名称自动显示推荐图标
3. 可直接点击推荐列表中的其他 Emoji 来更换图标
4. 点击 **「确定」** 保存修改

## 注意事项

### 标题为空时推荐区域的状态

- 菜单名称**未填写时**，推荐图标区域会呈现置灰状态，Emoji 图标不可点击
- 同时显示提示文案：**「请先填写菜单名称以获取推荐图标」**
- 填写菜单名称后，推荐区域**实时更新**，立即展示匹配的 Emoji

### 图标选择方式

- **只能通过点击推荐列表中的 Emoji 来选择图标**（系统设计如此，简化操作）
- 也可以在"图标内容"文本框中手动输入 Emoji，但建议使用推荐列表
- 图标类型切换为「图片」时，推荐 Emoji 区域不显示，仍走原有的图片上传流程

### 前端展示兼容性

- H5、App（Flutter）、微信小程序等用户端**无需任何改动**，直接渲染 Emoji 字符
- 已配置图片 URL 的旧菜单继续显示图片，**不受影响**
- Emoji 图标在各主流系统（iOS、Android、Windows、macOS）上均有良好的原生渲染支持

### 数据库兼容性

- 系统数据库已确认支持 UTF-8 4 字节 Emoji 字符（包括 🏥、🩺、💊 等复杂 Emoji）的存储和读取
- 无需任何数据库迁移操作
