## 功能简介

本次改造把 AI 首页顶栏右上角从原来的 **4 个外露控件** 精简为 **3 个**：

- 改造前：`浅蓝「标准模式」徽章` + `绿色「关怀模式」按钮` + `🎁` + `➕`（拥挤、不直观）
- 改造后：`标准模式 ▾`（下拉胶囊） + `🎁`（邀请好友） + `➕`（更多菜单）

合并后的下拉胶囊：

- **收起状态**：胶囊里只显示「当前模式文字 + 下拉箭头 ▾」，例如 `标准模式 ▾`。
- **展开状态**：点击胶囊后箭头翻转朝上 ▴，下方弹出小面板，列出「标准模式 / 关怀模式」两行；当前所在模式高亮并打勾 ✓。
- **切换行为**：点击另一个模式即触发切换（保存偏好 → 弹出提示「已切换到关怀模式 ✓」→ 跳转对应模式首页）；点击当前模式或面板外区域只收起面板、不切换。

> 切换的业务逻辑沿用现有实现，未做任何改动；🎁 邀请好友入口保留在外面，功能正常。

## 使用说明

### 在 H5 网页版体验

1. 打开 [AI 首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)。
2. 看右上角，会看到一个浅蓝色胶囊 `标准模式 ▾`，右侧依次是 🎁 和 ➕。
3. **点击胶囊**：下方弹出下拉面板，箭头翻转为 ▴，面板内有两行：
   - 「标准模式」——当前所在模式，高亮显示并带 ✓ 打勾。
   - 「关怀模式」——可切换选项。
4. **切换到关怀模式**：点击「关怀模式」一行，系统会保存你的模式偏好，弹出提示「已切换到关怀模式 ✓」，并自动跳转到关怀模式首页。
5. **不想切换**：点击当前模式「标准模式」一行，或点击面板外的任意区域，面板会收起、箭头复位为 ▾，不做任何切换。
6. **邀请好友**：点击胶囊右侧的 🎁 图标，进入邀请好友页面（功能与之前一致）。

### 在微信小程序体验

操作方式与 H5 完全一致：右上角「标准模式 ▾」胶囊 → 点击展开 → 选择「关怀模式」即切换并跳转，选择当前模式或点面板外仅收起。

## 注意事项

- **模式切换仅一处入口**：右上角不再有单独的「标准模式」徽章和「关怀模式」按钮，所有模式相关操作都通过下拉胶囊完成。
- **切换需要登录**：邀请好友、模式偏好保存等涉及用户态的操作需要先登录；未登录时小程序会引导登录。
- **偏好保存失败不阻塞跳转**：若网络异常导致偏好保存失败，系统仍会完成提示与跳转，下次仍可正常切换（已在本地兜底记录）。
- **防重复点击**：切换过程中按钮短暂禁用，避免连续点击导致重复跳转。
- **两端表现一致**：H5 与微信小程序的胶囊外观、下拉面板、打勾高亮、箭头翻转交互保持一致。
