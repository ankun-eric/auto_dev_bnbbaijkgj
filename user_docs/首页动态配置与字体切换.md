## 功能简介

本次更新实现了 **首页动态配置** 与 **字体大小切换** 两大核心功能：

1. **首页动态配置**：将首页的搜索栏、Banner 轮播、宫格菜单从硬编码改为后台动态配置，运营人员可在管理后台自助维护，无需开发介入
2. **Banner 点击跳转**：修复了三端首页 Banner 不可点击的问题，支持配置跳转到应用内页面、外部 H5 链接或微信小程序
3. **字体大小切换**：新增 3 档字体大小调节功能（标准/大/超大），方便中老年用户调整阅读体验
4. **三端统一管理**：后台配置一次，H5、微信小程序、Flutter App 三端同步生效

## 使用说明

### 一、管理后台操作指南

#### 1. 首页基础设置

**入口**：管理后台 → 首页配置 → 首页基础设置

此页面分为三个配置区域：

**搜索框配置**
- **搜索框显示**：开关控制，关闭后三端首页不显示搜索栏
- **提示文字**：设置搜索框的默认提示文字，修改后三端统一展示

**宫格配置**
- **菜单列数**：下拉选择 3列/4列/5列，控制首页菜单的排列布局

**字体切换配置**
- **字体切换功能**：开关控制，关闭后三端隐藏所有字体切换入口
- **新用户默认档位**：设置新用户首次使用时的默认字体档位（标准/大/超大）
- **档位参数**：可微调每个档位的基础字号（单位 px）

每个配置区域都有独立的"保存"按钮，修改后点击保存即可生效。

#### 2. 首页菜单管理

**入口**：管理后台 → 首页配置 → 首页菜单管理

**新增菜单**
1. 点击左上角"新增菜单"按钮
2. 在弹窗中填写：
   - 菜单名称（最长 10 个字符）
   - 图标类型：选择 Emoji 或图片
   - 图标内容：输入 Emoji 字符或上传图片
   - 跳转类型：应用内页面 / 外部 H5 链接 / 微信小程序
   - 跳转地址：对应的路径或 URL
   - 排序值：数值越小越靠前
   - 是否显示：控制是否在前端展示
3. 点击"确定"保存

**排序菜单**
- 使用表格中的"上移""下移"按钮调整菜单顺序

**编辑/删除**
- 点击"编辑"修改菜单项属性
- 点击"删除"移除菜单项（需二次确认）
- 直接切换"显示状态" Switch 可快速上下线

#### 3. 首页 Banner 管理

**入口**：管理后台 → 首页配置 → 首页 Banner 管理

操作方式与菜单管理类似：
- **新增 Banner**：上传 Banner 图片，配置跳转类型和地址
- **跳转类型**：不跳转 / 应用内页面 / 外部 H5 链接 / 微信小程序
- **排序**：使用上移/下移按钮
- **显示控制**：Switch 开关控制上下线

### 二、用户端字体切换

#### H5 端

**入口一：首页快捷按钮**
1. 打开 H5 首页，点击右上角"Aa"图标
2. 在弹出的底部浮层中，拖动滑块在"标准""大""超大"三个档位之间切换
3. 下方预览文字会实时展示字体效果
4. 关闭浮层，字体设置立即生效

**入口二：个人中心**
1. 进入个人中心页面
2. 找到"字体大小"菜单项，点击进入
3. 操作方式同上

**入口三：设置页**
1. 进入设置页面
2. 找到"字体大小"选项，点击进入

#### 微信小程序

**入口一：首页"Aa"按钮**
1. 打开小程序首页，点击顶部"Aa"图标
2. 在弹窗中拖动滑块调整字体大小
3. 点击"确认"保存设置

**入口二：个人中心**
1. 进入"我的"页面
2. 找到"字体大小"菜单项

#### Flutter App

**入口一：首页 AppBar**
1. 打开 App 首页，点击顶部"Aa"图标
2. 在底部弹出的面板中调整字体大小

**入口二：设置页**
1. 进入"设置"页面
2. 找到"字体大小"选项

## 注意事项

1. **权限要求**：首页配置管理功能仅管理员和运营人员可操作，普通用户无权访问管理后台
2. **生效时间**：后台修改配置后，用户下次打开首页时自动刷新生效（前端缓存 5 分钟）
3. **字体偏好独立**：每个设备/端独立存储字体偏好，不同设备之间不同步
4. **字体功能关闭**：如果后台关闭字体切换功能，已调整字体的用户保持当前字号不变，但入口会被隐藏
5. **菜单数量建议**：建议控制在 15 个以内，避免首页过长
6. **Banner 数量建议**：建议控制在 5 个以内，确保轮播体验流畅
7. **图片格式要求**：
   - 菜单图标：PNG/JPG/SVG，建议 96×96px，最大 500KB
   - Banner 图片：建议比例 16:9 或 3:1，最大 2MB
