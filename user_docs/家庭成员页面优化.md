## 功能简介

本次更新对「贝尼健康」系统中的家庭成员相关页面进行了六项体验优化：

1. **邀请区域移植** — 健康档案页 Hero 卡下方新增「邀请家人共管」入口
2. **邀请区域智能隐藏** — 查看已守护成员时自动隐藏邀请区域
3. **角标文案更新** — "被守护"统一改为"守护中"，更符合守护者视角
4. **成员信息补充** — 家庭成员卡片新增性别和年龄展示
5. **编辑功能修正** — 编辑按钮现在正确打开个人信息编辑面板
6. **提醒设置重设计** — 全新仪表盘式天蓝色主调设计

## 使用说明

### 一、健康档案页 — 邀请家人共管

1. 登录系统后进入「健康档案」页面
2. 在 Hero 个人信息卡片的正下方，可以看到一条浅蓝色背景的「💙 邀请家人共管」提示条
3. 点击「去邀请 ›」即可进入邀请流程
4. 当您在顶部成员切换条中切换到**已守护状态**的家人时，邀请区域会自动隐藏（因为该成员已在守护中）
5. 切换回本人或未守护成员时，邀请区域自动恢复显示

### 二、"守护中"角标

1. 在健康档案页的顶部成员切换条中，已建立守护关系的成员头像右上角会显示**天蓝色**「守护中」角标
2. 在 Hero 大头像处也会同步显示该角标
3. 角标采用天蓝色背景（`#0EA5E9`）+ 白色文字，清晰醒目

### 三、家庭成员列表页 — 查看性别与年龄

1. 进入「家庭成员」页面
2. 每张成员卡片中，在姓名和关系信息下方，新增了一行**性别 · 年龄**的展示
3. 展示格式示例：`男 · 35岁`
4. 如果您尚未填写性别或生日，对应字段会显示"未填写"

### 四、编辑家庭成员信息

1. 在家庭成员列表页，点击任意成员卡片底部的「编辑」按钮
2. 系统将弹出一个底部编辑面板（而非之前错误的新增弹窗）
3. 编辑面板支持修改以下信息：
   - **姓名**：直接输入修改
   - **性别**：点击胶囊按钮选择（男 / 女 / 其他）
   - **生日**：日期选择器
   - **身高**：数字输入（单位 cm）
   - **体重**：数字输入（单位 kg）
   - **关系**：仅编辑家人时显示
4. 修改完毕后点击「保存」按钮，信息将立即更新

### 五、提醒设置（全新设计）

1. 在家庭成员列表页，点击任意成员卡片底部的「提醒设置」按钮
2. 系统将弹出全新设计的提醒设置面板，采用仪表盘式天蓝色主调：

   **顶部状态卡片**（天蓝色渐变背景）：
   - 展示当前 AI 外呼的开启/关闭状态和手机号

   **AI 外呼提醒**：
   - 使用开关（Switch）控制开启/关闭，天蓝色激活态
   - 呼叫时机使用胶囊标签样式选择：准时拨打 / 延迟5分钟 / 延迟10分钟 / 延迟15分钟
   - 选中的标签为天蓝色填充，未选中为浅灰底

   **超时通知守护者**（仅查看家人时显示）：
   - 橙色图标标识
   - 触发时长同样使用胶囊标签：5分钟 / 10分钟 / 15分钟

   **查看提醒历史**：
   - 紫色图标标识，点击可查看历史提醒记录

3. 设置完成后点击底部天蓝色渐变「保存」按钮即可

## 注意事项

1. **登录要求**：使用以上功能前需先登录系统
2. **权限说明**：
   - 本人可编辑自己的档案信息和设置提醒
   - 守护者可编辑被守护成员的基本信息和设置提醒
3. **邀请区域显示规则**：
   - 切换到「本人」或「未守护成员」时显示邀请区域
   - 切换到「已守护成员」时自动隐藏
4. **性别/年龄显示**：如需补充信息，请通过编辑按钮进入编辑面板填写
5. **兼容性**：适配 iOS / Android 微信内置浏览器及主流移动端浏览器
