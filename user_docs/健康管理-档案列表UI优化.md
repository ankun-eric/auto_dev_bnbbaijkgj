## 功能简介

本次发布**不新增**任何业务功能，专注于优化档案列表相关页面的可读性与操作可发现性，覆盖：

1. **入口卡片文案精简**：用「已管理 X/Y」斜杠格式替换冗长的「已管理 X 人，还可添加 Y 人」。
2. **去除重复提示**：删除档案列表内页顶部「本套餐可管理 X 人」横条。
3. **去除截断/冗余文案**：会员级别行不再展示「可管理 N 位家人」（与"已管理 X/Y"重复，且过长被截断）。
4. **统一视觉层次**：页面背景改为浅灰 `#F5F7FA`，会员卡片与本人卡片之间增加 16px 间距。
5. **操作按钮平铺**：去掉「⋯」折叠菜单，所有可用操作（删除/解除守护/取消邀请/邀请记录等）**全部平铺为蓝色胶囊按钮**，自动换行。
6. **去重复"本人"**：本人卡片只保留姓名旁的灰色「本人」标签，不在头像下方再次出现。

## 使用说明

### 一、查看已管理档案数

1. 打开「健康档案首页」（菜单：**菜单 → 健康档案**）。
2. 第一行「档案列表」入口卡片，副标题显示 **`已管理 0/9`**：
   - 左侧 0 = 当前已建档的家庭成员数量（含本人）
   - 右侧 9 = 当前套餐的家庭守护成员上限
3. 套餐为「不限」档时，副标题简化为 **`已管理 X`**。

### 二、进入档案列表内页

1. 点击「档案列表」入口卡片，进入档案列表内页。
2. 页面顶部蓝色「邀请家人」卡片（含套餐名 + 已管理 X / 上限 Y 实时数字）。
3. 蓝色卡片下方间隔 16px 后，依次显示每位家庭成员的卡片，**本人卡片**始终置顶。

### 三、识别成员状态

每张成员卡片右上角的灰/蓝/绿/橙/红色小标签为状态标识：

| 标签 | 含义 |
|------|------|
| 本人（灰底） | 当前账户本人档案 |
| 已绑定（蓝底） | 已接受邀请并完成绑定 |
| 待邀请（绿底） | 已建档但尚未发起邀请 |
| 邀请中（橙底） | 已发出邀请，等待对方接受 |
| 已拒绝/已过期/已取消（红/灰底） | 邀请终态 |

### 四、对成员执行操作（按钮平铺，所见即所点）

每张成员卡片的右下角操作区会**直接平铺**所有当前可用的操作按钮：

- **主操作**（蓝色实心胶囊 `#1677FF`）：
  - 「查看档案」：查看该成员档案详情
  - 「去邀请」/「重新邀请」：发起或重新发起邀请
  - 「查看邀请码」：复制 / 分享当前邀请链接
- **次要操作**（蓝色描边胶囊；删除类用红色描边）：
  - 「邀请记录」：查看该成员的全部邀请历史
  - 「解除守护」（已绑定状态）：解除与该家庭成员的守护关系
  - 「取消邀请」（邀请中状态）：撤销正在进行的邀请
  - 「删除」（适用状态）：删除该家庭成员档案
  - 「AI 外呼额度」（本人卡片）：查看本套餐 AI 外呼/紧急呼叫剩余次数

> 💡 **变化说明**：以前次要操作藏在卡片右下角的「⋯」按钮里，需点开抽屉才能看到。**现在所有可用操作都一目了然地平铺在卡片底部**，一行放不下时自动换行。

### 五、本人卡片细节

- 姓名旁会显示一个灰色「本人」小标签，便于快速识别。
- 本人卡片右下角的「AI 外呼额度」按钮可一键查看常规外呼 / 紧急呼叫的余量。
- 本人档案**不可删除**（无「删除」按钮）。

### 六、会员中心同步优化

打开「会员中心」页面顶部的「邀请家人」卡片，主标题不再显示「普通会员 · 可管理 10 位家人」（避免在小屏被截断），改为仅显示套餐名「普通会员」。已管理 X / 上限 Y 仍在卡片副行清晰展示。

## 注意事项

1. 本次升级**无后端接口变更**，所有数据来源不变，刷新页面即可看到新 UI。
2. 部分浏览器存在静态资源缓存，如未看到新样式请按 `Ctrl+F5` / `Cmd+Shift+R` 强制刷新一次。
3. 按钮平铺后，当一个成员的可用操作较多（例如 4 个以上）时，按钮会自动换行，卡片高度会比之前略高，属正常表现。
4. 屏幕窄于 360px 的极小屏机型上，胶囊按钮可能一行只能放下 1~2 个，会换多行展示，仍能正常点击。
5. 「本套餐可管理 X 人」横条已删除，相同信息（已管理 X / 上限 Y）请在顶部蓝色「邀请家人」卡片或健康档案首页入口卡查看。
