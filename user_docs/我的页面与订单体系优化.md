## 功能简介

本次更新对 H5 端「我的」页面和管理后台进行了全面优化，主要包括：

1. **个人信息区域优化**：头像可点击跳转编辑页、昵称旁增加编辑图标、显示会员卡号、会员码入口视觉升级
2. **修改个人信息页面**：新增独立的个人信息编辑页，支持修改头像和昵称
3. **统计数据修复**：优惠券、收藏等统计项空值从「-」改为「0」
4. **订单快捷入口优化**：所有入口显示数量角标，「退款/售后」作为独立入口
5. **订单Tab栏重构**：H5端6个Tab，合并「待收货/待使用」，新增「已完成」「已取消」
6. **订单状态流转调整**：确认收货/核销后直接变为「已完成」，待评价从已完成中筛选
7. **管理后台菜单精简**：删除「订单管理」和「服务管理」，统一由「商品体系」承载
8. **订单明细页面增强**：9个Tab、完整查询条件、6张统计卡片
9. **订单统计页面升级**：12张统计卡片、6张图表（含退款趋势和退款原因分布）

## 使用说明

### 一、H5 端「我的」页面

#### 1. 查看和编辑个人信息

- 打开 H5 端，点击底部「我的」Tab 进入个人中心
- **点击头像**或**点击昵称旁的编辑图标**，跳转到「修改个人信息」页面
- 在编辑页面中：
  - 点击头像区域可上传新头像
  - 修改昵称输入框中的内容
  - 手机号仅展示，不可修改
- 修改完成后点击「保存」按钮

#### 2. 查看会员卡号

- 在「我的」页面，昵称下方直接显示您的会员号（用户编号）

#### 3. 查看会员码

- 在「我的」页面右上角，点击「二维码图标 + 会员码」按钮
- 进入会员卡页面，展示动态二维码

#### 4. 查看统计数据

- 积分、优惠券、收藏三个统计项位于头像区域下方
- 无数据时统一显示「0」
- 点击各项可跳转到对应详情页面

#### 5. 订单快捷入口

- 「我的订单」区域显示 5 个快捷入口图标：
  - **待付款** — 跳转订单列表待付款Tab
  - **待收货** — 跳转订单列表待收货/待使用Tab
  - **待使用** — 跳转订单列表待收货/待使用Tab
  - **待评价** — 跳转订单列表待评价Tab
  - **退款/售后** — 跳转独立的退款/售后列表页
- 每个图标右上角显示红色数字角标，表示对应状态的订单数量

### 二、H5 端订单列表

#### 1. Tab 栏说明

订单列表页共 6 个 Tab：

| Tab 名称 | 展示内容 |
|----------|---------|
| 全部 | 所有订单 |
| 待付款 | 尚未支付的订单 |
| 待收货/待使用 | 已发货待收货 + 到店待使用的订单（合并展示） |
| 已完成 | 已确认收货或已核销的订单 |
| 待评价 | 已完成但尚未评价的订单 |
| 已取消 | 用户取消、超时关闭、退款成功的订单 |

#### 2. 订单状态标签

- 普通取消订单显示「已取消」
- 退款成功的订单显示「已取消（已退款）」

#### 3. 订单操作

- **待付款订单**：可「取消订单」或「去支付」
- **待收货订单**：可「确认收货」→ 确认后直接变为「已完成」
- **已完成订单**（未评价）：可「去评价」

### 三、设置页面

- 设置菜单中已移除「个人信息」入口
- 个人信息编辑统一通过头像/昵称旁的编辑按钮进入

### 四、管理后台操作

#### 1. 菜单导航

- 侧边栏已移除「订单管理」和「服务管理」顶层菜单
- 所有订单和商品管理功能统一在「商品体系」菜单下
- 菜单结构：商品分类 / 商品管理 / **订单明细** / **订单统计** / 优惠券管理 / 核销管理 / 进店记录

#### 2. 订单明细（原「统一订单」）

- 页面顶部展示 6 张统计卡片：
  - 第1行：今日订单数、今日成交金额、今日退款金额
  - 第2行：本月订单数、本月成交金额、本月退款金额
- 9 个 Tab 可切换：全部 / 待付款 / 待发货 / 待收货 / 待使用 / 已完成 / 待评价 / 已取消 / 退款申请
- 完整查询条件：搜索框（订单号/昵称/手机号）、日期范围、订单状态、支付方式、退款状态、商品分类、金额范围

#### 3. 订单统计（原「数据统计」）

- 12 张统计卡片（三行四列）：
  - 第1行：今日订单数、今日成交金额、今日退款单数、今日退款金额
  - 第2行：本月订单数、本月成交金额、本月退款单数、本月退款金额
  - 第3行：累计订单数、累计成交金额、累计退款单数、累计退款金额
- 6 张图表：
  - 订单量趋势（折线图）
  - 成交金额趋势（折线图）
  - 退款金额趋势（折线图）
  - 商品分类占比（饼图）
  - 订单状态分布（柱状图）
  - 退款原因分布（饼图）

## 注意事项

1. **个人信息编辑**需要先登录才能使用，未登录用户点击头像会跳转登录页
2. **订单角标数量**会实时刷新，进入「我的」页面时自动获取最新数据
3. **已取消（已退款）**标签仅在退款成功的订单上显示，普通取消订单仍显示「已取消」
4. **待评价**筛选逻辑：从已完成的订单中自动筛选出未评价的订单
5. **管理后台**的「订单明细」保留全部 9 个状态Tab，比 H5 端更完整
6. **数据迁移**：原「服务管理」中的 5 个服务分类（健康食品、口腔服务、体检服务、专家咨询、养老服务）已自动迁移到商品分类中
