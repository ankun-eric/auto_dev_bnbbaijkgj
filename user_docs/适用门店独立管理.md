## 功能简介

本次新增了「**适用门店独立管理**」功能，将原本嵌套在商品编辑弹窗中的"适用门店"功能完全独立出来，作为「商品体系」下的独立子菜单。主要变化包括：

1. **新增独立菜单入口**：在管理后台侧边栏「商品体系」下新增「适用门店」菜单
2. **商品维度管理**：从商品角度查看和管理每个商品绑定了哪些门店
3. **门店维度管理**：从门店角度查看和管理每个门店关联了哪些商品
4. **批量操作**：支持批量绑定/解绑门店，大幅提升管理效率
5. **即时保存**：勾选/取消勾选后立即生效，无需额外点保存按钮
6. **商品编辑弹窗改造**：原有的门店多选组件已移除，改为只读显示已绑定门店数量 + "去管理"跳转链接

## 使用说明

### 一、进入「适用门店」管理页面

1. 使用管理员账号登录管理后台
2. 在左侧侧边栏找到「**商品体系**」菜单组
3. 展开后点击「**适用门店**」菜单项（位于"商品管理"之后、"预约表单库"之前）
4. 进入适用门店管理页面，默认显示「商品维度」Tab

### 二、商品维度管理（从商品角度管理门店绑定）

#### 查看商品列表

- 页面显示所有商品的列表，包含：商品图片、名称、分类、价格、状态、已绑定门店数
- 顶部搜索框支持按商品名称模糊搜索
- 已下架商品会以灰色展示，但仍可正常编辑绑定关系

#### 管理某个商品的门店绑定

1. 点击某个商品行中的「**已绑定门店数**」（蓝色数字链接）或「**管理门店**」按钮
2. 右侧弹出门店勾选抽屉，显示所有门店列表
3. **勾选门店**：在门店前的复选框打勾 → 立即绑定该门店（自动保存，无需额外操作）
4. **取消勾选**：取消某个门店的复选框 → 立即解绑该门店
5. 操作成功后会显示「操作成功」提示

#### 门店勾选抽屉内的筛选功能

- **搜索**：输入门店名称进行搜索
- **绑定状态筛选**：选择「全部」/「已绑定」/「未绑定」快速过滤

#### 批量操作

1. 在商品列表中通过前方复选框**勾选多个商品**
2. 页面底部会浮出操作栏，包含两个按钮：
   - **批量绑定门店**：将选中的多个商品统一绑定到某一个门店
   - **批量解绑门店**：将选中的多个商品统一从某一个门店解绑
3. 点击按钮后，在弹窗中选择一个目标门店
4. 确认操作信息后，点击「确认」执行批量操作
5. 完成后显示操作结果提示

### 三、门店维度管理（从门店角度管理商品关联）

#### 查看门店列表

- 切换到「**门店维度**」Tab
- 显示所有门店列表，包含：门店名称、编号、营业状态、已关联商品数
- 已关闭的门店以灰色展示，但仍可编辑关联关系

#### 管理某个门店的商品关联

1. 点击某个门店行中的「**已关联商品数**」或「**管理商品**」按钮
2. 右侧弹出商品勾选抽屉，显示所有商品
3. 勾选/取消勾选商品即可立即关联/取消关联（即时保存）

#### 商品勾选抽屉内的筛选功能

- **搜索**：输入商品名称进行搜索
- **分类筛选**：选择商品分类进行过滤
- **关联状态筛选**：选择「全部」/「已关联」/「未关联」

### 四、商品编辑弹窗中的变化

- 原来的"适用门店"多选组件和智能推荐功能已**移除**
- 编辑商品时，在弹窗中会看到：`已绑定 X 个门店` + `去管理 →` 链接
- 点击「去管理 →」会跳转到「适用门店」独立管理页面

### 五、新建商品后的引导

- 成功创建一个新商品后，会弹出提示：「商品创建成功！该商品尚未绑定任何门店，请前往「适用门店」完成门店绑定。」
- 点击「立即前往」：跳转到适用门店管理页面
- 点击「稍后处理」：关闭提示，留在当前页面

## 注意事项

1. **权限要求**：使用此功能需要拥有「商品管理」操作权限的管理员角色
2. **即时保存**：单个勾选/取消操作会立即生效，请谨慎操作
3. **批量操作需确认**：批量绑定/解绑操作会弹出确认框，确认后才执行
4. **下架商品可编辑绑定**：已下架的商品仍可正常编辑门店绑定关系
5. **已关闭门店可编辑绑定**：已关闭的门店仍可正常编辑绑定关系，方便门店重新开业后直接生效
6. **数据一致性**：门店绑定操作独立于商品编辑，不会出现"商品保存成功但门店绑定失败"的问题
7. **浏览器兼容**：支持 Chrome 80+、Firefox 78+、Safari 13+、Edge 80+
