## 使用说明

### 1. 查看成员 Tab 胶囊

打开健康档案首页，顶部横向滑动的成员胶囊栏现在只显示每位成员的**姓名或昵称**，不再显示"爸爸""妈妈"等关系名称。

### 2. 编辑成员关系

1. 点击页面上方的成员头像区域，进入编辑弹层
2. 非本人档案会显示**关系选择器**，包含完整的关系选项：爸爸、妈妈、老公、老婆、儿子、女儿、哥哥、姐姐、弟弟、妹妹、爷爷、奶奶、外公、外婆、其他
3. 选择关系后点击保存，关系信息会正确保存到后端

### 3. 查看就医资料

1. 在健康档案首页下方找到「就医资料」区域
2. 点击右上角的 **「全部 ›」** 可进入就医资料全部页面
3. 全部页面按 Tab 分类显示：病例单、体检报告、药物、其他
4. 数据会按当前选中的成员进行过滤

### 4. 就医资料定位跳转

1. 在健康档案首页展开就医资料分类卡片
2. 点击某个分类后，在弹出抽屉中点击**具体的某条资料**
3. 系统会跳转到全部页面对应 Tab，并自动滚动到该条资料并高亮显示
