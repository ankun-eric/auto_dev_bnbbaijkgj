### 2.3 操作步骤

1. **左侧大类列表**：点击切换大类（如「理疗保健」「居家服务」）。当前大类有 **绿色文字加粗 + 左侧 3px 主色竖条** 高亮提示。
2. **顶部子类横滑**：选中大类后，右上角出现该大类的所有子类按钮。点击「全部」可查看大类下所有商品；点击具体子类（如「推拿」）则按子类筛选。
3. **顶部搜索框**：
   - 永久常驻在页面顶部
   - 输入关键词后 **300ms 自动触发搜索**（无需点放大镜）
   - 搜索范围为 **全部分类**（不局限于当前大类）
   - 关键词同时匹配商品 **名称** 与 **症状标签**
4. **商品卡片**：每张卡片右上角根据履约类型显示彩色角标
   - 🟧 **暖橙「到店」**（`#FF8A3D`）— 需到店服务
   - 🟦 **科技蓝「快递」**（`#3B82F6`）— 实物快递配送
   - 🟪 **尊贵紫「虚拟」**（`#8B5CF6`）— 虚拟商品 / 电子券码
5. **下拉刷新 + 上拉加载更多**：商品列表支持无限滚动加载。

### 2.4 搜索无结果时的「热门推荐」

当搜索关键词没有匹配任何商品时，页面会自动显示「🔥 热门推荐」专区，展示 **近 30 天销量 Top 6** 的商品，方便用户继续浏览。

### 2.5 注意事项

- 搜索为实时模糊匹配，至少输入 1 个字即可触发；输入太短可能匹配过多结果。
- 履约角标的颜色与文案不可自定义，请商户在管理后台准确设置 `履约类型`。
- 若某大类暂无子类，子类横滑栏会自动隐藏，仅显示「全部」选项。

## 三、Bug ① 中医养生·体质检测 提交修复

### 入口

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/tcm](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/tcm)

### 修复内容

- **现象**：之前「自己」「家人」两种咨询人提交后必现「提交测评失败」。
- **现状**：现已修复。提交后将正确返回辨证分析与调理建议。

### 操作步骤

1. 打开「中医养生」入口，点击「体质检测」。
2. 选择咨询人（自己 或 家人）。
3. 完成 8 道体质问卷题目（每题选择「经常 / 偶尔 / 从不」等选项）。
4. 点击「提交测评」。
5. 等待 5~30 秒，系统将返回完整的：
   - 体质类型（如「气虚质」「平和质」「痰湿质」等）
   - 辨证分析
   - 调理建议（饮食、作息、运动等）

### 注意事项

- 答题不能为空，否则前端会提示「答题内容不能为空，请先完成体质问卷」。
- 当 AI 服务暂时不可用时，系统会兜底返回「平和质」并附上提示，**不再出现 500 错误**。
- 选择「家人」时，需先在「家庭管理」中添加家庭成员。

## 五、Bug ③ 优惠券·兑换码批次导出 CSV

### 入口

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/product-system/coupons](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/product-system/coupons)

### 修复内容

之前点「导出」会跳转到「Gateway OK」错误页面。现已修复：
- 前端改用带 token 的 `fetch + Blob` 下载，浏览器直接弹出文件保存框
- 后端响应 `Content-Type: text/csv; charset=utf-8` + `Content-Disposition: attachment` + UTF-8 BOM
- Excel 双击打开无中文乱码

### 操作步骤

#### 5.1 导出兑换码批次（CSV）

1. 进入「优惠券管理」。
2. 找到「兑换码批次」区域。
3. 在某个批次行点击「导出 CSV」按钮。
4. 浏览器自动下载 `batch_<id>_codes.csv`。

#### 5.2 导出领取记录（CSV）

1. 在某个优惠券行点击「领取记录」→ 弹窗中点「导出 CSV」。
2. 浏览器自动下载 `coupon_grants_<id>.csv`。

### 注意事项

- 若导出失败（如权限不足或批次已删除），页面会弹出 Toast 显示真实错误原因（如 `导出失败：HTTP 403`），不再无差别跳转到 Gateway OK 页面。
- 文件以 UTF-8 BOM 开头，Excel 直接双击即可显示中文，无需额外操作。
- 单批次数据量极大（> 10 万行）时下载可能耗时 30~60 秒，请耐心等待。

## 八、附：常见问题

**Q1：上传 Logo 后页面还是不显示？**
A：请按 Ctrl+F5 强制刷新浏览器，清除本地缓存（CDN 强缓存 30 天）。

**Q2：服务列表搜索没结果怎么办？**
A：页面会自动显示「🔥 热门推荐」6 个商品，可继续浏览。

**Q3：CSV 在 Excel 打开仍显示乱码？**
A：本期已加 UTF-8 BOM，正常情况下 Excel 双击即可。如仍乱码，请用 WPS 或 Notepad++ 以「UTF-8」编码打开。

**Q4：体质测评提交后等待时间过长？**
A：AI 分析通常 5~15 秒返回，最长可能 30 秒。若 60 秒仍无响应，请刷新页面重试。

**Q5：本期改造对小程序/安卓 App 是否有影响？**
A：本期未修改小程序与 App 客户端代码，仅后端 + Web 端（管理后台 + H5）有变更，无需重新打包发版。
