- 「兑换记录 ›」：跳转兑换记录页，查看历史兑换的券/服务/实物。

### 2.2 积分商城统计区

- 顶部浅绿卡片（`#C8E6C9`）：显示「我的积分」「用积分兑换好礼」。

- 开发中商品整卡置灰 + 遮罩，不可兑换。

### 2.3 兑换流程

1. 在商城点击商品 → 进入确认弹窗。
2. 体验服务商品会提示「服务券自兑换日起 30 天内有效，过期后积分不退还、预约失败也不退积分」。
3. 确认后原子执行：扣积分 → 扣库存 → 写 `point_exchange_records` → 写 `points_records`（积分扣减流水）→ 若为优惠券/体验服务，额外写入 `user_coupons`（服务券 `coupon_type=service_voucher`）。
4. 成功后自动跳转兑换记录页。

### 2.4 兑换记录页

- 按时间倒序列出所有兑换，支持下拉刷新、上拉加载更多。
- 每条记录包括：商品图/图标、类型徽标、状态徽标、名称、兑换时间、到期时间、消耗积分。
- 右下角按钮按类型联动：
  - **优惠券** → `查看我的券`（跳 `/coupons`）
  - **体验服务** → `去预约`（按 `ref_service_type` 跳到对应详情页，如专家问诊、体检、TCM、健康计划）
  - **实物商品** → `查看订单`（跳 `/unified-orders`）

## 五、管理后台

Admin 基础 URL：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin`

### 5.1 积分商城（/admin/points/mall）

- 「商品分类」选项：
  - 优惠券
  - 体验服务
  - 实物商品
  - 虚拟商品（开发中）← 置灰不可选
  - 第三方商品（开发中）← 置灰不可选
- 选择 **优惠券** 时：
  - 弹出「关联优惠券」下拉（自动加载上架未兑完的券），下方库存字段禁用，提示「跟随券本体」。
- 选择 **体验服务** 时：
  - 弹出「服务类型」下拉（专家问诊 / 体检 / TCM / 健康计划体验）+「关联服务ID」输入框。
  - 库存字段禁用，提示「由服务本体管控」。
- 选择 **实物商品** 时：需手动填写库存。
- 保存时，`ref_coupon_id=...;ref_service_type=...;ref_service_id=...` 会被写入 `description`，供兑换接口解析（无需新增字段，兼容旧数据）。

### 5.2 兑换记录管理（/admin/points/exchange-records）

- 列表字段：兑换单号、用户 ID、商品类型、商品名、消耗积分、数量、状态、兑换时间、到期时间、关联订单号。
- 支持按关键字（商品名）、类型（coupon/service/physical）、状态（success/used/expired/cancelled）、起止时间筛选。
- 分页，默认 20 条/页，最大 500 条/页，便于导出。

## 七、常见问题

**Q1：兑换了"体验服务"后去哪里预约？**
兑换记录页对应条目点击「去预约」，系统会根据服务类型跳转：专家 → `/experts/{id}`；体检 → `/physical-exam/{id}`；TCM → `/tcm/{id}`；健康计划 → `/health-plans/{id}`。

**Q2：服务券过期了积分会退吗？**
不退。30 天有效期内请尽快预约；预约失败（医生排班冲突等）也不退积分，由客服按券面规则处理。

**Q3：实物商品的兑换记录在哪里？**
既会出现在「兑换记录」列表（状态展示），也会在「我的订单」中生成一条 `order_type=points_exchange / payment_method=points` 的订单，可在订单中心查看物流。

**Q4：为什么虚拟商品、第三方商品选不了？**
当前版本仅开放优惠券 / 体验服务 / 实物三类，虚拟商品与第三方平台商品在后续版本接入中，后台已标注「开发中」。
