# 新增家庭成员优化（最终版）— 用户体验使用手册

> 适用端：H5、小程序、App（三端体验一致）
> 入口：AI 首页 → 左下角"咨询人" → 添加成员

---

## 一、功能介绍

本次更新把"添加家庭成员"这条线做了一次彻底的清理与升级，让整个流程更顺畅、提示更清晰：

1. **入口更干净**：添加成员页里那块独立的"去邀请"蓝色横幅已被移除，页面只保留"填写信息 + 保存"一条主线。
2. **名额先判断，再开表单**：点"新增咨询人 / 添加成员"按钮时，系统会先实时查询您当前账号可添加的家庭成员名额。
   - 名额未满 → 正常打开添加表单。
   - 名额已满 → 弹出"家庭成员名额已满"提示，并提供升级入口。
3. **保存后引导邀请**：成员保存成功后，会弹出"成员已添加成功 🎉"提示，您可以选择"暂不邀请"或直接"去邀请 TA"，跳转到二维码邀请页。
4. **二维码邀请页升级**：标题、说明、按钮配色全部按统一设计调整，更亲切、更醒目。
   - 标题：**邀请 TA 加入我的健康守护**
   - 说明：**让 TA 扫码或打开链接，确认后就能加入啦**
   - 有效期：**邀请 24 小时内有效**
   - 三个能力标签：📋 档案管理 · 💊 用药提醒 · 🔔 异常提醒
   - 按钮配色：保存到本地 / 复制链接 = 主色蓝渐变；转发微信好友 = 微信绿。

> 名额上限完全由后端会员配置决定（免费用户读 `free_member_quota.max_managed`，付费用户读当前套餐 `membership_plans.max_managed`），运营在后台调整后立刻生效，前端不会写死任何数字。

## 二、操作步骤

### 步骤 1：进入 AI 首页

在浏览器中打开下方链接（H5）：

- **首页**：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)
- **AI 助手页**：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai)

小程序与 App 端打开后操作一致，从底部"AI"或聊天入口进入。

### 步骤 2：点击"新增咨询人 / 添加成员"

- 系统会自动调用配额接口 `GET /api/family/member/quota` 查询当前可添加名额。
- 名额未满 → 直接进入添加成员表单。
- 名额已满 → 看到"家庭成员名额已满"弹框，文案示例："当前最多可添加 X 位家庭成员，升级会员可解锁更多名额。"

### 步骤 3a：名额已满 → 升级或稍后

- 点击 **"暂不升级"** 关闭弹框，本次操作结束。
- 点击 **"去升级"** 跳转到会员中心，可选择更高档套餐获得更大名额。

### 步骤 3b：名额未满 → 填写并保存

- 在表单中填写关系（如：父亲 / 母亲 / 配偶）、姓名、出生日期等基本信息。
- 点击"保存"按钮提交。
- 保存成功后，弹出 **"成员已添加成功 🎉"** 提示，文案："要现在邀请 TA 来一起管理健康吗？发个二维码给 TA，扫一扫就能加入。"

### 步骤 4：选择是否立即邀请

- 点击 **"暂不邀请"** → 关闭弹框，可稍后再操作。
- 点击 **"去邀请 TA"** → 自动跳转到二维码邀请页。
  - 邀请页直链：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family-invite](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family-invite)

### 步骤 5：在邀请页完成邀请

- 页面顶部显示 **"邀请 TA 加入我的健康守护"** 与三个 emoji 能力标签。
- 二维码下方有效期提示："邀请 24 小时内有效"。
- 操作按钮：
  - **保存到本地**（实心蓝）：将二维码图片保存到设备相册。
  - **复制链接**（实心蓝）：复制邀请链接，直接粘贴发送。
  - **转发微信好友**（微信绿）：H5 内将引导您使用浏览器右上角分享，小程序/App 内为原生分享。

被邀请者扫码或点开链接 → 在 `/family-auth` 页确认即可加入您的家庭健康守护圈。

## 三、注意事项

1. **名额数字以接口返回为准**：弹框中的"X 位"完全实时取自后端，运营调整套餐后无需更新客户端，立即生效。
2. **邀请有效期 24 小时**：超过 24 小时邀请将失效，需要在原成员条目上重新发起。
3. **同一手机号只能加入一个家庭**：被邀请方如果已在其它家庭，会在确认页提示。
4. **小程序与 App**：与 H5 流程一致，所有名额判断与文案均通过统一接口下发，三端体验保持一致。
5. **付费用户升级后立即可用**：从"名额已满"弹框点击"去升级"完成支付后，回到添加成员页可直接添加，无需重启 App。
6. **成员资料可后续完善**：保存时只需填必填项（姓名 + 关系），其它信息（生日、性别等）可后续在成员档案中补充。
