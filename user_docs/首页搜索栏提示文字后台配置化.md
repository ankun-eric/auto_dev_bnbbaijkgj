## 功能简介

本次更新实现了「首页搜索栏提示文字后台配置化」功能，使运营人员可以通过管理后台灵活修改各客户端（H5 Web、Flutter App、微信小程序）首页搜索栏的 placeholder 提示文字，无需发版即可更新。

主要改动：
- **H5 Web 端**：搜索栏 placeholder 从硬编码的"搜索文章、视频、服务、商品"改为从后台接口动态获取，缓存时长从 5 分钟提升至 1 小时
- **Flutter App 端**：默认兜底值统一更新为"搜索健康知识、服务、商品"
- **微信小程序端**：默认兜底值统一更新为"搜索健康知识、服务、商品"
- **后端**：默认值统一更新为"搜索健康知识、服务、商品"
- **统一兜底策略**：当后台未配置或接口异常时，所有端统一显示"搜索健康知识、服务、商品"

## 使用说明

### 运营人员：修改搜索栏提示文字

1. 登录管理后台：[https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 进入「首页基础设置」页面
3. 找到「搜索框配置」模块
4. 在「提示文字」输入框中输入新的提示文案（最多 50 个字符）
5. 点击「保存」按钮，系统提示"搜索框配置保存成功"
6. 各客户端将在缓存过期后（最多 1 小时）自动更新为新的提示文字

### 用户端：体验搜索栏提示文字

1. 打开首页（H5 Web / Flutter App / 微信小程序任一端）
2. 观察首页搜索栏中灰色的提示文字
3. 该文字即为运营人员在后台配置的内容
4. 若运营人员未配置或清空了提示文字，将显示默认值"搜索健康知识、服务、商品"

## 注意事项

1. **缓存延迟**：运营人员在后台修改提示文字后，各客户端最多需要 1 小时才能看到更新（因为缓存时长为 1 小时）。如需立即查看效果，可清除浏览器/App 缓存后重新打开首页。
2. **字符限制**：提示文字最多支持 50 个字符，建议简洁明了，涵盖主要搜索范围。
3. **兜底策略**：当后台未配置提示文字、或提示文字被清空、或网络异常时，所有端将统一显示默认值"搜索健康知识、服务、商品"。
4. **多端一致性**：所有客户端（H5、Flutter App、微信小程序）共用同一后台配置，运营人员设置的文案需兼顾各端的展示效果。
5. **搜索框显示开关**：如果在后台将「搜索框显示」设置为关闭，则首页不会显示搜索栏，提示文字配置也不会生效。
