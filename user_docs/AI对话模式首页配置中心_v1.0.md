## 功能简介

本次 PRD-405 v1.0 在原 v0.5 基础上把 AI 对话模式首页（H5 端 `ai-home`）所有视觉与交互元素**全面**下放给运营后台自助配置：

### 一、admin 后台「AI 对话模式首页配置」6 Tab 容器

按 PRD 设计将原来零散的 11 张配置卡片重组为 **6 个 Tab 锚点导航**：

| Tab | 名称 | 内含模块 |
|-----|------|---------|
| 1 | 欢迎区 | 主标题（含 `{昵称}` 占位）+ 副标题 + 头像图（emoji/图片二选一） |
| 2 | 首屏内容 | 今日健康贴士轮播（间隔 3~5 秒）+ 推荐问列表（双字段：显示文案 ≤ 8 字 + 实际发送内容 ≤ 200 字）+ 空对话占位 |
| 3 | 功能宫格 | 1~6 项卡片，每项 **7 字段**：主文案/副说明/跳转/图标/渐变色起始+结束/角标/启用 |
| 4 | 输入栏 | 占位文字 + 语音输入开关 + **家庭成员咨询胶囊**（必含 `{name}` 占位符）+ 查看档案跳转 |
| 5 | 会话策略 | **7 个全局字段**：单次回答字数（100~5000）/ 加载动画 / 单日免费次数（1~999）/ 回答风格 / 敏感词过滤 / 上下文记忆轮数（3/5/10/20）/ 免责声明（≤ 100 字） |
| 6 | 全局开关 | **9 个模块级开关**（与各 Tab 内启用字段联动取并集）+ 打卡悬浮按钮 4 字段 |

### 二、H5 端「晓医风格」首页 1:1 还原

按设计图重构 `ai-home` 页面：

1. **顶栏**：默认隐藏（设计图无顶栏，仅保留右上角 ☰ 历史记录入口）
2. **欢迎区**：左头像 + 右文字 **横向布局**（替换原纵向布局）
3. **今日健康贴士**：紫色渐变大卡轮播图（图片做整张卡片背景，无叠加文字）
4. **功能宫格**：3 项含**主文案 + 副说明 + 渐变色 + 角标**（替换原通用 funcButtons）
5. **推荐问**：底部**横向滚动胶囊**（替换原纵向列表）
6. **空对话占位**：大图标 + 主标题 居中
7. **输入栏（双层）**：
   - 第一层：🎤/⌨️ 切换图标 + 输入框 + 智能隐藏发送按钮（丁香派策略：空时灰显、有文字时亮起、语音模式时整体隐藏）
   - 第二层：紫色「为(XX)咨询 ⇆」胶囊 + 「查看档案 ›」文字链接
8. **打卡悬浮按钮**：右下，胶囊形带文字"健康打卡 ✅"（保留未完成任务红点逻辑）

### 三、运营自助配置能力

- **零研发介入**：所有文案、图标、跳转、显隐、颜色、模板均可后台自助调整，**保存即生效**，所有 H5 用户立刻看到最新效果
- **保存机制**：每个 Tab 内每张卡片独立「保存本节」按钮 + 底部全局「全部保存」
- **校验机制**：必填 + 格式（URL/HEX 颜色）+ 长度（每个字段独立上限）全前端拦截，红框提示
- **操作日志**：每次保存均写入操作日志（90 天保留），支持按时间/操作人/模块筛选 + diff 抽屉

## 操作步骤

### 1. 进入 admin 配置中心

1. 用**超级管理员**账号登录 [admin 后台](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 在左侧菜单选择「首页配置 → AI 对话模式首页配置」
3. 进入主页面后顶部出现 **6 Tab 锚点导航**（粘性置顶），点击任意 Tab 自动滚动到对应卡片

### 2. 配置欢迎区（Tab 1）

- **主标题**：输入新文案，支持 `{昵称}` 占位符自动填充用户昵称（≤ 30 字）
- **副标题**：输入新文案（≤ 50 字）
- **头像**：选择 Emoji 或上传图片（≤ 1MB，PNG/JPG/WebP，建议 1:1 比例）
- 完成后点击右上角「保存本节」

### 3. 配置首屏内容（Tab 2）

- **今日健康贴士**：复用「轮播图」模块的图片（在「H5 → 首页 Banner」单独管理图片），本卡片只控制是否显示、轮播间隔（3~5 秒）、是否显示底部小圆点
- **推荐问列表**：每条 4 字段
  - 显示文案（按钮上展示，≤ 8 字，如"体检解读"）
  - 实际发送内容（点击后发给 AI 的完整提问，≤ 200 字，可与显示文案不同）
  - 图标（Emoji，可选）
  - 是否启用（开/关）
  - 拖拽排序
- **空对话占位**：占位图标（默认 💬）+ 主标题（≤ 20 字，默认"还没有对话记录"）

### 4. 配置功能宫格（Tab 3）

- 默认 3 项：AI诊室 / 看报告 / 健康档案
- 每项 7 字段（主文案 ≤ 8 字 / 副说明 ≤ 12 字 / 跳转链接（必须 `/` 开头或 `http(s)://`）/ 图标 / 渐变起始色 + 结束色（合法 HEX）/ 角标 ≤ 4 字 / 启用开关）
- 项目数量限制：**1~6 项**
- 卡片右下方实时显示渐变背景预览
- 上下箭头按钮调整顺序

### 5. 配置输入栏 + 家庭成员咨询胶囊（Tab 4）

- **输入框占位文字**：≤ 30 字，如"发消息或按住说话..."
- **是否启用语音输入**：控制左侧麦克风/键盘切换图标
- **是否显示家庭成员咨询胶囊**：开/关
- **胶囊默认文案模板**：必须包含 `{name}` 占位符（如"为({name})咨询"），≤ 20 字
- **是否显示「查看档案」按钮**：开/关
- **查看档案跳转链接**：必须 `/` 开头

### 6. 配置会话策略（Tab 5）

7 个全局字段：

- 单次回答最大字数（100~5000）
- 单日免费提问次数上限（1~999）
- 上下文记忆轮数（3/5/10/20 单选）
- 默认回答风格（专业医学术语 / 通俗易懂 / 温馨亲切）
- AI 思考中加载动画开关
- 敏感词过滤开关
- 免责声明文案（≤ 100 字，每次回答末尾自动追加）

### 7. 配置全局开关（Tab 6）

9 个模块级总开关，控制每个区块整体显隐：

| # | 开关 |
|---|------|
| 1 | 欢迎区 - 整块显隐 |
| 2 | 今日健康贴士 - 整块显隐 |
| 3 | 功能宫格 - 整块显隐 |
| 4 | 推荐问 - 整块显隐 |
| 5 | 空对话占位 - 整块显隐 |
| 6 | 输入栏 - 家庭成员咨询胶囊 |
| 7 | 输入栏 - 查看档案按钮 |
| 8 | 输入栏 - 语音输入图标 |
| 9 | 打卡悬浮按钮 - 整块显隐 |

> **注意**：与对应 Tab 内字段「是否启用」联动取**并集**——只要其中一处关闭，模块即不显示。

### 8. 查看操作日志

- 点击页面右上角「操作日志」按钮 → 进入日志列表页
- 可按起止日期、操作人、模块筛选
- 点击任意行 → 弹出 diff 抽屉对比保存前后内容
- 90 天前的日志自动清理

### 9. 在 H5 端验证效果

保存后立即在 [H5 AI 对话首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home) 刷新查看（H5 端有 5 分钟本地缓存，必要时清空 localStorage 立刻生效）。

### 10. H5 终端用户：切换咨询对象

1. 进入 [AI 对话首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home)
2. 输入框下方紫色胶囊显示当前咨询对象（默认"为(本人)咨询"）
3. 点击胶囊 → 弹出"切换咨询对象"列表：本人 / 已添加家庭成员 / 「+ 新建家庭成员」
4. 选中后胶囊文案立即变更为「为({新成员名})咨询」，AI 后续回答会带上该成员的画像（年龄/性别/慢病史等）

## 注意事项

1. **权限**：仅**超级管理员**可见到「AI 对话模式首页配置」菜单与编辑权限。普通运营即便手动拼接 URL 进入也会被 403 拦截
2. **配置全局唯一**：整个 AI 对话模式首页只有一份配置，所有 H5 用户共享，不做多租户
3. **保存即生效**：保存后立即同步给所有 C 端用户，**无草稿/发布二次确认机制，无历史版本回滚**
4. **图片上传**：≤ 1MB，仅支持 PNG/JPG/WebP，且需通过文件头魔数二次校验（防文件名伪装）
5. **跳转链接**：除浮动按钮强制 `/` 开头外，功能宫格 / 查看档案 / 推荐问的跳转支持 `/` 开头或 `http(s)://`
6. **H5 端兜底**：当配置项异常缺失或接口失败时，H5 端展示前端硬编码的默认值（避免页面崩溃）
7. **图标双形态**：所有"图标"字段同时支持 Emoji 输入和图片上传，二选一，优先级以最近一次保存的为准
8. **缓存策略**：H5 端 `localStorage.__ai_home_config_cache__` 存 5 分钟，过期后自动重新拉取最新配置
9. **菜单模式独立**：本配置中心**仅作用于 AI 对话模式首页**，与菜单模式首页（菜单版）的「首页菜单管理」**完全独立、互不影响**
10. **家庭成员**：复用菜单版的家庭成员后端 API（创建/编辑/删除/列表），UI 在 ai-home 页面下独立新写

## 与原 v0.5 的差异

本次 v1.0 在保留 v0.5 全部字段（兼容性）基础上新增：

| 模块 | v1.0 新增字段 |
|------|--------------|
| 欢迎区 | `main_title`（含占位符）/ `sub_title` |
| 顶栏 | `visible`（默认 false，匹配晓医风格无顶栏） |
| 输入栏 | `family_consult.{enabled,template,show_archive_link,archive_path}` |
| 会话策略 | `strategy.{max_answer_chars,show_loading,daily_free_quota,answer_style,sensitive_filter,context_memory_rounds,disclaimer}` |
| 功能宫格 | `items[]` 7 字段：`main_text`/`sub_text`/`target_path`/`icon`/`gradient_start`/`gradient_end`/`badge`/`enabled`/`sort` |
| 推荐问 | 显示文案 + 实际发送内容**双字段已强校验**（≤ 8 字 / ≤ 200 字） |
| 健康贴士轮播 | `health_tips.{visible,interval_seconds(3~5),show_indicator}` |
| 空对话占位 | `empty_placeholder.{icon,main_title}` |
| 全局开关 | `global_switches`（9 个模块级开关） |
