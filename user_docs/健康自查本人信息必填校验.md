## 功能简介

本功能对健康自查流程中"为谁咨询"面板的**"本人"**信息编辑区域进行了必填校验增强，确保 AI 健康分析所需的基本用户信息完整。

具体包括：

- **姓名**、**性别**、**出生日期**三个字段标记红色 `*` 必填标识
- 点击"确认并开始分析"时，对三个必填字段进行校验
- 校验不通过时，字段下方显示红色错误提示文案，同时弹出 Toast 汇总提示
- 校验未通过时阻止提交，确保用户填写完整信息后才能进入 AI 分析

**覆盖范围**：H5 网页端、微信小程序端、Flutter App 端，三端行为一致。

## 使用说明

### 操作步骤

1. **打开健康自查页面**
   - 进入应用后，点击底部导航栏中的相关入口，或从首页找到"健康自查"功能入口
   - 访问地址：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/symptom](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/symptom)

2. **完成症状选择流程**
   - **第一步**：选择不适的身体部位（如头部、胸部、腹部等）
   - **第二步**：选择具体症状（可多选），并可补充描述和选择持续时间
   - **第三步**：查看症状摘要，点击"AI智能分析"按钮

3. **为谁咨询 — 选择"本人"**
   - 系统弹出"为谁咨询"面板，默认选中"本人"
   - 展示健康档案编辑区域，包含以下字段：

   | 字段 | 必填 | 说明 |
   |------|------|------|
   | 姓名 | ✅ 必填 | 带红色 `*` 标识，输入框形式 |
   | 性别 | ✅ 必填 | 带红色 `*` 标识，男/女按钮选择 |
   | 出生日期 | ✅ 必填 | 带红色 `*` 标识，日期选择器 |
   | 身高 | 选填 | 单位 cm |
   | 体重 | 选填 | 单位 kg |
   | 既往病史 | 选填 | 标签选择 + 自定义输入 |
   | 过敏史 | 选填 | 标签选择 + 自定义输入 |
   | 家族遗传病史 | 选填 | 标签选择 + 自定义输入 |

4. **数据自动回填**
   - 如果您之前已填写过个人资料，系统会自动将已有的姓名、性别、出生日期等信息回填到对应字段
   - 回填后的数据可直接使用，无需重新填写

5. **填写必填信息**
   - 确保**姓名**已输入、**性别**已选择、**出生日期**已选择
   - 红色 `*` 标识的字段为必填项

6. **点击"确认并开始分析"**
   - 如果三个必填字段都已填写 → 系统保存数据并跳转到 AI 咨询页面
   - 如果有未填写的必填字段 → 系统会：
     - 在未填写的字段下方显示红色错误提示（如"请输入姓名"）
     - 弹出 Toast 提示"请填写完整的必填信息"
     - 阻止提交，直到所有必填项填写完整

### 校验不通过时的提示示例

| 缺失字段 | 字段下方提示 |
|----------|-------------|
| 姓名为空 | 请输入姓名 |
| 性别未选 | 请选择性别 |
| 出生日期未选 | 请选择出生日期 |

同时会弹出全局 Toast 提示：**"请填写完整的必填信息"**

## 注意事项

1. **仅针对"本人"生效**：此必填校验仅在选择"本人"作为咨询对象时触发。选择其他家庭成员时，使用的是已有的家庭成员必填校验逻辑
2. **数据会自动保存**：确认提交后，系统会自动将您填写的姓名、性别、出生日期等信息保存到您的健康档案，下次使用时无需重复填写
3. **为何要求必填**：姓名、性别、出生日期是 AI 健康分析的基础信息，缺失这些信息可能影响分析结果的准确性
4. **三端行为一致**：H5 网页端、微信小程序端、Flutter App 端的必填校验逻辑和提示信息完全一致
