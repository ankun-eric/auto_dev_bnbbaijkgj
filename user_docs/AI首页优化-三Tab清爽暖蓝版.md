# AI 首页优化使用手册（三 Tab · 清爽排版 · 暖蓝关怀配色）

> 适用端：手机浏览器 H5 端
> 版本：V1.0（2026-06-01）
> 设计目标：面向 40–70 岁用户，字大、图标大、按钮好戳、不飘东西、清爽温暖。

---

## 一、功能介绍

本次把「AI 首页」整体做了清爽改版，主要变化有四点：

1. **顶部新增 3 个 Tab**：档案 / 咨询 / 服务，打开后默认停在「咨询」。
2. **页面做减法、排清爽**：删掉了右下角会误碰的悬浮头像球，铃铛收进右上角固定。
3. **配色升级为「暖蓝关怀」**：以柔和天蓝为主色，少量暖橙点缀，看着清爽又温暖。
4. **常用操作收进「+ 菜单」**：切换模式、邀请好友等不再占用正文空间。

## 二、页面布局总览

### 顶栏（从左到右）

| 位置 | 元素 | 作用 |
|------|------|------|
| 左上角 | 「三」汉堡图标 | 点开查看历史会话 |
| 中间 | 档案 / 咨询 / 服务 三个 Tab | 切换不同功能页，选中项下方有一条蓝色小横线 |
| 右上角 | 「+ 圆圈」图标 | 打开更多菜单 |
| 右上角 | 🔔 小铃铛 | 固定不动；有新提醒（如待办、用药）时显示红点 |

### 咨询页正文（从上到下只剩 5 块）

1. **一句问候语**：例如「Hi~ 健康管理用小康」。
2. **一张健康贴士图**：一张静态大图，不再轮播。
3. **4 个大图标**：等大、圆角方块、统一底色、整齐一排（默认：AI 诊室 / 看报告 / 查用药 / 找医院，具体以后台配置为准）。
4. **3 条「大家都在问」**：不知道问什么时，照着点即可。
5. **底部 AI 输入框**：随时打字或语音向小康提问。

## 三、操作步骤

### 1. 切换三个 Tab

- 打开 App 默认停在 **「咨询」**。
- 点顶部 **「档案」**：进入家庭健康档案页，可在顶部切换「本人 / 妈妈 / 爸爸」查看不同人的档案。
- 点顶部 **「服务」**：进入健康服务聚合页（健康管理 / 就医服务 / 关怀守护 / 我的）。
- 点回 **「咨询」**：回到 AI 咨询首页。选中的 Tab 下方会有一条蓝色小横线提示当前所在页。

### 2. 查看历史会话

- 点左上角 **「三」** 图标，打开历史会话列表。

### 3. 查看提醒（铃铛）

- 右上角 🔔 铃铛固定在顶栏，不会再飘动。
- 当有新的待办或用药提醒时，铃铛上会出现 **红点**；点铃铛即可查看详情。

### 4. 使用「+ 圆圈」更多菜单

点右上角 **「+ 圆圈」**，弹出 4 个选项：

1. **发起新对话**：开启一段全新的咨询。
2. **切换模式**：切换 AI 回答模式（带当前模式标签，默认「标准模式」）。
3. **邀请好友**：把小康分享给家人朋友。
4. **帮助与反馈**：遇到问题可在这里反馈。

### 5. 向小康提问

- 在底部输入框直接打字提问，或点右下角 **语音** 按钮说话。
- 左下角可切换「为谁咨询」（帮爸妈问健康问题时很有用）。
- 输入完点 **发送** 即可。

## 四、注意事项

1. **默认停在「咨询」**：每次进入 App 都会停在咨询页，无需手动切换。
2. **Tab 切换说明**：「咨询」是停留在当前页；点「档案」「服务」会进入各自的功能页，属正常跳转。
3. **铃铛红点**：红点代表有未处理的提醒（待办 / 用药等），点开处理后红点会消失。
4. **不再有悬浮球**：原来右下角的小康头像悬浮球已删除，避免误碰，相关功能已整合进顶栏和菜单。
5. **4 个图标内容**：由后台统一配置，显示的图标和数量可能随运营调整而变化，属正常现象。
6. **适老化设计**：正文字号、按钮点击区域均已加大，方便长辈点按；若仍觉得字小，可在「+ 菜单 → 字体大小」（如有）中调整。

## 五、配色说明（暖蓝关怀）

- **主色天蓝**（`#3FA9F5`）：用于选中 Tab 下划线、主按钮、发送键、链接。
- **顶栏淡蓝**（`#EAF6FF`）、**页面近白**（`#F7FBFF`）：整体干净清爽。
- **暖橙点缀**（`#FFB26B`）：贴士条小竖线、推荐问小竖线、铃铛红点，少量使用、点到为止，让页面温暖不冷淡。
