## 一、功能介绍

本次更新围绕「健康档案」做了两件事，让填写体验更顺手、更统一：

1. **编辑改为底部半屏抽屉**
   健康档案页顶部 Hero 卡片右上角的「编辑」，点开后不再是占满整屏的大表单，而是**从屏幕底部向上滑出的半屏抽屉**（上方仍能看到背景），与「邀请家庭成员」弹框观感一致；同时修复了原来「身高 / 体重」输入框被屏幕边缘挤压、截断的问题。

2. **新增并统一三项健康信息**
   把「过敏史 / 手术史 / 家族病史 / 个人习惯」整合成一个**公共填写区**，在「编辑健康档案」与「添加 / 邀请家庭成员」两处**完全一致**地使用。其中**手术史、家族病史**为本次新增，可结构化添加与删除。

## 二、操作步骤

### 场景 A：编辑自己 / 家人的健康档案

1. 打开健康档案页：
   `https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile`
2. 点击顶部卡片右上角的 **「编辑」**，底部弹出半屏抽屉「编辑健康档案」。
3. **基础信息**（默认展开）：
   - 填写 / 修改 **姓名**；
   - **性别**：点选「男 / 女」胶囊；
   - 选择 **出生日期**、**血型**；
   - **身高（cm）** 与 **体重（kg）** 现已并排两列，各自独立、不再被截断。
   - 关系：本人 Tab 显示「本人」且只读；非本人 Tab 可修改关系。
4. 点开 **「其他（选填）」** 折叠区，可维护：
   - **既往病史（慢病）**：点选常见慢病胶囊；
   - **过敏史**：药物 / 食物 / 其他，输入后点「添加」生成标签，点标签上的 `×` 删除；
   - **手术史（新增）**：点「+ 添加手术史」→ 填手术名称、时间、备注 → 保存；列表中可「编辑 / 删除」；
   - **家族病史（新增）**：点「+ 添加家族病史」→ 选与本人关系、填疾病名称、备注 → 保存；列表中可「编辑 / 删除」；
   - **个人习惯**：抽烟（有/无）、饮酒（有/无）、运动频率（无/偶尔/经常）、饮食偏好（清淡/重口味/素食/其他）。
5. 底部点击 **「保存」**，提示保存成功后数据即落库；点「取消」放弃修改。

### 场景 B：添加 / 邀请家庭成员时顺手填写

1. 打开家庭成员管理页：
   `https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/family-guardian-list`
2. 点击「添加成员 / 邀请家人」，底部弹出新增成员抽屉。
3. 填写昵称、关系等基础信息后，展开 **「其他（选填）」**，可见与编辑档案**完全一致**的健康信息填写区（过敏史 / 手术史 / 家族病史 / 个人习惯）。
4. 一并填好后提交，系统会先创建成员，再自动把这些扩展健康信息保存到该成员的档案中。

## 三、注意事项

1. **「其他（选填）」均为可选**：不填不影响保存基础档案；只填写需要的部分即可。
2. **手术史 / 家族病史是逐条添加**：每条单独「保存」，列表里可随时「编辑」或「删除」。
3. **身高 / 体重为数字**：建议合理范围（如身高 0–300cm、体重 0–300kg），留空表示未填写。
4. **本人关系不可改**：本人 Tab 的关系固定为「本人」；要维护家人请切到对应成员或在添加成员处操作。
5. **添加成员时的扩展信息**：若成员创建成功但扩展健康信息（手术史等）保存失败，不会影响成员本身已创建成功，可稍后在该成员档案的「编辑」中补填。
6. **两处填写区一致**：编辑档案与添加成员处为同一套填写组件，操作方式完全相同。

## 四、常见问题（FAQ）

- **Q：点「编辑」为什么是从下往上弹出而不是整屏？**
  A：这是本次优化，统一为底部半屏抽屉，便于单手操作且不遮挡全部背景。

- **Q：身高体重输入框以前会被截断，现在呢？**
  A：已修复，改为同一行两列并排，各自独立完整显示。

- **Q：手术史 / 家族病史填了在哪能看到？**
  A：保存后会出现在「其他（选填）」对应分区的列表中，可继续编辑或删除。
