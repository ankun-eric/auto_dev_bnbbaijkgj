## 功能简介

本次更新解决了血压详情页的两个体验问题，并把更直观的操作方式同步到了血糖详情页：

1. **AI 本次解读按钮更省心**：「🤖 AI 解读本次血压」按钮始终显示。点击时系统自动判断——如果还没有任何血压记录，会在屏幕中央弹出轻提示「暂无血压记录，请先录入一次再点击解读。」，不会进入解读流程；有记录时则正常给出 AI 解读结果。
2. **历史记录可以改 / 删，且一眼可见**：每条历史记录右侧新增了「…」三点入口，点击后从屏幕底部滑出操作面板，包含「修改」和「删除」两个选项。
   - **修改**：可以完整修改这条记录的血压数值、测量时段和测量时间，保存后直接更新原记录（不会多出一条重复记录）。
   - **删除**：会先弹出二次确认「确认删除这条记录？此操作不可撤销」，确认后删除成功并提示「已删除」，列表实时刷新。
3. **血糖详情页同步优化**：血糖的历史记录此前只能左滑出现删除按钮，不够直观。本次为血糖每条记录也补上了「…」三点入口，点开同样从底部滑出含「修改 / 删除」的操作面板，与血压完全一致。

## 使用说明

### 一、AI 本次解读

1. 进入「健康档案」，点击「血压」卡片进入血压详情页。
2. 在趋势图下方可以看到「🤖 AI 解读本次血压」按钮（始终显示）。
3. 点击该按钮：
   - 若当前没有任何血压记录 → 屏幕中央会一闪而过提示「暂无血压记录，请先录入一次再点击解读。」，请先用「手工录入」录入一次血压。
   - 若已有记录 → 正常弹出 AI 解读结果。

### 二、修改一条血压历史记录

1. 在血压详情页下方「历史记录」列表中，找到要修改的记录。
2. 点击该记录右侧的「…」三点图标。
3. 从底部滑出的面板中点击「修改」。
4. 在弹出的编辑面板中修改收缩压、舒张压、测量时段。
5. 点击「保存」，记录会被就地更新，列表自动刷新。

### 三、删除一条血压历史记录

1. 在「历史记录」列表中找到要删除的记录，点击右侧「…」。
2. 在底部面板中点击「删除」。
3. 在二次确认弹层中确认「确认删除这条记录？此操作不可撤销」。
4. 点击「确认删除」，提示「已删除」后列表实时刷新。

### 四、血糖历史记录操作（同步新增）

血糖详情页的操作方式与血压完全一致：每条历史记录右侧点击「…」→ 底部面板「修改 / 删除」。修改完整更新原记录，删除有二次确认 +「已删除」提示。

## 注意事项

- 详情页需登录并选择健康档案后访问；建议从「健康档案」页面点击血压/血糖卡片进入。
- 「修改」为完整更新原记录，不会新增重复记录；「删除」为物理删除，删除后不可恢复，请谨慎操作。
- 血压与血糖的「…」底部操作面板样式、动画、文案保持完全统一。
- 微信小程序中的血压/血糖详情页通过内嵌网页方式展示，会自动同步本次 H5 优化，无需单独更新小程序。
