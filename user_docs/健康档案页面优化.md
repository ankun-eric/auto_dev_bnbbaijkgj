> 版本：v1.0　|　发布日期：2026-06-01　|　所属模块：家庭健康管理 · 健康档案

## 一、本次优化做了什么

「健康档案」是您管理家里人健康的核心页面。这次优化围绕**更直观、更一致**做了 3 处改进：

1. **叫法更直白**：原来的「档案列表 / 健康档案」统一改名为 **「家庭成员」**，一眼就能看懂这是"管理家里人"的地方。
2. **人数对得上**：入口卡片上显示的"已管理 X 人"，现在和点进去后列表里实际统计的人数**始终一致**，不再出现两边数字对不上的困惑。
3. **新增成员更顺手**：点击「邀请家庭成员」按钮后，弹出的填写页面与 AI 首页「新增咨询人」**完全一样**；填写保存后自动回到「家庭成员」列表，新成员立即出现，人数自动 +1。

## 三、操作步骤

### 1. 进入「家庭成员」

1. 打开健康档案首页。
2. 找到原来的「档案列表 / 健康档案」入口卡片，现已显示为 **「家庭成员」**。
3. 卡片上的 **「已管理 X / 上限 Y」** 即家庭成员人数（含本人）。
4. 点击卡片进入 **「家庭成员」** 列表页，页面顶部标题为 **「家庭成员」**。

### 2. 核对人数是否一致

- 进入列表页后，列表中统计出来的成员数量，与刚才入口卡片上的「已管理 X」**完全相同**。
- 无论新增、删除成员，两处数字都会同步变化。

### 3. 新增家庭成员（复用「新增咨询人」流程）

1. 在「家庭成员」列表页顶部，点击 **「邀请家庭成员」** 按钮。
2. 弹出的新增成员填写页面，与 AI 首页「咨询人 → 新增咨询人」**完全一致**：填写昵称、与本人关系、性别等信息。
3. 点击保存，**成功后自动回到「家庭成员」列表**。
4. 新增的成员**立即出现在列表中**，人数**自动 +1**，入口卡片人数同步更新。

## 四、注意事项

1. **人数含本人**：人数口径为"家庭成员总数（含本人）"，因此新账号进入时通常已有 1 人（您自己）。
2. **两处数字始终一致**：入口卡片与列表页采用同一数据来源，若发现不一致请刷新页面重试。
3. **已删除的成员不计入**：被删除（软删）的成员不会出现在列表中，也不计入人数。
4. **按钮文字不变**：「邀请家庭成员」按钮文字保持不变，仅点击后的行为改为复用「新增咨询人」填写页面。
5. **小程序需重新打开页面**：小程序更新后，建议重新进入「健康档案」页面以加载最新统计。

## 五、常见问题（FAQ）

**Q1：为什么入口卡片显示的人数比我添加的成员多 1？**
A：人数包含您本人，所以是"本人 + 已添加成员"的总数。

**Q2：点「邀请家庭成员」后弹出的页面，和以前不一样了？**
A：是的，现在统一复用与「新增咨询人」相同的填写页面，体验更一致。

**Q3：保存成功后没看到新成员？**
A：保存成功会自动回到列表并刷新，新成员应立即出现；若未出现，请下拉或重新进入页面刷新。
