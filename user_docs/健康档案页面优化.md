## 一、本次优化能为你做什么

本次升级把"健康档案"这张承担最多信息的核心页面做了一次全面重构，让你能用更少的步骤、更直观的视觉，看清「我正在管理谁」、「TA 今天还要吃几次药」、「我们是否已建立守护关系」。

升级后，你将感受到以下变化：

1. **顶部标题栏 + 咨询人切换条整体吸顶**，无论你向下滚动多远，都能立刻切换到下一个家人
2. **切换咨询人 = 切换整页世界**：包括"今日用药数"、用药计划、各项健康卡片都按 TA 重新加载，不会再有"上一个人数据闪现"
3. **头像右上角胶囊角标"被守护"**：一眼分清哪些家人已经被你守护
4. **Hero 卡更专注**：去掉了"既往病史 / 过敏史 / 家族遗传"3 个低频字段，新增「今日用药 · N ›」直达打卡页
5. **共管区域三态切换**：本人态显示"已守护 N 人 ›"，未守护非本人显示"+ 邀请共管"，已守护非本人显示"管理"
6. **AI 外呼提醒整体迁移到「家庭守护列表」**：一份配置覆盖该人所有用药计划，避免逐条计划设置
7. **设备入口整理**：本人态走"我的"中"我的设备"，被守护人的设备只在「家庭守护列表」内以只读视图查看

## 三、操作步骤详解

### 3.1 切换咨询人

1. 打开「健康档案」页
2. 顶部标题栏下方的"咨询人切换条"提供横向滚动头像
3. 点击任意头像即可切换：
   * Hero 卡瞬时刷新成新咨询人的资料
   * 「今日用药 · N ›」自动按 TA 重新计数
   * 下方所有人维度的卡片全部一刀切刷新
4. 滚动浏览过程中，标题栏 + 切换条始终吸顶，任何时刻都能切换

### 3.2 识别"被守护"角标

* 头像右上角出现蓝色胶囊「被守护」，意味着 **你（当前主账号）正在守护这个家人**
* 角标在切换条头像和 Hero 大头像上同步显示
* 本人自己不会出现该角标

### 3.3 查看 / 跳转「今日用药」

1. 在 Hero 卡内找到「💊 今日用药 · N ›」
2. 数字 N 表示当天该咨询人**应打卡总次数**（含已打 / 漏打 / 待打）
3. 点击直接跳转到该咨询人的打卡页
4. 即使 N=0，也仍可点击进入

### 3.4 共管区域三种形态

| 当前选中咨询人 | 区域显示 | 点击行为 |
|----------------|----------|----------|
| 本人 | 「已守护 N 人 ›」（N=0 时显示"暂未守护任何人，去邀请家人 ›"） | 进入「家庭守护列表」 |
| 非本人 + 未守护 | 按钮「+ 邀请共管」 | 进入"邀请共管"页（生成邀请链接发给 TA） |
| 非本人 + 已守护 | 按钮「🔧 管理」 | 跳到「家庭守护列表」中 TA 的详情页 |

### 3.5 在「家庭守护列表」管理 AI 外呼提醒

1. 进入「家庭守护列表」（任意上述入口）
2. 列表显示：**本人对自己** + 所有已守护家人
3. 点击任意一行进入"守护详情"
4. 在详情页可配置：
   * **AI 外呼总开关**：关闭后该家人名下所有用药计划都不再外呼
   * **免打扰时段**：起止时间（支持跨天，例如 22:00 - 07:00）
   * **外呼对象**：被守护人本人 / 守护者
5. 配置即时保存，无需手动提交

### 3.6 查看被守护人的设备（只读）

1. 进入「守护详情」页（仅对已守护的非本人显示）
2. 找到"📱 TA 的设备"模块
3. 可看到 TA 名下设备列表（设备名 + 最近同步时间），所有设备均为**只读**
4. 若对方暂未绑定设备，会出现「提醒 TA 绑定设备 ›」按钮
5. 点击按钮系统会向 TA 发送一条提醒（站内 + 微信通道）

### 3.7 解除守护关系

1. 在"守护详情"页底部，点击「解除守护」
2. 弹窗二次确认后即可解除
3. 解除后：
   * 切换条头像上的"被守护"角标消失
   * 健康档案页中的"管理"按钮自动恢复为"+ 邀请共管"
   * 相关数据按未守护权限重新拉取

### 3.8 打卡提醒设置入口

* 在健康档案页下方仍保留「📞 打卡提醒设置 ›」单行入口
* 点击直接跳转到家庭守护列表里"对自己"的 AI 外呼设置
* 当选中咨询人是"非本人 + 未守护"时该入口自动隐藏

## 四、注意事项

1. **站内消息 / 微信消息为系统默认通道**，本次升级后不再向用户暴露开关，永远开启
2. **AI 外呼每个被守护人只有一份配置**，所有用药计划共用，不再支持单条计划差异化设置
3. **每台设备只能绑定一个登录账号**：主账号无法直接增 / 删 / 解绑家人名下的设备，必须由 TA 本人在自己的微信里操作
4. **既往病史 / 过敏史 / 家族遗传**已从 Hero 卡移除，仍保留在"完整档案详情页"内可查
5. 健康档案页右上角的"设备 Logo"已下线；本人管理自己的设备请走"我的"/个人中心
6. **邀请链接有 7 天时效**，被邀请人需在自己微信中完成授权才正式建立守护关系
7. **跨天免打扰时段**（如 22:00 - 07:00）系统会按跨天逻辑正确判断，不会遗漏
8. **若选择外呼对象 = 守护者，但当前没有守护关系**，系统会自动回退为"被守护人本人"

## 五、常见问题

**Q1：切换咨询人后，为什么用药计划仍然停留在上一个人？**

A：本次优化已修复该问题。如仍出现，请下拉刷新或退出页面重进。

**Q2：被守护的家人能看到我给他配置的 AI 外呼吗？**

A：可以。AI 外呼是对**他本人手机**的拨打行为，他在自己账号内也能查看与覆盖配置。

**Q3：我可以不守护就给家人配置 AI 外呼吗？**

A：不行。必须先建立守护关系（通过"+ 邀请共管"流程）后才能配置。

**Q4：家人换了手机号，AI 外呼怎么处理？**

A：拨打默认走他在系统里登记的手机号。请提醒家人在个人资料中更新手机号。

**Q5：以前每条用药计划上的 AI 外呼开关 / 时段，现在保留了哪些？**

A：本次升级时系统已对每个用户做一次性迁移：根据历史多数情况推算"总开关"，免打扰时段沿用最近一次有效配置，外呼对象沿用最近一次有效配置；无历史则默认"关闭 / 22:00-07:00 / 被守护人本人"。
