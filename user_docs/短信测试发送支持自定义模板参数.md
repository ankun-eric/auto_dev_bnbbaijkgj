## 功能简介

本次更新为短信管理模块新增了**自定义模板参数**功能，主要包含以下增强：

1. **模板变量配置**：在短信模板的新增/编辑流程中，支持为每个模板定义变量信息（变量名称、变量说明、默认值），方便管理员预先配置模板参数。
2. **测试发送参数自定义**：管理员在测试发送短信时，可以手动指定模板变量参数的具体值，支持自动模式（根据模板配置自动生成输入框并填充默认值）和高级输入模式（手动输入逗号分隔的参数）。
3. **发送前预览确认**：点击发送后会弹出预览确认框，展示手机号、模板信息、参数列表和短信预览内容，确认无误后才真正发送。
4. **测试结果增强展示**：发送完成后（无论成功或失败）展示参数列表和完整短信预览内容。

## 使用说明

### 一、模板变量配置

#### 1. 进入短信管理页面

登录管理后台后，在左侧菜单中点击 **「系统管理」→「短信管理」**，进入短信管理页面。

#### 2. 配置模板变量

1. 切换到 **「短信模板」** 标签页
2. 点击 **「新增模板」** 按钮，或点击已有模板的 **「编辑」** 按钮
3. 在弹出的模板编辑窗口中，除了填写基本信息（模板名称、服务商、模板ID、模板内容、签名名称、用途场景）外，在底部可以看到 **「模板变量配置」** 区域
4. 点击 **「添加变量」** 按钮，每点击一次新增一行变量配置：
   - **变量名称**（必填）：如"验证码"、"有效时间"
   - **变量说明**（选填）：如"6位数字验证码"、"单位为分钟"
   - **默认值**（选填）：如"123456"、"5"
5. 变量的排列顺序与云厂商模板中的 `{1}`、`{2}`、`{3}` 等占位符一一对应
6. 每行变量右侧有删除按钮，可删除该行
7. 配置完成后点击 **「保存」** 即可

#### 3. 查看模板变量状态

在模板列表中，**「变量说明」** 列会显示已配置的变量名称列表。未配置变量的模板会显示黄色的 **「未配置」** 标签，提醒管理员去补全。

---

### 二、测试发送短信（自定义参数）

#### 1. 进入测试发送

切换到 **「测试发送」** 标签页。

#### 2. 填写基本信息

- **服务商**：选择腾讯云或阿里云
- **手机号**：输入11位手机号（支持正则校验 `1` 开头的11位数字）
- **短信模板**：从下拉列表中选择已配置的短信模板（**必选**）

#### 3. 填写模板参数

选择模板后，系统会根据模板是否配置了变量信息自动展示对应的输入方式：

**自动模式（默认）：**
- 如果所选模板已配置变量信息，系统会自动生成对应数量的输入框
- 每个输入框标注变量名称和说明
- 默认值会自动填充，管理员可直接使用或修改
- 如果模板未配置变量信息，会显示提示信息，建议先补全变量配置或切换到高级模式

**高级输入模式：**
- 点击 **「切换到高级输入」** 按钮，切换到文本框输入方式
- 在文本框中按英文逗号分隔输入所有参数值，如：`123456,5`

#### 4. 预览并确认发送

1. 点击 **「发送测试短信」** 按钮
2. 系统弹出 **预览确认框**，显示以下信息：
   - 收信手机号
   - 服务商信息
   - 模板名称和模板ID
   - 各变量参数及其值（如：验证码 = 123456，有效时间 = 5）
   - **短信预览内容**：将参数值填入模板后生成的最终短信文本
3. 确认无误后，点击 **「确认发送」** 真正发送测试短信
4. 如需修改，点击 **「取消」** 返回修改

#### 5. 查看发送结果

发送完成后，表单下方会展示结果卡片：

- **成功时**：绿色样式，显示"发送成功"
- **失败时**：红色样式，显示失败原因

结果区域同时展示：
- **参数列表**：本次使用的各变量参数名称和值
- **短信预览内容**：将参数填入模板后的完整短信文本

---

### 三、发送记录查看

切换到 **「发送记录」** 标签页，可以查看所有短信发送记录。测试发送的记录会标注 **「测试」** 标签，记录中包含本次使用的模板参数信息。

## 注意事项

1. **权限要求**：只有系统管理员（admin 角色）才能访问短信管理模块的所有功能
2. **变量顺序**：模板变量的顺序必须与云厂商模板中的占位符（`{1}`、`{2}`等）严格对应，请确认一致
3. **旧模板兼容**：已有的旧模板未配置变量信息时，系统不会报错，可通过高级模式手动输入参数，或先编辑模板补录变量配置
4. **参数校验**：
   - 自动模式下，所有变量输入框必须填写，否则无法发送
   - 高级模式下，如果模板已配置变量信息，系统会校验输入的参数个数是否匹配
5. **测试发送仅用于验证**：本功能仅用于测试短信效果，不影响业务发送逻辑
6. **预览内容为参考**：预览内容基于本地模板 content 生成，实际短信以云厂商审核通过的模板内容为准
