## 功能简介

在 AI 首页右上角有一个「🔔 铃铛」，点开后里面有一块「💊 用药提醒」。

**本次修复前的问题**：当你在首页切换过某个家庭成员、或从识药卡点过「用药提醒」按钮后，再点铃铛，里面会**混进家庭成员的用药**（例如把家人在吃的「阿莫西林克拉维酸钾片」也列了出来），而你只想看自己的。

**本次修复后**：

- 铃铛里的「用药提醒」**恒定只显示登录账号本人**的用药，完全不受"当前选了哪个咨询人/家庭成员"的影响。
- 「用药提醒-全部」页在正常进入（未指定具体家庭成员）时，也**只显示本人**的用药，与铃铛口径完全一致。
- 铃铛里每条用药的**状态徽章与操作按钮**（已服用 / 即将服用 / 已超时 / 未到时间）与「用药提醒-全部」页逐条一致；铃铛里的「待办计数」= 全部页的「待服用」数字。
- 在铃铛里打卡 / 撤销后，回到「全部页」状态会实时同步。

> 本次更新仅涉及 H5 网页端，后端、微信小程序、手机 App 均无需更新，继续使用即可。

## 使用说明

### 一、查看铃铛里的本人用药提醒

1. 打开 [AI 首页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/)。
2. 点击页面**右上角的「🔔 铃铛」**图标，打开提醒抽屉。
3. 在抽屉里找到「💊 用药提醒」区块——这里现在**只会显示你本人**的用药，不再混入任何家庭成员的用药条目。
4. 每条用药右侧会显示对应的状态与操作按钮：

   | 状态 | 状态徽章 | 操作按钮 |
   |------|----------|----------|
   | 已服用 | 已服用（绿色） | ✓ 完成（绿色描边，点击可在 5 分钟内撤销） |
   | 即将服用 | 即将服用（橙色） | 打卡（橙色） |
   | 已超时 | ⚠️ 已超时（红色） | 补打卡（红色） |
   | 未到时间 | 未到时间（灰色） | 打卡（蓝色） |

### 二、在铃铛里打卡 / 撤销

1. 对「即将服用 / 未到时间 / 已超时」的用药，点击对应的「打卡 / 补打卡」按钮即可标记为已服用。
2. 对「已服用」的用药，在 5 分钟内点击「✓ 完成」可撤销本次打卡。
3. 打卡或撤销后，铃铛里的「待办计数」会实时更新；切换到「用药提醒-全部」页，状态也会同步一致。

### 三、查看完整用药列表（全部页）

1. 进入 [用药提醒-全部页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home/medication-reminder/)。
2. 从「健康档案 - 本人 Tab」正常进入时，这里**只显示本人**的用药，与铃铛完全一致。
3. 只有当你从首页明确选择了某个具体家庭成员再进入时，才会按该成员展示其用药。

## 注意事项

- **登录后才能看到用药数据**：用药提醒需要登录账号后查看，未登录会提示需要登录。
- **铃铛永远是本人口径**：无论你之前在首页切换过哪个咨询人 / 家庭成员，铃铛里的用药始终只显示你本人的，这是设计预期，不是 Bug。
- **家庭成员的用药在哪看**：如需查看某个家庭成员的用药，请在首页明确选择该成员后，进入「用药提醒-全部」页查看。
- **撤销有时间限制**：已服用的打卡仅在 5 分钟内可撤销，超时后按钮不再支持撤销。
- 本次更新为网页端即时生效，无需重新下载或安装任何客户端。
