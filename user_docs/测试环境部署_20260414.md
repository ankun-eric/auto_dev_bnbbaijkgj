## 功能简介

本次操作将 Bini Health 健康管理系统从正式环境（newbb.bangbangvip.com）部署到测试环境服务器（newbb.test.bangbangvip.com），使用全新的部署唯一标识 `6b099ed3-7175-4a78-91f4-44570c84ed27`。

Bini Health 是一款综合健康管理系统，包含以下核心功能：

- **AI 健康助手**：智能对话问诊、症状分析、中医诊断
- **体检报告管理**：OCR 识别上传、智能分析、趋势追踪
- **健康计划**：个性化健康方案生成、每日打卡任务
- **药品查询**：药品搜索、药品识别、用药建议
- **家庭管理**：家庭成员绑定、健康数据共享、SOS 紧急求助
- **健康服务**：专家咨询、服务预约、订单管理
- **积分系统**：签到积分、积分商城兑换
- **内容资讯**：健康文章浏览、收藏

## 使用说明

### H5 用户端

1. **打开页面**：在手机浏览器或微信中打开 H5 链接
2. **注册/登录**：首次使用需通过手机号短信验证码注册，之后可直接登录
3. **首页导航**：登录后进入首页，可看到 Banner 轮播、功能菜单入口、健康资讯等
4. **功能使用**：
   - 点击 **AI 助手** 进入智能对话
   - 点击 **体检报告** 上传和查看报告
   - 点击 **健康计划** 制定和执行每日健康任务
   - 点击 **药品查询** 搜索或拍照识别药品
   - 点击 **家庭管理** 绑定家庭成员
   - 点击 **积分中心** 查看积分余额和签到

### 管理后台

1. **打开页面**：在 PC 浏览器中打开管理后台链接
2. **登录**：使用管理员账号登录
3. **功能管理**：
   - **首页配置**：管理 Banner、菜单入口、搜索提示文字
   - **内容管理**：文章发布与管理
   - **用户管理**：查看和管理注册用户
   - **服务管理**：配置健康服务分类和项目
   - **专家管理**：管理专家信息和排班
   - **积分管理**：配置积分规则和商城商品
   - **消息通知**：发送系统通知

## 注意事项

1. **测试环境说明**：本次部署为测试环境，数据与正式环境完全独立，请放心测试
2. **浏览器兼容性**：H5 用户端推荐使用手机浏览器或微信内置浏览器访问；管理后台推荐使用 Chrome、Edge 等现代浏览器
3. **网络要求**：需要能访问 `newbb.test.bangbangvip.com` 域名，请确保网络通畅
4. **数据初始化**：首次部署时数据库为空，需要通过管理后台配置基础数据（如菜单、Banner 等），用户端才能正常展示
5. **客户端应用**：Flutter APP 和微信小程序在本测试环境中需要手动修改 API 地址才能连接（API 地址已在代码中更新为测试环境地址）
