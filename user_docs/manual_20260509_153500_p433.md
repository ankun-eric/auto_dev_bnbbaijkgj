## 功能简介

本次升级把 AI 对话模式首页（`/ai-home`）的「**无气泡纯文本流**」（PRD-429 落地版）整体升级为「**AI 卡片 + 用户气泡**」分侧聊天样式，与晓医、ChatGPT、文心一言、通义千问等主流 AI 健康助手视觉风格对齐，提升 AI 回答的视觉边界感与对话可读性，同时让用户消息回到主流右侧位置。

## 使用说明

### 1. 用户消息（右侧浅蓝气泡）

- 在 AI 对话首页底部输入框输入消息后点击 ➤ 发送
- 您的消息将显示在屏幕右侧，呈现为**浅蓝色淡彩气泡**（背景 `#E6F0FF`、文字 `#1F2937`）
- 气泡最大宽度 ≤ 屏幕宽度的 75%，超出自动换行
- 气泡距屏幕右边缘 16px，**不显示头像**（与 ChatGPT、Claude、文心一言等主流 AI 一致）

### 2. AI 回答（左侧白色卡片）

- AI 回复将以**白色卡片**形式呈现在屏幕左侧
- 卡片占屏 88~90%，圆角 12~14px，有 1px 极浅灰描边（`#EAEBED`），无阴影
- 卡片**外部上方**独立一行显示「🌿 小康」头像和名称
- 卡片内部从上到下依次为：
  1. 咨询对象档案折叠卡片（PRD-432，可点击展开/收起 7 项核心档案）
  2. AI 回答正文（支持 Markdown：标题/列表/链接/代码块/表格/图片等）
  3. **参考资料**列表（**容错渲染**：仅当后端返回 `references` 字段时显示）
  4. **AI 生成内容仅供参考，不作为诊断依据** 一行小字
  5. 操作按钮行：📋 复制 / 🔗 分享 / 🔁 朗读（朗读靠右）

### 3. AI 生成中状态（Loading 卡片）

- 用户发送消息后、AI 首字未到达前，会显示一张与正常 AI 卡片视觉一致的**白底卡片**
- 卡片内显示「**小康正在思考中…**」文字 + **三点跳动**加载动画
- 首字到达后 Loading 内容自动淡出，开始**流式输出**（逐字打字机效果，**去除了末尾光标**）
- 流式过程中操作按钮 / 免责 / 参考资料暂不显示，待完整接收后再淡入显示

### 4. 时间显示（时间分隔条）

- 不再在每条消息上显示时间戳
- 当前消息与上一条消息**间隔超过 5 分钟**时，在两条消息之间插入一条**时间分隔条**
- 时间格式（与微信一致）：
  - 今天：`HH:mm`（如 14:30）
  - 昨天：`昨天 HH:mm`
  - 7 天内：`周X HH:mm`（如 周二 14:30）
  - 超过 7 天：`YYYY/MM/DD HH:mm`

### 5. 卡片底部三个操作按钮

| 按钮 | 行为 |
|------|------|
| 📋 复制 | 复制 AI 回答内容到剪贴板，复制成功后 toast 提示「已复制」 |
| 🔗 分享 | 调起对应端的分享面板（H5 站内分享卡 / 小程序转发 / APP 系统分享 sheet） |
| 🔁 朗读 | 调起 TTS 朗读 AI 回答内容；再次点击停止；TTS 配置复用「AI 对话模式首页配置-输入栏」+「AI 咨询配置-TTS 语音配置」既有能力 |

> 朗读按钮位置参考晓医样式，**位于按钮行最右侧**，与左侧两个按钮拉开距离（视觉对称）

### 6. 历史会话抽屉

- 顶部导航栏左上角的 ☰ 菜单图标点击后从左侧滑出抽屉
- 抽屉内：顶部用户头像 + 昵称、中部历史会话列表（按时间倒序：今天 / 昨天 / 7 天内 / 更早 分组）
- 点击历史会话项可恢复该会话
- 关闭方式：点击右侧透明遮罩 / 左滑手势 / 抽屉内顶部关闭按钮

### 7. 输入栏

- 圆角调整为 **22px 胶囊式**，背景 `#F5F7FA`
- 发送按钮配色调整为与用户气泡浅蓝调性协调
- 多模态输入能力（文本 / 语音 / 拍照）**完全保留**，本次仅视觉微调

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 用户消息位置 | 始终靠右、AI 消息始终靠左，**不允许配置反转** |
| 卡片占屏比 | AI 卡片固定 88~90%，**不随消息数量、内容长度变化** |
| 用户气泡宽度 | 严格 ≤ 75vw，超出强制换行 |
| 时间分隔条阈值 | 固定为 **5 分钟**，不做配置项 |
| 免责声明 | 文案固定为「AI 生成内容仅供参考，不作为诊断依据」，每条 AI 卡片必显，**不可关闭** |
| 参考资料 | 采用**容错渲染**：后端无 `references` 字段则不显示，前端不报错 |
| 操作按钮 | 固定为复制/分享/朗读三项，**不引入收藏/点赞/点踩** |
| TTS 配置 | 完全复用「AI 对话模式首页配置-输入栏」+「AI 咨询配置-TTS 语音配置」既有能力，本次**未新增任何后台配置开关** |
| 4 端一致性 | 4 端使用统一设计 token（颜色 / 间距 / 圆角 / 字号），**端间无偏差** |
| 流式输出 | 输出过程中操作按钮 / 免责 / 参考资料**延迟到正文完整后再显示**（避免抖动） |
| 已知不在范围 | 暗黑模式、新建会话流程、多模态输入功能调整、AI 后端接口扩展 references、消息撤回 / 编辑、字体大小调节（沿用现有「···」菜单内能力，本期未动） |
