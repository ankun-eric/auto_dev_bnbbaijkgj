## 功能简介

健康档案页面在切换到某个家人（非本人 Tab）时，右侧的「守护 TA 的人」卡片此前存在三个问题：

1. 部分家人点了卡片**没反应**——尤其是那些一个守护人都没有的家人。
2. 卡片副标题里的「守护者 X / 上限 Y」**显示的是你本人的数字**，不是被切换到的那个家人的。
3. 卡片标题**写死成「守护 TA 的人」**，不直观，分不清在看谁。

本次更新一次性把这三件事修好：

- ✅ 不管这个家人有没有守护人，**点一下都能弹出只读小窗**。
- ✅ 统计数字改为统计**当前这个家人**的真实守护人数与上限。
- ✅ 卡片标题与弹窗标题统一改为 **「XX 的守护人」**（XX = 这个家人的名字 / 昵称，中间一个空格；姓名为空时回退「TA 的守护人」）。

整个交互**纯只读**，不包含任何添加 / 编辑 / 删除 / 邀请按钮。

## 操作步骤

1. **进入健康档案首页**
   - 在 H5 客户端中打开：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile/)
   - 登录后进入。

2. **切换到某个家人**
   - 在页面顶部 Tab 中，从「本人」切到任意一个家人（如「妈妈」「张三」等）。

3. **查看「XX 的守护人」卡片**
   - 看右侧那张带 💚 图标的卡片：
     - **标题**变为「{家人名字} 的守护人」（如「妈妈 的守护人」）；如果家人没填名字，显示「TA 的守护人」。
     - **副标题**「守护者 X / 上限 Y」中：
       - **X** = 当前这个家人**真实**的守护人数量；
       - **Y** = 当前这个家人各自的会员上限（未注册或读不到时回退默认 3）。

4. **点击卡片打开只读弹窗**
   - 不论这位家人**有没有**守护人，点一下卡片都会**从底部弹出**一个只读小窗。
   - 弹窗标题与卡片标题一致：「{家人名字} 的守护人」。
   - 有守护人时：列表平铺展示每个守护人的**头像、姓名、关系、是否主守护人、加入时间**。
   - 一个守护人都没有时：弹窗内显示「**暂无守护人**」空状态。

5. **关闭弹窗**
   - 点击弹窗外的灰色蒙层区域；
   - 或点击弹窗右上角的 ✕ 按钮。

## 注意事项

- 本次改动**仅影响健康档案非本人 Tab** 的「守护 TA 的人」卡片。本人 Tab 的「守护我的人」卡片、左侧「家庭成员」入口卡**保持原样不变**。
- 整个流程**纯只读**：弹窗内不再提供编辑 / 邀请 / 删除等按钮。
- 上限 Y 的取值规则：
  - 该家人**未注册成正式账号** → 上限默认 **3**；
  - 该家人**已注册账号** → 读他自己的会员套餐上限；
  - 兜底（注册了但读不到上限 / 接口失败） → **3**。
- 如果切换 Tab 后副标题数字仍是「0 / 3」，请稍等 1~2 秒等待后台拉取，或刷新页面。
