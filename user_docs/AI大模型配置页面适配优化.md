## 功能简介

本次优化了管理后台「AI 大模型配置」页面的 UI 体验，修复了以下问题：

1. **表格横向溢出问题**：精简了配置列表和模板管理两个 Tab 的表格列数，并添加了横向滚动条支持
2. **无分页问题**：为两个 Tab 的表格均启用了分页功能（默认每页 10 条，支持切换每页条数）
3. **状态列体验差**：统一了状态列展示风格（绿色/橙色 Tag），将状态切换操作移至操作列，并增加了确认弹窗和 Tooltip 提示

## 使用说明

### 一、进入 AI 大模型配置页面

1. 打开管理后台：[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/)
2. 登录管理员账号
3. 在左侧菜单栏中，展开「AI管理」，点击「AI模型配置」

### 二、配置列表 Tab

进入页面后默认显示「配置列表」Tab，表格展示所有 AI 模型配置，包含以下列：

| 列名 | 说明 |
|------|------|
| 服务商 | AI 服务提供商名称 |
| 模型名称 | 使用的模型标识 |
| 参数配置 | 显示 Max Tokens / Temperature 的简要摘要，鼠标悬浮可查看详细参数 |
| 状态 | 绿色「活跃」标签 或 橙色「未启用」标签 |
| 最近测试 | 最近一次连接测试的结果和时间 |
| 操作 | 启用/停用、编辑、测试、删除 |

#### 状态查看

- **绿色标签「活跃」**：表示该配置当前正在被系统使用
- **橙色标签「未启用」**：表示该配置未启用

#### 启用/停用配置

1. 在操作列中找到「启用」或「停用」文字按钮
2. 鼠标悬浮可看到操作提示（如"点击启用"或"点击停用"）
3. 点击后会弹出确认弹窗："确定要启用/停用该配置吗？"
4. 点击「确定」完成操作，状态标签颜色会随之切换

#### 查看参数配置详情

- 在「参数配置」列，鼠标悬浮到数字上方，会显示 Tooltip 气泡，展示完整参数信息：
  - Max Tokens: 4096
  - Temperature: 0.7

#### 分页操作

- 表格底部显示分页器，默认每页 10 条数据
- 左侧显示总条数（如"共 5 条"）
- 右侧可切换每页条数：10 / 20 / 50
- 支持翻页浏览

#### 编辑配置

- 点击操作列中的「编辑」，弹出编辑弹窗
- 在弹窗中可查看和修改 API Key、Base URL 等完整字段
- 修改后点击「保存」

#### 测试配置

- 点击操作列中的「测试」，弹出测试弹窗
- 可输入测试消息并发送，查看模型响应结果

#### 删除配置

- 点击操作列中红色的「删除」文字
- 弹出确认弹窗后点击「确定」完成删除

### 三、模板管理 Tab

点击页面顶部的「模板管理」Tab，表格展示所有模型模板，包含以下列：

| 列名 | 说明 |
|------|------|
| 图标 | 模板对应的服务商图标 |
| 模板名称 | 模板显示名称 |
| 模型名称 | 模板关联的模型标识 |
| 说明 | 模板的简短描述 |
| 状态 | 绿色「已启用」标签 或 橙色「未启用」标签 |
| 操作 | 停用/启用、编辑、删除 |

#### 启用/停用模板

1. 在操作列中找到「启用」或「停用」文字按钮
2. 鼠标悬浮可看到操作提示
3. 点击后弹出确认弹窗
4. 确认后完成操作

#### 分页操作

- 与配置列表相同，支持分页浏览和切换每页条数

---

## 注意事项

1. **API Key 和 Base URL**：这两个敏感字段已从表格列表中移除，仅在编辑弹窗中可查看和修改，更加安全
2. **启用配置**：同一时间只能有一个配置处于「活跃」状态，启用新配置会自动停用当前活跃配置
3. **停用活跃配置**：如果停用了当前唯一的活跃配置，系统将没有可用的 AI 模型配置，相关 AI 功能可能无法正常使用
4. **表格滚动**：在较窄的屏幕上（如 1366px 宽度），表格会出现横向滚动条，可左右滑动查看所有列
5. **权限要求**：需要管理员权限才能访问和操作 AI 大模型配置页面
