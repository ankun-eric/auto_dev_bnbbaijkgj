## 功能简介

本次更新为 bini-health 健康管理系统新增了**打卡送积分**功能，将三类健康打卡行为（健康习惯打卡、用药提醒打卡、健康计划任务打卡）统一纳入积分体系。管理员可在后台配置打卡积分规则，用户打卡时自动发放积分并给予实时反馈。

### 核心功能

- **管理后台**：积分规则页面新增"打卡规则"Tab，支持配置启用开关、每次打卡积分值、每日积分上限
- **用户端**：打卡成功后自动获得积分并弹出Toast提示，打卡页面显示今日积分进度条
- **积分明细**：积分流水中清晰标注打卡来源类型

## 使用说明

### 一、管理员配置打卡积分规则

1. 登录管理后台：[管理后台入口](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单中找到 **积分管理 → 积分规则**
3. 页面顶部有两个 Tab 页签：**"通用规则"** 和 **"打卡规则"**
4. 点击 **"打卡规则"** Tab，进入打卡积分配置
5. 配置项说明：
   - **启用打卡送积分**：总开关，关闭后所有打卡均不送积分
   - **每次打卡奖励积分**：用户每次完成打卡可获得的积分数（默认 2 积分）
   - **每日积分上限**：每天通过打卡最多可获得的积分总量（默认 50 积分）
6. 配置完成后，点击页面右上角 **"保存规则"** 按钮
7. 保存成功后会弹出提示："积分规则保存成功"
8. 规则保存后**立即生效**，用户下次打卡即按新规则发放积分

> 💡 提示：总开关关闭时，下方的积分值和上限输入框会自动置灰，不可编辑。

### 二、用户端打卡获得积分（H5 / APP / 小程序）

#### 步骤一：进入打卡页面

- **H5端**：打开 [H5主页](https://newbb.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)，进入"健康计划"模块
- **APP端**：打开 APP，进入"健康计划"页面
- **小程序端**：打开微信小程序，进入"健康计划"页面

#### 步骤二：查看今日打卡积分进度

在打卡页面顶部，您会看到一个**积分进度条**：

- 显示格式：`今日打卡积分：X / Y`（X=已获积分，Y=每日上限）
- 进度条会随打卡积分增长而填充
- 达到上限后，进度条变为绿色，显示 **"今日打卡积分已满 ✓"**

> 注意：只有管理员开启了"打卡送积分"功能后，积分进度条才会显示。

#### 步骤三：完成打卡

支持以下三种打卡行为获得积分：

| 打卡类型 | 操作方式 | 积分来源标注 |
|----------|----------|-------------|
| 健康习惯打卡 | 在"健康打卡"页面，点击打卡按钮完成每日习惯打卡 | 健康习惯打卡 |
| 用药提醒打卡 | 在"用药提醒"页面，点击确认今日已用药 | 用药提醒打卡 |
| 健康计划任务打卡 | 在"健康计划"页面，点击完成计划中的具体任务 | 健康计划任务打卡 |

#### 步骤四：查看积分反馈

打卡成功后，会自动弹出 Toast 提示：

- **正常获得积分**：`"打卡成功，获得 X 积分！"`
- **到达每日上限**：`"打卡成功！今日打卡积分已达上限"`
- **积分功能关闭**：仅显示 `"打卡成功"`

### 三、查看积分明细

1. 进入"我的" → "我的积分" → "积分明细"
2. 在积分流水列表中，可以看到每笔打卡积分记录
3. 打卡积分记录会标注具体的打卡类型（如"健康习惯打卡"、"用药提醒打卡"等）
4. 积分变动值显示为正数（如 +2）

## 注意事项

1. **管理员权限**：只有管理员账号才能在后台配置打卡积分规则，普通用户无法访问管理后台
2. **积分上限**：每天通过打卡获得的积分有上限（管理员可配置），达到上限后继续打卡不再获得积分，但打卡操作本身不受影响
3. **跨天重置**：每日积分上限在每天 00:00 自动重置，不受前一天影响
4. **即时生效**：管理员修改积分规则后立即生效，用户下次打卡即按新规则发放
5. **功能关闭**：如管理员关闭"打卡送积分"开关，所有打卡均不再发放积分，打卡页面也不显示积分进度条
6. **积分通用**：通过打卡获得的积分与其他途径（如签到）获得的积分统一管理，可在积分商城兑换商品
