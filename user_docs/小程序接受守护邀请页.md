## 功能简介

本次为小程序新增/升级了「接受守护邀请」页（路径 `pages/family-auth/index`），实现老人在微信扫码 / H5 唤起 / 群卡片打开后，在小程序内**闭环完成**守护邀请的接受、签署一次性授权，并在成功后引导通知邀请方关注公众号。

页面在原有简易接受/拒绝交互上，按 PRD 全面补齐：

- 顶部品牌条 + 客服入口
- 邀请人卡片（头像 / 昵称 / 关系标签 / 邀请到期时间）
- 关系说明卡片（接受后将发生什么 + 当前已被多少人守护 / 上限提示）
- 健康档案合并预览（逐字段「将合并 / 保留原值」可切换）
- 一次性授权协议（未勾选时「接受」按钮置灰）
- 拒绝二次确认
- 接受成功后强制展示「关注公众号」引导弹层
- 完整的失效态文案矩阵（已过期 / 已被他人接受 / 已取消 / 自邀 / 已达守护者上限）
- 未登录引导（登录后自动回到本页）
- 网络异常时的「重试」按钮

后端 `GET /api/family/invitation/{code}` 增加了：邀请人头像、邀请人手机号、成员 id、关系类型、当前用户已被守护人数、最大上限、`is_self_invite`、`invalid_reason`、`merge_preview`。`POST /api/family/invitation/{code}/accept` 新增可选 `merge_fields` 列表参数，允许前端控制合并字段范围。

## 使用说明

### 受邀者（老人 / 家人）

1. **打开邀请页**：通过以下任一方式进入小程序 `pages/family-auth/index`：
   - 用微信扫描子女分享的小程序码
   - 在 H5 邀请页 `/family-auth?code=xxx` 点击「在小程序内接受」
   - 在微信群里收到的小程序卡片直接点开
2. **登录提示**：若尚未在小程序登录，会先弹出「需要登录」提示，点击「去登录」按手机号登录，登录成功后会自动回到本页继续流程
3. **查看邀请人卡片**：页面顶部展示邀请人头像、昵称、关系（如「您的儿子」）、邀请到期时间，请确认是您认识的家人
4. **阅读关系说明**：卡片中明确告知接受后子女将获得哪些查看 / 推送权限，以及您随时可在「我的-家庭」中解除守护；同时显示您当前已被守护的人数（最多 3 位）
5. **查看健康档案合并预览**：如有可合并字段（昵称 / 生日 / 身高 / 体重 / 既往病史等），逐条展示「您现有的值 → 邀请方填写的值」，每行右侧可切换「将合并 / 保留原值」（默认全选「将合并」）
6. **勾选授权协议**：勾选「我已阅读《家庭健康守护授权协议》…」后，「接受」按钮才可点击；可点击协议链接查看完整条款
7. **接受或拒绝**：
   - 点「接受」：成功后页面顶部出现绿色横幅「已成功成为「XX」的被守护者」，并弹出公众号关注引导弹层
   - 点「拒绝」：会弹出二次确认「确定拒绝吗？拒绝后该邀请将作废」，确认后将提示「您已拒绝此守护邀请」
8. **关注公众号引导（接受成功后必看）**：弹层会指引「让 XX 关注公众号」（即让邀请方关注），关注后子女才能在公众号收到您的体检异常推送；您也可以截图发给 TA。点「我已通知 XX 关注」关闭弹层
9. **返回首页 / 查看我的家人**：成功页底部可选择返回首页或跳转「我的家人」列表查看新的守护关系

### 失效态处理

| 进入页面后看到的提示 | 含义 | 可执行操作 |
|---|---|---|
| 邀请已过期 | 邀请超过 24 小时未被接受 | 点「联系邀请人」会自动复制邀请人手机号 |
| 邀请已被接受 | 该邀请已被其他用户接受 | 点「返回首页」 |
| 邀请已取消 | 邀请人主动撤销了邀请 | 点「返回首页」 |
| 无法接受 / 不能接受自己发起的邀请 | 您是邀请人本人，不可自接受 | 点「返回首页」 |
| 已达守护者上限 | 您已被 3 位家人守护 | 点「前往『我的家庭』」先解除一位 |

## 注意事项

1. **登录前提**：接受守护邀请属于敏感操作，必须先登录小程序。未登录时会引导前往登录页，登录完成后再回到本页继续
2. **24 小时有效期**：邀请生成后默认 24 小时内有效，超时后受邀者将看到「邀请已过期」，需邀请人重新生成
3. **守护者上限 3 位**：单个用户最多被 3 位家人守护，超过后必须在「我的家庭」先解除一位
4. **合并是单向、可选**：仅会在您**当前为空**的字段上回填邀请方填写的值；您已经填过的字段会被保留，并显示「→」之后的值仅作为对比展示
5. **关注公众号是必要补充**：仅完成小程序内的接受不会自动推送公众号消息，必须让邀请方关注公众号后才能收到体检异常推送
6. **拒绝不可恢复**：拒绝后该邀请被作废，若想再守护需邀请方重新生成邀请
7. **网络异常自动重试**：网络抖动时页面会出现「重试」按钮，无需手动刷新

### 体验步骤

1. **下载压缩包**：点击上方链接，将 zip 压缩包下载到本地电脑
2. **解压文件**：将下载的 zip 文件解压到任意目录（记住解压后的文件夹路径）
3. **下载微信开发者工具**：如尚未安装，请前往 [微信开发者工具官方下载页面](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装
4. **打开微信开发者工具**：启动开发者工具，使用微信扫码登录
5. **导入项目**：
   - 点击开发者工具首页的「导入项目」（或「+」号）
   - 在「目录」栏点击浏览，选择第 2 步解压后的 `miniprogram` 子目录
   - 「AppID」栏可填入项目的 AppID，或选择「测试号」进行体验
   - 点击「导入」按钮
6. **预览体验**：导入成功后，开发者工具会自动编译并在模拟器中展示小程序界面
7. **快速进入本页**：在「编译模式」中新增自定义编译模式，启动页面填 `pages/family-auth/index`，启动参数填 `code=<真实邀请码>`；保存后点击「编译」即可直达接受邀请页
8. **真机预览**（推荐）：点击右上角「预览」，用微信扫码即可在手机上看到完整效果
