## 功能简介

本次更新修复了管理后台登录后出现「Application error: a client-side exception has occurred」错误的 Bug，并同步优化了管理后台所有页面的数据加载逻辑。

### 修复内容

1. **Dashboard 数据概览页崩溃修复**：修复了登录后跳转到数据概览页面时，因后端返回数据格式与前端不匹配导致的客户端渲染崩溃问题。
2. **Dashboard 数据增强**：后端 API 现已返回完整的仪表盘数据，包括用户增长趋势、订单量趋势、最近订单列表和 AI 调用次数。
3. **管理后台全页面数据对接优化**：修复了用户管理、订单管理、服务管理、内容管理、积分管理、专家管理、AI配置等所有管理页面的数据加载问题。

## 使用说明

### 步骤一：登录管理后台

1. 打开管理后台链接：[https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 输入管理员手机号和密码
3. 点击「登 录」按钮

### 步骤二：查看数据概览

登录成功后，系统自动跳转到「数据概览」页面，您可以看到：

- **统计卡片**：总用户数、今日订单、AI调用次数、总收入
- **用户增长趋势图**：展示最近7天每日新增用户数量的柱状图
- **订单量趋势图**：展示最近7天每日订单数量的柱状图
- **最近订单列表**：展示最近5笔订单的订单号、用户、服务、金额、状态和时间

### 步骤三：使用各管理功能

通过左侧导航菜单，您可以访问以下管理模块：

- **用户管理**：查看用户列表、搜索用户、启用/禁用用户
- **商家管理**：门店管理、商家账号管理
- **AI模型配置**：配置和测试 AI 服务
- **服务管理**：管理服务分类和服务项目
- **订单管理**：查看和处理订单
- **内容管理**：管理文章和视频内容
- **专家管理**：管理健康专家信息
- **积分体系**：配置积分规则、管理积分商城、设置会员等级
- **客服工作台**：处理客服咨询
- **系统管理**：系统设置、短信管理

## 注意事项

1. **登录账号**：需使用管理员账号（role 为 admin）登录，普通用户账号无法登录管理后台
2. **浏览器兼容性**：建议使用 Chrome、Edge、Firefox 等现代浏览器访问
3. **数据展示**：数据概览页面的趋势图和统计数据为实时查询，首次登录如果数据库中尚无数据，将显示系统预设的示例数据
4. **操作权限**：所有管理操作需要管理员权限验证（JWT Token），如 Token 过期将自动跳转到登录页面
