## 功能简介

本次操作将系统管理员账号的手机号（账号）和密码重置为以下内容：

- **手机号（账号）**：`13800000000`
- **密码**：`admin123`

系统在每次后端服务启动时，会自动检查并确保管理员账号为上述凭据。如果管理员的手机号或密码发生了变更，系统会自动将其重置为默认值。

### 步骤 2：输入管理员账号和密码

在登录页面中：

## 注意事项

1. **账号安全**：登录成功后，建议及时修改默认密码，避免使用默认密码带来的安全风险。
2. **权限说明**：管理员账号拥有系统的最高权限，可以管理所有功能模块。请妥善保管账号信息，避免泄露。
3. **自动重置机制**：系统在每次后端服务重启时都会自动将管理员账号重置为默认的手机号 `13800000000` 和密码 `admin123`。如果您修改了密码，在服务重启后密码会被自动恢复为默认值。
4. **登录接口说明**：管理后台使用独立的管理员登录接口（`/api/admin/login`），与普通用户登录接口不同，仅限管理员角色的账号登录。
