## 功能简介

本次 Bug 修复（v1.2）针对 v1.2 上线后发现的 4 类问题进行了全面修复与功能补齐：

### 1) 后台两个核心页面卡片化改造

把「紧急呼叫触发源管理」与「守护关系管理」两个页面从原 antd Table 列表，改造为统一的**响应式卡片网格**视觉规范：

- **顶部 120px 天蓝渐变 Hero 区**：含标题、副标题，以及 4 个统计数字卡片
- **响应式卡片网格**：屏幕越宽，每行展示卡片越多（≥1600px 一行 4 张，≥1200px 一行 3 张，<1200px 一行 2 张）
- **主色统一为 #1890FF**，圆角 20px，柔和阴影 + hover 上浮 2px 反馈
- **角色徽章天蓝同色系**：主守护人天蓝实心，普通守护人浅底描边

### 2) 中文乱码修复

后台两个核心接口（紧急呼叫触发源、守护关系管理）以及新增的详情接口，统一在响应头显式声明 `Content-Type: application/json; charset=utf-8`，并禁用 ensure_ascii，从根本上解决中文字段被部分浏览器/反代以 Latin-1 误解码的问题。

### 3) 守护关系管理字段补齐 + 筛选器

守护关系列表新增 5 个核心列：
- **角色**（主守护人 / 普通守护人）
- **优先级**（串行外呼顺位）
- **会员等级**（普通 / 健康 / 尊享）
- **本月紧急 AI 呼叫额度剩余**（含进度条 + 颜色预警）
- **本月 AI 外呼额度剩余**（含进度条 + 颜色预警）

新增 3 个筛选器：
- 仅看主守护人
- 仅看普通守护人
- 仅看付费会员

### 4) 守护关系只读详情抽屉（6 分区）

点击守护关系卡片右下角"详情"按钮，打开宽 560px 的只读抽屉，分 6 个分区完整展示：

1. **基本信息** — 被守护人、关系、性别年龄、我相对的角色、优先级、守护开始时间
2. **会员与配额** — 套餐、会员等级、套餐到期、本月双配额（紧急/AI 外呼）、守护他人上限
3. **代付开关状态** — 主守护人是否开启代付被守护人 AI 外呼额度
4. **关联守护人列表** — 该被守护人身上所有守护人（含主/普通徽章、关系、手机号脱敏）
5. **最近一次紧急 AI 呼叫** — 触发时间、触发源、扣谁的额度
6. **最近一次 AI 外呼** — 提醒名称、设置者、扣额度对象（含代付标识）

### 5) 紧急呼叫触发源 CRUD 权限分层

- **内置 4 种**（健康数据异常 / 烟雾报警器 / 水位报警器 / 紧急呼叫器）：仅可启停，编辑/删除操作菜单灰色禁用，后端对内置项 PUT/DELETE 返回 403
- **自定义触发源**：可完整 CRUD，删除前二次确认弹窗

### 6) 客户端入口接入闭环

- **首页右上角 MoreMenu 新增「会员中心」入口**（替换"我的硬件"语义，因健康档案已有硬件入口），最终菜单项：扫一扫 / 字体大小 / 立即分享 / **会员中心**
- **健康档案"我守护的人"双卡片整块可点击**，跳转至 `/health-profile/i-guard`（v1.2 直筒列表），带 hover 阴影加深反馈
- **「我守护的人」本人行的「我的 AI 外呼额度」按钮**跳转至 `/member-center#quota`，自动锚定到"本月配额"板块

## 使用说明

### A. 管理员 — 使用「紧急呼叫触发源管理」（卡片化新版）

1. 登录管理后台 → 左侧菜单展开「健康档案管理」→ 点击「紧急呼叫触发源管理」
2. 顶部 Hero 区即时显示当前 4 项统计：总数 / 内置 / 自定义 / 启用中
3. 每张卡片含图标 + 名称 + 启停开关（顶部）+ 描述（中部）+ 内置/自定义徽章 + 适用设备 + 启用/停用徽章（底部）
4. 右下角 ⋯ 菜单：
   - 内置触发源 → 编辑/删除菜单项灰色禁用，鼠标悬停提示"内置触发源仅可启停，不可编辑/删除"
   - 自定义触发源 → 编辑/删除菜单项可用，删除前会弹窗二次确认
5. 启停 Switch 直接点击即可生效，无需保存
6. 点击右上角「新增触发源」按钮，弹窗输入编码、名称、描述、适用设备、排序、启用状态即可

### B. 管理员 — 使用「守护关系管理」（卡片化新版 + 详情抽屉）

1. 登录管理后台 → 左侧菜单「健康档案管理」→ 点击「守护关系管理」
2. 顶部 Hero 区即时显示 4 项统计：总关系 / 主守护人 / 普通守护人 / 付费会员
3. 筛选栏支持：
   - **关键字搜索**：在搜索框输入昵称或手机号
   - **角色筛选**：全部 / 仅看主守护人 / 仅看普通守护人
   - **付费筛选**：全部 / 仅看付费 / 仅看免费
4. 每张卡片采用**上下分区式**：
   - 顶部：头像 + 被守护人姓名 + 角色徽章（主守护人天蓝实心 / 普通守护人浅底描边）
   - 中部：会员徽章 + 优先级（串行外呼顺位）+ 守护开始时间
   - 底部：紧急 AI 呼叫剩余进度条 + AI 外呼额度剩余进度条
     - 剩余 ≤2 → 黄色预警
     - 剩余 = 0 → 红色
     - 其他 → 主色天蓝
5. 点击卡片右下角「详情」按钮 → 打开宽 560px 只读详情抽屉（6 分区，无任何编辑/保存按钮）
6. 点击「解除」按钮可强制解除该守护关系（含二次确认）

### C. 终端用户 — 通过 H5 首页 MoreMenu 进入会员中心

1. 打开 H5 客户端首页 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/)
2. 登录后，点击右上角「⋯」三道杠按钮
3. 弹出菜单包含：扫一扫 / 字体大小 / 立即分享 / **会员中心**
4. 点击「会员中心」即可跳转至会员中心页（自动锚定到本月配额板块）

### D. 终端用户 — 通过健康档案进入「我守护的人」

1. 打开 H5 健康档案首页 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)
2. 在页面中部找到「我守护的人」双卡片区（左卡片）
3. 点击整块即可跳转至 `/health-profile/i-guard` 列表页（v1.2 直筒列表）
4. 鼠标悬停时整块卡片有轻微阴影加深反馈

### E. 终端用户 — 从我守护的人列表跳到会员中心

1. 进入「我守护的人」列表页
2. 在本人那一行（顶部）操作位有 3 个按钮：编辑档案 / 邀请记录 / **我的 AI 外呼额度**
3. 点击「我的 AI 外呼额度」即可跳转至会员中心的本月配额板块

## 注意事项

1. **内置触发源仅可启停**：4 种内置触发源（健康数据异常 / 烟雾报警器 / 水位报警器 / 紧急呼叫器）不可编辑名称、描述等业务字段，也不可删除；只能启停。如需禁用，使用卡片右上角 Switch 即可。
2. **删除自定义触发源不可恢复**：删除前会弹窗二次确认，请仔细确认后再删除。
3. **守护关系详情抽屉为只读**：抽屉内 6 分区全部以展示态呈现，无任何编辑/保存按钮；如需调整守护关系（如设主、解除等），请前往 H5 客户端「我守护的人 → 守护管理」抽屉操作。
4. **进度条颜色预警**：紧急 AI 呼叫剩余 ≤2 时进度条变黄；剩余 = 0 时变红；其他保持系统天蓝。请运营/客服重点关注红黄色的卡片。
5. **MoreMenu 不再有"我的硬件"**：由于健康档案首页已有「我的设备」入口，原 MoreMenu 中"我的硬件"语义已删除，新增"会员中心"作为对应入口。
6. **筛选器组合生效**：关键字 / 角色 / 付费 三种筛选可任意组合使用，列表会实时更新。

|------|------|
