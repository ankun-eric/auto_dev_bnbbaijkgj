## 一、功能介绍

本次升级让「心率」模块在视觉与交互上和「血压」完全一致：

- **小卡片**：右上角带彩色状态胶囊（偏慢 / 正常 / 偏快），底部显示「时间 · 来源」行；无数据时自动隐藏胶囊与时间行，数值显示 `—`。
- **详情页**：浅蓝灰底 + 蓝色系主题，顶部大号居中数值 + 居中彩色胶囊 + 底卡按档位变色；提供两个 AI 解读按钮（解读本次 / 解读趋势）；趋势图支持日 / 周切换、点击数据点弹窗、带正常参考线（60 / 100 bpm）；历史记录每条可编辑、可删除并带档位配色。
- **判定规则**：心率正常区间 60～100 bpm。正常为蓝色，偏慢 / 偏快为橙色。

同时，小程序端的**血压、血糖**小卡片也已同步胶囊化、补齐「时间 · 来源」行，全端样式统一。

## 三、操作步骤

### 1. 查看心率小卡片
1. 打开「健康档案」首页。
2. 找到「心率」卡片：
   - 有数据时：右上角显示彩色胶囊（偏慢/正常/偏快），底部显示如「今天 08:30 · 手动录入」。
   - 无数据时：数值显示 `—`，不显示胶囊与时间行。

### 2. 进入心率详情页
1. 点击心率小卡片，进入详情页。
2. 顶部主卡：查看大号心率数值与档位胶囊，底卡颜色随档位变化。

### 3. AI 解读
- 点击主卡下方的 **「AI 解读本次心率」**：获取对最近一次心率的解读。
- 点击趋势图下方的 **「AI 解读趋势」**：获取对一段时间心率趋势的解读。

### 4. 查看趋势图
1. 在趋势图区域切换 **「日 / 周」**。
2. 点击图中任一数据点，弹出该次测量的详细信息（数值、时间、档位）。
3. 图中两条参考线分别为 60 与 100 bpm，用于直观判断是否在正常区间。

### 5. 手工录入 / 绑定设备
- 点击 **「手工录入」**，填写心率值与活动状态后保存。
- 点击 **「绑定设备」** 可关联测量设备（如已支持）。

### 6. 编辑 / 删除历史记录
1. 在「历史记录」区域，点击某条记录右侧的 **「⋯」**。
2. 在弹出的操作面板中选择 **「修改」** 或 **「删除」**。
3. 修改：调整数值后点「保存」。
4. 删除：在二次确认弹窗中点「确认删除」。

## 四、注意事项

1. **正常区间**：心率正常范围为 60～100 bpm；低于 60 为「偏慢」，高于 100 为「偏快」，均以橙色胶囊提示。
2. **无数据状态**：未录入任何心率数据时，小卡片仅显示 `—`，不显示胶囊与时间行，属正常现象。
3. **AI 解读**：解读结果为健康参考信息，不能替代专业医疗诊断；如有异常请及时就医。
4. **两端一致**：H5 与小程序为同批上线，若发现样式不一致，请刷新页面或重新进入小程序。
5. **数据时间·来源**：显示的是该指标的测量时间与录入来源（如手动录入 / 设备同步）。

## 五、常见问题（FAQ）

- **Q：小程序里的心率详情页和网页端为什么长得一样？**
  A：小程序详情页通过内嵌网页（web-view）加载 H5 精装详情页，因此两端视觉与交互完全一致。

- **Q：胶囊颜色代表什么？**
  A：蓝色 = 正常（60～100），橙色 = 偏慢（<60）或偏快（>100）。

- **Q：编辑/删除后趋势图没更新？**
  A：操作完成后列表与趋势会自动刷新；若未刷新，可下拉或重新进入页面。
