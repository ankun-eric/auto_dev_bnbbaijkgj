## 功能简介

本次对「健康档案 - 居家安全设备」页面进行了体验优化，让您进入页面后能一眼看清当前家庭成员名下各类安全设备的数量分布，并把「添加设备」入口放到更顺手的位置。

主要变化有两点：

1. **新增「设备统计卡片」**：在顶部家庭成员切换 Tab 的下方，新增一块统计卡片，集中展示当前成员名下的设备数量。
2. **调整「+ 添加设备」按钮位置**：从原来的页面底部固定位置，挪到了统计卡片的下方，操作更顺手。

## 使用说明

### 1. 进入页面

打开 [居家安全设备页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/home-safety)，或从健康档案中点击进入「居家安全设备」。

### 2. 查看设备统计卡片

进入页面后，顶部家庭成员切换 Tab 的正下方，即可看到「设备统计卡片」，自上而下包含：

- **已绑定总数（大数字）**：当前选中成员名下**所有类型**已绑定设备的总数量（包含紧急呼叫器、烟雾报警器、水浸报警器，以及门磁、燃气报警器等其它类型）。
- **三个分类小项**（并列展示）：
  - 紧急呼叫器：当前成员名下「紧急呼叫器」数量
  - 烟雾报警器：当前成员名下「烟雾报警器」数量
  - 水浸报警器：当前成员名下「水浸报警器」数量

> 说明：由于「已绑定总数」统计的是所有类型设备之和，当存在其它类型（如门磁、燃气报警器）的设备时，总数可能会大于三个分类项之和，这是正常现象。

### 3. 切换家庭成员

点击顶部的家庭成员 Tab（本人 / 父亲 / 母亲 / 配偶 …），统计卡片中的数字会自动切换，只反映**当前选中成员**名下的设备。例如切到「父亲」，卡片就只统计父亲名下的设备。

### 4. 无设备时的提示

当所选成员名下一台设备都没有时：

- 统计卡片照常显示，「已绑定总数」与三个分类项的数字均为 **0**；
- 卡片下方会出现一句引导文案：**「暂无设备，快去添加吧」**。

### 5. 添加设备

点击统计卡片**下方**的「+ 添加设备」按钮，即可打开绑定弹窗，按提示填写设备类型、归属成员、设备备注、网关 ID、设备 SN、紧急联系手机后完成绑定。该按钮的功能与原来一致，仅位置发生了变化。

### 6. 统计实时刷新

成功**添加**或**解绑**一台设备后，统计卡片中的数字会**自动实时刷新**，无需手动刷新页面。

## 注意事项

- 设备统计卡片仅统计**当前选中成员**名下的设备，切换成员后数据随之变化。
- 「已绑定总数」按所有类型口径累加；三个分类小项只统计各自对应类型的数量。
- 紧急呼叫器、烟雾报警器、水浸报警器是页面单独列出的三类设备；门磁、燃气报警器等其它类型不单独列项展示，但会计入「已绑定总数」。
- 本次优化仅涉及健康档案 H5 端页面与后端统计接口，未改动微信小程序、安卓 / 苹果 App、桌面端等客户端，无需重新安装或导入任何客户端。
