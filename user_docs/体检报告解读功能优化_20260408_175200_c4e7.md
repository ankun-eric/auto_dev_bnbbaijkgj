## 功能简介

本次更新为比尼健康的**体检报告解读功能**带来全面升级，从原先的纯文本解读升级为专业的健康解读平台，主要新增/优化内容包括：

- **AI 综合健康评分**：AI 自动为每份报告生成 0-100 分的综合健康评分，以圆环仪表盘形式展示，直观呈现整体健康状态
- **五级风险等级系统**：每个指标按「优秀 / 正常 / 轻度异常 / 中度异常 / 严重异常」五级分类，配合绿色-蓝色-黄色-橙色-红色的色彩标识
- **详细指标建议**：每项异常指标提供 200-300 字的专业建议，涵盖指标含义科普、可能原因、饮食建议、运动建议、生活习惯、复查建议、就医指引共七大模块
- **报告对比分析**：支持任意两份报告的横向对比，展示评分变化趋势、指标改善/恶化情况和 AI 建议
- **优化加载体验**：分析过程中展示分步加载动画和进度提示文字，提升等待体验

## 使用说明

### 一、上传体检报告

1. 打开 H5 页面（或对应的小程序/App）
2. 进入「体检报告」模块
3. 点击「上传报告」按钮
4. 选择体检报告图片（支持多图上传）
5. 确认上传后等待 OCR 识别完成

### 二、查看 AI 智能解读

报告上传并分析后，进入报告详情页查看 AI 解读结果：

1. **健康评分仪表盘**（页面顶部）
   - 圆环仪表盘展示 0-100 分的综合健康评分
   - 评分下方显示 AI 短评（如「整体良好，注意血脂」）
   - 评分颜色根据分数段自动变化（绿色 ≥ 80、蓝色 ≥ 60、黄色 ≥ 40、橙色 ≥ 20、红色 < 20）

2. **汇总卡片**（仪表盘下方）
   - 显示检查总项数、异常项数、优秀项数

3. **分类指标详情**（页面主体）
   - 按分类（如血常规、肝功能、肾功能等）分组展示
   - 每个分类可展开/折叠
   - 每项指标显示五级风险标签：
     - 🟢 优秀 — 指标表现优异
     - 🔵 正常 — 指标在正常范围
     - 🟡 轻度异常 — 需关注
     - 🟠 中度异常 — 建议就医复查
     - 🔴 严重异常 — 需尽快就医
   - 异常指标默认展开，正常指标默认折叠

4. **详细建议**（点击指标展开）
   - 每项指标可展开查看 AI 生成的详细建议，内容包含：
     - 📋 指标含义科普
     - 🔍 可能原因
     - 🥗 饮食建议
     - 🏃 运动建议
     - 🌙 生活习惯建议
     - 🔄 复查建议
     - 🏥 就医指引

### 三、报告对比分析

1. 在报告列表页，点击「对比模式」按钮
2. 选择两份需要对比的报告（勾选复选框）
3. 点击「开始对比分析」按钮
4. 查看对比结果：
   - **AI 综合摘要**：总结两次体检的变化趋势
   - **评分对比**：两份报告的健康评分对比和分差
   - **指标变化列表**：逐项展示变好（↑）、变差（↓）、不变（→）的状态
   - **筛选功能**：可按「全部 / 变好 / 变差 / 不变」过滤查看
   - **AI 建议**：针对每项变化的个性化建议

### 四、报告列表

- 每份报告卡片上显示小型健康评分圆环，方便快速了解各报告的评分
- 支持长按（App）或勾选模式进入对比选择

## 注意事项

1. **AI 解读仅供参考**：所有 AI 生成的解读结论和建议仅供健康参考，不能替代专业医生的诊断和建议
2. **报告质量要求**：上传的体检报告图片需清晰可读，模糊或残缺的图片可能导致 OCR 识别不准确
3. **分析耗时**：首次分析报告时，AI 需要一定的处理时间（通常 10-30 秒），请耐心等待加载动画完成
4. **网络要求**：功能需要网络连接，请确保设备已联网
5. **iOS 安装说明**：iOS IPA 为未签名版本，需要使用 AltStore、Sideloadly 等第三方工具安装
6. **对比分析**：需要至少两份已分析的报告才能使用对比功能
