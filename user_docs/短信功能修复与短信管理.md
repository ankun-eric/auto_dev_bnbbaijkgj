## 功能简介

本次更新包含以下三项内容：

1. **修复短信验证码发送 BUG**：修复了 H5 端点击发送验证码后，界面显示"已发送"但用户手机实际未收到短信的问题。根因为后端腾讯云短信配置参数未正确注入到 Docker 容器环境变量中，导致短信服务被静默跳过。
2. **改造 H5 前端短信发送提示逻辑**：H5 端现在会根据后端返回的实际发送结果来展示提示——发送成功提示"验证码已发送，请注意查收短信"，发送失败提示具体的错误原因。
3. **新增管理后台"短信管理"模块**：在管理后台的"系统管理"菜单下新增"短信管理"子菜单，包含短信配置、发送记录查看和测试发送三个功能。

## 使用说明

### 一、H5 端短信验证码登录（已修复）

1. 打开 H5 端页面 [https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/)
2. 在登录页面输入 11 位手机号
3. 点击「获取验证码」按钮
4. 如果短信发送成功，页面会提示"验证码已发送，请注意查收短信"，并开始 60 秒倒计时
5. 如果短信发送失败，页面会提示具体的失败原因（如"短信发送失败，请稍后重试"或"发送过于频繁，请60秒后重试"）
6. 在手机上收到验证码后，输入 6 位验证码并点击「登录/注册」按钮完成登录

### 二、管理后台 - 短信管理模块

#### 进入方式

1. 登录管理后台 [https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/admin/)
2. 在左侧菜单栏找到「系统管理」，点击展开
3. 点击「短信管理」进入短信管理页面

#### 功能一：短信配置

1. 进入短信管理页面后，默认显示「短信配置」标签页
2. 填写腾讯云短信服务的配置参数：
   - **SecretID**：腾讯云 API 密钥 ID
   - **SecretKey**：腾讯云 API 密钥 Key（密文显示，输入后无法查看明文，只能重新输入覆盖）
   - **SDK AppID**：短信应用 SDK AppID
   - **短信签名**：已审核通过的短信签名内容
   - **模板ID**：已审核通过的短信模板 ID
   - **App Key**（可选）：短信应用 App Key
3. 点击「保存配置」按钮保存

> 注意：SecretKey 等敏感参数一旦保存后，页面只会显示"已设置"状态，无法回显明文。如需修改，直接输入新值并保存即可覆盖原值。

#### 功能二：发送记录

1. 点击「发送记录」标签页
2. 页面以表格形式展示所有短信发送记录，包含：
   - **发送时间**：短信发送的具体时间
   - **手机号**：接收短信的手机号（中间 4 位脱敏显示，如 138\*\*\*\*1234）
   - **验证码**：发送的验证码内容
   - **发送状态**：绿色标签表示成功，红色标签表示失败
   - **失败原因**：发送失败时显示具体原因
   - **测试发送**：蓝色标签标记通过测试功能发送的短信
3. 支持按手机号搜索和按发送状态（成功/失败）筛选
4. 支持分页浏览

#### 功能三：测试发送

1. 点击「测试发送」标签页
2. 输入接收测试短信的手机号（11位手机号格式）
3. 模板ID 为可选项，留空则使用配置中的默认模板 ID
4. 点击「发送测试短信」按钮
5. 发送后页面会显示发送结果：
   - 成功：绿色提示框显示"发送成功"
   - 失败：红色提示框显示失败原因
6. 测试发送的记录会同步记录到「发送记录」标签页中，并标记为"测试"

## 注意事项

1. **短信管理功能仅管理员可访问**：所有短信管理 API 和页面均需管理员权限，普通用户无法访问
2. **配置优先级**：系统优先使用管理后台中配置的短信参数，如果管理后台未配置，则使用服务器环境变量中的配置
3. **SecretKey 安全**：SecretKey 在数据库中加密存储，前端页面不返回明文，只能重新输入覆盖
4. **发送频率限制**：同一手机号 60 秒内只能发送一次验证码，频繁请求会收到"发送过于频繁"的提示
5. **短信费用**：每次发送（包括测试发送）都会产生腾讯云短信费用，请注意控制测试频率
6. **首次使用**：部署后需要先在管理后台的「短信配置」中填写正确的腾讯云短信参数，短信功能才能正常工作
