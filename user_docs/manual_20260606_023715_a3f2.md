## 功能简介

本次更新（v3.0）包含三大功能模块的改造升级：

### 1. 健康档案共管数据合并 (F1-F4)

守护人为被守护人录入的健康数据在共管关系建立后自动同步到被守护人本人名下，被守护人打开自己 App 即可看到完整历史数据。

- **共管数据自动同步**：邀请接受后，守护人名下的用药记录、检查报告、健康指标等数据全量同步
- **权限分级管理**：主守护人可查看编辑完整档案，普通守护人只能查看概览视图
- **解绑极简视图**：解除关系后仍可看到 Hero 卡片和健康看板（只读）

### 2. 设备场景分类改造 (F5-F11)

设备管理按"安全守护、健康守护、日夜守护、其他"四大场景分类展示，支持后台灵活配置。

- **四大场景分类**：我的设备页面按场景分组展示，每类显示绑定数量统计
- **后台灵活管理**：分类名称可编辑、排序可调、可启用/禁用
- **设备目录增强**：支持自定义图标（图片 URL）和跳转链接

### 3. 解除守护短信验证 (F12-F13)

所有解除守护入口统一增加短信验证码保护，防止误操作和恶意解除。

- **统一验证流程**：确认 → 短信验证码 → 校验 → 执行解除
- **验证码保护**：6位数字验证码，5分钟有效，1分钟限发1次
- **/family 页面删除**：功能已合并到家庭成员列表页面

## 使用说明

### 一、健康档案共管数据合并

#### 1.1 共管数据同步

**守护人操作**：
1. 进入「健康档案」→「家庭成员列表」
2. 点击"邀请共管"，选择要邀请的家人，发送邀请
3. 被守护人接受邀请后，系统自动将守护人名下属于被守护人的数据同步到被守护人本人名下
4. 同步的数据包括：用药记录、检查报告、健康指标、健康打卡、健康事件等

**被守护人操作**：
1. 收到共管邀请后，点击接受
2. 接受后，打开自己的健康档案，即可看到完整历史数据
3. 无需额外操作，数据会自动出现在个人档案中

#### 1.2 权限分级

| 角色 | 可查看内容 | 可编辑 |
|------|-----------|--------|
| 本人 | 完整健康档案 | 是 |
| 主守护人 | 被守护人的完整健康档案 | 是 |
| 普通守护人 | Hero 卡片 + 守护人卡片 + 健康看板 | 否（只读） |
| 已解除守护 | Hero 卡片 + 守护人卡片 + 健康看板 | 只读 |

#### 1.3 解除守护后的极简视图

解除守护关系后：
- 被守护人的 Tab 仍然可见（不会消失）
- 但只显示三样内容：Hero 卡片（只读）、守护人卡片（只读）、健康看板（只读）
- 其他详细数据（用药记录、检查报告、健康指标等）不再展示

### 二、设备场景分类

#### 2.1 我的设备页面

进入「我的设备」页面后，设备按四大场景分类展示：

```
我的设备
├── 安全守护（已绑 X 台）—— 如智能门锁、烟雾报警器等
├── 健康守护（已绑 X 台）—— 如血压计、血糖仪等
├── 日夜守护（已绑 X 台）—— 如智能床垫、夜灯等
└── 其他（已绑 X 台）—— 未分类设备
```

- 每个分类右上角显示"已绑 X 台"
- 禁用的分类不会展示
- 分类下无设备时显示"暂无设备"
- 点击设备卡片可跳转到详情页

#### 2.2 管理后台配置（管理员）

**设备分类管理**（管理后台 → 设备管理 → 设备分类管理）：
- 查看分类列表（名称、排序、启用状态、设备数量）
- 新增/编辑分类名称
- 调整排序（上移/下移）
- 启用或禁用某个分类
- 删除分类（仅当分类下无设备时）

**设备目录管理**（管理后台 → 设备管理 → 设备目录管理）：
- 查看设备列表（名称、所属分类、图标、跳转链接）
- 新增/编辑设备信息
- 上传自定义设备图标（PNG/JPG/SVG）
- 设置设备详情跳转链接
- 删除设备

### 三、解除守护短信验证码

#### 3.1 统一验证流程

所有"解除守护"入口现在都需要短信验证码：

1. 点击"解除守护"按钮
2. 系统弹出确认框 → 您确定要解除守护吗？
3. 确认后，系统自动发送 6 位验证码到您的手机
4. 输入收到的验证码
5. 验证通过后执行解除操作

#### 3.2 验证码规则

| 规则 | 说明 |
|------|------|
| 验证码长度 | 6 位数字 |
| 有效期 | 5 分钟，过期需重新获取 |
| 发送频率 | 同一手机号 1 分钟内限发 1 次 |
| 验证次数 | 最多尝试 3 次，超过需重新获取 |

#### 3.3 两个解除入口

| 入口 | 路径 | 说明 |
|------|------|------|
| 入口 B | 健康档案 → 家庭成员列表 → 解除守护 | 守护人主动解除 |
| 入口 C | 健康档案 → 我的守护人 → 移除 | 被守护人主动移除 |

> 注：原 /family 页面已删除，功能合并到家庭成员列表页面。

## 注意事项

1. **数据同步**：共管数据同步在邀请接受时自动触发，无需手动操作。同步过程采用事务保护，确保数据不丢失。
2. **权限安全**：解除守护需要短信验证码确认，防止误操作和恶意解除。验证码发送到操作人自己的手机号。
3. **设备分类**：设备未指定分类时自动归入"其他"分类。禁用分类后前端不展示，但数据保留不删除。
4. **数据库变更**：本次新增了 `device_scene_group` 设备场景分类表，原有的 `device_binding` 老表已废弃。
5. **/family 页面**：原 /family 页面已删除，相关功能已合并到健康档案的家庭成员列表页面。
6. **普通守护人权限**：普通守护人只能查看被守护人的概览信息（Hero卡片+守护人卡片+健康看板），不包含详细的健康数据。
7. **账号信息**：管理后台默认账号 `admin` / `admin123`。
