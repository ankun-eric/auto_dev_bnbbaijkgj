## 一、功能介绍

关怀模式是专为**老年用户**打造的简化版首页，强调"大字、大图标、一眼看懂、操作简单"。本次优化对关怀模式首页做了整体升级：

- **顶部栏与标准模式完全统一**，三种模式看起来一致，不再混乱。
- **更亲切的欢迎区**：按时段问候、卡通机器人头像、并直接显示当日用药提醒。
- **5 张大字整行卡片**重组核心功能入口，老人一眼能找到常用功能。
- **新增「今日健康数据」页**，集中查看血压、血糖、心率、血氧、睡眠。
- **新增「紧急呼叫 SOS」页**，长按一键求助、一键拨打 120/110、维护紧急联系人。
- **新增「个人信息卡」**，一张证件式名片，含二维码，可分享/保存为图片，走失或紧急情况下方便他人快速了解老人信息。

## 二、如何进入关怀模式首页

1. 打开应用（H5 网页或微信小程序）。
2. 在顶部栏右侧的**模式切换**处，选择「关怀模式」。
3. 进入后即可看到全新的关怀模式首页。

- H5 直达地址：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home`

## 三、页面结构与操作步骤

### 1. 顶部固定栏

从左到右依次为：

- **☰ 菜单**：打开侧边菜单。
- **「小康」**：品牌标题。
- **模式切换 / 🎁 礼物 / ⊕ 加号**：与标准模式完全一致。

### 2. 欢迎区（蓝绿渐变）

- **大字问候语**：随时间自动变化 —— 早上好 ☀️ / 中午好 🌤️ / 晚上好 🌙。
- **小字介绍**：「我是小康，有健康问题随时问我~」。
- **今日用药提醒**：自动显示**最新一条**用药提醒，例如：
  「🔔 今日提醒：下午 14:00 请按时服用'氨氯地平'」。
- **右侧头像**：「宾尼小康」机器人 LOGO（窄白边白圈样式）。

### 3. 核心入口区 — 5 张大字整行卡片

| 顺序 | 卡片 | 点击后 |
|---|---|---|
| 1 | **用药提醒** | 进入「健康档案 → 本人 → 用药提醒」，直接看到完整用药提醒列表 |
| 2 | **健康记录** | 进入新页面「今日健康数据」 |
| 3 | **居家安全设备** | 进入「健康档案 → 本人 → 居家安全设备」页面 |
| 4 | **紧急呼叫** | 进入「SOS 紧急求助」页 |
| 5 | **个人信息卡** | 进入证件/名片式个人信息卡 |

## 四、各新功能详细说明

### 4.1 今日健康数据

- 页面包含 5 张卡片，顺序固定：**血压 → 血糖 → 心率 → 血氧 → 睡眠**。
- 卡片样式与「健康档案 → 本人」里的数据卡完全一致。
- **点击任一卡片**（如血压），进入与健康档案中完全一致的详情/曲线页。
- H5 地址：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home/today-health`

> 说明：本页数据直接复用健康档案的数据与详情页，健康档案如有调整，本页自动跟随，无需重复维护。

### 4.2 居家安全设备（设备名已统一）

设备名称统一显示为：

1. **紧急呼叫器**
2. **烟雾报警器**
3. **水浸报警器**

两端一致：在「健康档案 → 本人 → 居家安全设备」中看到的也是这三个名称。

### 4.3 紧急呼叫 SOS

- H5 地址：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home/sos`
- 页面包含：
  - 顶部大红色 **SOS 按钮**：**长按 3 秒**触发求助，下方显示当前定位。
  - **紧急联系人**：儿子 / 女儿 / 家庭医生等（从联系人维护中读取）。
  - **一键呼叫 120 / 110**。
  - **分享位置**。
  - **紧急联系人维护**入口：可添加/编辑联系人姓名、电话，并填写**家庭住址**。

**操作步骤（维护联系人）：**
1. 点击「紧急联系人维护」。
2. 填写姓名、关系、电话，点击「添加联系人」。
3. 在「家庭住址」处填写后点击「保存家庭住址」。
4. 维护后的内容会自动同步到「个人信息卡」。

### 4.4 个人信息卡

- H5 地址：`https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-ai-home/info-card`
- 进入即一张做好的**名片式卡片**，展示：
  - 姓名、年龄、出生日期、性别
  - 既往病史、过敏史（来自健康档案）
  - 家庭住址
  - 紧急联系人姓名 + 电话
- **某项为空**时仍会显示该项，内容为「暂无 / 未填写」，提示家人补充。
- 卡片**右上角二维码**：他人扫码后打开一个网页，显示完整信息，并提供**直接拨打紧急联系人电话**的按钮。
- **底部按钮**：
  - **分享图片**：将卡片以图片形式分享（可发微信、存相册）。
  - **保存图片**：保存到本地相册。

## 五、注意事项

1. **数据来源**：个人信息卡的大部分内容来自「健康档案 → 本人」，请先在健康档案中完善基础信息（姓名、出生日期、性别、病史、过敏史等）。
2. **紧急联系人/家庭住址**：需在「SOS 页面 → 紧急联系人维护」中填写，个人信息卡只负责读取展示。
3. **SOS 定位**：首次使用需在浏览器/小程序中**允许定位权限**，否则位置可能显示为空。
4. **二维码网页**：可被任何人扫码查看（无需登录），便于紧急情况下他人快速了解信息，请勿在卡片中填写不希望公开的内容。
5. **拨打电话**：120/110 及紧急联系人拨号需在支持拨号的设备上使用。
