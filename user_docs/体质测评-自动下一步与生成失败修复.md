## 二、管理员：在后台如何配置"自动下一步"

### 2.1 入口

打开 Web 管理后台：  
👉 `https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/function-buttons`

找到任意一个**问卷类**功能按钮（如「中医体质测评」），点击「编辑」。

### 2.2 新分组：呈现配置

在编辑抽屉里，「AI 解读 Prompt 模板」上方新增了一个「**呈现配置**」分组，包含三个新参数：

| 参数 | 取值 | 说明 |
| --- | --- | --- |
| **呈现容器** | 抽屉 / 对话内插入 | 「对话内插入」表示直接在 AI 聊天气泡里渲染问卷 |
| **每页题数** | 1 ~ 999 整数 | 抽屉模式下控制每页显示几道题；为 1 时即"一题一屏" |
| **自动下一步** | 开 / 关 | 仅当 容器=抽屉 且 每页题数=1 时可以打开 |

### 2.3 联动规则（不用记，UI 会自动管控）

- 切到「对话内插入」→「每页题数」隐藏、「自动下一步」自动关闭并置灰；
- 「每页题数」从 1 改为 >1 → 弹窗确认「自动下一步将失效，是否继续？」，确认后自动关闭；
- 「每页题数」输入 0、负数、小数、>999 → 红字报错「请输入 1~999 之间的整数」；
- 问卷模板包含非单选题（多选/填空） → 自动下一步仍可开，但**只对单选题生效**，其余题型保留「下一步」按钮。

### 2.4 默认值

- **新建按钮**：自动下一步默认 = **开**
- **存量按钮**：自动下一步默认 = **关**（保守迁移，老体验保持原样）
- **中医体质测评（王琦 36 题）按钮**：迁移脚本已主动把它设为 **开**

## 五、常见问题

**Q1：之前一直没出报告的那次测评，可以补出来吗？**  
A：进入「我的 → 测评记录」可以查看历史记录；如果之前确实卡死了没有结果，请重新进入体质测评再答一次，新链路已经修好，60 秒内一定有响应。

**Q2：自动跳题太快，没看清题干怎么办？**  
A：自动跳题只在「单选题」上触发；如果误选，可以点页面左上角「← 上一题」回到上一题重新选。

**Q3：管理员手滑把"自动下一步"全员打开了，怎么撤回？**  
A：到「功能按钮」列表里把对应按钮的「自动下一步」关掉即可，**用户已经在答题的会话不会受影响**，下次进入时按最新配置渲染。

**Q4：每页题数填了 50，自动下一步还能开吗？**  
A：不能。「自动下一步」必须 「每页题数 = 1」 才允许打开，前端会自动置灰。
