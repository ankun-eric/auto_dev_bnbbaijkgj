## 功能简介

本次更新对「健康自查」流程进行了全面增强，主要包括以下改进：

### 1. Bug 修复
- **修复**：聊天页面现在能正确读取 URL `msg` 参数，症状信息可自动发送给 AI
- **修复**：健康自查不再使用临时 ID，改为调用后端创建真实的聊天会话（Session），历史消息加载正常

### 2. 家庭成员选择（全新功能）
- 点击「AI智能分析」后，先弹出「为谁咨询」选择弹窗
- 支持选择「为自己」或已有家庭成员，也可以新建家庭成员并填写健康档案
- 支持记录家庭成员的基本信息和健康档案（既往病史、过敏史等）

### 3. AI 上下文增强
- 后端自动将家庭成员健康档案注入 AI system prompt
- AI 以亲切语气回复，自然引用咨询对象的基本情况

### 4. 聊天页增强
- 顶部展示症状摘要条（AI 首次回复后自动收起）
- 第一条消息渲染为绿色症状卡片（默认收起，点击可展开）
- 输入框旁新增「切换咨询对象」小图标

## 使用说明

### 流程一：发起健康自查并选择咨询对象

**步骤 1：进入健康自查**
1. 打开 H5 应用，在首页找到「健康自查」入口，或直接访问 [健康自查页面](https://newbb.test.bangbangvip.com/autodev/3b7b999d-e51c-4c0d-8f6e-baf90cd26857/symptom)
2. 进入三步引导流程：

**步骤 2：选择不适部位**
- 点击对应的身体部位图标（如：头部、胸部、腹部等）
- 点击「下一步」

**步骤 3：描述症状**
- 从常见症状标签中勾选（支持多选）
- 可在下方文本框补充详细描述
- 点击「下一步」

**步骤 4：补充信息**
- 选择症状持续时间（如：今天刚开始 / 1-3天 / 一周内等）
- 确认症状摘要卡片无误后，点击「AI智能分析」

**步骤 5：选择咨询对象（新功能）**

屏幕底部弹出「为谁咨询」选择面板：

| 选项 | 说明 |
|------|------|
| 为自己 | 默认选项，AI 将基于您自己的健康档案进行分析 |
| 家庭成员（已有） | 从列表中选择，下方展开该成员健康档案供查看/编辑 |
| 添加家庭成员 | 填写新成员信息并创建档案 |

**选择已有成员时**：
- 选中后，下方自动展开该成员的健康档案信息
- 可直接修改性别、出生日期、身高、体重、既往病史、过敏史
- 修改后会自动保存

**新建家庭成员时**，需填写以下信息：
- **关系类型**（必填）：父亲 / 母亲 / 配偶 / 子女 / 兄弟姐妹 / 祖父母 / 外祖父母 / 其他
- **昵称**（必填）：如「王大明」
- **出生日期**：点击日期选择器选择（无需手动输入）
- **性别**：男 / 女
- **身高**：数字输入（单位 cm）
- **体重**：数字输入（单位 kg）
- **既往病史**：从标签中多选（高血压、糖尿病、冠心病等 12 个常见病） + 可自由输入其他
- **过敏史**：从标签中多选（青霉素、头孢类、海鲜等 10 种常见过敏原）+ 可自由输入其他

**步骤 6：确认并开始分析**
- 点击「确认并开始分析」按钮
- 系统自动创建真实聊天会话并跳转到 AI 聊天页面

---

### 流程二：在 AI 聊天页面体验增强功能

进入聊天页面后，您将看到以下新增元素：

**顶部症状摘要条**
- 进入时展开显示完整摘要（部位 · 症状 · 持续时间 · 咨询对象）
- AI 首次回复后自动收起为一行
- 随时点击可重新展开

**症状自查卡片（第一条消息）**
- 绿色边框 + 绿色「✚ 健康自查摘要」标题
- 默认收起显示一行摘要
- 点击右侧箭头展开查看完整症状信息

**AI 回复风格**
- AI 以亲切自然的语气开始回复（如「您好，我注意到这次是为您的爸爸咨询...」）
- 自然融入家庭成员的健康档案信息进行分析

**切换咨询对象**
- 在输入框左侧有一个「👤」小图标
- 点击弹出成员选择列表
- 可切换到「为自己」或其他家庭成员
- 切换后 AI 后续回复将基于新的健康档案

---

### 流程三：其他入口使用说明

对于「通用健康咨询」「中医养生」「用药参考」等其他聊天入口：
- **不强制弹出**家庭成员选择弹窗，默认为「自己」
- 进入聊天页后，可随时点击输入框旁的「👤」图标切换咨询对象

## 注意事项

1. **健康档案数据安全**：家庭成员的健康档案信息不会在 URL 中明文传输，仅通过后端安全通道处理，请放心使用。

2. **AI 建议仅供参考**：所有 AI 分析结果均为健康参考信息，不构成医疗诊断或治疗建议。如有不适，请及时就医。

3. **家庭成员档案**：填写的既往病史和过敏史信息有助于 AI 提供更精准的建议，建议尽量完整填写。

4. **切换咨询对象**：在已开始的对话中切换咨询对象后，AI 将基于新对象的档案继续当前对话（不会开启新会话）。

5. **出生日期输入**：请使用日期选择器选择出生日期，不支持手动文字输入。

6. **既往病史和过敏史**：支持多选标签 + 自由文字输入，点击标签即可选中/取消，也可在「其他」输入框中填写不在列表中的项目。
