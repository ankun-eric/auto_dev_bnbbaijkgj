## 功能简介

本次「体质测评优化」共落地 4 个核心点 + 1 项品牌视觉统一：

1. **测评结果卡片排版优化**：体质标签「主：特禀质」一行完整显示、不被挤断；如有兼夹体质，「兼：xxx」自动换到下一行，主/兼分行清晰，主流手机屏幕下不溢出、不截断。
2. **详情页内容改为后台可运营配置**：详情页下方「专属膳食套餐」「门店服务」两块，由原来写死的假数据改为后台可运营配置，按用户体质类型智能匹配；若该体质未配置任何内容，则整块隐藏，不再出现「暂未上架 / 敬请期待」等占位假文案。
3. **右上角返回直达 AI 首页**：点击测评结果页右上角返回，直接回到「AI 首页」；已废弃的旧测评列表页彻底弃用，不再作为返回目标。
4. **分享方式重新设计**：
   - **好友/群转发卡片**：微信小程序原生一键转发（右上角「转发给好友」），卡片标题动态带上用户体质（如「我的体质是「特禀质」，快来测测你是什么体质？」），使用统一的圆形品牌 Logo 封面，好友点开直达测评。
   - **朋友圈长图海报**：点「保存海报」生成竖版长图（清爽留白版：浅蓝底 + 白卡片），含品牌、体质结果（主/兼）、3 条按体质自动匹配的调理建议、小程序码与「长按识别测测你的体质」引导语。
5. **品牌视觉统一**：全端统一为「宾尼小康」（不再出现「小白健康」），主色调统一为天蓝色 `#0EA5E9`（不再用紫色），Logo 统一为圆形头像 `binni-xiaokang-logo.png`，测评时长文案统一为「约 3 分钟完成」。

## 使用说明

### 一、体验测评结果新排版（优化点 1）

1. 进入「AI 首页」，点击「体质测评」入口（H5 入口：[体质测评页](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/tcm/)）。
2. 页面会提示「约 3 分钟完成」，按引导完成测评。
3. 完成后进入结果卡片：
   - 「主：xx质」标签会**一行完整显示**，不会被挤断。
   - 若你存在兼夹体质，「兼：xx质」会自动排到**下一行**，主、兼分行清晰。

### 二、查看专属膳食套餐 / 门店服务（优化点 2）

1. 在结果卡片点击进入「详情页」。
2. 向下滑动，可看到「专属膳食套餐」「门店服务」两块内容：
   - 这些内容由运营人员在后台按体质类型配置，**会根据你的体质智能匹配**。
   - 如果你的体质暂未配置内容，对应整块会**自动隐藏**，不会再出现「暂未上架 / 敬请期待」这类占位文案。

> 运营人员维护入口：登录运营后台 → 左侧菜单「体质测评运营内容」（[配置页面](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/admin/constitution-content/)），按「体质类型 + 板块（膳食 / 门店）」增删改内容卡，停用或不配置则用户端不展示。

### 三、返回 AI 首页（优化点 3）

- 在测评结果页点击**右上角返回**，会直接回到「AI 首页」。
- 旧的测评列表页已弃用，不会再跳到它。

### 四、分享你的体质（优化点 4）

**转发给好友 / 群（小程序）**

1. 在测评结果页点击右上角「···」→「转发给好友」（小程序原生转发）。
2. 转发卡片标题会自动带上你的体质（如「我的体质是「特禀质」，快来测测你是什么体质？」），封面为统一的圆形品牌 Logo。
3. 好友点开卡片，**直接进入体质测评**，路径最短。

**保存朋友圈长图海报**

1. 在测评结果页点击「保存海报」。
2. 生成一张竖版清爽长图（浅蓝底 + 白卡片）：
   - 顶部：宾尼小康品牌 + 大标题；
   - 中部：高亮你的体质结果（主 / 兼）；
   - 下部：3 条按你的体质自动匹配的调理建议；
   - 底部：小程序码 + 「长按识别测测你的体质」引导语。
3. 保存到相册后即可发朋友圈。

### 体验步骤

1. **下载压缩包**：点击上方链接，将 zip 压缩包下载到本地电脑。
2. **解压文件**：将下载的 zip 文件解压到任意目录（记住解压后的文件夹路径）。
3. **下载微信开发者工具**：如尚未安装，请前往 [微信开发者工具官方下载页面](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装。
4. **打开微信开发者工具**：启动开发者工具，使用微信扫码登录。
5. **导入项目**：
   - 点击开发者工具首页的「导入项目」（或「+」号）；
   - 在「目录」栏点击浏览，选择第 2 步解压后的文件夹；
   - 「AppID」栏可填入项目的 AppID，或选择「测试号」进行体验；
   - 点击「导入」按钮。
6. **预览体验**：导入成功后，开发者工具会自动编译并在模拟器中展示小程序界面，进入「体质测评」即可体验本次新功能（结果卡片排版、详情页配置内容、返回直达首页、原生转发与海报）。
