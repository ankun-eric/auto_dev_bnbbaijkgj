# 体质测评优化 · 用户体验使用手册

> 产品：宾尼小康　|　模块：AI 首页 → 体质测评　|　版本：v1.0
> 适用端：微信小程序（主战场）、H5、APP（安卓 / 苹果）

---

## 一、功能介绍

本次对「体质测评」做了 4 项体验升级，并统一为「宾尼小康」天蓝色品牌视觉：

1. **测评结果卡片排版更清爽**：主体质（如「主：特禀质」）一行完整显示不再被挤断，兼夹体质（「兼：xxx」）自动排到下一行，主、兼分行清晰，主流手机不溢出、不截断。
2. **「专属膳食套餐 / 门店服务」改为后台可运营**：内容由运营在后台按体质类型配置，结果页按你的体质智能匹配展示；**某体质没有配置内容时，对应整块自动隐藏**，不再出现「暂未上架 / 敬请期待」这类占位假文案。
3. **右上角返回直达 AI 首页**：从测评结果页点返回，稳定回到「AI 首页」，不再跳到已废弃的旧列表页。
4. **分享更方便**：
   - **转发给好友（小程序原生一键转发）**：标题自动带上你的体质，例如「我的体质是「特禀质」，快来测测你是什么体质？」，使用统一固定封面（圆形头像 Logo），好友点开即可直接进入测评。
   - **朋友圈长图海报**：点「保存海报」生成清爽留白版竖图（浅蓝底 + 白卡片），含宾尼小康品牌、大标题、高亮体质结果、3 条按体质自动匹配的调理建议，以及「长按识别测测你的体质」引导语。

## 二、操作步骤

### 1. 进入体质测评并查看结果
1. 打开「AI 首页」，进入「体质测评」（约 3 分钟完成）。
2. 完成测评后进入结果详情页，自上而下依次为：体质名片、深度解读、个性化养生方案、专属膳食套餐、门店服务、分享区。
3. 结果卡片顶部的「主 / 兼」体质标签现已分行清晰展示。

### 2. 查看后台配置的套餐 / 门店服务
- 「专属膳食套餐」「门店服务」会根据你的体质类型自动展示对应内容。
- 若该体质暂无配置，对应板块不会出现（不再有占位文案）。

### 3. 返回 AI 首页
- 在结果页点击右上角返回（小程序为顶栏左侧「‹」），将直接回到 AI 首页。

### 4. 转发给好友
- **小程序**：点击右上角「···」→「转发给好友」，或点击页面底部「转发给好友，一起测体质」按钮，标题会自动带上你的体质，好友点开即进入测评。
- **H5 / APP**：点击「转发给好友」，调用系统分享或复制分享文案。

### 5. 保存朋友圈长图海报（H5）
- 在 H5 结果页点击「保存海报」，稍候生成清爽留白版长图，长按或点击保存到相册后即可发朋友圈。

### 6. 运营人员：配置套餐 / 门店服务内容
1. 登录后台管理系统。
2. 进入「AI 配置中心 → 体质测评运营内容」。
3. 按「体质类型 + 板块（专属膳食套餐 / 门店服务）」新增、编辑、启用 / 停用、删除内容卡。
4. 仅「启用」的内容会展示给对应体质用户；某体质未配置任何启用项时，前端对应整块自动隐藏。

## 四、注意事项

1. **品牌统一**：全端统一为「宾尼小康」名称、天蓝色主色与圆形头像 Logo，不再出现「小白健康」与紫色。
2. **测评时长文案**：统一为「约 3 分钟完成」。
3. **内容由运营配置**：膳食套餐 / 门店服务为运营在后台维护的真实内容；如未看到某板块，说明该体质暂无启用内容，属正常隐藏逻辑。
4. **小程序转发封面**：使用统一固定封面（圆形头像 Logo），便于维护，不做千人千面封面。
5. **海报为清爽留白版（方案 A）**：浅蓝底 + 白卡片，3 条调理建议按体质自动匹配，非写死。
6. **APP 体质结果页**：与小程序对齐，膳食套餐 / 门店服务无配置时整块隐藏，分享标题动态带体质。
