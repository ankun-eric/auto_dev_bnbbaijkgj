## 功能简介

本次升级对「档案管理 · 用药计划模块」做了系统性精简与结构化改造，让填写更简单、数据更干净，同时把全局性的 AI 外呼提醒迁移到「健康提醒 / 共管」统一管理。

主要变化：

1. **顶部温馨提示**：新增/编辑用药计划页**最上方**固定一行 `⚠️ 用药请遵循医嘱`，浅黄底深棕字。
2. **表单字段精简**：移除了"拍照识药入口"、"单条用药提醒开关"、"单条 AI 外呼开关"和"关联疾病"选择。
3. **结构化输入**：
   - 每日次数（默认 3 次，1~4 次）→ 自动生成对应数量的时间选择器，并按经验值预填（如 3 次 → 08:00 / 12:00 / 20:00）。
   - 每次剂量 = 数值（1/4、1/2、1、2、3…500）+ 单位（片/粒/袋/支/瓶/贴/g/mL/mg/μg），均为预设下拉，**不允许手输**。
   - 服用周期 = 开始日期 + 服用天数（默认今天 + 5 天），下方实时显示"结束日期"。
   - 用药指导：餐前 / 餐后 / 空腹 / 随餐服用 / 睡前 单选。
   - 备注最多 30 字，实时计数。
4. **AI 外呼提醒全局开关**：迁移到「健康档案 → 共管与提醒」Tab 中的「📞 用药 AI 外呼提醒」开关，默认关闭。共管家人也可代为开启，两处共用同一份数据。
5. **列表卡片**：
   - 当 AI 外呼提醒**全局开启**且该计划**进行中**时，卡片底部用药时间旁会出现 📞 电话图标。
   - 当 AI 外呼提醒**关闭**时，列表顶部展示一行轻量横幅：`📞 开启 AI 外呼提醒 可电话提醒按时吃药 →`，点击直达健康提醒；横幅可关闭，关闭后 7 天内不再展示。
6. **生命周期**：服用周期到期后自动归档到"历史用药"，列表"进行中"页面不再展示。
7. **同名药品去重**：不允许同时存在两条"进行中"的同名药品。重复创建会弹窗提示"您已有一条进行中的【XX】用药计划，是否要去查看/修改？"（按钮：去查看 / 取消）。
8. **数据保留**："关联疾病"字段仅在数据库中保留，由系统/医生在后台维护，**用户端任何位置均不展示**。

## 使用说明

### 1. 进入用药计划页

打开 H5，进入"健康档案 → 用药计划"，或直接访问：

[https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/medication-plans](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/medication-plans)

页面顶部为"用药计划"标题与"+ 新增用药计划"按钮，下方为「进行中 / 历史用药」Tab。

### 2. 新增用药计划

1. 点击右上角「+ 新增用药计划」。
2. 表单顶部固定一行温馨提示：`⚠️ 用药请遵循医嘱`。
3. 填写以下字段：
   - **药品名称**：手动输入，1~30 字（如：阿司匹林肠溶片）。
   - **每日次数**：下拉选择 1 / 2 / 3 / 4 次，默认 3 次。改变次数后下方"用药时间"会自动重新生成对应数量的时间选择器并预填默认值。
   - **用药时间**：每个时间点独立选择，24 小时制；同一计划内不允许两个相同的时间点。
   - **每次剂量**：左侧选数值，右侧选单位；两个下拉独立，不联动。
   - **服用周期**：开始日期默认今天，服用天数默认 5 天；下方"结束日期"自动按 `开始日期 + 天数 - 1` 显示。
   - **用药指导**：从「餐前 / 餐后 / 空腹 / 随餐服用 / 睡前」中单选。
   - **备注**：可选，最多 30 字。
4. 点击「保存」。

### 3. 列表卡片

新建后回到列表，每张卡片展示：

- 药品名称
- 剂量（如 `1/2 片`）
- 用药指导（如 `餐后`）
- 服用周期（如 `2026-05-16 ~ 2026-05-20`）
- 用药时间点：当 AI 外呼全局开启且该计划进行中时，每个时间点前显示 📞 蓝色电话图标
- 备注（如有）
- 编辑 / 删除按钮（沿用原实现）

切换到「历史用药」Tab 可查看已结束的用药计划（不再展示编辑/删除按钮）。

### 4. 开启 AI 外呼提醒

> AI 外呼提醒会按用药计划中的服药时间，自动 AI 电话提醒按时吃药。

1. 在用药计划页顶部如看到引导横幅 `📞 开启 AI 外呼提醒 可电话提醒按时吃药 →`，直接点击横幅，会跳转到「健康档案 → 共管与提醒」。
2. 也可以直接访问 [健康档案](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile)，切换到「共管与提醒」Tab。
3. 在第一块「📞 用药 AI 外呼提醒」中，把开关拨到打开。
4. 返回用药计划页：所有"进行中"用药计划卡片底部的时间旁会出现 📞 图标，顶部横幅自动消失。

> ✅ 该开关在「健康提醒」与「共管」两个入口共用同一份数据：家人在「共管」侧帮你开启，你在「健康提醒」侧也能看到。

### 5. 同名药品提醒

当你尝试为已经在"进行中"的同名药品再创建一条计划时，系统会弹出提示：

```
同名药品提醒
您已有一条进行中的【XX】用药计划，是否要去查看/修改？
[取消] [去查看]
```

- 点击「去查看」会回到列表并打开对应的编辑窗口。
- 同名判定不区分大小写、忽略前后空白；仅针对"进行中"的计划生效，**已结束/已归档的同名药品不会阻拦新建**。

### 6. 自动归档

服用周期结束后，该用药计划会在你下次进入「用药计划」页面时**自动归档**：

- 不再出现在「进行中」Tab；
- 出现在「历史用药」Tab，可查看，但不显示编辑/删除按钮。

## 注意事项

1. **关联疾病字段对用户不可见**：保留在数据库中作为系统能力字段，由运营/医生在后台维护，用药计划的用户端表单与列表均不展示。
2. **每日次数上限为 4**：少数眼药水类高频用药用户暂不能覆盖，可在备注中补充说明。
3. **剂量数值/单位严格从预设挡位选择**：如未来发现挡位不够，运营会根据真实数据补充。
4. **AI 外呼提醒默认关闭**：需要主动到「健康提醒 / 共管」中开启；开启后跟随每条进行中计划的"用药时间"自动外呼。
5. **同名药区分时机**：仅限"进行中"。如果你之前的计划已经过了结束日期或被归档，再创建同名药不会被阻拦。
