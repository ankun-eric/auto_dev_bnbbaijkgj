## 功能简介

本次更新为 AI 健康咨询聊天页面新增了以下核心功能：

### 1. 功能按钮栏
在聊天输入框上方新增了一排可横向滚动的功能按钮，提供丰富的快捷操作入口，包括：

| 按钮 | 功能说明 |
|------|----------|
| 📞 打电话 | 进入数字人 AI 健康顾问视频通话模式 |
| 📷 拍照识药 | 拍照或选择图片，AI 自动识别药品信息 |
| 📄 上传报告 | 上传体检报告图片，AI 智能解读各项指标 |
| 🩺 症状自查 | AI 引导式问诊，帮助初步分析症状 |
| 💊 用药提醒 | AI 对话式创建用药提醒计划 |
| 📝 健康日记 | AI 引导录入血压、血糖、体重等健康数据 |
| 🏥 预约挂号 | AI 引导选择科室和医生进行预约 |
| 🚑 急救指南 | 查看常见急救场景的详细步骤指导 |

### 2. 数字人对话模式
点击"打电话"按钮后，进入全屏数字人视频通话页面，与 AI 健康顾问进行语音对话：
- 数字人视频全屏播放，AI 说话时有口型动作
- 底部显示对话气泡字幕
- 支持语音输入，自动检测说话结束

### 3. 后台管理功能
管理后台新增三个管理页面：
- **功能按钮管理**：自定义配置按钮名称、图标、类型、排序
- **数字人形象管理**：管理数字人视频素材和 TTS 音色
- **语音服务配置**：配置腾讯云语音密钥和 VAD 参数

## 使用说明

### 一、功能按钮栏使用

1. 打开 AI 健康咨询页面（首页底部点击「AI」Tab，选择任意咨询类型进入对话）
2. 在聊天输入框上方可以看到一排功能按钮
3. 左右滑动可浏览更多按钮
4. 点击对应按钮即可使用相应功能

### 二、数字人对话（打电话）

1. 在功能按钮栏中点击「📞 打电话」按钮
2. 系统会请求麦克风权限，请点击"允许"
3. 进入全屏数字人通话页面，看到数字人视频和"连接中..."提示
4. 连接成功后，直接对着手机说话即可
5. AI 识别到您说完后，会以语音和文字回复
6. 底部的对话气泡流会显示对话内容，可上滑查看历史
7. 点击底部红色「挂断」按钮结束通话
8. 通话结束后，对话内容会自动同步到聊天记录中

### 三、拍照识药

1. 点击「📷 拍照识药」按钮
2. 选择"拍照"或"从相册选择"
3. 拍摄或选择药品图片后，图片会发送到对话中
4. AI 将识别药品名称、规格、用法用量、注意事项等信息

### 四、上传报告

1. 点击「📄 上传报告」按钮
2. 选择"拍照"、"从相册选择"或"选择文件"
3. 上传体检报告图片后，AI 将自动识别并解读各项指标
4. 解读内容包括异常指标标注、通俗解释和健康建议

### 五、症状自查

1. 点击「🩺 症状自查」按钮
2. AI 会询问您的不适症状
3. 按照提示描述症状持续时间、严重程度、伴随症状等
4. AI 会给出初步分析和建议就诊科室

### 六、其他功能按钮

- **💊 用药提醒**：AI 引导您设置药品名称、剂量、服药时间等，自动创建用药提醒
- **📝 健康日记**：AI 引导您录入血压、血糖、体重等日常健康数据
- **🏥 预约挂号**：AI 根据症状推荐科室，引导完成预约
- **🚑 急救指南**：展示心肺复苏、烫伤、骨折等急救场景的详细操作步骤

### 七、后台管理配置

管理员登录后台管理系统后：

1. 在左侧菜单找到「AI咨询配置」
2. 选择子菜单进入对应管理页面：
   - **功能按钮管理**：新增/编辑/删除/排序功能按钮
   - **数字人形象管理**：管理数字人视频素材
   - **语音服务配置**：配置腾讯云密钥和 VAD 参数

## 注意事项

1. **麦克风权限**：使用数字人对话功能需要授权麦克风权限，请在弹出权限请求时点击"允许"
2. **网络要求**：数字人对话需要稳定的网络连接，建议在 WiFi 环境下使用
3. **AI 回复声明**：AI 提供的健康建议仅供参考，不构成医疗诊断。如有健康问题请及时就医
4. **图片上传限制**：拍照识药和上传报告单次最多上传 5 张图片，每张不超过 10MB
5. **后台管理权限**：
   - 管理员可管理功能按钮和数字人形象
   - 语音服务配置（腾讯云密钥）仅超级管理员可操作
6. **通话记录保存**：数字人通话结束后，对话内容会自动同步到聊天记录，可随时回看
