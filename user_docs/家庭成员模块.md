> 版本：v2.0
> 上线日期：2026-05-18
> 适用端：H5（浏览器 / App 内嵌 WebView） & 微信小程序

---

## 一、功能简介

「家庭成员模块」是 bini-health 健康档案的核心入口之一，用于记录您家中其他成员（父母、配偶、子女、祖辈等）的基础健康档案信息，为家庭健康风险预测、家族病史分析、家庭医生服务提供数据基础。

本次 v2.0 版本完成全面重构，主要改进：

- **入口统一**：AI 对话页咨询人抽屉 + 健康档案页顶部「+」按钮，进入的是同一个新版表单
- **关系网格**：15 种常见亲属关系一屏铺开，无需层层翻找
- **智能预填**：选定关系后，性别自动锁定、出生年份按辈分智能默认
- **硬校验**：父母、子女、祖辈年龄合理性自动校验，避免脏数据
- **字徽头像**：列表头像采用统一的「圆形主色 + 白字」字徽（爸→「爸」、儿子→「儿」、本人→「我」），辈分分色区分
- **年龄显示**：列表副信息行直接展示年龄（如「妈妈 · 女 · 58 岁」），无需心算

## 三、操作步骤

### 步骤 1：进入新增表单

**方式 A：从健康档案页**
1. 打开 https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/health-profile
2. 点击右上角「+」按钮，弹出底部表单抽屉

**方式 B：从 AI 对话页**
1. 打开 https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/ai-home
2. 点击左下角「咨询人」按钮 → 抽屉底部「+ 新增家庭成员」

> 若您的本人档案缺失出生年，系统会弹窗提示「请先完善您的个人档案」，必须先完善本人档案才能继续添加家庭成员。

### 步骤 2：（可选）邀请家人共管

- 表单顶部为蓝色渐变「邀请家人加入健康守护计划」横幅
- 点击「去邀请」按钮可跳转邀请共管页

### 步骤 3：选择成员关系（必填）

- 在 4 列网格中点选 15 个关系胶囊之一：
  爸爸、妈妈、老公、老婆、儿子、女儿、哥哥、姐姐、弟弟、妹妹、爷爷、奶奶、外公、外婆、其他
- **唯一关系**（爸爸/妈妈/老公/老婆/爷爷/奶奶/外公/外婆）若已添加过，该胶囊会置灰不可点
- 选择「其他」后，会显示「关系」输入框（1-8 字符），用于填写如「舅舅、表妹」等自定义称呼

### 步骤 4：填写基础信息（必填）

| 字段 | 说明 |
|---|---|
| 姓名 | 1-12 字符，必填 |
| 性别 | 选择关系后自动锁定（如「爸爸」=男），不可改；「其他」可手动选择 |
| 出生日期 | 自动按辈分预填年份（如「爸爸」= 本人+25 岁、「儿子」= 本人-25 岁），月日固定 1.1，可调整 |

### 步骤 5：（可选）展开「其他（选填）」补充健康信息

点击「其他（选填）」可展开附加字段：

- 身高（cm，0-300）
- 体重（kg，0-300）
- **既往病史**：10 个慢病胶囊多选（高血压 / 糖尿病 / 高血脂 / 冠心病 / 脑卒中 / 慢阻肺 / 哮喘 / 慢性肾病 / 甲状腺 / 痛风）
- **过敏史**：分 3 组录入（药物 / 食物 / 其他），多个用逗号或空格分隔，如「青霉素、头孢」

### 步骤 6：保存

- 点击底部「保存」按钮
- 校验通过后会自动关闭抽屉并弹出 Toast「添加成功」
- 校验失败时表单顶部出现红色错误条，对应字段标红

## 四、字徽与年龄展示

新增成功后，您会在以下两个位置看到家庭成员的统一字徽：

1. **健康档案页 · 家庭成员列表**
   - 本人卡片置顶放大，字徽显示「我」+ 主色渐变背景
   - 其他成员按添加顺序排列
   - 副信息行：`关系 · 性别 · X 岁`（如「妈妈 · 女 · 58 岁」）

2. **AI 对话页 · 咨询人抽屉**
   - 本人卡片置顶，主色渐变背景
   - 其他成员按辈分分色字徽
   - 副信息行：`关系 · 性别 · X 岁`

### 字徽取字与颜色规则

| 关系 | 字徽 | 颜色 |
|---|---|---|
| 本人 | 我 | 主色渐变（白字） |
| 爸爸/妈妈/爷爷/奶奶/外公/外婆 | 爸/妈/爷/奶/外公/外婆 | 深蓝 `#0284C7` |
| 老公/老婆/哥哥/姐姐/弟弟/妹妹 | 夫/妻/哥/姐/弟/妹 | 天蓝 `#0EA5E9` |
| 儿子/女儿 | 儿/女 | 浅蓝绿 `#06B6D4` |
| 其他 | 自定义关系首字 | 灰蓝 `#64748B` |
| 老数据（缺关系字段） | 姓名首字 | 灰蓝 + 右上角「待补充」橙色角标 |

## 五、校验规则一览

| 校验项 | 规则 | 触发时机 |
|---|---|---|
| 必填校验 | 关系、姓名、性别、出生日期 4 项 | 失焦 / 保存 |
| 姓名长度 | 1-12 字符 | 失焦 / 保存 |
| 出生日期上限 | ≤ 今天 | 选择 / 保存 |
| 关系合理性 | 父母/祖辈年龄必须 > 本人；子女年龄必须 < 本人 | 保存 |
| 关系唯一性 | 唯一关系已存在时胶囊置灰 | 进入表单 |
| 其他关系名 | 必填、1-8 字符 | 失焦 / 保存 |
| 本人档案 | 出生年必须存在 | 进入表单 |

报错文案统一为：`关系与出生日期不符，请检查后重新填写`

## 六、注意事项

1. **不支持删除**：当前版本暂不提供家庭成员删除入口
2. **被邀请的家人**：被邀请共管的家庭成员关系数据仅可查看，不可编辑
3. **老数据兼容**：缺失字段在列表上会显示 `-`；进入编辑页时会引导您补全
4. **网络异常保存**：保留已填数据，提示「网络异常，请稍后重试」
5. **保存按钮防重**：保存按钮一旦点击会立即置 loading 防止重复提交
