## 功能简介

本次更新包含两项优化：

### 需求一：欢迎区 LOGO 布局调整

标准版 AI 首页和长辈版 AI 首页的欢迎区进行了视觉优化：

- **LOGO 尺寸统一为 52px**：宾尼小康头像在欢迎区更醒目、更大
- **三行左边缘统一对齐**：LOGO、欢迎语「我是宾尼小康，聊聊健康问题吧~」、用药提醒卡片三行的左边缘现在完全对齐，视觉更整齐
- **用药提醒新增独立浅色背景框**：用药提醒从原先的透明样式改为白色半透明独立圆角卡片，更容易被注意到和点击

配色保持不变：标准版为蓝绿渐变，长辈版为暖橙渐变。「去长辈版 / 去标准版」切换按钮位置不变（右上角）。

### 需求二：长辈版新增「乐龄游戏」卡片

长辈版 AI 首页在现有的 6 张服务卡片末尾新增第 7 张：「乐龄游戏」卡片。

| 属性 | 值 |
|------|-----|
| 图标 | 🧩（拼图） |
| 标题 | 乐龄游戏 |
| 描述 | 趣味益智游戏，锻炼大脑活力 |
| 卡片颜色 | 紫色渐变 |
| 点击链接 | `/brain-game` |

卡片排列顺序（更新后）：

| 序号 | 卡片名称 |
|------|----------|
| 1 | 数字安全绳 |
| 2 | 用药提醒 |
| 3 | 健康记录 |
| 4 | 居家安全 |
| 5 | 紧急呼叫 |
| 6 | 个人信息卡 |
| **7** | **乐龄游戏（新增）** |

## 使用说明

### 标准版 AI 首页

1. 打开 [标准版 AI 首页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/ai-home)
2. 顶部欢迎区可以看到更新后的宾尼小康 LOGO（52px）+ 时段问候语
3. 下方「我是宾尼小康，聊聊健康问题吧~」欢迎语与 LOGO 左边缘完美对齐
4. 再下方用药提醒卡片有独立的白色半透明圆角背景框，点击可进入用药提醒页面
5. 右上角「去长辈版」按钮可切换到关怀模式

### 长辈版 AI 首页

1. 打开 [长辈版 AI 首页](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/care-ai-home)
2. 顶部暖橙渐变欢迎区同样有 52px LOGO + 时段问候语 + 对齐欢迎语 + 独立背景框用药提醒
3. 向下滚动可以看到 7 张服务卡片，第 7 张是新增的「🧩 乐龄游戏」卡片（紫色渐变）
4. 点击「乐龄游戏」卡片即可进入益智乐园

### 乐龄游戏

1. 进入 [乐龄游戏页面](https://6b099ed3-7175-4a78-91f4-44570c84ed27.noob-ai.test.bangbangvip.com/brain-game)
2. 包含数学游戏（3 级难度可选）、排行榜、组队挑战等功能
3. 适老化设计，大字体、大按钮，方便长辈操作
