## 功能简介

本次更新围绕「适老化关怀」一次性上线了 4 大核心能力：

1. **版本选择页**（首次进入）：温柔引导用户从「关怀模式 / 标准模式」中二选一，并支持「暂不选择，先进去看看」跳过
2. **关怀版 AI 首页**：顶栏暖橙「👵 关怀模式 ▼」角标、特大字号欢迎语、4 个超大快捷胶囊（档案 / 用药 / 拍照问 AI / 家人）、横向滚动推荐问、AI 主动推送 4 张关怀卡（健康简报含血糖、用药提醒、居家安全、SOS 关怀）
3. **SOS 一键求救 4 状态流程**：
   - 状态 1：5 秒倒计时确认页（满屏红色 + 巨型倒计时数字 + 可撤销）
   - 状态 2：选择呼叫对象（120 / 家人）
   - 状态 3：呼叫中（自动并行：拨 120、发位置短信、附健康摘要、通知家人）
   - 状态 4：求救已送达 + 进度时间线
4. **SOS 智能触发**：高危词单触 + 症状词+程度词双触 + 否定词/疑问句过滤的三道防误报机制，关键词支持后台热配置

## 使用说明

### 一、首次进入：选择适合自己的模式

1. 打开 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/welcome-mode/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/welcome-mode/)
2. 页面会展示两个大卡片：
   - 👵 **关怀模式**（推荐）：超大字、超大按钮、语音优先、AI 主动关怀、SOS 一键求救
   - 🧑‍💼 **标准模式**：常规字号、功能更全、操作更高效、数据可视化
3. 点击任一卡片即可选定，并自动跳转到对应模式的首页
4. 也可以点击底部「暂不选择，先进去看看 →」跳过，默认进入**标准模式**

> 💡 小康的建议：如果您是为长辈准备的，或自己对手机操作不太熟悉，强烈推荐选择「关怀模式」。

### 二、体验关怀版 AI 首页

1. 打开 [https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-home/](https://newbb.test.bangbangvip.com/autodev/6b099ed3-7175-4a78-91f4-44570c84ed27/care-home/)
2. **顶栏右上角**有暖橙「👵 关怀模式 ▼」角标，点击可弹出切换面板，随时切回标准模式
3. **欢迎区**会根据当前时段自动生成问候（早上好/中午好/下午好/晚上好/夜深了）
4. **4 个快捷胶囊**：档案、用药、拍照问 AI、家人，点击直达对应功能
5. **横向滚动推荐问**：6 条常用问句（血压、睡眠、饮食、报告、用药、问医生），点击直接发送给 AI
6. **AI 主动推送 3 张卡片**：
   - 🟢 **健康简报**：2×2 网格显示血压、血糖（异常变红）、睡眠、步数
   - 🟠 **用药提醒**：今天 1-2 条未完成的用药计划 + 一键「已吃」打卡
   - 🩵 **居家安全**：紧急呼叫器、烟感报警器在线状态与电量（离线或电量<20% 整格变橙）

### 三、SOS 一键求救（最重要）

#### 触发方式 1：点 SOS 悬浮球

页面右下角始终有一个红色「SOS」呼吸光晕悬浮球，点击即可立刻进入 5 秒倒计时确认页。

#### 触发方式 2：在 AI 对话中触发

底部输入框中输入或语音说出以下任意内容会自动弹出「SOS 关怀卡」：
- **高危词**（单独触发）：救命 / 晕倒 / 摔倒 / 抽搐 / 打120 / ……
- **症状词 + 程度词**（双词触发）：例如「胸闷得厉害」「头晕受不了」「肚子疼得要命」

#### 三道防误报保险

下列情形 **不会** 误触发 SOS：
1. **否定词**：「胸口不闷」「头不晕了」「没有摔倒」
2. **疑问句**：「是不是要胸闷？」「会不会头晕？」
3. **5 秒撤销窗口**：即使倒计时启动，5 秒内点「我没事 · 取消」即可撤销

#### 完整 4 状态流程

1. **状态 1**：满屏红色，巨型「5、4、3、2、1」倒计时 → 用户可点白色「我没事 · 取消」按钮撤销
2. **状态 2**：倒计时结束自动进入「请选择呼叫对象」页 → 「🔴 呼叫 120 · 急救中心」/「👨‍👩‍👧 呼叫家人」/「先取消，我再看看」
3. **状态 3**：呼叫中页 → 自动并行执行：拨 120 电话 + 短信发位置 + 短信附健康摘要 + 推送家人
4. **状态 4**：求救已送达页 → 时间线显示「120 接听 / 救护车出发 / 预计到达时间」+ 家人「已查看·正在赶来」状态 + 「我已安全 / 再次呼叫」按钮

### 四、切换模式

- **入口 1**：首页右上角「👵 关怀模式 ▼」角标 → 点击弹出切换面板 → 选「🧑‍💼 标准模式」即可
- **入口 2**：版本选择页（首次进入显示，后续可在设置中重新打开）
- 切换 **不丢任何数据**（健康档案、对话历史、用药记录完全保留）

## 注意事项

1. **关怀模式数据持久化**：模式选择会同时写入本地缓存与后端账号偏好，多端登录同账号会自动同步
2. **SOS 关键词热配置**：后台修改关键词后客户端无需发版即可生效（通过 `GET /api/care-v1/sos/keywords` 拉取）
3. **SOS 误报率控制**：本次新增的三道防误报保险（否定词 + 疑问句 + 5 秒撤销）已通过 16 项 E2E 验证
4. **首期不含硬件联动**：状态 2 中**不含**「呼叫家中紧急呼叫器」按钮，将在二期接入
5. **视觉规格**：关怀模式下所有正文 ≥18pt、主按钮 ≥56px、SOS 按钮 ≥88px，符合 WCAG AAA 级对比度
6. **后端测试**：本次后端新增 19 个 pytest 用例 + 16 项 HTTPS E2E 测试，**全部 100% 通过**
