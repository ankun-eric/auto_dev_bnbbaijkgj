## 功能简介

本次对健康档案页面（`/health-profile`）进行了 12 项功能优化，旨在提升中老年用户（40-70 岁）的使用体验，包括：

- 精简信息显示，减少信息噪音
- 优化编辑流程，提升个人信息管理效率
- 修复数据统计不准确的问题
- 新增快捷操作入口，缩短常用操作路径
- 优化用药提醒卡片，信息极简易懂

## 使用说明

### 1. 成员切换（优化后）

打开健康档案页面后，顶部横向排列的成员胶囊现在**只显示名字/昵称**，不再显示关系前缀。

- 点击不同成员胶囊即可切换查看该成员的健康档案
- 最右侧 "+" 按钮可添加新的家庭成员

### 2. Hero 卡片（优化后）

名字下方新增了一行基本信息，格式为：`性别 · 年龄 · 身高 · 体重`

- 仅显示已填写的字段，缺失的不显示
- 示例：若只填了性别和生日，则显示 `男 · 28岁`
- 健康标签（慢病、过敏等）直接展示，无多余标题文字

### 3. 编辑个人信息（优化后）

点击 Hero 卡片右上角的「编辑」按钮打开编辑弹层：

- **字段顺序**：关系 → 姓名 → 性别 → 生日 → 身高/体重 → 血型
- **布局**：标签在左，输入内容在右，一行一个字段
- **关系**：本人档案固定显示"本人"不可改；非本人档案可选择（爸爸、妈妈、老公、老婆等）
- **性别**：仅"男""女"两个选项
- **身高/体重**：同一行两个输入框，左侧身高(cm)，右侧体重(kg)
- **血型**：A、B、AB、O 四个选项

填写完成后点击「保存」按钮即可。

### 4. 邀请家人区域

- 选中**本人**时，不再显示"邀请家人加入守护计划"区域
- 选中**非本人且未守护**的家庭成员时，才显示邀请区域

### 5. 我的设备与用药计划统计

- 每次进入健康档案页面时，设备数量和用药计划数量都会重新拉取最新数据
- 统计数与实际绑定设备数和有效用药计划数完全一致

### 6. 用药计划快捷操作

用药计划区域标题栏右侧新增两个快捷入口：

- **"+新增"**：点击直接进入新增用药流程
- **"全部 ›"**：点击跳转到用药计划完整列表页面（含"服药中/未开始/已结束"三个分类）

### 7. 就医资料快捷操作

就医资料区域标题栏右侧新增两个快捷入口：

- **"+新增"**：点击直接进入新增资料页面
- **"全部 ›"**：点击跳转到就医资料全部页面

### 8. 就医资料全部页面（新增）

全部页面按资料类型分为 4 个 Tab：

| Tab | 对应分类 |
|-----|----------|
| 病例单 | 病例单类资料 |
| 体检报告 | 体检报告类资料 |
| 药物 | 药物类资料 |
| 其他 | 其他类资料 |

- 点击健康档案首页的就医资料卡片后，会自动跳转到对应分类的 Tab
- 支持搜索和查看详情

### 9. 今日健康数据 — 用药提醒（优化后）

用药提醒卡片采用极简设计，面向中老年用户：

**三种状态展示：**

| 状态 | 显示内容 | 卡片样式 |
|------|----------|----------|
| 有待服用药物 | ⏰ 下一次：14:00 美托洛尔 — 还剩N次 | 天蓝色卡片 |
| 仅剩最后一次 | ⏰ 下一次：14:00 美托洛尔 — 最后1次 | 天蓝色卡片 |
| 今日全部完成 | 今日用药已全部完成 ✅ | 渐变绿色卡片 |

- 点击"全部 ›"可跳转到用药提醒完整列表

### 10. 今日健康数据 — 刷新功能

标题行右侧新增更新时间和刷新按钮：

- 显示格式：`更新时间：HH:MM 🔄`
- 点击 🔄 图标可手动刷新今日健康数据
- 刷新后时间自动更新

## 注意事项

1. **登录要求**：使用健康档案功能需要先登录账号
2. **数据准确性**：每次进入页面会自动刷新统计数据，如数据显示异常可点击刷新按钮
3. **编辑权限**：
   - 本人档案：可编辑自己的个人信息，关系字段固定为"本人"
   - 家庭成员：被守护人的信息由守护人可编辑
4. **兼容性**：页面适配 iOS Safari 和 Android Chrome 主流版本，支持 320px-428px 屏幕宽度
5. **年龄显示**：未满 1 岁显示"<1岁"，超过 100 岁正常显示
